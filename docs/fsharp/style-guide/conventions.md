### <a name="carefully-apply-autoopen"></a><span data-ttu-id="dbb99-122">请仔细应用 `[<AutoOpen>]`</span><span class="sxs-lookup"><span data-stu-id="dbb99-122">Carefully apply `[<AutoOpen>]`</span></span>

## <a name="error-management"></a><span data-ttu-id="dbb99-163">错误管理</span><span class="sxs-lookup"><span data-stu-id="dbb99-163">Error management</span></span>

<span data-ttu-id="dbb99-164">大型系统中的错误管理是一项复杂且能体现细微差别工作，并且没有确保您的系统中的项目符号是容错的和的行为也没有 silver。</span><span class="sxs-lookup"><span data-stu-id="dbb99-164">Error management in large systems is a complex and nuanced endeavor, and there are no silver bullets in ensuring your systems are fault-tolerant and behave well.</span></span> <span data-ttu-id="dbb99-165">以下指南应提供导航此困难空间中的指导。</span><span class="sxs-lookup"><span data-stu-id="dbb99-165">The following guidelines should offer guidance in navigating this difficult space.</span></span>

### <a name="represent-error-cases-and-illegal-state-in-types-intrinsic-to-your-domain"></a><span data-ttu-id="dbb99-166">表示错误情况和你的域中的内部类型中的非法状态</span><span class="sxs-lookup"><span data-stu-id="dbb99-166">Represent error cases and illegal state in types intrinsic to your domain</span></span>

<span data-ttu-id="dbb99-167">与[可区分联合](../language-reference/discriminated-unions.md)，F#使你能够在类型系统中表示发生故障的程序状态。</span><span class="sxs-lookup"><span data-stu-id="dbb99-167">With [Discriminated Unions](../language-reference/discriminated-unions.md), F# gives you the ability to represent faulty program state in your type system.</span></span> <span data-ttu-id="dbb99-168">例如：</span><span class="sxs-lookup"><span data-stu-id="dbb99-168">For example:</span></span>

```fsharp

    | CardExpired of DateTime
    | UndisclosedFailure
```

<span data-ttu-id="dbb99-169">在这种情况下，有三种已知的方式从银行帐户提取资金可能会失败。</span><span class="sxs-lookup"><span data-stu-id="dbb99-169">In this case, there are three known ways that withdrawing money from a bank account can fail.</span></span> <span data-ttu-id="dbb99-170">每个错误用例表示在类型中，并因此地进行处理安全地在整个程序。</span><span class="sxs-lookup"><span data-stu-id="dbb99-170">Each error case is represented in the type, and can thus be dealt with safely throughout the program.</span></span>

```fsharp

    match w with

    | UndisclosedFailure -> printfn "Failed: unknown"
```
