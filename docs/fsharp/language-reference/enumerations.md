# <a name="enumerations"></a><span data-ttu-id="aca82-103">枚举</span><span class="sxs-lookup"><span data-stu-id="aca82-103">Enumerations</span></span>

<span data-ttu-id="aca82-104">*枚举*，也称为*枚举*、 是整型类型的标签分配到值的子集。</span><span class="sxs-lookup"><span data-stu-id="aca82-104">*Enumerations*, also known as *enums*, , are integral types where labels are assigned to a subset of the values.</span></span> <span data-ttu-id="aca82-105">可以使用它们来代替文本，使代码更具可读性且更易维护。</span><span class="sxs-lookup"><span data-stu-id="aca82-105">You can use them in place of literals to make code more readable and maintainable.</span></span>

...
