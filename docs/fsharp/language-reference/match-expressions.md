# <a name="match-expressions"></a><span data-ttu-id="c0f6d-103">匹配表达式</span><span class="sxs-lookup"><span data-stu-id="c0f6d-103">Match expressions</span></span>

<span data-ttu-id="c0f6d-104">`match`表达式提供的表达式与一组模式的比较结果为基础的分支控制。</span><span class="sxs-lookup"><span data-stu-id="c0f6d-104">The `match` expression provides branching control that is based on the comparison of an expression with a set of patterns.</span></span>

## <a name="syntax"></a><span data-ttu-id="c0f6d-105">语法</span><span class="sxs-lookup"><span data-stu-id="c0f6d-105">Syntax</span></span>

```fsharp
// Match expression.
match test-expression with
| pattern1 [ when condition ] -> result-expression1
| pattern2 [ when condition ] -> result-expression2
| ...

// Pattern matching function.
function
| pattern1 [ when condition ] -> result-expression1
| pattern2 [ when condition ] -> result-expression2
| ...
```

## <a name="remarks"></a><span data-ttu-id="c0f6d-106">备注</span><span class="sxs-lookup"><span data-stu-id="c0f6d-106">Remarks</span></span>

<span data-ttu-id="c0f6d-107">模式匹配表达式允许基于与一组模式的测试表达式比较复杂的分支。</span><span class="sxs-lookup"><span data-stu-id="c0f6d-107">The pattern matching expressions allow for complex branching based on the comparison of a test expression with a set of patterns.</span></span> <span data-ttu-id="c0f6d-108">在中`match`表达式中，*测试表达式*中打开，并找到匹配项，相应与每个模式进行比较*结果表达式*计算和生成的值是返回与匹配表达式的值。</span><span class="sxs-lookup"><span data-stu-id="c0f6d-108">In the `match` expression, the *test-expression* is compared with each pattern in turn, and when a match is found, the corresponding *result-expression* is evaluated and the resulting value is returned as the value of the match expression.</span></span>

<span data-ttu-id="c0f6d-109">模式匹配上一个语法中所示的函数是在哪种模式匹配是立即对自变量执行一个 lambda 表达式。</span><span class="sxs-lookup"><span data-stu-id="c0f6d-109">The pattern matching function shown in the previous syntax is a lambda expression in which pattern matching is performed immediately on the argument.</span></span> <span data-ttu-id="c0f6d-110">等效于以下模式匹配函数，在上述语法中所示。</span><span class="sxs-lookup"><span data-stu-id="c0f6d-110">The pattern matching function shown in the previous syntax is equivalent to the following.</span></span>

```fsharp
    | pattern2 [ when condition ] -> result-expression2
    | ...
```

<span data-ttu-id="c0f6d-111">有关 lambda 表达式的详细信息，请参阅[Lambda 表达式：`fun`关键字](functions/lambda-expressions-the-fun-keyword.md)。</span><span class="sxs-lookup"><span data-stu-id="c0f6d-111">For more information about lambda expressions, see [Lambda Expressions: The `fun` Keyword](functions/lambda-expressions-the-fun-keyword.md).</span></span>
