# <a name="compiler-error-cs0545"></a><span data-ttu-id="2e611-102">编译器错误 CS0545</span><span class="sxs-lookup"><span data-stu-id="2e611-102">Compiler Error CS0545</span></span>

 <span data-ttu-id="2e611-104">尝试了定义属性访问器的重写，而此时基类没有这样的定义可供重写。</span><span class="sxs-lookup"><span data-stu-id="2e611-104">A try was made to define an override for a property accessor when the base class has no such definition to override.</span></span> <span data-ttu-id="2e611-105">可通过以下方法解决该错误：</span><span class="sxs-lookup"><span data-stu-id="2e611-105">You can resolve this error by:</span></span>

## <a name="example"></a><span data-ttu-id="2e611-111">示例</span><span class="sxs-lookup"><span data-stu-id="2e611-111">Example</span></span>  
 <span data-ttu-id="2e611-112">下面的示例生成CS0545。</span><span class="sxs-lookup"><span data-stu-id="2e611-112">The following sample generates CS0545.</span></span>  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
