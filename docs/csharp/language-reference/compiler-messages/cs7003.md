# <a name="compiler-error-cs7003"></a><span data-ttu-id="60c80-102">编译器错误 CS7003</span><span class="sxs-lookup"><span data-stu-id="60c80-102">Compiler Error CS7003</span></span>

<span data-ttu-id="60c80-103">意外使用了未绑定的通用名称</span><span class="sxs-lookup"><span data-stu-id="60c80-103">Unexpected use of an unbound generic name</span></span>

<span data-ttu-id="60c80-104">如果使用一个泛型类型，该类型需要一个参数类型，但不在尖括号之间传递任何泛型参数类型名称，则会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="60c80-104">This error occurs if you use a generic type needing one parameter type without passing any generic parameter type name between the angle brackets.</span></span> <span data-ttu-id="60c80-105">这种用法可能用于变量声明或对象实例化。</span><span class="sxs-lookup"><span data-stu-id="60c80-105">This use may be a variable declaration, or an object instantiation.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="60c80-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="60c80-106">To correct this error</span></span>

<span data-ttu-id="60c80-107">使用泛型类型时，请在尖括号中提供一个参数类型名称。</span><span class="sxs-lookup"><span data-stu-id="60c80-107">Provide one parameter type name in angle brackets when using a generic type.</span></span>

## <a name="example"></a><span data-ttu-id="60c80-108">示例</span><span class="sxs-lookup"><span data-stu-id="60c80-108">Example</span></span>

<span data-ttu-id="60c80-109">下面的示例生成 CS7003：</span><span class="sxs-lookup"><span data-stu-id="60c80-109">The following example generates CS7003:</span></span>

```csharp
// CS7003.cs
class Program
{
    static void Main(string[] args)
    {
        var myVar1 = new MyGenericClass<>();  //CS7003

        MyGenericClass<> var2;                //CS7003
    }
}

public class MyGenericClass<T> { }
```
