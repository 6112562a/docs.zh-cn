# <a name="compiler-error-cs0413"></a><span data-ttu-id="13e71-102">编译器错误 CS0413</span><span class="sxs-lookup"><span data-stu-id="13e71-102">Compiler Error CS0413</span></span>
<span data-ttu-id="13e71-103">由于类型参数“type parameter”既没有类类型约束，也没有“class”约束，因此不能与“as”运算符一起使用</span><span class="sxs-lookup"><span data-stu-id="13e71-103">The type parameter 'type parameter' cannot be used with the 'as' operator because it does not have a class type constraint nor a 'class' constraint</span></span>  
  
<span data-ttu-id="13e71-104">如果泛型类型使用 [as](../operators/type-testing-and-cast.md#as-operator) 运算符，但该泛型类型不具有类类型约束，则发生此错误。</span><span class="sxs-lookup"><span data-stu-id="13e71-104">This error occurs if a generic type uses the [as](../operators/type-testing-and-cast.md#as-operator) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="13e71-105">`as` 运算符只允许用于引用和可为 null 的值类型，因此必须约束类型参数，保证它不是值类型。</span><span class="sxs-lookup"><span data-stu-id="13e71-105">The `as` operator is only allowed with reference and nullable value types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="13e71-106">若要避免此错误，请使用类类型约束或引用类型约束。</span><span class="sxs-lookup"><span data-stu-id="13e71-106">To avoid this error, use a class type constraint or a reference type constraint.</span></span>
  
<span data-ttu-id="13e71-107">这是因为 `as` 运算符能够返回 `null`（它不可能是值类型的值），并且必须将类型参数视为值类型，除非它是类类型约束或引用类型约束。</span><span class="sxs-lookup"><span data-stu-id="13e71-107">This is because the `as` operator could return `null`, which is not a possible value for a value type, and the type parameter must be treated as a value type unless it is a class type constraint or a reference type constraint.</span></span>
  
## <a name="example"></a><span data-ttu-id="13e71-108">示例</span><span class="sxs-lookup"><span data-stu-id="13e71-108">Example</span></span>

<span data-ttu-id="13e71-109">下面的示例生成 CS0413。</span><span class="sxs-lookup"><span data-stu-id="13e71-109">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
