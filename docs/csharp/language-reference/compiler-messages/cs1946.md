# <a name="compiler-error-cs1946"></a><span data-ttu-id="66bad-102">编译器错误 CS1946</span><span class="sxs-lookup"><span data-stu-id="66bad-102">Compiler Error CS1946</span></span>

<span data-ttu-id="66bad-103">无法将匿名方法表达式转换为表达式树。</span><span class="sxs-lookup"><span data-stu-id="66bad-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="66bad-104">[匿名方法](../operators/delegate-operator.md)表示一组语句，但表达式树不得包含一条语句。</span><span class="sxs-lookup"><span data-stu-id="66bad-104">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="66bad-105">因此，不能由表达式树表达匿名方法。</span><span class="sxs-lookup"><span data-stu-id="66bad-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="66bad-106">若要更正此错误，请将匿名方法更改为 [Lambda 表达式](../../programming-guide/statements-expressions-operators/lambda-expressions.md)。</span><span class="sxs-lookup"><span data-stu-id="66bad-106">To correct this error, change the anonymous method to a [lambda expression](../../programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="66bad-107">示例</span><span class="sxs-lookup"><span data-stu-id="66bad-107">Example</span></span>

<span data-ttu-id="66bad-108">下面的示例生成 CS1946：</span><span class="sxs-lookup"><span data-stu-id="66bad-108">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
