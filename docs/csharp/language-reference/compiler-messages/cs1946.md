# <a name="compiler-error-cs1946"></a><span data-ttu-id="15821-102">编译器错误 CS1946</span><span class="sxs-lookup"><span data-stu-id="15821-102">Compiler Error CS1946</span></span>

<span data-ttu-id="15821-103">无法将匿名方法表达式转换为表达式树。</span><span class="sxs-lookup"><span data-stu-id="15821-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
<span data-ttu-id="15821-104">匿名方法表示一组语句，但表达式树不得包含一条语句。</span><span class="sxs-lookup"><span data-stu-id="15821-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="15821-105">因此，不能由表达式树表达匿名方法。</span><span class="sxs-lookup"><span data-stu-id="15821-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="15821-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="15821-106">To correct this error</span></span>

1. <span data-ttu-id="15821-107">将匿名方法更改为 Lambda 表达式。</span><span class="sxs-lookup"><span data-stu-id="15821-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="15821-108">示例</span><span class="sxs-lookup"><span data-stu-id="15821-108">Example</span></span>

<span data-ttu-id="15821-109">下面的示例生成 CS1946：</span><span class="sxs-lookup"><span data-stu-id="15821-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
