# <a name="compiler-error-cs0052"></a><span data-ttu-id="6a7f5-102">编译器错误 CS0052</span><span class="sxs-lookup"><span data-stu-id="6a7f5-102">Compiler Error CS0052</span></span>

<span data-ttu-id="6a7f5-103">可访问性不一致：字段类型“type”的可访问性低于字段“field”</span><span class="sxs-lookup"><span data-stu-id="6a7f5-103">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>

 <span data-ttu-id="6a7f5-104">字段类型的可访问性不能低于字段本身的可访问性，因为所有公共构造函数都必须返回一个可公开访问的对象。</span><span class="sxs-lookup"><span data-stu-id="6a7f5-104">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>

## <a name="example"></a><span data-ttu-id="6a7f5-105">示例</span><span class="sxs-lookup"><span data-stu-id="6a7f5-105">Example</span></span>

 <span data-ttu-id="6a7f5-106">下面的示例生成 CS0052：</span><span class="sxs-lookup"><span data-stu-id="6a7f5-106">The following sample generates CS0052:</span></span>

```csharp
// CS0052.cs
public class MyClass2
{
    // The following line causes an error because the field, M, is declared
    // as public, but the type, MyClass, is private. Therefore the type is
    // less accessible than the field.
    public MyClass M;   // CS0052

    private class MyClass
    {
    }
    // One way to resolve the error is to change the accessibility of the type
    // to public.
    //public class MyClass
    // Another solution is to change the accessibility of the field to private.
    // private MyClass M;
}

public class MainClass
{
    public static void Main()
    {
    }
}
```
