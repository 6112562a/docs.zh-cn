# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="0fd15-102">编译器警告（等级 2）CS0108</span><span class="sxs-lookup"><span data-stu-id="0fd15-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="0fd15-103">“member1”会隐藏继承的成员“member2”。</span><span class="sxs-lookup"><span data-stu-id="0fd15-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="0fd15-104">如果打算隐藏，请使用 new 关键字。</span><span class="sxs-lookup"><span data-stu-id="0fd15-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="0fd15-105">声明变量所使用的名称与基类中的变量名称相同。</span><span class="sxs-lookup"><span data-stu-id="0fd15-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="0fd15-106">但没有使用 [new](../../../csharp/language-reference/keywords/new.md) 关键字。</span><span class="sxs-lookup"><span data-stu-id="0fd15-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="0fd15-107">此警告通知你应使用 new；按照已在声明中使用了 new 来声明变量。</span><span class="sxs-lookup"><span data-stu-id="0fd15-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="0fd15-108">以下示例生成 CS0108：</span><span class="sxs-lookup"><span data-stu-id="0fd15-108">The following sample generates CS0108:</span></span>  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
