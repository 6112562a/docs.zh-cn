# <a name="compiler-error-cs0579"></a><span data-ttu-id="effdd-102">编译器错误 CS0579</span><span class="sxs-lookup"><span data-stu-id="effdd-102">Compiler Error CS0579</span></span>
<span data-ttu-id="effdd-103">“attribute”属性重复</span><span class="sxs-lookup"><span data-stu-id="effdd-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="effdd-104">不可多次指定相同的属性，除非该属性在其 [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md) 中指定 AllowMultiple=true  。</span><span class="sxs-lookup"><span data-stu-id="effdd-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="effdd-105">示例</span><span class="sxs-lookup"><span data-stu-id="effdd-105">Example</span></span>  
 <span data-ttu-id="effdd-106">下面的示例生成 CS0579。</span><span class="sxs-lookup"><span data-stu-id="effdd-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
