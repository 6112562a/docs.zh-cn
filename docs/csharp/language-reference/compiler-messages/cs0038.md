 <span data-ttu-id="98b8f-104">类中的字段并不自动可用于嵌套类。</span><span class="sxs-lookup"><span data-stu-id="98b8f-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="98b8f-105">若要可用于嵌套类，该字段必须是[静态](../keywords/static.md)字段。</span><span class="sxs-lookup"><span data-stu-id="98b8f-105">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="98b8f-106">否则，必须创建外部类的实例。</span><span class="sxs-lookup"><span data-stu-id="98b8f-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="98b8f-107">有关详细信息，请参阅[嵌套类型](../../programming-guide/classes-and-structs/nested-types.md)。</span><span class="sxs-lookup"><span data-stu-id="98b8f-107">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>

```csharp
// CS0038.cs

{
    public int count;
    // Try the following line instead.
    // public static int count;

    class InnerClass
    {
        void Func()
        {
            // or, create an instance
            // OuterClass class_inst = new OuterClass();
            // int count2 = class_inst.count;
            int count2 = count;   // CS0038
        }
    }

    public static void Main()
    {
    }
}
```
