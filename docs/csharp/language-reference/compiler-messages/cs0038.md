# <a name="compiler-error-cs0038"></a><span data-ttu-id="913c5-102">编译器错误 CS0038</span><span class="sxs-lookup"><span data-stu-id="913c5-102">Compiler Error CS0038</span></span>
<span data-ttu-id="913c5-103">无法通过嵌套类型“type2”访问外部类型“type1”的非静态成员</span><span class="sxs-lookup"><span data-stu-id="913c5-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="913c5-104">类中的字段并不自动可用于嵌套类。</span><span class="sxs-lookup"><span data-stu-id="913c5-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="913c5-105">若要可用于嵌套类，该字段必须是[静态](../keywords/static.md)字段。</span><span class="sxs-lookup"><span data-stu-id="913c5-105">To be available to a nested class, the field must be [static](../keywords/static.md).</span></span> <span data-ttu-id="913c5-106">否则，必须创建外部类的实例。</span><span class="sxs-lookup"><span data-stu-id="913c5-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="913c5-107">有关详细信息，请参阅[嵌套类型](../../programming-guide/classes-and-structs/nested-types.md)。</span><span class="sxs-lookup"><span data-stu-id="913c5-107">For more information, see [Nested Types](../../programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="913c5-108">下面的示例生成 CS0038：</span><span class="sxs-lookup"><span data-stu-id="913c5-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
