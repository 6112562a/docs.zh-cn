## <a name="example"></a><span data-ttu-id="34527-110">示例</span><span class="sxs-lookup"><span data-stu-id="34527-110">Example</span></span>

<span data-ttu-id="34527-111">下面的示例生成 CS1936：</span><span class="sxs-lookup"><span data-stu-id="34527-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```
