# <a name="compiler-error-cs0188"></a><span data-ttu-id="4a708-102">编译器错误 CS0188</span><span class="sxs-lookup"><span data-stu-id="4a708-102">Compiler Error CS0188</span></span>

<span data-ttu-id="4a708-103">在给“this”对象的所有字段赋值之前，无法使用该对象</span><span class="sxs-lookup"><span data-stu-id="4a708-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="4a708-104">构造函数必须给 `struct` 中的所有字段都赋值，然后才能调用 `struct` 中的方法。</span><span class="sxs-lookup"><span data-stu-id="4a708-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="4a708-105">如果尝试在结构构造函数中初始化属性时看到此错误，解决方法是直接初始化该属性的支持字段而不是属性本身。</span><span class="sxs-lookup"><span data-stu-id="4a708-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="4a708-106">示例</span><span class="sxs-lookup"><span data-stu-id="4a708-106">Example</span></span>

 <span data-ttu-id="4a708-107">以下示例生成 CS0188：</span><span class="sxs-lookup"><span data-stu-id="4a708-107">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
