# <a name="compiler-error-cs0016"></a>编译器错误 CS0016
未能写入输出文件“file”—“reason”  
  
 编译器未能写入输出文件。 检查文件路径，确保存在该路径。 如果该位置已存在一个以前生成的文件，请确保该文件可写入，并且当前没有进程将该文件锁定。 例如，确保在尝试构建时可执行文件未在运行。
