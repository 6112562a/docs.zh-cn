# <a name="compiler-error-cs0122"></a>编译器错误 CS0122
“member”不可访问，因为它具有一定的保护级别  
  
 类成员的[访问修饰符](../keywords/modifiers.md)禁止访问该成员。 有关详细信息，请参阅[访问修饰符](../../programming-guide/classes-and-structs/access-modifiers.md)。  
  
 导致出现此错误（未在下面的示例中显示）的一个原因是：在友元程序集的目标上省略了 /out 编译器标志  。 有关详细信息，请参阅[友元程序集](../../../standard/assembly/friend.md)和 [/out（C# 编译器选项）](../compiler-options/out-compiler-option.md)  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0122：  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
