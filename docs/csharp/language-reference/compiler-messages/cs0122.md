## <a name="example"></a><span data-ttu-id="f2612-108">示例</span><span class="sxs-lookup"><span data-stu-id="f2612-108">Example</span></span>

 <span data-ttu-id="f2612-109">下面的示例生成 CS0122：</span><span class="sxs-lookup"><span data-stu-id="f2612-109">The following sample generates CS0122:</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
