# <a name="compiler-error-cs1721"></a><span data-ttu-id="1316a-102">编译器错误 CS1721</span><span class="sxs-lookup"><span data-stu-id="1316a-102">Compiler Error CS1721</span></span>

<span data-ttu-id="1316a-103">类“class”不能具有多个基类：“class_1”和“class_2”</span><span class="sxs-lookup"><span data-stu-id="1316a-103">Class 'class' cannot have multiple base classes: 'class_1' and 'class_2'</span></span>

<span data-ttu-id="1316a-104">此错误消息最常见的原因是尝试使用多重继承。</span><span class="sxs-lookup"><span data-stu-id="1316a-104">The most common cause of this error message is attempting to use multiple inheritance.</span></span> <span data-ttu-id="1316a-105">C# 中的类只能直接从一个类继承。</span><span class="sxs-lookup"><span data-stu-id="1316a-105">A class in C# may only inherit directly from one class.</span></span> <span data-ttu-id="1316a-106">但是，一个类可以实现任何数量的接口。</span><span class="sxs-lookup"><span data-stu-id="1316a-106">However, a class can implement any number of interfaces.</span></span>

## <a name="example"></a><span data-ttu-id="1316a-107">示例</span><span class="sxs-lookup"><span data-stu-id="1316a-107">Example</span></span>

<span data-ttu-id="1316a-108">下面的示例演示可能生成 CS1721 的一种方法：</span><span class="sxs-lookup"><span data-stu-id="1316a-108">The following example shows one way in which CS1721 is generated:</span></span>

```csharp
// CS1721.cs
public class A {}
public class B {}
public class MyClass : A, B {}   // CS1721
```

## <a name="to-correct-this-error"></a><span data-ttu-id="1316a-109">更正此错误</span><span class="sxs-lookup"><span data-stu-id="1316a-109">To correct this error</span></span>

<span data-ttu-id="1316a-110">使用以下不同方法，可更正此错误：</span><span class="sxs-lookup"><span data-stu-id="1316a-110">The following are different ways to correct this error:</span></span>

- <span data-ttu-id="1316a-111">使类 `B` 继承自 `A`，`MyClass` 继承自 `B`：</span><span class="sxs-lookup"><span data-stu-id="1316a-111">Make class `B` inherit from `A`, and `MyClass` inherit from `B`:</span></span>

    ```csharp
    public class A {}
    public class B : A {}
    public class MyClass : B {}
    ```

- <span data-ttu-id="1316a-112">将 `B` 声明为接口。</span><span class="sxs-lookup"><span data-stu-id="1316a-112">Declare `B` as an interface.</span></span> <span data-ttu-id="1316a-113">使 `MyClass` 继承自接口 `B` 和类 `A`：</span><span class="sxs-lookup"><span data-stu-id="1316a-113">Make `MyClass` inherit from the interface `B`, and the class `A`:</span></span>

    ```csharp
    public class A {}
    public interface B {}
    public class MyClass : A, B {}
    ```
