
<span data-ttu-id="eb52b-126">使用 [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) 运算符以获取表示指定类型的 <xref:System.Type?displayProperty=nameWithType> 实例：</span><span class="sxs-lookup"><span data-stu-id="eb52b-126">Use the [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
// System.Double
```
