# <a name="using-c-reference"></a>using（C# 参考）

`using` 关键字有三个主要用途：
- [using 语句](using-statement.md)定义一个范围，在此范围的末尾将释放对象。 
- using [指令](using-directive.md)创建命名空间别名，或导入在其他命名空间中定义的类型。 
- [using static 指令](using-static.md)导入单个类的成员。
