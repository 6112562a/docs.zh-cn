# <a name="ascending-c-reference"></a>ascending（C# 参考）
查询表达式中的 [orderby 子句](../../../csharp/language-reference/keywords/orderby-clause.md) 中使用 `ascending` 上下文关键字指定排序顺序为从小到大。 由于 `ascending` 为默认排序顺序，因此无需加以指定。
