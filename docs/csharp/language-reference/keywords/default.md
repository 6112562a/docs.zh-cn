# <a name="default-c-reference"></a>default（C# 参考）

`default` 关键字有两种用法：
- 指定默认标签的 [`switch` 语句](switch.md)。
- 生成类型的默认值的[默认值表达式](../../programming-guide/statements-expressions-operators/default-value-expressions.md)。
