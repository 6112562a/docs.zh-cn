# <a name="selection-statements-c-reference"></a>选择语句（C# 参考）

选择语句会导致基于特定条件是否为 `true` 而将程序控件转移给特定的流。

选择语句中使用下列关键字：

- [if](if-else.md)

- [else](if-else.md)

- [switch](switch.md)

- [case](switch.md)

- [default](switch.md)
