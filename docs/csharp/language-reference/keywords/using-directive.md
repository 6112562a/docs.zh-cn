# <a name="using-directive-c-reference"></a><span data-ttu-id="8b0f4-102">using 指令（C# 参考）</span><span class="sxs-lookup"><span data-stu-id="8b0f4-102">using Directive (C# Reference)</span></span>
<span data-ttu-id="8b0f4-103">`using` 指令有三种用途：</span><span class="sxs-lookup"><span data-stu-id="8b0f4-103">The `using` directive has three uses:</span></span>  
  
-   <span data-ttu-id="8b0f4-104">允许在命名空间中使用类型，这样无需在该命名空间中限定某个类型的使用：</span><span class="sxs-lookup"><span data-stu-id="8b0f4-104">To allow the use of types in a namespace so that you do not have to qualify the use of a type in that namespace:</span></span>  
  
    ```csharp  
    using System.Text;  
    ```  
  
-   <span data-ttu-id="8b0f4-105">允许访问类型的静态成员和嵌套类型，而无需限定使用类型名称进行访问。</span><span class="sxs-lookup"><span data-stu-id="8b0f4-105">To allow you to access static members and nested types of a type without having to qualify the access with the type name.</span></span> 
  
    ```csharp  
    using static System.Math;  
    ```  
     
    <span data-ttu-id="8b0f4-106">有关详细信息，请参阅 [using static 指令](using-static.md)。</span><span class="sxs-lookup"><span data-stu-id="8b0f4-106">For more information, see the [using static directive](using-static.md).</span></span>

-   <span data-ttu-id="8b0f4-107">为命名空间或类型创建别名。</span><span class="sxs-lookup"><span data-stu-id="8b0f4-107">To create an alias for a namespace or a type.</span></span> <span data-ttu-id="8b0f4-108">这称为 *using 别名指令*。</span><span class="sxs-lookup"><span data-stu-id="8b0f4-108">This is called a *using alias directive*.</span></span>  
  
    ```csharp  
    using Project = PC.MyCompany.Project;  
    ```  
  
 <span data-ttu-id="8b0f4-109">`using` 关键字还用于创建 using 语句，此类语句有助于确保正确处理 <xref:System.IDisposable> 对象（如文件和字体）。</span><span class="sxs-lookup"><span data-stu-id="8b0f4-109">The `using` keyword is also used to create *using statements*, which help ensure that <xref:System.IDisposable> objects such as files and fonts are handled correctly.</span></span> <span data-ttu-id="8b0f4-110">有关详细信息，请参阅 [using 语句](../../../csharp/language-reference/keywords/using-statement.md)。</span><span class="sxs-lookup"><span data-stu-id="8b0f4-110">See [using Statement](../../../csharp/language-reference/keywords/using-statement.md) for more information.</span></span>  
  
## <a name="using-static-type"></a><span data-ttu-id="8b0f4-111">Using Static 类型</span><span class="sxs-lookup"><span data-stu-id="8b0f4-111">Using Static Type</span></span>  
 <span data-ttu-id="8b0f4-112">你可以访问类型的静态成员，而无需限定使用类型名称进行访问：</span><span class="sxs-lookup"><span data-stu-id="8b0f4-112">You can access static members of a type without having to qualify the access with the type name:</span></span>  
  
```csharp  
using static System.Console;   
using static System.Math;  
class Program   
{   
    static void Main()   
    {   
        WriteLine(Sqrt(3*3 + 4*4));   
    }   
}  
```
