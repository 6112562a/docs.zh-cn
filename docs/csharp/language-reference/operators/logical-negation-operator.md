# <a name="-operator-c-reference"></a>! 运算符（C# 参考）
逻辑非运算符 (`!`) 是一种否定其操作数的一元运算符。 它针对 `bool` 定义，当且仅当其操作数为 `false` 时返回 `true`。  
  
## <a name="remarks"></a>备注  
 用户定义的类型可以重载 `!` 运算符（请参阅[运算符](../../../csharp/language-reference/keywords/operator.md)）。
