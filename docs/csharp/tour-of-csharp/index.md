# <a name="a-tour-of-the-c-language"></a><span data-ttu-id="87e59-104">C# 语言介绍</span><span class="sxs-lookup"><span data-stu-id="87e59-104">A tour of the C# language</span></span>

<span data-ttu-id="87e59-105">C#（读作“See Sharp”）是一种新式编程语言，不仅面向对象，还类型安全。</span><span class="sxs-lookup"><span data-stu-id="87e59-105">C# (pronounced "See Sharp") is a modern, object-oriented, and type-safe programming language.</span></span> <span data-ttu-id="87e59-106">C# 源于 C 语言系列，C、C++、Java 和 JavaScript 程序员很快就可以上手使用。</span><span class="sxs-lookup"><span data-stu-id="87e59-106">C# has its roots in the C family of languages and will be immediately familiar to C, C++, Java, and JavaScript programmers.</span></span>

<span data-ttu-id="87e59-107">本教程概述了 C# 8 及更高版本中该语言的主要组件。</span><span class="sxs-lookup"><span data-stu-id="87e59-107">This tour provides an overview of the major components of the language in C# 8 and earlier.</span></span> <span data-ttu-id="87e59-108">如果想要通过交互式示例探索语言，请尝试 [C# 简介](../tutorials/intro-to-csharp/index.md)教程。</span><span class="sxs-lookup"><span data-stu-id="87e59-108">If you want to explore the language through interactive examples, try the [introduction to C#](../tutorials/intro-to-csharp/index.md) tutorials.</span></span>

<span data-ttu-id="87e59-109">C# 是一种面向对象的语言。不仅如此，C# 还进一步支持***面向组件的***编程。</span><span class="sxs-lookup"><span data-stu-id="87e59-109">C# is an object-oriented language, but C# further includes support for ***component-oriented*** programming.</span></span> <span data-ttu-id="87e59-110">当代软件设计越来越依赖采用自描述的独立功能包形式的软件组件。</span><span class="sxs-lookup"><span data-stu-id="87e59-110">Contemporary software design increasingly relies on software components in the form of self-contained and self-describing packages of functionality.</span></span> <span data-ttu-id="87e59-111">此类组件的关键是它们提供包含属性、方法和事件的编程模型。</span><span class="sxs-lookup"><span data-stu-id="87e59-111">Key to such components is that they present a programming model with properties, methods, and events.</span></span> <span data-ttu-id="87e59-112">它们包含提供组件相关声明性信息的属性。</span><span class="sxs-lookup"><span data-stu-id="87e59-112">They have attributes that provide declarative information about the component.</span></span> <span data-ttu-id="87e59-113">它们合并了自己的文档。</span><span class="sxs-lookup"><span data-stu-id="87e59-113">They incorporate their own documentation.</span></span> <span data-ttu-id="87e59-114">C# 提供了语言构造来直接支持这些概念，让 C# 成为一种非常自然的语言，可用于创建和使用软件组件。</span><span class="sxs-lookup"><span data-stu-id="87e59-114">C# provides language constructs to support directly these concepts, making C# a natural language in which to create and use software components.</span></span>

<span data-ttu-id="87e59-115">多项 C# 功能有助于构造可靠耐用的应用程序。</span><span class="sxs-lookup"><span data-stu-id="87e59-115">Several C# features aid in the construction of robust and durable applications.</span></span> <span data-ttu-id="87e59-116">“垃圾回收”会自动回收无法访问的未使用对象所占用的内存。</span><span class="sxs-lookup"><span data-stu-id="87e59-116">***Garbage collection*** automatically reclaims memory occupied by unreachable unused objects.</span></span> <span data-ttu-id="87e59-117">“异常处理”提供了一种结构化且可扩展的方法来进行错误检测和恢复。</span><span class="sxs-lookup"><span data-stu-id="87e59-117">***Exception handling*** provides a structured and extensible approach to error detection and recovery.</span></span> <span data-ttu-id="87e59-118">该语言的“类型安全”设计使其无法读取未初始化的变量、无法索引超出其范围的数组或执行未经检查的类型转换。</span><span class="sxs-lookup"><span data-stu-id="87e59-118">The ***type-safe*** design of the language makes it impossible to read from uninitialized variables, to index arrays beyond their bounds, or to perform unchecked type casts.</span></span>

<span data-ttu-id="87e59-119">C# 采用***统一的类型系统***。</span><span class="sxs-lookup"><span data-stu-id="87e59-119">C# has a ***unified type system***.</span></span> <span data-ttu-id="87e59-120">所有 C# 类型（包括 `int` 和 `double` 等基元类型）均继承自一个根 `object` 类型。</span><span class="sxs-lookup"><span data-stu-id="87e59-120">All C# types, including primitive types such as `int` and `double`, inherit from a single root `object` type.</span></span> <span data-ttu-id="87e59-121">因此，所有类型共用一组通用运算，任何类型的值都可以一致地进行存储、传输和处理。</span><span class="sxs-lookup"><span data-stu-id="87e59-121">Thus, all types share a set of common operations, and values of any type can be stored, transported, and operated upon in a consistent manner.</span></span> <span data-ttu-id="87e59-122">此外，C# 还支持用户定义的引用类型和值类型，从而支持对象动态分配以及轻量级结构的内嵌式存储。</span><span class="sxs-lookup"><span data-stu-id="87e59-122">Furthermore, C# supports both user-defined reference types and value types, allowing dynamic allocation of objects as well as in-line storage of lightweight structures.</span></span>

<span data-ttu-id="87e59-123">为了确保 C# 程序和库能够持续兼容，C# 设计非常注重***版本控制***。</span><span class="sxs-lookup"><span data-stu-id="87e59-123">To ensure that C# programs and libraries can evolve over time in a compatible manner, much emphasis has been placed on ***versioning*** in C#'s design.</span></span> <span data-ttu-id="87e59-124">许多编程语言很少关注此问题。</span><span class="sxs-lookup"><span data-stu-id="87e59-124">Many programming languages pay little attention to this issue.</span></span> <span data-ttu-id="87e59-125">因此，当引入新版依赖库时，用这些语言编写的程序会出现更多不必要的中断现象。</span><span class="sxs-lookup"><span data-stu-id="87e59-125">As a result, programs written in those other languages break more often than necessary when newer versions of dependent libraries are introduced.</span></span> <span data-ttu-id="87e59-126">C# 设计中受版本控制加强直接影响的方面包括：单独的 `virtual` 和 `override` 修饰符，关于方法重载决策的规则，以及对显式接口成员声明的支持。</span><span class="sxs-lookup"><span data-stu-id="87e59-126">Aspects of C#'s design that were directly influenced by versioning considerations include the separate `virtual` and `override` modifiers, the rules for method overload resolution, and support for explicit interface member declarations.</span></span>

<span data-ttu-id="87e59-127">在较新的版本中，C# 包含了其他编程范例。</span><span class="sxs-lookup"><span data-stu-id="87e59-127">In more recent versions, C# has embraced other programming paradigms.</span></span> <span data-ttu-id="87e59-128">C# 包含支持 lambda 表达式等功能编程技术的功能。</span><span class="sxs-lookup"><span data-stu-id="87e59-128">C# has included features that support functional programming techniques like lambda expressions.</span></span> <span data-ttu-id="87e59-129">其他新功能支持分离数据和算法，例如模式匹配。</span><span class="sxs-lookup"><span data-stu-id="87e59-129">Other new features support separating data and algorithms, like pattern matching.</span></span>
