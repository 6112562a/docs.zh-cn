# <a name="compiler-warning-level-4-cs0628"></a><span data-ttu-id="7e7b2-102">编译器警告（等级 4）CS0628</span><span class="sxs-lookup"><span data-stu-id="7e7b2-102">Compiler Warning (level 4) CS0628</span></span>
<span data-ttu-id="7e7b2-103">“member”：在密封类中声明了新的保护成员</span><span class="sxs-lookup"><span data-stu-id="7e7b2-103">'member' : new protected member declared in sealed class</span></span>  
  
 <span data-ttu-id="7e7b2-104">[密封](../../csharp/language-reference/keywords/sealed.md) 类不能引入 [保护](../../csharp/language-reference/keywords/protected.md) 成员，因为其他类不能继承 `sealed` 类并使用 `protected` 成员。</span><span class="sxs-lookup"><span data-stu-id="7e7b2-104">A [sealed](../../csharp/language-reference/keywords/sealed.md) class cannot introduce a [protected](../../csharp/language-reference/keywords/protected.md) member because no other class will be able to inherit from the `sealed` class and use the `protected` member.</span></span>  
  
 <span data-ttu-id="7e7b2-105">下面的示例生成 CS0628：</span><span class="sxs-lookup"><span data-stu-id="7e7b2-105">The following sample generates CS0628:</span></span>

```csharp  
{  
   protected int i;   // CS0628  
}

{  
   public static void Main()  
   {  
   }  
}  
```
