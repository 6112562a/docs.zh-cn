# <a name="compiler-error-cs0743"></a><span data-ttu-id="d2433-102">编译器错误 CS0743</span><span class="sxs-lookup"><span data-stu-id="d2433-102">Compiler Error CS0743</span></span>
<span data-ttu-id="d2433-103">应为上下文关键字 "on"</span><span class="sxs-lookup"><span data-stu-id="d2433-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="d2433-104">`join` 子句的模式是 `join`...`in`...`on`...`equals`，如本示例所示：</span><span class="sxs-lookup"><span data-stu-id="d2433-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d2433-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="d2433-105">To correct this error</span></span>  
  
1. <span data-ttu-id="d2433-106">将 `on` 关键字添加到 `join` 子句中。</span><span class="sxs-lookup"><span data-stu-id="d2433-106">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d2433-107">示例</span><span class="sxs-lookup"><span data-stu-id="d2433-107">Example</span></span>  
 <span data-ttu-id="d2433-108">下面的代码生成 CS0743：</span><span class="sxs-lookup"><span data-stu-id="d2433-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
