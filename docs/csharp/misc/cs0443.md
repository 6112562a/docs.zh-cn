# <a name="compiler-error-cs0443"></a>编译器错误 CS0443
语法错误，应为值  
  
 当引用数组而没有指定数组索引的值时，将发生此错误。  
  
## <a name="example"></a>示例  
 以下代码生成 CS0443：  
  
```csharp  
// CS0443.cs   
using System;   
class MyClass   
{  
    public static void Main()      
    {  
        int[,] x = new int[1,5];  
        if (x[] == 5) {} // CS0443  
        // if (x[0, 0] == 5) {}   
    }  
}  
```
