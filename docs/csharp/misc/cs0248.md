# <a name="compiler-error-cs0248"></a>编译器错误 CS0248
无法创建大小为负值的数组  
  
 使用负数指定了数组大小。 有关详细信息，请参阅[数组](../../csharp/programming-guide/arrays/index.md)。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0248：  
  
```csharp  
// CS0248.cs  
class MyClass  
{  
    public static void Main()  
    {  
        int[] myArray = new int[-3] {1,2,3};   // CS0248, pass a nonnegative number  
    }  
}  
```
