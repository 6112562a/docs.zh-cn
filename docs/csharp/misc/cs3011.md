## <a name="example"></a><span data-ttu-id="9be6c-107">示例</span><span class="sxs-lookup"><span data-stu-id="9be6c-107">Example</span></span>  
 <span data-ttu-id="9be6c-108">下面的示例生成 CS3011：</span><span class="sxs-lookup"><span data-stu-id="9be6c-108">The following example generates CS3011:</span></span>  
  
```csharp  
// CS3011.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public abstract class I  
{  
    [CLSCompliant(false)]  
    public abstract int M();   // CS3011  
  
    // OK  
    [CLSCompliant(false)]  
    public void M2()  
    {  
    }  
}  
  
public class C : I  
{  
    public override int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
