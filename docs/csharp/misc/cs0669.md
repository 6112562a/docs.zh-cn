<span data-ttu-id="d1296-103">具有 ComImport 特性的类不能有用户定义的构造函数。</span><span class="sxs-lookup"><span data-stu-id="d1296-103">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="d1296-104">公共语言运行时中的 COM 互操作层为 [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) 类提供构造函数。</span><span class="sxs-lookup"><span data-stu-id="d1296-104">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="d1296-105">因此，COM 对象可以用作运行时中的托管对象。</span><span class="sxs-lookup"><span data-stu-id="d1296-105">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
