# <a name="compiler-error-cs1565"></a>编译器错误 CS1565
指定的冲突选项：Win32 资源文件;Win32 图标  
  
 不可在同一编译中同时指定 [/win32res](../language-reference/compiler-options/win32res-compiler-option.md) 编译器选项和 [/win32ico](../language-reference/compiler-options/win32icon-compiler-option.md) 编译器选项。
