<span data-ttu-id="72423-103">无法获取给定表达式的地址</span><span class="sxs-lookup"><span data-stu-id="72423-103">Cannot take the address of the given expression</span></span>  
  
 <span data-ttu-id="72423-104">可以采用字段、局部变量和指针间接寻址的地址，但不能采用两个局部变量之和的地址等。</span><span class="sxs-lookup"><span data-stu-id="72423-104">You can take the address of fields, local variables, and indirection of pointers, but you cannot take, for example, the address of the sum of two local variables.</span></span> <span data-ttu-id="72423-105">有关详细信息，请参阅[不安全代码和指针](../programming-guide/unsafe-code-pointers/index.md)。</span><span class="sxs-lookup"><span data-stu-id="72423-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="72423-106">下面的示例生成 CS0211：</span><span class="sxs-lookup"><span data-stu-id="72423-106">The following sample generates CS0211:</span></span>  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
