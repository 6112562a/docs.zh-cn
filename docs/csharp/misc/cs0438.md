 <span data-ttu-id="51838-104">当源文件中的类型与另一个源文件中的命名空间冲突时，将发生此错误。</span><span class="sxs-lookup"><span data-stu-id="51838-104">This error occurs when a type in a source file conflicts with a namespace in another source file.</span></span> <span data-ttu-id="51838-105">这种情况通常发生在其中之一或二者均来自添加的模块时。</span><span class="sxs-lookup"><span data-stu-id="51838-105">This typically happens when one or both come from an added module.</span></span> <span data-ttu-id="51838-106">若要解决此问题，请对导致冲突的类型或命名空间进行重命名。</span><span class="sxs-lookup"><span data-stu-id="51838-106">To resolve, rename the type or the namespace that caused the conflict.</span></span>

 <span data-ttu-id="51838-109">再编译以下文件：</span><span class="sxs-lookup"><span data-stu-id="51838-109">Then compile this file:</span></span>

 <span data-ttu-id="51838-110">然后编译以下文件：</span><span class="sxs-lookup"><span data-stu-id="51838-110">And then compile this file:</span></span>
