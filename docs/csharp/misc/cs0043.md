# <a name="compiler-error-cs0043"></a>编译器错误 CS0043

PDB 文件“file”的格式不正确或过时。 请将其删除并重新生成。

 删除此编译生成的 .pdb 文件并重新编译。

> [!NOTE]
> Roslyn 中不再使用此编译器错误。
