# <a name="compiler-error-cs0218"></a><span data-ttu-id="a6379-102">编译器错误 CS0218</span><span class="sxs-lookup"><span data-stu-id="a6379-102">Compiler Error CS0218</span></span>
<span data-ttu-id="a6379-103">类型（“type”）必须包含运算符 true 和运算符 false 的声明</span><span class="sxs-lookup"><span data-stu-id="a6379-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="a6379-104">如果某个用户定义类型重载[& 运算符](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-)或[&#124;运算符](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-)，它还必须定义[true 和 false](../language-reference/operators/true-false-operators.md)运算符，以使短路[& & 运算符](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-)或[ &#124; &#124;运算符](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-)定义。</span><span class="sxs-lookup"><span data-stu-id="a6379-104">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/operators/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="a6379-105">以下示例生成 CS0218：</span><span class="sxs-lookup"><span data-stu-id="a6379-105">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
