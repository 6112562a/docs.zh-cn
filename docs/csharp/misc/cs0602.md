# <a name="compiler-warning-level-1-cs0602"></a>编译器警告（等级 1）CS0602
功能“old_feature”已弃用。 请改为使用“new_feature”  
  
 虽然目前仍然支持 (*old_feature*) 代码中使用的语言功能，但在以后的版本中可能会删除该支持。 应转而使用建议的语法 (*new_feature*)。
