# <a name="compiler-error-cs0041"></a>编译器错误 CS0041
“type”的完全限定名对于调试信息而言太长。 请在不使用“/debug”选项的情况下编译。  
  
 使用 [/debug](../language-reference/compiler-options/debug-compiler-option.md) 编译器选项时会出现此错误。 如果你遇到此错误，请尝试删除 bin 目录中的 PDB 文件并重新编译。 如果仍然遇到此错误, 则可能需要修复或重新安装 Visual Studio。
