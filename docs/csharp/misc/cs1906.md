# <a name="compiler-error-cs1906"></a>编译器错误 CS1906
选项“option”无效；资源可见性必须是“public”或“private”  
  
 此错误表示无效的[/resource （嵌入资源文件）](../language-reference/compiler-options/resource-compiler-option.md)或[-linkresource （链接到 .NET Framework 资源）](../language-reference/compiler-options/linkresource-compiler-option.md)命令行选项。 检查 **/resource** 或 **/linkresource** 命令行选项的语法，确保使用的可访问性修饰符为 **public** 或 `private`。
