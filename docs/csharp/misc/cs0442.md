# <a name="compiler-error-cs0442"></a>编译器错误 CS0442
“Property”：抽象属性不能具有专用访问器  
  
 使用访问修饰符“private”修饰抽象访问器时，将出现此错误。 解决方法是使用其他访问修饰符或将该属性改为非抽象。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0442：  
  
```csharp  
// CS0442.cs  
public abstract class MyClass   
{  
    public abstract int AbstractProperty   
    {  
        get;  
        private set;   // CS0442  
        // Try this instead:  
        // set;  
    }  
}  
```
