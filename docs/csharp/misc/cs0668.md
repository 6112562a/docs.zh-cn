# <a name="compiler-error-cs0668"></a><span data-ttu-id="c80ce-102">编译器错误 CS0668</span><span class="sxs-lookup"><span data-stu-id="c80ce-102">Compiler Error CS0668</span></span>
<span data-ttu-id="c80ce-103">两个索引器的名称不同；在类型中的每个索引器上的 IndexerName 特性都必须使用相同的名称</span><span class="sxs-lookup"><span data-stu-id="c80ce-103">Two indexers have different names; the IndexerName attribute must be used with the same name on every indexer within a type</span></span>  
  
 <span data-ttu-id="c80ce-104">对于类型中的所有索引器，传递给 **IndexerName** 特性的值必须是相同的。</span><span class="sxs-lookup"><span data-stu-id="c80ce-104">The values passed to the **IndexerName** attribute must be the same for all indexers in a type.</span></span> <span data-ttu-id="c80ce-105">有关 **IndexerName** 特性的详细信息，请参阅 [IndexerNameAttribute 类](xref:System.Runtime.CompilerServices.IndexerNameAttribute)。</span><span class="sxs-lookup"><span data-stu-id="c80ce-105">For more information on the **IndexerName** attribute, see [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="c80ce-106">下面的示例生成 CS0668：</span><span class="sxs-lookup"><span data-stu-id="c80ce-106">The following sample generates CS0668:</span></span>  
  
```csharp
// CS0668.cs  
using System;  
using System.Runtime.CompilerServices;  
  
class IndexerClass  
{  
   [IndexerName("IName1")]  
   public int this [int index]   // indexer declaration  
   {  
      get  
      {  
         return index;  
      }  
      set  
      {  
      }  
   }  
  
   [IndexerName("IName2")]  
   public int this [string s]    // CS0668, change IName2 to IName1  
   {  
      get  
      {  
         return int.Parse(s);  
      }  
      set  
      {  
      }  
   }  
  
   void Main()  
   {  
   }  
}  
```
