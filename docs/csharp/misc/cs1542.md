# <a name="compiler-error-cs1542"></a>编译器错误 CS1542
“dll”无法添加到此程序集中，因为它已经是程序集；请使用“/R”选项  
  
 使用 [/addmodule](../language-reference/compiler-options/addmodule-compiler-option.md) 编译器选项引用的文件不是使用 [/target:module](../language-reference/compiler-options/target-module-compiler-option.md)生成的；在此编译中使用 [/reference](../language-reference/compiler-options/reference-compiler-option.md) 来引用该文件。
