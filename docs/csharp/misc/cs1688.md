<span data-ttu-id="8cba1-103">无法将不含参数列表的匿名方法块转换为委托类型“delegate”，原因是该方法块具有一个或多个 out 参数</span><span class="sxs-lookup"><span data-stu-id="8cba1-103">Cannot convert anonymous method block without a parameter list to delegate type 'delegate' because it has one or more out parameters</span></span>  
  
 <span data-ttu-id="8cba1-104">大多数情况下，编译器允许省略匿名方法块中的参数。</span><span class="sxs-lookup"><span data-stu-id="8cba1-104">The compiler allows parameters to be omitted from an anonymous method block in most cases.</span></span> <span data-ttu-id="8cba1-105">如果匿名方法块不具有参数列表，而委托具有 `out` 参数，则将出现此错误。</span><span class="sxs-lookup"><span data-stu-id="8cba1-105">This error arises when the anonymous method block does not have a parameter list, but the delegate has an `out` parameter.</span></span> <span data-ttu-id="8cba1-106">由于编译器需要忽略 `out` 参数的存在（而这不太可能是正确行为），因此它不允许这种情况。</span><span class="sxs-lookup"><span data-stu-id="8cba1-106">The compiler does not allow this situation because it would need to ignore the presence of the `out` parameter, which is unlikely to be the correct behavior.</span></span>
