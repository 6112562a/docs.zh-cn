# <a name="compiler-error-cs0833"></a>编译器错误 CS0833
匿名类型不能有多个同名属性。  
  
 匿名类型与其他类型一样，不能有两个同名称的属性。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 为类型中的每个属性指定唯一名称。  
  
## <a name="example"></a>示例  
 以下示例生成 CS0833：  
  
```csharp  
// cs0833.cs  
using System;  
  
public class C  
{  
    public static int Main()  
    {  
        var c = new { p1 = 1, p1 = 2 }; // CS0833  
        return 1;  
    }  
}  
```
