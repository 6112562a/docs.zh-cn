## <a name="example"></a><span data-ttu-id="81722-107">示例</span><span class="sxs-lookup"><span data-stu-id="81722-107">Example</span></span>  
  
```csharp  
// CS0011_1.cs

public class Outer
{  
   public class B { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="81722-108">示例</span><span class="sxs-lookup"><span data-stu-id="81722-108">Example</span></span>  
 <span data-ttu-id="81722-109">第二个文件创建一个 DLL，它定义了派生自上一示例中所创建 `C` 类的 `B` 类。</span><span class="sxs-lookup"><span data-stu-id="81722-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

// CS0011_2.cs  
// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll  
public class C : Outer.B {}  
```  
  
## <a name="example"></a><span data-ttu-id="81722-110">示例</span><span class="sxs-lookup"><span data-stu-id="81722-110">Example</span></span>

// compile with: /target:library /out:cs0011_1.dll

```  
  
## <a name="example"></a><span data-ttu-id="81722-112">示例</span><span class="sxs-lookup"><span data-stu-id="81722-112">Example</span></span>
