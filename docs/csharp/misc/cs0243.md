# <a name="compiler-error-cs0243"></a><span data-ttu-id="3db7f-102">编译器错误 CS0243</span><span class="sxs-lookup"><span data-stu-id="3db7f-102">Compiler error CS0243</span></span>

<span data-ttu-id="3db7f-103">Conditional 特性在“method”上无效，因为它是重写方法</span><span class="sxs-lookup"><span data-stu-id="3db7f-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="3db7f-104">在<xref:System.Diagnostics.ConditionalAttribute>使用[重写](../language-reference/keywords/override.md)关键字标记的方法上不允许该属性。</span><span class="sxs-lookup"><span data-stu-id="3db7f-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="3db7f-105">有关详细信息，请参阅[知道何时使用覆盖和新建关键字](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)。</span><span class="sxs-lookup"><span data-stu-id="3db7f-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="3db7f-106">编译器从不绑定以重写方法。</span><span class="sxs-lookup"><span data-stu-id="3db7f-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="3db7f-107">它仅绑定到基方法，并且公共语言运行时根据需要调用重写。</span><span class="sxs-lookup"><span data-stu-id="3db7f-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="3db7f-108">以下代码生成 CS0243：</span><span class="sxs-lookup"><span data-stu-id="3db7f-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
