 <span data-ttu-id="74d93-104">正如其他各个声明一样，范围变量的声明必须具有在变量声明空间中唯一的标识符。</span><span class="sxs-lookup"><span data-stu-id="74d93-104">The declaration of a range variable, just like every other declaration, must have an identifier which is unique within the variable's declaration space.</span></span>

## <a name="example"></a><span data-ttu-id="74d93-107">示例</span><span class="sxs-lookup"><span data-stu-id="74d93-107">Example</span></span>  
 <span data-ttu-id="74d93-108">下面的代码生成 CS1931，因为该标识符 `x` 既用作 `Main` 中的本地变量，也用作查询表达式中的范围变量：</span><span class="sxs-lookup"><span data-stu-id="74d93-108">The following code generates CS1931 because the identifier `x` is used both as a local variable in `Main` and as the range variable in the query expression:</span></span>  
  
```csharp  
// cs1931.cs  
class Test  
    {  
        static void Main()  
        {  
            int x = 1;  
            var y = from x in Enumerable.Range(1, 100) // CS1931  
                    select x;  
        }  
    }  
```
