# <a name="compiler-error-cs0757"></a><span data-ttu-id="e0937-102">编译器错误 CS0757</span><span class="sxs-lookup"><span data-stu-id="e0937-102">Compiler Error CS0757</span></span>

 <span data-ttu-id="e0937-104">分部方法必须包含一个（且仅限一个）定义声明（签名），并可包含一个或零个实现声明（主体）。</span><span class="sxs-lookup"><span data-stu-id="e0937-104">A partial method consists of exactly one defining declaration (signature) and one or zero implementing declarations (body).</span></span> <span data-ttu-id="e0937-105">同一定义声明不允许有多个实现声明。</span><span class="sxs-lookup"><span data-stu-id="e0937-105">Multiple implementing declarations for the same identical defining declarations are not allowed.</span></span> <span data-ttu-id="e0937-106">可以重载分部方法，并且每个重载版本都可以有一个或零个实现方法。</span><span class="sxs-lookup"><span data-stu-id="e0937-106">Partial methods may be overloaded, and each overloaded version may have one or zero implementing declarations.</span></span>

## <a name="example"></a><span data-ttu-id="e0937-109">示例</span><span class="sxs-lookup"><span data-stu-id="e0937-109">Example</span></span>  
 <span data-ttu-id="e0937-110">下面的示例生成 CS0757：</span><span class="sxs-lookup"><span data-stu-id="e0937-110">The following example generates CS0757:</span></span>  
  
```csharp  
// cs0757.cs  
using System;  
  
    public partial class C  
    {  
        // Defining declaration.  
        partial void Part();  
  
        // Implementing declaration.  
        partial void Part()  
        {  
            //...Do something.  
        }  
  
        // Second implementing declaration.  
        partial void Part() // CS0757  
        {  
            //...Do something.  
        }  
  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
