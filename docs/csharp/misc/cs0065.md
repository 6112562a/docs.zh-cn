# <a name="compiler-error-cs0065"></a><span data-ttu-id="036a3-102">编译器错误 CS0065</span><span class="sxs-lookup"><span data-stu-id="036a3-102">Compiler Error CS0065</span></span>
<span data-ttu-id="036a3-103">“event”：事件属性必须同时具有 add 和 remove 访问器</span><span class="sxs-lookup"><span data-stu-id="036a3-103">'event' : event property must have both add and remove accessors</span></span>  
  
 <span data-ttu-id="036a3-104">并不是字段的事件必须有两种访问方法。</span><span class="sxs-lookup"><span data-stu-id="036a3-104">An event that is not a field must have both access methods.</span></span>  
  
 <span data-ttu-id="036a3-105">以下示例生成 CS0065：</span><span class="sxs-lookup"><span data-stu-id="036a3-105">The following sample generates CS0065:</span></span>  
  
```csharp  
// CS0065.cs  
using System;  
public delegate void Eventhandler(object sender, int e);  
public class MyClass  
{  
   public event EventHandler Click   // CS0065,  
   {  
      // to fix, uncomment the add and remove definitions  
      /*  
      add  
      {  
         Click += value;  
      }  
      remove  
      {  
         Click -= value;  
      }  
      */  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
