 <span data-ttu-id="a94d6-104">具有 [void](../language-reference/keywords/void.md) 返回类型的方法不能返回值。</span><span class="sxs-lookup"><span data-stu-id="a94d6-104">A method with a [void](../language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="a94d6-105">有关详细信息，请参阅[方法](../programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="a94d6-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="a94d6-106">下面的示例生成 CS0127：</span><span class="sxs-lookup"><span data-stu-id="a94d6-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
