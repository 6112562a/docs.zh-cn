# <a name="compiler-error-cs0568"></a>编译器错误 CS0568
结构不能包含显式的无参数构造函数  
  
 每个[结构](../../csharp/language-reference/keywords/struct.md)已具有将对象初始化为零的无参数构造函数。 因此，你可以为结构创建构造函数，该构造函数必须采用一个或多个参数。  
  
 下面的示例生成 CS0568：  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
