# <a name="compiler-error-cs0529"></a><span data-ttu-id="f3def-102">编译器错误 CS0529</span><span class="sxs-lookup"><span data-stu-id="f3def-102">Compiler Error CS0529</span></span>
<span data-ttu-id="f3def-103">继承接口“interface1”在“interface2”的接口层次结构中导致一个循环</span><span class="sxs-lookup"><span data-stu-id="f3def-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="f3def-104">[接口](../language-reference/keywords/interface.md) 的继承列表包括对其自身的直接或间接引用。</span><span class="sxs-lookup"><span data-stu-id="f3def-104">The inheritance list for an [interface](../language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="f3def-105">接口不能从自身继承。</span><span class="sxs-lookup"><span data-stu-id="f3def-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="f3def-106">下面的示例生成 CS0529：</span><span class="sxs-lookup"><span data-stu-id="f3def-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
