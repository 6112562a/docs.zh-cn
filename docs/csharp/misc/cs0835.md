# <a name="compiler-error-cs0835"></a><span data-ttu-id="3fd78-102">编译器错误 CS0835</span><span class="sxs-lookup"><span data-stu-id="3fd78-102">Compiler Error CS0835</span></span>
<span data-ttu-id="3fd78-103">无法将 lambda 转换为其类型实参“type”不是委托类型的表达式树。</span><span class="sxs-lookup"><span data-stu-id="3fd78-103">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="3fd78-104">如果将 lambda 表达式转换为表达式树，表达式树必须针对其参数有一个委托类型。</span><span class="sxs-lookup"><span data-stu-id="3fd78-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="3fd78-105">此外，lambda 表达式必须可转换为委托类型。</span><span class="sxs-lookup"><span data-stu-id="3fd78-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3fd78-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="3fd78-106">To correct this error</span></span>  
  
1. <span data-ttu-id="3fd78-107">将类型参数从 `int` 更改为委托类型，例如 `Func<int,int>`。</span><span class="sxs-lookup"><span data-stu-id="3fd78-107">Change the type parameter from `int` to a delegate type, for example `Func<int,int>`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3fd78-108">示例</span><span class="sxs-lookup"><span data-stu-id="3fd78-108">Example</span></span>  
 <span data-ttu-id="3fd78-109">以下示例生成 CS0835：</span><span class="sxs-lookup"><span data-stu-id="3fd78-109">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
