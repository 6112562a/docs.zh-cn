# <a name="compiler-error-cs2003"></a>编译器错误 CS2003
已多次包含响应文件“file”  
  
 已多次将响应文件传递到编译器。 每个输出文件只能将响应文件传递到编译器一次。  
  
 有关响应文件的详细信息，请参阅 [@（指定响应文件）](../../csharp/language-reference/compiler-options/response-file-compiler-option.md)。
