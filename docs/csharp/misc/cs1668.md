# <a name="compiler-warning-level-2-cs1668"></a>编译器警告（等级 2）CS1668
在“path string”中指定的路径“path”无效 - “系统错误消息”  
  
 提供给命令行的 [/lib](../language-reference/compiler-options/lib-compiler-option.md) 的路径无效，或 LIB 环境变量中的路径无效。 检查所用的路径以验证它是否存在且可供访问。 单引号中的错误消息是从操作系统返回的错误。
