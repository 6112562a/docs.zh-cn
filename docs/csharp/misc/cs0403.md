## <a name="example"></a><span data-ttu-id="2110c-108">示例</span><span class="sxs-lookup"><span data-stu-id="2110c-108">Example</span></span>  
 <span data-ttu-id="2110c-109">下面的示例生成 CS0403。</span><span class="sxs-lookup"><span data-stu-id="2110c-109">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
