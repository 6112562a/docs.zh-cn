# <a name="compiler-error-cs1621"></a><span data-ttu-id="fb7de-102">编译器错误 CS1621</span><span class="sxs-lookup"><span data-stu-id="fb7de-102">Compiler Error CS1621</span></span>
<span data-ttu-id="fb7de-103">不能在匿名方法或 lambda 表达式内使用 yield 语句</span><span class="sxs-lookup"><span data-stu-id="fb7de-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="fb7de-104">yield 语句不能处于迭代器的匿名方法块内。</span><span class="sxs-lookup"><span data-stu-id="fb7de-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fb7de-105">示例</span><span class="sxs-lookup"><span data-stu-id="fb7de-105">Example</span></span>  
 <span data-ttu-id="fb7de-106">下面的示例生成 CS1621：</span><span class="sxs-lookup"><span data-stu-id="fb7de-106">The following sample generates CS1621:</span></span>  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
