# <a name="compiler-error-cs1621"></a><span data-ttu-id="b1bc8-102">编译器错误 CS1621</span><span class="sxs-lookup"><span data-stu-id="b1bc8-102">Compiler Error CS1621</span></span>
<span data-ttu-id="b1bc8-103">不能在匿名方法或 lambda 表达式内使用 yield 语句</span><span class="sxs-lookup"><span data-stu-id="b1bc8-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="b1bc8-104">不能在[匿名方法](../language-reference/operators/delegate-operator.md)或[lambda 表达式](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md)中使用[yield](../../csharp/language-reference/keywords/yield.md)语句。</span><span class="sxs-lookup"><span data-stu-id="b1bc8-104">You cannot use the [yield](../../csharp/language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="b1bc8-105">下面的示例生成 CS1621：</span><span class="sxs-lookup"><span data-stu-id="b1bc8-105">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
