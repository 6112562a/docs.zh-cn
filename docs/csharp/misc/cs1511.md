# <a name="compiler-error-cs1511"></a><span data-ttu-id="8e519-102">编译器错误 CS1511</span><span class="sxs-lookup"><span data-stu-id="8e519-102">Compiler Error CS1511</span></span>
<span data-ttu-id="8e519-103">关键字“base”在静态方法中不可用</span><span class="sxs-lookup"><span data-stu-id="8e519-103">Keyword 'base' is not available in a static method</span></span>  
  
 <span data-ttu-id="8e519-104">[base](../../csharp/language-reference/keywords/base.md) 关键字用在 [静态](../../csharp/language-reference/keywords/static.md) 方法中。</span><span class="sxs-lookup"><span data-stu-id="8e519-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used in a [static](../../csharp/language-reference/keywords/static.md) method.</span></span> <span data-ttu-id="8e519-105">只能在实例构造函数、实例方法或实例访问器中调用`base` 。</span><span class="sxs-lookup"><span data-stu-id="8e519-105">`base` can only be called in an instance constructor, instance method, or instance accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8e519-106">示例</span><span class="sxs-lookup"><span data-stu-id="8e519-106">Example</span></span>  
 <span data-ttu-id="8e519-107">以下示例生成 CS1511。</span><span class="sxs-lookup"><span data-stu-id="8e519-107">The following sample generates CS1511.</span></span>  
  
```csharp  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
