 <span data-ttu-id="ddaff-104">尝试创建委托失败，因为未使用正确的签名。</span><span class="sxs-lookup"><span data-stu-id="ddaff-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="ddaff-105">必须使用与委托声明相同的签名声明委托实例。</span><span class="sxs-lookup"><span data-stu-id="ddaff-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>

 <span data-ttu-id="ddaff-108">以下示例生成 CS0123。</span><span class="sxs-lookup"><span data-stu-id="ddaff-108">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
