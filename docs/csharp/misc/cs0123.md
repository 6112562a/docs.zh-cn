# <a name="compiler-error-cs0123"></a><span data-ttu-id="8876a-102">编译器错误 CS0123</span><span class="sxs-lookup"><span data-stu-id="8876a-102">Compiler Error CS0123</span></span>
<span data-ttu-id="8876a-103">“method”的重载均与委托“delegate”不匹配</span><span class="sxs-lookup"><span data-stu-id="8876a-103">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="8876a-104">尝试创建委托失败，因为未使用正确的签名。</span><span class="sxs-lookup"><span data-stu-id="8876a-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="8876a-105">必须使用与委托声明相同的签名声明委托实例。</span><span class="sxs-lookup"><span data-stu-id="8876a-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="8876a-106">你可通过调整方法或委托签名来解决此错误。</span><span class="sxs-lookup"><span data-stu-id="8876a-106">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="8876a-107">有关详细信息，请参阅[委托](../../csharp/programming-guide/delegates/index.md)。</span><span class="sxs-lookup"><span data-stu-id="8876a-107">For more information, see [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="8876a-108">以下示例生成 CS0123。</span><span class="sxs-lookup"><span data-stu-id="8876a-108">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
