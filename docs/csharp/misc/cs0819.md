# <a name="compiler-error-cs0819"></a><span data-ttu-id="54346-102">编译器错误 CS0819</span><span class="sxs-lookup"><span data-stu-id="54346-102">Compiler Error CS0819</span></span>

<span data-ttu-id="54346-103">隐式类型的变量不能有多个声明符。</span><span class="sxs-lookup"><span data-stu-id="54346-103">Implicitly-typed variables cannot have multiple declarators.</span></span>

 <span data-ttu-id="54346-104">显式类型声明中允许有多个声明符，而隐式类型化的变量不允许。</span><span class="sxs-lookup"><span data-stu-id="54346-104">Multiple declarators are allowed in explicit type declarations, but not with implicitly typed variables.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="54346-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="54346-105">To correct this error</span></span>

<span data-ttu-id="54346-106">有两种选项：</span><span class="sxs-lookup"><span data-stu-id="54346-106">There are two options:</span></span>

1. <span data-ttu-id="54346-107">如果变量具有相同的类型，请使用显式声明。</span><span class="sxs-lookup"><span data-stu-id="54346-107">If the variables are of the same type, use explicit declarations.</span></span>
1. <span data-ttu-id="54346-108">在单独的行上声明每个隐式类型化的局部变量并为其赋值。</span><span class="sxs-lookup"><span data-stu-id="54346-108">Declare and assign a value to each implicitly typed local variable on a separate line.</span></span>

## <a name="example-1"></a><span data-ttu-id="54346-109">示例 1</span><span class="sxs-lookup"><span data-stu-id="54346-109">Example 1</span></span>

<span data-ttu-id="54346-110">下面的代码生成 CS0819：</span><span class="sxs-lookup"><span data-stu-id="54346-110">The following code generates CS0819:</span></span>

```csharp
// cs0819.cs
class Program
{
    public static void Main()
    {
        var a = 3, b = 2; // CS0819

        // First correction option.
        //int a = 3, b = 2;

        // Second correction option.
        //var a = 3;
        //var b = 2;
    }
}
```
