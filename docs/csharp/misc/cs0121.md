# <a name="compiler-error-cs0121"></a>编译器错误 CS0121

调用在以下方法或属性之间不明确："方法1"和"方法 2"

由于隐式转换，编译器无法调用一种形式的重载方法。 您可以通过以下方式之一解决此错误：

- 指定方法参数的方式是隐式转换不发生。
- 删除方法的所有重载。
- 在调用方法之前，强制转换为正确的类型。
- 使用命名参数。

## <a name="example"></a>示例

以下示例生成编译器错误 CS0121：

```csharp
public class Program
{
    static void f(int i, double d)
    {
    }

    static void f(double d, int i)
    {
    }

    public static void Main()
    {
        f(1, 1);   // CS0121

        // Try the following code instead:
        // f(1, 1.0);
        // or
        // f(1.0, 1);
        // or
        // f(1, (double)1);   // Cast and specify which method to call.
        // or
        // f(i: 1, 1);
        // or
        // f(d: 1, 1);

        // f(i: 1, d: 1); // This still gives CS0121
    }
}
```

## <a name="example"></a>示例

```csharp
class Program2
{
    static int ol_invoked = 0;

    delegate int D1(int x);
    delegate T D1<T>(T x);
    delegate T D1<T, U>(U u);

    static void F(D1 d1) { ol_invoked = 1; }
    static void F<T>(D1<T> d1t) { ol_invoked = 2; }
    static void F<T, U>(D1<T, U> d1t) { ol_invoked = 3; }

    static int Test001()
    {
        F(delegate(int x) { return 1; }); // CS0121
        if (ol_invoked == 1)
            return 0;
        else
            return 1;
    }

    static int Main()
    {
        return Test001();
    }
}
```
