# <a name="compiler-error-cs0210"></a><span data-ttu-id="acc3f-102">编译器错误 CS0210</span><span class="sxs-lookup"><span data-stu-id="acc3f-102">Compiler Error CS0210</span></span>
<span data-ttu-id="acc3f-103">必须在 fixed 或者 using 语句声明中提供初始值设定项</span><span class="sxs-lookup"><span data-stu-id="acc3f-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="acc3f-104">必须声明并初始化在 [fixed 语句](../language-reference/keywords/fixed-statement.md)中的变量。</span><span class="sxs-lookup"><span data-stu-id="acc3f-104">You must declare and initialize the variable in a [fixed statement](../language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="acc3f-105">有关详细信息，请参阅[不安全代码和指针](../programming-guide/unsafe-code-pointers/index.md)。</span><span class="sxs-lookup"><span data-stu-id="acc3f-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="acc3f-106">以下示例生成 CS0210：</span><span class="sxs-lookup"><span data-stu-id="acc3f-106">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="acc3f-107">以下的示例也生成 CS0210，因为 [using 语句](../language-reference/keywords/using-statement.md) 没有初始值设定项。</span><span class="sxs-lookup"><span data-stu-id="acc3f-107">The following sample also generates CS0210 because the [using statement](../language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test
{  
   static void Main()
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
