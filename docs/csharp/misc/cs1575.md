# <a name="compiler-error-cs1575"></a><span data-ttu-id="6b0d1-102">编译器错误 CS1575</span><span class="sxs-lookup"><span data-stu-id="6b0d1-102">Compiler Error CS1575</span></span>

 <span data-ttu-id="6b0d1-104">使用 [stackalloc](../language-reference/operators/stackalloc.md)所请求的分配的大小必须在方括号中指定。</span><span class="sxs-lookup"><span data-stu-id="6b0d1-104">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="6b0d1-105">下面的示例生成 CS1575：</span><span class="sxs-lookup"><span data-stu-id="6b0d1-105">The following sample generates CS1575:</span></span>  
  
```csharp

{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575

   }  
}  
```
