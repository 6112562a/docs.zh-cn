# <a name="compiler-error-cs0828"></a><span data-ttu-id="f108d-102">编译器错误 CS0828</span><span class="sxs-lookup"><span data-stu-id="f108d-102">Compiler Error CS0828</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="f108d-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="f108d-105">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="f108d-107">示例</span><span class="sxs-lookup"><span data-stu-id="f108d-107">Example</span></span>

```csharp  
public class C  
{  
    public static int Main()  
    {

    }

```
