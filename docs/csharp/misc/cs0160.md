# <a name="compiler-error-cs0160"></a><span data-ttu-id="99651-102">编译器错误 CS0160</span><span class="sxs-lookup"><span data-stu-id="99651-102">Compiler Error CS0160</span></span>
<span data-ttu-id="99651-103">上一个 catch 子句已经捕获了此类型或超类型（“type”）的所有异常</span><span class="sxs-lookup"><span data-stu-id="99651-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
<span data-ttu-id="99651-104">一系列 [catch](../language-reference/keywords/try-catch.md) 语句必须按派生程度降序排列。</span><span class="sxs-lookup"><span data-stu-id="99651-104">A series of [catch](../language-reference/keywords/try-catch.md) statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="99651-105">例如，派生程度最高的对象必须显示在最前。</span><span class="sxs-lookup"><span data-stu-id="99651-105">For example, the most derived objects must appear first.</span></span>
  
 <span data-ttu-id="99651-106">有关详细信息, 请参阅[异常和异常处理](../programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="99651-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="99651-107">下面的示例生成 CS0160：</span><span class="sxs-lookup"><span data-stu-id="99651-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
