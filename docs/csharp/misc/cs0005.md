# <a name="compiler-error-cs0005"></a>编译器错误 CS0005
编译器选项“compiler_option”后面必须有参数  
  
 某些编译器选项需要参数。 若未传递编译器选项所需的传递参数，则会生成 CS0005。  
  
 有关详细信息，请参阅 [C# 编译器选项](../language-reference/compiler-options/index.md)。
