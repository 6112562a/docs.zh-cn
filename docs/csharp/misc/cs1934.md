## <a name="example"></a><span data-ttu-id="95cd1-110">示例</span><span class="sxs-lookup"><span data-stu-id="95cd1-110">Example</span></span>  
 <span data-ttu-id="95cd1-111">以下示例演示了生成 CS1934 的一种方式：</span><span class="sxs-lookup"><span data-stu-id="95cd1-111">The following example shows one way to produce CS1934:</span></span>  
  
```csharp

using System.Collections;

{  
    public static void Main()  
    {
```  
