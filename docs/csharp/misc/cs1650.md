# <a name="compiler-error-cs1650"></a><span data-ttu-id="1d896-102">编译器错误 CS1650</span><span class="sxs-lookup"><span data-stu-id="1d896-102">Compiler Error CS1650</span></span>
<span data-ttu-id="1d896-103">无法给静态只读字段“identifier”的字段赋值（在静态构造函数或变量初始值设定项中除外）</span><span class="sxs-lookup"><span data-stu-id="1d896-103">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="1d896-104">当你试图修改静态只读字段中不允许修改的成员时，将发生此错误。</span><span class="sxs-lookup"><span data-stu-id="1d896-104">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="1d896-105">若要解决此错误，请将只读字段的赋值限定为构造函数或变量初始值设定项，或从该字段的声明中删除 `readonly` 关键字。</span><span class="sxs-lookup"><span data-stu-id="1d896-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
