## <a name="example"></a><span data-ttu-id="7fa2d-109">示例</span><span class="sxs-lookup"><span data-stu-id="7fa2d-109">Example</span></span>

<span data-ttu-id="7fa2d-110">下面的示例生成 CS0837：</span><span class="sxs-lookup"><span data-stu-id="7fa2d-110">The following example generates CS0837:</span></span>  
  
```csharp  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```
