 <span data-ttu-id="4b71c-104">在赋值语句中，右侧的值赋值给了左侧。</span><span class="sxs-lookup"><span data-stu-id="4b71c-104">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="4b71c-105">左侧必须是变量、属性或索引器。</span><span class="sxs-lookup"><span data-stu-id="4b71c-105">The left-hand side must be a variable, property, or indexer.</span></span>

## <a name="example"></a><span data-ttu-id="4b71c-108">示例</span><span class="sxs-lookup"><span data-stu-id="4b71c-108">Example</span></span>  
 <span data-ttu-id="4b71c-109">以下示例生成 CS0131。</span><span class="sxs-lookup"><span data-stu-id="4b71c-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="4b71c-110">示例</span><span class="sxs-lookup"><span data-stu-id="4b71c-110">Example</span></span>  
 <span data-ttu-id="4b71c-111">如果你尝试在赋值运算符的左侧执行算术运算，也可能出现此错误，如下例所示。</span><span class="sxs-lookup"><span data-stu-id="4b71c-111">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>

```csharp  
// CS0131b.cs  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)  
            return 0;  
        return 1;  
    }  
}  
```
