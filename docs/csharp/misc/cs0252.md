 <span data-ttu-id="8c418-104">编译器正在执行引用比较。</span><span class="sxs-lookup"><span data-stu-id="8c418-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="8c418-105">如果想要比较字符串值，请将表达式右侧强制转换为 `type`。</span><span class="sxs-lookup"><span data-stu-id="8c418-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="8c418-106">下面的示例生成 CS0252：</span><span class="sxs-lookup"><span data-stu-id="8c418-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
