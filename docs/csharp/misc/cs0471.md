## <a name="example"></a><span data-ttu-id="da190-106">示例</span><span class="sxs-lookup"><span data-stu-id="da190-106">Example</span></span>  
 <span data-ttu-id="da190-107">下面的示例生成 CS0471。</span><span class="sxs-lookup"><span data-stu-id="da190-107">The following example generates CS0471.</span></span>  
  
```csharp  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```
