# <a name="compiler-error-cs0471"></a>编译器错误 CS0471
“name”方法不是泛型方法。 如果原打算使用表达式列表，请用括号将 < 表达式括起来。  
  
 如果代码中包含的表达式列表不带括号，则会生成此错误。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0471。  
  
```csharp  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```
