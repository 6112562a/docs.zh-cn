# <a name="compiler-error-cs1559"></a>编译器错误 CS1559
无法对 Main 方法使用“object”，因为它是被导入的  
  
 为 [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) 编译器选项指定的类无效；该类不能用作 [Main](../../csharp/programming-guide/main-and-command-args/index.md) 方法的位置。
