# <a name="compiler-error-cs0216"></a><span data-ttu-id="b809f-102">编译器错误 CS0216</span><span class="sxs-lookup"><span data-stu-id="b809f-102">Compiler Error CS0216</span></span>
<span data-ttu-id="b809f-103">运算符“operator”要求也要定义匹配的运算符“missing_operator”</span><span class="sxs-lookup"><span data-stu-id="b809f-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="b809f-104">用户定义的 [true](../language-reference/keywords/true-false-operators.md) 运算符要求用户定义 [false](../language-reference/keywords/true-false-operators.md) 运算符，反之亦然。</span><span class="sxs-lookup"><span data-stu-id="b809f-104">A user-defined [true](../language-reference/keywords/true-false-operators.md) operator requires a user-defined [false](../language-reference/keywords/true-false-operators.md) operator, and vice versa.</span></span>
  
 <span data-ttu-id="b809f-105">下面的示例生成 CS0216：</span><span class="sxs-lookup"><span data-stu-id="b809f-105">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
