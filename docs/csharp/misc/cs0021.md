<span data-ttu-id="85088-105">在 C++ 程序集中尝试使用索引器时，可能会遇到 CS0021。</span><span class="sxs-lookup"><span data-stu-id="85088-105">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="85088-106">在这种情况下，请用 `DefaultMember` 特性修饰 C++ 类，让 C# 编译器能够识别默认索引器。</span><span class="sxs-lookup"><span data-stu-id="85088-106">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="85088-107">下面的示例生成 CS0021。</span><span class="sxs-lookup"><span data-stu-id="85088-107">The following sample generates CS0021.</span></span>

## <a name="example"></a><span data-ttu-id="85088-108">示例</span><span class="sxs-lookup"><span data-stu-id="85088-108">Example</span></span>

<span data-ttu-id="85088-109">下面C++的示例将编译为 .dll 文件。</span><span class="sxs-lookup"><span data-stu-id="85088-109">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="85088-110">请注意， [DefaultMember](xref:System.Reflection.DefaultMemberAttribute)属性会被注释掉，以生成错误。</span><span class="sxs-lookup"><span data-stu-id="85088-110">Note that the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>

```cpp
// CPP0021.cpp
// compile with: /clr /LD
using namespace System::Reflection;
// Uncomment the following line to resolve
//[DefaultMember("myItem")]
public ref class MyClassMC
{
        public:
        property int myItem[int]
        {
            int get(int i){  return 5; }
            void set(int i, int value) {}
        }
};
```

<span data-ttu-id="85088-111">下面C#的示例调用 .dll 文件。</span><span class="sxs-lookup"><span data-stu-id="85088-111">The following C# example calls the .dll file.</span></span> <span data-ttu-id="85088-112">它尝试通过索引器访问类，但由于没有将任何成员声明为默认索引器，因此会生成错误。</span><span class="sxs-lookup"><span data-stu-id="85088-112">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="85088-113">可以通过取消注释行 `[DefaultMember("myItem")]` 来解决该错误。</span><span class="sxs-lookup"><span data-stu-id="85088-113">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>

```csharp
// CS0021.cs
// compile with: /reference:CPP0021.dll
public class MyClass
{
    public static void Main()
    {
        MyClassMC myMC = new MyClassMC();
        int j = myMC[1]; // CS0021
    }
}
```
