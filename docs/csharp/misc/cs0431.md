<span data-ttu-id="59d44-103">无法将别名“identifier”与“::”一起使用，因为该别名引用了类型。</span><span class="sxs-lookup"><span data-stu-id="59d44-103">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="59d44-104">请改用“.”。</span><span class="sxs-lookup"><span data-stu-id="59d44-104">Use '.' instead.</span></span>  
  
 <span data-ttu-id="59d44-105">同时使用了“::”与引用类型的别名。</span><span class="sxs-lookup"><span data-stu-id="59d44-105">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="59d44-106">若要解决此错误，请使用“.”运算符。</span><span class="sxs-lookup"><span data-stu-id="59d44-106">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="59d44-107">下面的示例生成 CS0431:</span><span class="sxs-lookup"><span data-stu-id="59d44-107">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer
{  
   public class Inner
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
