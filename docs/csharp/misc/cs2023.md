# <a name="compiler-warning-level-1-cs2023"></a>编译器警告（等级 1）CS2023
/noconfig 选项是在响应文件中指定的，因此被忽略  
  
 已在响应文件中指定 [/Noconfig](../language-reference/compiler-options/noconfig-compiler-option.md) 编译器选项，这是不允许的。  
  
 此警告不能由 [/nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) 选项禁止。
