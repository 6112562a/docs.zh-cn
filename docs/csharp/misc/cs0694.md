# <a name="compiler-error-cs0694"></a><span data-ttu-id="8ba9a-102">编译器错误 CS0694</span><span class="sxs-lookup"><span data-stu-id="8ba9a-102">Compiler Error CS0694</span></span>

## <a name="example"></a><span data-ttu-id="8ba9a-105">示例</span><span class="sxs-lookup"><span data-stu-id="8ba9a-105">Example</span></span>  
 <span data-ttu-id="8ba9a-106">以下示例生成 CS0694。</span><span class="sxs-lookup"><span data-stu-id="8ba9a-106">The following sample generates CS0694.</span></span>

