<span data-ttu-id="f9fcd-103">不能将类型“type1”用作泛型类型或方法“name”中的类型参数“name”。</span><span class="sxs-lookup"><span data-stu-id="f9fcd-103">The type 'type1' cannot be used as type parameter 'name' in the generic type or method 'name'.</span></span> <span data-ttu-id="f9fcd-104">没有从“type1”到“type2”的装箱转换或类型参数转换。</span><span class="sxs-lookup"><span data-stu-id="f9fcd-104">There is no boxing conversion or type parameter conversion from 'type1' to 'type2'.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="f9fcd-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="f9fcd-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="f9fcd-107">在下面的示例中，将 `where T : ClassConstraint` 添加到类 `B`中。</span><span class="sxs-lookup"><span data-stu-id="f9fcd-107">In the example that follows, add `where T : ClassConstraint` to class `B`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f9fcd-108">示例</span><span class="sxs-lookup"><span data-stu-id="f9fcd-108">Example</span></span>  
 <span data-ttu-id="f9fcd-109">下面的代码生成 CS0314：</span><span class="sxs-lookup"><span data-stu-id="f9fcd-109">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
