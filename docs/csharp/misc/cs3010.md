## <a name="example"></a><span data-ttu-id="3cebc-107">示例</span><span class="sxs-lookup"><span data-stu-id="3cebc-107">Example</span></span>

```csharp

public interface I  
{  
    [CLSCompliant(false)]

    }

    {  
    }  
}
```
