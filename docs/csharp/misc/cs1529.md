## <a name="example"></a><span data-ttu-id="d5a75-105">示例</span><span class="sxs-lookup"><span data-stu-id="d5a75-105">Example</span></span>

```csharp

    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {

}

```
