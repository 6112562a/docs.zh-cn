# <a name="compiler-error-cs0026"></a>编译器错误 CS0026
关键字 "this" 在静态属性、静态方法或静态字段初始值设定项中无效  
  
 [this](../../csharp/language-reference/keywords/this.md) 关键字是指一个对象，其为某类型的实例。 由于静态方法与包含类的任何实例均无关，因此“this”关键字不含任何意义，因此不受允许。 有关详细信息，请参阅[静态类和静态类成员](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md)并[对象](../../csharp/programming-guide/classes-and-structs/objects.md)。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0026：  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;     
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
