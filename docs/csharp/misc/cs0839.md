# <a name="compiler-error-cs0839"></a>编译器错误 CS0839
缺少参数。  
  
 方法调用中缺少参数。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 仔细检查方法的签名，然后找到缺少的参数并提供该参数。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0839：  
  
```csharp  
// cs0839.cs  
using System;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Add(int i, int j)  
        {  
            return i + j;  
        }  
  
        static int Main()   
        {  
            int i = Test.Add( , 5); // CS0839  
            return 1;  
  
        }  
    }  
}  
```
