# <a name="compiler-error-cs0689"></a>编译器错误 CS0689
“identifier”是一个类型形参，无法从它进行派生  
  
 类型参数无法指定泛型类型的基类或接口。 从特定类或接口派生，或改用从特定泛型类型派生，或包含未知类型作为成员。  
  
 以下示例生成 CS0689：  
  
```csharp
// CS0689.cs  
class A<T> : T   // CS0689  
{  
}  
```
