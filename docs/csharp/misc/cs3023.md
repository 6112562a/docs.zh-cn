## <a name="example"></a><span data-ttu-id="3360e-106">示例</span><span class="sxs-lookup"><span data-stu-id="3360e-106">Example</span></span>  
 <span data-ttu-id="3360e-107">下面的示例生成警告 CS3023：</span><span class="sxs-lookup"><span data-stu-id="3360e-107">The following example generates warning CS3023:</span></span>

    }
