# <a name="compiler-error-cs0167"></a>编译器错误 CS0167
委托“delegate”缺少 Invoke 方法  
  
 你导入并使用了用另一编译器创建的托管程序（即使用 .NET Framework 公共语言运行时的托管程序）。 此编译器允许了格式错误的 [delegate](../language-reference/builtin-types/reference-types.md)。 因此， `Invoke` 方法不可用。 有关详细信息，请参阅[委托](../programming-guide/delegates/index.md)。
