# <a name="compiler-warning-level-1-cs1633"></a>编译器警告（等级 1）CS1633
无法识别的 #pragma 指令  
  
 使用的杂注不是 C# 编译器支持的已知杂注之一。 若要解决此错误，请仅使用受支持的杂注。  
  
 以下示例生成 CS1633：  
  
```csharp  
// CS1633.cs  
// compile with: /W:1  
#pragma unknown  // CS1633  
  
class C  
{  
   public static void Main()  
   {  
   }  
}  
```
