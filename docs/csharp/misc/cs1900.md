# <a name="compiler-error-cs1900"></a>编译器错误 CS1900
警告等级必须在 0-4 的范围内  
  
 [-警告](../language-reference/compiler-options/warn-compiler-option.md)编译器选项只能接受五个可能值（0、1、2、3或4）之一。 传递给 **/warn** 的其他任何值均将导致 CS1900。  
  
 下面的示例生成 CS1900：  
  
```csharp  
// CS1900.cs  
// compile with: /W:5  
// CS1900 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
