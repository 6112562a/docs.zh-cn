# <a name="compiler-error-cs0410"></a>编译器错误 CS0410
“method”的重载均没有正确的参数类型和返回类型  
  
 如果尝试使用具有错误参数类型的函数对委托进行实例化，将出现此错误。 委托的参数类型必须与你要分配给该委托的函数相匹配。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0410：  
  
```csharp  
// CS0410.cs  
// compile with: /langversion:ISO-1  
  
class Test  
{  
    delegate void D(double d );  
    static void F(int i) { }  
  
    static void Main()  
    {  
        D d = new D(F);  // CS0410  
    }  
}  
```
