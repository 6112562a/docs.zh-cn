# <a name="compiler-error-cs0656"></a>编译器错误 CS0656
缺少编译器要求的成员“object.member”  
  
 存在以下问题之一：  
  
-   你安装的公共语言运行时已损坏。  
  
-   你具有对某个程序集的引用，该程序集定义同样位于公共语言运行库中的类型。 但是，程序集类型的定义方式与 C# 编译器所期望的不同。  
  
 检查引用，确保你使用的是正确版本的公共语言运行时。
