# <a name="compiler-error-cs2022"></a>编译器错误 CS2022
选项“/out”和“/target”必须出现在源文件名之前  
  
 在命令行中指定时， [-out （设置输出文件名）](../language-reference/compiler-options/out-compiler-option.md)和[-Target （指定输出文件格式）](../language-reference/compiler-options/target-compiler-option.md)编译器选项必须位于源代码文件之前。
