## <a name="example"></a><span data-ttu-id="78b57-110">示例</span><span class="sxs-lookup"><span data-stu-id="78b57-110">Example</span></span>  
 <span data-ttu-id="78b57-111">下面的示例生成 CS1059，因为 `i` 是常量而非变量，并且 `E` 为 `Enum` 类型，该类型的元素也是常量值。</span><span class="sxs-lookup"><span data-stu-id="78b57-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```csharp  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
