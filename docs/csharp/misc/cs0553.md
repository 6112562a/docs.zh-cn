 <span data-ttu-id="f1375-104">不允许用户定义的到基类值的转换；你不需要这样的运算符。</span><span class="sxs-lookup"><span data-stu-id="f1375-104">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="f1375-105">下面的示例生成 CS0553：</span><span class="sxs-lookup"><span data-stu-id="f1375-105">The following sample generates CS0553:</span></span>  
  
```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
