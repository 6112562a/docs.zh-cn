# <a name="compiler-error-cs0588"></a>编译器错误 CS0588
内部编译器错误: 步骤“LEX”  
  
 尝试确定编译器是否因为无法分析意外的语法而产生故障。 如果这不是这种情况，请尝试[与我们交流](/visualstudio/ide/talk-to-us)。
