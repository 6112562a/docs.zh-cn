# <a name="compiler-error-cs0517"></a>编译器错误 CS0517
“class”没有基类，无法调用基构造函数  
  
 仅当 .NET Framework 公共语言运行时编译对象类的源代码，而该类是唯一没有基类的类时，才发生 CS0517。
