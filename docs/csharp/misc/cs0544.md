 <span data-ttu-id="5c276-104">尝试将非属性数据类型重写为 [属性](../../csharp/programming-guide/classes-and-structs/properties.md)，这是不允许的。</span><span class="sxs-lookup"><span data-stu-id="5c276-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

   public int i;

      {  
         return 0;  
      }  
   }
