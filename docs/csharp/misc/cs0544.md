# <a name="compiler-error-cs0544"></a><span data-ttu-id="ffb26-102">编译器错误 CS0544</span><span class="sxs-lookup"><span data-stu-id="ffb26-102">Compiler Error CS0544</span></span>
<span data-ttu-id="ffb26-103">“property overrid”：不能重写，因为“non-property”不是属性</span><span class="sxs-lookup"><span data-stu-id="ffb26-103">'property override': cannot override because 'non-property' is not a property</span></span>  
  
 <span data-ttu-id="ffb26-104">尝试将非属性数据类型重写为 [属性](../programming-guide/classes-and-structs/properties.md)，这是不允许的。</span><span class="sxs-lookup"><span data-stu-id="ffb26-104">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="ffb26-105">以下示例生成 CS0544：</span><span class="sxs-lookup"><span data-stu-id="ffb26-105">The following sample generates CS0544:</span></span>  
  
```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
