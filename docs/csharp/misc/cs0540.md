# <a name="compiler-error-cs0540"></a><span data-ttu-id="18863-102">编译器错误 CS0540</span><span class="sxs-lookup"><span data-stu-id="18863-102">Compiler Error CS0540</span></span>

 <span data-ttu-id="18863-104">你试图在非派生自 [接口](../language-reference/keywords/class.md) 的 [类](../language-reference/keywords/interface.md)中实现接口成员。</span><span class="sxs-lookup"><span data-stu-id="18863-104">You attempted to implement an interface member in a [class](../language-reference/keywords/class.md) that does not derive from the [interface](../language-reference/keywords/interface.md).</span></span> <span data-ttu-id="18863-105">应删除接口成员的实现，或将接口添加到类的基类列表。</span><span class="sxs-lookup"><span data-stu-id="18863-105">You should either delete the implementation of the interface member or add the interface to the base-class list of the class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="18863-106">示例</span><span class="sxs-lookup"><span data-stu-id="18863-106">Example</span></span>  
 <span data-ttu-id="18863-107">下面的示例生成 CS0540。</span><span class="sxs-lookup"><span data-stu-id="18863-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
