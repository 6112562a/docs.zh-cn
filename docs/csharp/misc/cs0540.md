## <a name="example"></a><span data-ttu-id="d0d6a-106">示例</span><span class="sxs-lookup"><span data-stu-id="d0d6a-106">Example</span></span>  
 <span data-ttu-id="d0d6a-107">下面的示例生成 CS0540。</span><span class="sxs-lookup"><span data-stu-id="d0d6a-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
