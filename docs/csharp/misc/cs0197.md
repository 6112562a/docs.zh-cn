# <a name="compiler-warning-level-1-cs0197"></a><span data-ttu-id="091a4-102">编译器警告（等级 1）CS0197</span><span class="sxs-lookup"><span data-stu-id="091a4-102">Compiler Warning (level 1) CS0197</span></span>
<span data-ttu-id="091a4-103">由于“argument”是引用封送类的字段，因此，将它作为 ref 或 out 参数传递或获取它的地址可能导致运行时异常</span><span class="sxs-lookup"><span data-stu-id="091a4-103">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="091a4-104">从 <xref:System.MarshalByRefObject> 直接或间接派生的任何类都是引用封送类。</span><span class="sxs-lookup"><span data-stu-id="091a4-104">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="091a4-105">这样的类可以跨进程和计算机边界引用封送。</span><span class="sxs-lookup"><span data-stu-id="091a4-105">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="091a4-106">因此，此类的实例可以是远程对象的代理。</span><span class="sxs-lookup"><span data-stu-id="091a4-106">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="091a4-107">不能将代理对象的字段作为 [ref](../language-reference/keywords/ref.md) 或 [out](../language-reference/keywords/out-parameter-modifier.md)传递。因此，不能传递 `ref` 或 `out`这类字段，除非实例是 [this](../language-reference/keywords/this.md)，它不能为代理对象。</span><span class="sxs-lookup"><span data-stu-id="091a4-107">You cannot pass a field of a proxy object as [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="091a4-108">示例</span><span class="sxs-lookup"><span data-stu-id="091a4-108">Example</span></span>  
 <span data-ttu-id="091a4-109">下面的示例生成 CS0197。</span><span class="sxs-lookup"><span data-stu-id="091a4-109">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
