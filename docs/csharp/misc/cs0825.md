## <a name="example"></a><span data-ttu-id="5c143-107">示例</span><span class="sxs-lookup"><span data-stu-id="5c143-107">Example</span></span>  
 <span data-ttu-id="5c143-108">下面的代码生成 CS0825，因为 `var` 在类字段上使用：</span><span class="sxs-lookup"><span data-stu-id="5c143-108">The following code generates CS0825 because `var` is used on a class field:</span></span>  
  
```csharp  
// cs0825.cs  
class Test  
{  
    // Both of these declarations trigger CS0825
    private var genreName;   
    private var bookTitles = new List<string>();
  
    static int Main()  
    {  
        var totalBooks = 42; // var is OK here  
        return -1;  
    }  
}  
```
