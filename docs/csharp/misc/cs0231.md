 <span data-ttu-id="95982-104">[Params](../language-reference/keywords/params.md) 参数支持可变数目的实参，并且必须位于所有其他形参之后。</span><span class="sxs-lookup"><span data-stu-id="95982-104">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="95982-105">有关详细信息，请参阅[方法](../programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="95982-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="95982-106">以下示例生成 CS0231：</span><span class="sxs-lookup"><span data-stu-id="95982-106">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs

{  
   public void TestMethod(params int[] p, int i) {} // CS0231

   {  
   }

```
