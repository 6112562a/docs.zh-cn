# <a name="compiler-error-cs0077"></a><span data-ttu-id="39c59-102">编译器错误 CS0077</span><span class="sxs-lookup"><span data-stu-id="39c59-102">Compiler Error CS0077</span></span>
<span data-ttu-id="39c59-103">as 运算符必须与引用类型或可以为 null 的类型一起使用（“int”是不可以为 null 值的类型）。</span><span class="sxs-lookup"><span data-stu-id="39c59-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="39c59-104">向 [as](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) 运算符传递了 [值类型](../../csharp/language-reference/keywords/value-types.md)。</span><span class="sxs-lookup"><span data-stu-id="39c59-104">The [as](../../csharp/language-reference/operators/type-testing-and-conversion-operators.md#as-operator) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="39c59-105">因为 `as` 可以返回 [null](../../csharp/language-reference/keywords/null.md)，所以只能向其传递 [引用类型](../../csharp/language-reference/keywords/reference-types.md) 或可以为 null 的类型。</span><span class="sxs-lookup"><span data-stu-id="39c59-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="39c59-106">有关可以为 null 的类型的详细信息，请参阅[为 Null 的类型](../../csharp/programming-guide/nullable-types/index.md)。</span><span class="sxs-lookup"><span data-stu-id="39c59-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="39c59-107">下面的示例生成 CS0077：</span><span class="sxs-lookup"><span data-stu-id="39c59-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
