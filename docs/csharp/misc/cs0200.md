# <a name="compiler-error-cs0200"></a>编译器错误 CS0200
属性或索引器"属性"不能分配给 -- 它是只读的

尝试将值分配给[属性](../programming-guide/classes-and-structs/using-properties.md)，但该属性没有集访问器，或者赋值不在构造函数之外。 通过添加 set 访问器解决该错误。 有关详细信息，请参阅[如何声明和使用读取写入属性](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md)。

## <a name="examples"></a>示例
以下示例生成 CS0200：

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

以下示例使用[自动实现的属性](../programming-guide/classes-and-structs/auto-implemented-properties.md)、[对象初始化器](../programming-guide/classes-and-structs/object-and-collection-initializers.md)，并且仍然生成 CS0200：

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

分配给属性或索引器"属性"是只读的，可以通过添加集访问器或分配给对象构造函数中的属性来实现。

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
