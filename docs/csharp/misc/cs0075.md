# <a name="compiler-error-cs0075"></a><span data-ttu-id="db758-102">编译器错误 CS0075</span><span class="sxs-lookup"><span data-stu-id="db758-102">Compiler Error CS0075</span></span>
<span data-ttu-id="db758-103">若要强制转换某个负值，必须将该值放在括号内</span><span class="sxs-lookup"><span data-stu-id="db758-103">To cast a negative value, you must enclose the value in parentheses</span></span>  
  
 <span data-ttu-id="db758-104">如果使用标识预定义类型的关键字进行强制转换，则不需要括号。</span><span class="sxs-lookup"><span data-stu-id="db758-104">If you are casting using a keyword that identifies a predefined type, then you do not need parentheses.</span></span> <span data-ttu-id="db758-105">否则，必须用括号括起来，因为 (x) –y 不会被视为强制转换表达式。</span><span class="sxs-lookup"><span data-stu-id="db758-105">Otherwise, you must put the parentheses because (x) –y will not be considered a cast expression.</span></span> <span data-ttu-id="db758-106">摘自 C# 规范的第 7.6.6 节：</span><span class="sxs-lookup"><span data-stu-id="db758-106">From the C# Specification, Section 7.6.6:</span></span>  
  
 <span data-ttu-id="db758-107">*从消除歧义规则，它遵循的如果 x 和 y 是标识符，(x) y、 (x)(y)，和 (可以是强制转换表达式，但 (x)-y 不是，即使 x 标识的类型。但是，如果 x 是一个关键字，用于标识预定义的类型 （如 int)，然后所有四个窗体是强制转换表达式 （因为这种关键字不可能是表达式本身）。*</span><span class="sxs-lookup"><span data-stu-id="db758-107">*From the disambiguation rule it follows that, if x and y are identifiers, (x)y, (x)(y), and (x)(-y) are cast-expressions, but (x)-y is not, even if x identifies a type. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="db758-108">以下代码生成 CS0075：</span><span class="sxs-lookup"><span data-stu-id="db758-108">The following code generates CS0075:</span></span>  
  
```csharp  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
