# <a name="compiler-error-cs1100"></a><span data-ttu-id="8b72f-102">编译器错误 CS1100</span><span class="sxs-lookup"><span data-stu-id="8b72f-102">Compiler Error CS1100</span></span>
<span data-ttu-id="8b72f-103">方法“name”具有一个不在第一个参数上的“this”参数修饰符。</span><span class="sxs-lookup"><span data-stu-id="8b72f-103">Method 'name' has a parameter modifier 'this' which is not on the first parameter.</span></span>  
  
 <span data-ttu-id="8b72f-104">`this` 修饰符仅允许用于方法的第一个参数，这将向编译器显示该方法是一个扩展方法。</span><span class="sxs-lookup"><span data-stu-id="8b72f-104">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>
