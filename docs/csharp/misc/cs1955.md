<span data-ttu-id="7664a-109">以下代码生成 CS1955，因为代码尝试使用[调用表达式`()`](../language-reference/operators/member-access-operators.md#invocation-expression-)调用字段和属性。</span><span class="sxs-lookup"><span data-stu-id="7664a-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation expression `()`](../language-reference/operators/member-access-operators.md#invocation-expression-).</span></span> <span data-ttu-id="7664a-110">不能调用字段或属性。</span><span class="sxs-lookup"><span data-stu-id="7664a-110">You cannot call a field or a property.</span></span> <span data-ttu-id="7664a-111">使用[成员访问表达式`.`](../language-reference/operators/member-access-operators.md#member-access-expression-)访问它存储的值。</span><span class="sxs-lookup"><span data-stu-id="7664a-111">Use the [member access expression `.`](../language-reference/operators/member-access-operators.md#member-access-expression-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
