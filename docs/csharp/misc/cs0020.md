# <a name="compiler-error-cs0020"></a>编译器错误 CS0020
被常数零除  
  
 表达式在除法运算的分母中使用了文本（不是变量）值零。 未定义除数为零，因此无效。  
  
 下面的示例生成 CS0020:  
  
```csharp  
// CS0020.cs  
namespace x  
{  
   public class b  
   {  
      public static void Main()  
      {  
         1 / 0;   // CS0020  
      }  
   }  
}  
```
