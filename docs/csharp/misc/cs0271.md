<span data-ttu-id="1c8a1-103">由于 get 访问器不可访问，因此不能在此上下文中使用属性或索引器“property/indexer”</span><span class="sxs-lookup"><span data-stu-id="1c8a1-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="1c8a1-104">当你尝试访问无法访问的 `get` 访问器时出现此错误。</span><span class="sxs-lookup"><span data-stu-id="1c8a1-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="1c8a1-105">若要解决此错误，增加访问器的可访问性，或更改调用位置。</span><span class="sxs-lookup"><span data-stu-id="1c8a1-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="1c8a1-106">有关详细信息，请参阅[访问器辅助功能](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md)和[属性](../programming-guide/classes-and-structs/properties.md)。</span><span class="sxs-lookup"><span data-stu-id="1c8a1-106">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="1c8a1-107">以下示例生成 CS0271：</span><span class="sxs-lookup"><span data-stu-id="1c8a1-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
