# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="67026-102">编译器警告（等级 4）CS0078</span><span class="sxs-lookup"><span data-stu-id="67026-102">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="67026-103">“l”后缀容易与数字“1”混淆；为清楚起见，请使用“L”</span><span class="sxs-lookup"><span data-stu-id="67026-103">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="67026-104">编译器会在检测到使用小写 l 而不是大写 L 强制转换为 long 时发出警告。</span><span class="sxs-lookup"><span data-stu-id="67026-104">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="67026-105">以下示例生成 CS0078：</span><span class="sxs-lookup"><span data-stu-id="67026-105">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
