# <a name="compiler-error-cs0754"></a>编译器错误 CS0754
分部方法不能显式实现接口方法。  
  
 分部方法不能声明为显式实现接口中定义的方法。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 从方法声明中删除显式接口限定。  
  
## <a name="example"></a>示例  
 下面的代码生成 CS0754:  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
