# <a name="compiler-error-cs0761"></a><span data-ttu-id="b4830-102">编译器错误 CS0761</span><span class="sxs-lookup"><span data-stu-id="b4830-102">Compiler Error CS0761</span></span>

 <span data-ttu-id="b4830-104">如果分部方法具有实现，则泛型类型约束必须与方法签名上定义的约束相同。</span><span class="sxs-lookup"><span data-stu-id="b4830-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b4830-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="b4830-105">To correct this error</span></span>
