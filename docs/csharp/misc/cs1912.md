# <a name="compiler-error-cs1912"></a><span data-ttu-id="b4cca-102">编译器错误 CS1912</span><span class="sxs-lookup"><span data-stu-id="b4cca-102">Compiler Error CS1912</span></span>
<span data-ttu-id="b4cca-103">成员“name”的初始化重复。</span><span class="sxs-lookup"><span data-stu-id="b4cca-103">Duplicate initialization of member 'name'.</span></span>  
  
 <span data-ttu-id="b4cca-104">一个对象初始值设定项仅能对每个成员进行一次初始化。</span><span class="sxs-lookup"><span data-stu-id="b4cca-104">An object initializer can initialize each member only one time.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b4cca-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="b4cca-105">To correct this error</span></span>  
  
1. <span data-ttu-id="b4cca-106">删除对象初始值设定项中对该成员的第二次初始化。</span><span class="sxs-lookup"><span data-stu-id="b4cca-106">Remove the second initialization of the member in the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b4cca-107">示例</span><span class="sxs-lookup"><span data-stu-id="b4cca-107">Example</span></span>  
 <span data-ttu-id="b4cca-108">下面的代码生成 CS1912，因为 `memberA` 被初始化了两次：</span><span class="sxs-lookup"><span data-stu-id="b4cca-108">The following code generates CS1912 because `memberA` is initialized two times:</span></span>  
  
```csharp  
// cs1912.cs  
using System.Linq;  
  
public class TestClass  
{  
    public int memberA { get; set; }  
    public int memberB { get; set; }  
}  
  
public class Test  
{  
    static void Main()  
    {  
        TestClass tc = new TestClass() { memberA = 5, memberA = 6, memberB = 2}; // CS1912  
    }  
}  
```
