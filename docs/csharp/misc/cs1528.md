# <a name="compiler-error-cs1528"></a><span data-ttu-id="1b156-102">编译器错误 CS1528</span><span class="sxs-lookup"><span data-stu-id="1b156-102">Compiler Error CS1528</span></span>
<span data-ttu-id="1b156-103">应输入 ";" 或 "="(无法在声明中指定构造函数参数)</span><span class="sxs-lookup"><span data-stu-id="1b156-103">Expected ; or = (cannot specify constructor arguments in declaration)</span></span>  
  
 <span data-ttu-id="1b156-104">类引用的构造就像正在创建类的对象一样。</span><span class="sxs-lookup"><span data-stu-id="1b156-104">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="1b156-105">例如，尝试将变量传递给构造函数。</span><span class="sxs-lookup"><span data-stu-id="1b156-105">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="1b156-106">使用 [new](../../csharp/language-reference/operators/new-operator.md) 运算符创建类的对象。</span><span class="sxs-lookup"><span data-stu-id="1b156-106">Use the [new](../../csharp/language-reference/operators/new-operator.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="1b156-107">下面的示例生成 CS1528：</span><span class="sxs-lookup"><span data-stu-id="1b156-107">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
