## <a name="example"></a><span data-ttu-id="8db3d-105">示例</span><span class="sxs-lookup"><span data-stu-id="8db3d-105">Example</span></span>  
 <span data-ttu-id="8db3d-106">以下示例生成 CS0448。</span><span class="sxs-lookup"><span data-stu-id="8db3d-106">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="8db3d-107">示例</span><span class="sxs-lookup"><span data-stu-id="8db3d-107">Example</span></span>  
 <span data-ttu-id="8db3d-108">以下示例生成 CS0448。</span><span class="sxs-lookup"><span data-stu-id="8db3d-108">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448_b.cs  
public struct S  
{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448

{

   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
