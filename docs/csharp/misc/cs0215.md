# <a name="compiler-error-cs0215"></a><span data-ttu-id="1f7f9-102">编译器错误 CS0215</span><span class="sxs-lookup"><span data-stu-id="1f7f9-102">Compiler Error CS0215</span></span>
<span data-ttu-id="1f7f9-103">运算符 True 或 False 的返回类型必须是 bool</span><span class="sxs-lookup"><span data-stu-id="1f7f9-103">The return type of operator True or False must be bool</span></span>  
  
<span data-ttu-id="1f7f9-104">用户定义的[true 和 false](../language-reference/operators/true-false-operators.md)运算符必须具有[bool](../language-reference/builtin-types/bool.md)的返回类型。</span><span class="sxs-lookup"><span data-stu-id="1f7f9-104">User-defined [true and false](../language-reference/operators/true-false-operators.md) operators must have a return type of [bool](../language-reference/builtin-types/bool.md).</span></span>
  
<span data-ttu-id="1f7f9-105">以下示例生成 CS0215：</span><span class="sxs-lookup"><span data-stu-id="1f7f9-105">The following sample generates CS0215:</span></span>  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
