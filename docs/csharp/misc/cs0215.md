# <a name="compiler-error-cs0215"></a>编译器错误 CS0215
运算符 True 或 False 的返回类型必须是 bool  
  
 用户定义[true 和 false](../language-reference/operators/true-false-operators.md)运算符必须具有返回类型为[bool](../language-reference/keywords/bool.md)。 有关详细信息，请参阅[可重载运算符](../programming-guide/statements-expressions-operators/overloadable-operators.md)和[运算符](../language-reference/keywords/operator.md)关键字文章。  
  
 以下示例生成 CS0215：  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
