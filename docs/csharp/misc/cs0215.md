# <a name="compiler-error-cs0215"></a><span data-ttu-id="b360a-102">编译器错误 CS0215</span><span class="sxs-lookup"><span data-stu-id="b360a-102">Compiler Error CS0215</span></span>
<span data-ttu-id="b360a-103">运算符 True 或 False 的返回类型必须是 bool</span><span class="sxs-lookup"><span data-stu-id="b360a-103">The return type of operator True or False must be bool</span></span>  
  
 <span data-ttu-id="b360a-104">用户定义[true 和 false](../language-reference/keywords/true-false-operators.md)运算符必须具有返回类型为[bool](../language-reference/keywords/bool.md)。</span><span class="sxs-lookup"><span data-stu-id="b360a-104">User-defined [true and false](../language-reference/keywords/true-false-operators.md) operators must have a return type of [bool](../language-reference/keywords/bool.md).</span></span> <span data-ttu-id="b360a-105">有关详细信息，请参阅[可重载运算符](../programming-guide/statements-expressions-operators/overloadable-operators.md)和[运算符](../language-reference/keywords/operator.md)关键字文章。</span><span class="sxs-lookup"><span data-stu-id="b360a-105">For more information, see [Overloadable Operators](../programming-guide/statements-expressions-operators/overloadable-operators.md) and the [operator](../language-reference/keywords/operator.md) keyword article.</span></span>  
  
 <span data-ttu-id="b360a-106">以下示例生成 CS0215：</span><span class="sxs-lookup"><span data-stu-id="b360a-106">The following sample generates CS0215:</span></span>  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
