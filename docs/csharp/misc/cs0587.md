# <a name="compiler-error-cs0587"></a>编译器错误 CS0587

内部编译器错误: 步骤“stage”

 尝试确定编译器是否由于无法分析意外的语法而产生故障。 如果不是这样，请尝试[与我们交谈](/visualstudio/ide/feedback-options)。

> [!NOTE]
> Roslyn 中不再使用此编译器错误。
