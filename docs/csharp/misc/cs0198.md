 <span data-ttu-id="b2513-104">[readonly](../language-reference/keywords/readonly.md) 变量必须与构造函数具有相同的 [静态](../language-reference/keywords/static.md) 用法，在此构造函数中你要对其进行初始化。</span><span class="sxs-lookup"><span data-stu-id="b2513-104">A [readonly](../language-reference/keywords/readonly.md) variable must have the same [static](../language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="b2513-105">有关详细信息，请参阅[静态构造函数](../programming-guide/classes-and-structs/static-constructors.md)。</span><span class="sxs-lookup"><span data-stu-id="b2513-105">For more information, see [Static Constructors](../programming-guide/classes-and-structs/static-constructors.md).</span></span>

```csharp  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
