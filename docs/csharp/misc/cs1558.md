# <a name="compiler-error-cs1558"></a><span data-ttu-id="c92d5-102">编译器错误 CS1558</span><span class="sxs-lookup"><span data-stu-id="c92d5-102">Compiler Error CS1558</span></span>
<span data-ttu-id="c92d5-103">“class”没有合适的静态 Main 方法</span><span class="sxs-lookup"><span data-stu-id="c92d5-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="c92d5-104">[/main](../language-reference/compiler-options/main-compiler-option.md) 编译器选项指定了一个在其中查找 **Main** 方法的类。</span><span class="sxs-lookup"><span data-stu-id="c92d5-104">The [/main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="c92d5-105">但是， [Main](../programming-guide/main-and-command-args/index.md) 方法未正确定义。</span><span class="sxs-lookup"><span data-stu-id="c92d5-105">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="c92d5-106">以下示例由于无效的返回类型而生成 CS1558。</span><span class="sxs-lookup"><span data-stu-id="c92d5-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
