 <span data-ttu-id="54950-255">当表达式中出现两个或两个以上具有相同优先级的运算符时，将根据结合性计算它们。</span><span class="sxs-lookup"><span data-stu-id="54950-255">When two or more operators that have the same precedence are present in an expression, they are evaluated based on associativity.</span></span> <span data-ttu-id="54950-256">左结合运算符按从左到右的顺序计算。</span><span class="sxs-lookup"><span data-stu-id="54950-256">Left-associative operators are evaluated in order from left to right.</span></span> <span data-ttu-id="54950-257">例如，`x * y / z` 将计算为 `(x * y) / z`。</span><span class="sxs-lookup"><span data-stu-id="54950-257">For example, `x * y / z` is evaluated as `(x * y) / z`.</span></span> <span data-ttu-id="54950-258">右结合运算符按从右到左的顺序计算。</span><span class="sxs-lookup"><span data-stu-id="54950-258">Right-associative operators are evaluated in order from right to left.</span></span> <span data-ttu-id="54950-259">例如，赋值运算符是右关联的。</span><span class="sxs-lookup"><span data-stu-id="54950-259">For example, the assignment operator is right associative.</span></span> <span data-ttu-id="54950-260">如果不是，下面的代码将导致错误。</span><span class="sxs-lookup"><span data-stu-id="54950-260">If it were not, the following code would result in an error.</span></span>

//(a = b) = c;
