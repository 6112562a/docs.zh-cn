## <a name="namespaces-overview"></a>命名空间概述  

命名空间具有以下属性：  
  
- 它们组织大型代码项目。  
- 通过使用 `.` 运算符分隔它们。  
- `using` 指令可免去为每个类指定命名空间的名称。  
- `global` 命名空间是“根”命名空间：`global::System` 始终引用 .NET <xref:System> 命名空间。
