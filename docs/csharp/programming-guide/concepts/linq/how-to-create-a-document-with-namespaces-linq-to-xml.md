## <a name="example"></a><span data-ttu-id="88006-104">示例</span><span class="sxs-lookup"><span data-stu-id="88006-104">Example</span></span>  
 <span data-ttu-id="88006-105">若要创建一个属于命名空间的元素或属性，请首先声明和初始化一个 <xref:System.Xml.Linq.XNamespace> 对象。</span><span class="sxs-lookup"><span data-stu-id="88006-105">To create an element or an attribute that is in a namespace, you first declare and initialize an <xref:System.Xml.Linq.XNamespace> object.</span></span> <span data-ttu-id="88006-106">然后使用加法运算符重载来组合命名空间和本地名称（以字符串表示）。</span><span class="sxs-lookup"><span data-stu-id="88006-106">You then use the addition operator overload to combine the namespace with the local name, expressed as a string.</span></span>  
  
 <span data-ttu-id="88006-107">下面的示例创建一个包含一个命名空间的文档。</span><span class="sxs-lookup"><span data-stu-id="88006-107">The following example creates a document with one namespace.</span></span> <span data-ttu-id="88006-108">默认情况下，[!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] 使用默认命名空间序列化此文档。</span><span class="sxs-lookup"><span data-stu-id="88006-108">By default, [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] serializes this document with a default namespace.</span></span>  
  
```csharp  
// Create an XML tree in a namespace.  
XNamespace aw = "http://www.adventure-works.com";  
XElement root = new XElement(aw + "Root",  
    new XElement(aw + "Child", "child content")  
);  
Console.WriteLine(root);  
```  
  
 <span data-ttu-id="88006-109">该示例产生下面的输出：</span><span class="sxs-lookup"><span data-stu-id="88006-109">This example produces the following output:</span></span>  
  
```xml

  <Child>child content</Child>  
</Root>  
```  
  
## <a name="example"></a><span data-ttu-id="88006-110">示例</span><span class="sxs-lookup"><span data-stu-id="88006-110">Example</span></span>  
 <span data-ttu-id="88006-111">下面的示例创建一个包含一个命名空间的文档。</span><span class="sxs-lookup"><span data-stu-id="88006-111">The following example creates a document with one namespace.</span></span> <span data-ttu-id="88006-112">另外，还创建一个属性，该属性声明具有命名空间前缀的命名空间。</span><span class="sxs-lookup"><span data-stu-id="88006-112">It also creates an attribute that declares the namespace with a namespace prefix.</span></span> <span data-ttu-id="88006-113">若要创建一个声明具有前缀的命名空间的属性，请创建一个属性，其中该属性的名称为命名空间前缀，并且此名称位于 <xref:System.Xml.Linq.XNamespace.Xmlns%2A> 命名空间中。</span><span class="sxs-lookup"><span data-stu-id="88006-113">To create an attribute that declares a namespace with a prefix, you create an attribute where the name of the attribute is the namespace prefix, and this name is in the <xref:System.Xml.Linq.XNamespace.Xmlns%2A> namespace.</span></span> <span data-ttu-id="88006-114">此属性的值即是命名空间的 URI。</span><span class="sxs-lookup"><span data-stu-id="88006-114">The value of this attribute is the URI of the namespace.</span></span>  
  
```csharp  
// Create an XML tree in a namespace, with a specified prefix  
XNamespace aw = "http://www.adventure-works.com";  
XElement root = new XElement(aw + "Root",  
    new XAttribute(XNamespace.Xmlns + "aw", "http://www.adventure-works.com"),  
    new XElement(aw + "Child", "child content")  
);  
Console.WriteLine(root);  
```  
  
 <span data-ttu-id="88006-115">该示例产生下面的输出：</span><span class="sxs-lookup"><span data-stu-id="88006-115">This example produces the following output:</span></span>  
  
```xml  
<aw:Root xmlns:aw="http://www.adventure-works.com">  
  <aw:Child>child content</aw:Child>  
</aw:Root>  
```  
  
## <a name="example"></a><span data-ttu-id="88006-116">示例</span><span class="sxs-lookup"><span data-stu-id="88006-116">Example</span></span>  
 <span data-ttu-id="88006-117">下面的示例演示如何创建一个包含两个命名空间的文档。</span><span class="sxs-lookup"><span data-stu-id="88006-117">The following example shows the creation of a document that contains two namespaces.</span></span> <span data-ttu-id="88006-118">一个是默认命名空间。</span><span class="sxs-lookup"><span data-stu-id="88006-118">One is the default namespace.</span></span> <span data-ttu-id="88006-119">另一个是具有前缀的命名空间。</span><span class="sxs-lookup"><span data-stu-id="88006-119">Another is a namespace with a prefix.</span></span>  
  
 <span data-ttu-id="88006-120">通过在根元素中包括命名空间属性，命名空间进行了序列化，从而 `http://www.adventure-works.com` 是默认命名空间，而 `www.fourthcoffee.com` 用“fc”前缀进行了序列化。</span><span class="sxs-lookup"><span data-stu-id="88006-120">By including namespace attributes in the root element, the namespaces are serialized so that `http://www.adventure-works.com` is the default namespace, and `www.fourthcoffee.com` is serialized with a prefix of "fc".</span></span> <span data-ttu-id="88006-121">若要创建一个声明默认命名空间的属性，请创建一个名称为“xmlns”的属性，而无需命名空间。</span><span class="sxs-lookup"><span data-stu-id="88006-121">To create an attribute that declares a default namespace, you create an attribute with the name "xmlns", without a namespace.</span></span> <span data-ttu-id="88006-122">该属性的值即是默认命名空间 URI。</span><span class="sxs-lookup"><span data-stu-id="88006-122">The value of the attribute is the default namespace URI.</span></span>  
  
```csharp  
// The http://www.adventure-works.com namespace is forced to be the default namespace.  
XNamespace aw = "http://www.adventure-works.com";  
XNamespace fc = "www.fourthcoffee.com";  
XElement root = new XElement(aw + "Root",  
    new XAttribute("xmlns", "http://www.adventure-works.com"),  
    new XAttribute(XNamespace.Xmlns + "fc", "www.fourthcoffee.com"),  
    new XElement(fc + "Child",  
        new XElement(aw + "DifferentChild", "other content")  
    ),  
    new XElement(aw + "Child2", "c2 content"),  
    new XElement(fc + "Child3", "c3 content")  
);  
Console.WriteLine(root);  
```  
  
 <span data-ttu-id="88006-123">该示例产生下面的输出：</span><span class="sxs-lookup"><span data-stu-id="88006-123">This example produces the following output:</span></span>  
  
```xml  
<Root xmlns="http://www.adventure-works.com" xmlns:fc="www.fourthcoffee.com">  
  <fc:Child>  
    <DifferentChild>other content</DifferentChild>  
  </fc:Child>  
  <Child2>c2 content</Child2>  
  <fc:Child3>c3 content</fc:Child3>  
</Root>  
```  
  
## <a name="example"></a><span data-ttu-id="88006-124">示例</span><span class="sxs-lookup"><span data-stu-id="88006-124">Example</span></span>  
 <span data-ttu-id="88006-125">下面的示例创建一个包含两个命名空间的文档，这两个命名空间都具有命名空间前缀。</span><span class="sxs-lookup"><span data-stu-id="88006-125">The following example creates a document that contains two namespaces, both with namespace prefixes.</span></span>
