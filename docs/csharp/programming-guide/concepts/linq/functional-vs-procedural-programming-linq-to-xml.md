<span data-ttu-id="edf12-103">XML 应用程序有多种类型：</span><span class="sxs-lookup"><span data-stu-id="edf12-103">There are various types of XML applications:</span></span>  
  
- <span data-ttu-id="edf12-104">有些应用程序采用源 XML 文档并生成与源文档形状不同的新 XML 文档。</span><span class="sxs-lookup"><span data-stu-id="edf12-104">Some applications take source XML documents, and produce new XML documents that are in a different shape than the source documents.</span></span>

- <span data-ttu-id="edf12-106">有些应用程序采用源 XML 文档并将记录插入数据库。</span><span class="sxs-lookup"><span data-stu-id="edf12-106">Some applications take source XML documents, and insert records into a database.</span></span>

 <span data-ttu-id="edf12-108">这些并不是所有的 XML 应用程序类型，但它们是 XML 程序员必须实现的一组有代表性的功能类型。</span><span class="sxs-lookup"><span data-stu-id="edf12-108">These are not all of the types of XML applications, but these are a representative set of the types of functionality that an XML programmer has to implement.</span></span>  
  
 <span data-ttu-id="edf12-109">对于所有这些类型应用程序，开发人员可以采用两种对比方法：</span><span class="sxs-lookup"><span data-stu-id="edf12-109">With all of these types of applications, there are two contrasting approaches that a developer can take:</span></span>  
  
- <span data-ttu-id="edf12-110">使用声明性方法的函数构造。</span><span class="sxs-lookup"><span data-stu-id="edf12-110">Functional construction using a declarative approach.</span></span>  
  
- <span data-ttu-id="edf12-111">使用过程代码的内存中 XML 树修改法。</span><span class="sxs-lookup"><span data-stu-id="edf12-111">In-memory XML tree modification using procedural code.</span></span>  
  
 <span data-ttu-id="edf12-112">LINQ to XML 同时支持这两种方法。</span><span class="sxs-lookup"><span data-stu-id="edf12-112">LINQ to XML supports both approaches.</span></span>  
  
 <span data-ttu-id="edf12-113">使用函数方法时，编写可采用源文档并生成具有所需形状的全新结果文档的转换。</span><span class="sxs-lookup"><span data-stu-id="edf12-113">When using the functional approach, you write transformations that take the source documents and generate completely new result documents with the desired shape.</span></span>  
  
 <span data-ttu-id="edf12-114">就地修改 XML 树时，编写可遍历内存中 XML 树节点并在其中导航以便根据需要插入、删除和修改节点的代码。</span><span class="sxs-lookup"><span data-stu-id="edf12-114">When modifying an XML tree in place, you write code that traverses and navigates through nodes in an in-memory XML tree, inserting, deleting, and modifying nodes as necessary.</span></span>
