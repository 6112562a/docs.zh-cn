 <span data-ttu-id="9a98c-111">类是引用类型。</span><span class="sxs-lookup"><span data-stu-id="9a98c-111">A class is a reference type.</span></span> <span data-ttu-id="9a98c-112">创建类的对象后，向其分配对象的变量仅保留对相应内存的引用。</span><span class="sxs-lookup"><span data-stu-id="9a98c-112">When an object of the class is created, the variable to which the object is assigned holds only a reference to that memory.</span></span> <span data-ttu-id="9a98c-113">将对象引用分配给新变量后，新变量会引用原始对象。</span><span class="sxs-lookup"><span data-stu-id="9a98c-113">When the object reference is assigned to a new variable, the new variable refers to the original object.</span></span> <span data-ttu-id="9a98c-114">通过一个变量所做的更改将反映在另一个变量中，因为它们引用相同的数据。</span><span class="sxs-lookup"><span data-stu-id="9a98c-114">Changes made through one variable are reflected in the other variable because they both refer to the same data.</span></span>
