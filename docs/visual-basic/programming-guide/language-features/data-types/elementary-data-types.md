# <a name="elementary-data-types-visual-basic"></a>基本数据类型 (Visual Basic)
Visual Basic 提供一组预定义的数据类型，可用于许多编程元素。 本部分介绍了这些类型以及如何使用它们。  
  
> [!NOTE]
> Visual Basic 中的每个基本数据类型都受结构或 <xref:System> 命名空间中的类支持。 编译器使用每个数据类型关键字作为基础结构或类的别名。 例如，使用保留字 `Byte` 声明变量与使用完全限定的结构名称 <xref:System.Byte?displayProperty=nameWithType>声明变量相同。
