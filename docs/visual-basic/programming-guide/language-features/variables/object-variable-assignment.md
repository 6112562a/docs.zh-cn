Dim thisObject As Object

## <a name="initialization"></a><span data-ttu-id="9e494-106">初始化</span><span class="sxs-lookup"><span data-stu-id="9e494-106">Initialization</span></span>

Dim ver As New System.Version(6, 1, 0)  
```  
  
## <a name="disassociation"></a><span data-ttu-id="9e494-112">解除关联</span><span class="sxs-lookup"><span data-stu-id="9e494-112">Disassociation</span></span>

```  
  
 <span data-ttu-id="9e494-116">如果您的变量是指该对象是另一个应用程序中，该应用程序是否已终止或只是失效的对象无法确定此测试。</span><span class="sxs-lookup"><span data-stu-id="9e494-116">If the object your variable refers to is in another application, this test cannot determine whether that application has terminated or just invalidated the object.</span></span>  
  
 <span data-ttu-id="9e494-117">值为对象变量`Nothing`也称为*null 引用*。</span><span class="sxs-lookup"><span data-stu-id="9e494-117">An object variable with a value of `Nothing` is also called a *null reference*.</span></span>  
  
## <a name="current-instance"></a><span data-ttu-id="9e494-118">当前实例</span><span class="sxs-lookup"><span data-stu-id="9e494-118">Current Instance</span></span>  
 <span data-ttu-id="9e494-119">*当前实例*的对象是当前正在其中执行代码。</span><span class="sxs-lookup"><span data-stu-id="9e494-119">The *current instance* of an object is the one in which the code is currently executing.</span></span> <span data-ttu-id="9e494-120">在过程内执行所有代码，因为当前实例是在其中调用该过程。</span><span class="sxs-lookup"><span data-stu-id="9e494-120">Since all code executes inside a procedure, the current instance is the one in which the procedure was invoked.</span></span>  
  
 <span data-ttu-id="9e494-121">`Me`关键字作为对象变量引用当前实例。</span><span class="sxs-lookup"><span data-stu-id="9e494-121">The `Me` keyword acts as an object variable referring to the current instance.</span></span> <span data-ttu-id="9e494-122">如果不是一个过程[共享](../../../../visual-basic/language-reference/modifiers/shared.md)，可以使用`Me`关键字来获取当前实例的指针。</span><span class="sxs-lookup"><span data-stu-id="9e494-122">If a procedure is not [Shared](../../../../visual-basic/language-reference/modifiers/shared.md), it can use the `Me` keyword to obtain a pointer to the current instance.</span></span> <span data-ttu-id="9e494-123">共享的过程不能与类的特定实例相关联。</span><span class="sxs-lookup"><span data-stu-id="9e494-123">Shared procedures cannot be associated with a specific instance of a class.</span></span>  
  
 <span data-ttu-id="9e494-124">使用`Me`对于当前的实例传递到另一个模块中的过程非常有用。</span><span class="sxs-lookup"><span data-stu-id="9e494-124">Using `Me` is particularly useful for passing the current instance to a procedure in another module.</span></span> <span data-ttu-id="9e494-125">例如，假设你有多个 XML 文档，想要将一些标准文本添加到所有这些。</span><span class="sxs-lookup"><span data-stu-id="9e494-125">For example, suppose you have a number of XML documents and wish to add some standard text to all of them.</span></span> <span data-ttu-id="9e494-126">下面的示例定义一个过程来执行此操作。</span><span class="sxs-lookup"><span data-stu-id="9e494-126">The following example defines a procedure to do this.</span></span>  
  
```  
Sub addStandardText(XmlDoc As System.Xml.XmlDocument)  
    XmlDoc.CreateTextNode("This text goes into every XML document.")  
End Sub  
```  
  
 <span data-ttu-id="9e494-127">然后，每个 XML 文档对象无法调用该过程，并作为参数传递它的当前实例。</span><span class="sxs-lookup"><span data-stu-id="9e494-127">Every XML document object could then call the procedure and pass its current instance as an argument.</span></span> <span data-ttu-id="9e494-128">下面的示例演示这一操作。</span><span class="sxs-lookup"><span data-stu-id="9e494-128">The following example demonstrates this.</span></span>  
  
```  
addStandardText(Me)  
```
