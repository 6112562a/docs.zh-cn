## <a name="example"></a><span data-ttu-id="9fb19-106">示例</span><span class="sxs-lookup"><span data-stu-id="9fb19-106">Example</span></span>  
 <span data-ttu-id="9fb19-107">下面的示例使用 <xref:System.Xml.Linq.XElement.Attribute%2A> 方法。</span><span class="sxs-lookup"><span data-stu-id="9fb19-107">The following example uses the <xref:System.Xml.Linq.XElement.Attribute%2A> method.</span></span>  
  
```vb

                           <Phone type="home">555-555-5555</Phone>  
                           <Phone type="work">555-555-6666</Phone>  
                       </PhoneNumbers>

    Console.WriteLine(e.@type)  
Next  
```

## <a name="example"></a><span data-ttu-id="9fb19-110">示例</span><span class="sxs-lookup"><span data-stu-id="9fb19-110">Example</span></span>  
 <span data-ttu-id="9fb19-111">如果希望检索属性的值，可以强制转换该值，就像使用 <xref:System.Xml.Linq.XElement> 对象时一样。</span><span class="sxs-lookup"><span data-stu-id="9fb19-111">If you want to retrieve the value of the attribute, you can cast it, just as you do for with <xref:System.Xml.Linq.XElement> objects.</span></span> <span data-ttu-id="9fb19-112">下面的示例演示这一操作。</span><span class="sxs-lookup"><span data-stu-id="9fb19-112">The following example demonstrates this.</span></span>  
  
```vb  
Dim cust As XElement = <PhoneNumbers>  
                           <Phone type="home">555-555-5555</Phone>  
                           <Phone type="work">555-555-6666</Phone>  
                       </PhoneNumbers>  
Dim elList As IEnumerable(Of XElement) = _  
    From el In cust...<Phone> _  
    Select el  
For Each el As XElement In elList  
    Console.WriteLine(el.@type)  
Next  
```  
  
 <span data-ttu-id="9fb19-113">此代码生成以下输出：</span><span class="sxs-lookup"><span data-stu-id="9fb19-113">This code produces the following output:</span></span>


## <a name="example"></a><span data-ttu-id="9fb19-115">示例</span><span class="sxs-lookup"><span data-stu-id="9fb19-115">Example</span></span>
