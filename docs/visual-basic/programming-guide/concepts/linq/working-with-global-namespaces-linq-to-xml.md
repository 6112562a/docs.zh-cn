## <a name="examples-of-global-namespaces"></a><span data-ttu-id="50b30-115">全局命名空间的示例</span><span class="sxs-lookup"><span data-stu-id="50b30-115">Examples of Global Namespaces</span></span>  
 <span data-ttu-id="50b30-116">下面的示例通过使用 `Imports` 语句声明一个默认的全局命名空间，然后使用 XML 文本在该命名空间中初始化一个 <xref:System.Xml.Linq.XElement> 对象：</span><span class="sxs-lookup"><span data-stu-id="50b30-116">The following example declares a default global namespace by using the `Imports` statement, and then uses an XML literal to initialize an <xref:System.Xml.Linq.XElement> object in that namespace:</span></span>

 <span data-ttu-id="50b30-119">该示例产生下面的输出：</span><span class="sxs-lookup"><span data-stu-id="50b30-119">This example produces the following output:</span></span>

## <a name="global-namespaces-and-embedded-expressions"></a><span data-ttu-id="50b30-120">全局命名空间和嵌入式表达式</span><span class="sxs-lookup"><span data-stu-id="50b30-120">Global Namespaces and Embedded Expressions</span></span>  
 <span data-ttu-id="50b30-121">使用 XML 文本声明的命名空间不会延续到嵌入的表达式。</span><span class="sxs-lookup"><span data-stu-id="50b30-121">Namespaces that are declared in XML literals do not carry over into embedded expressions.</span></span> <span data-ttu-id="50b30-122">下面的示例声明一个默认命名空间。</span><span class="sxs-lookup"><span data-stu-id="50b30-122">The following example declares a default namespace.</span></span> <span data-ttu-id="50b30-123">然后对 `Child` 元素使用一个嵌入式表达式。</span><span class="sxs-lookup"><span data-stu-id="50b30-123">It then uses an embedded expression for the `Child` element.</span></span>  
  
```vb

    <Root xmlns="http://www.adventure-works.com">  
        <%= <Child/> %>  
    </Root>  
Console.WriteLine(root)  
```  
  
 <span data-ttu-id="50b30-124">该示例产生下面的输出：</span><span class="sxs-lookup"><span data-stu-id="50b30-124">This example produces the following output:</span></span>  
  
```xml  
<Root xmlns="http://www.adventure-works.com">  
  <Child xmlns="" />  
</Root>  
```  
  
 <span data-ttu-id="50b30-125">您可以看到，生成的 XML 包括一个默认命名空间声明，从而使 `Child` 元素不在任何命名空间中。</span><span class="sxs-lookup"><span data-stu-id="50b30-125">As you can see, the resulting XML includes a declaration of a default namespace so that the `Child` element is in no namespace.</span></span>  
  
 <span data-ttu-id="50b30-126">您可以在嵌入式表达式中重新声明该命名空间，如下所示：</span><span class="sxs-lookup"><span data-stu-id="50b30-126">You could re-declare the namespace in the embedded expression, as follows:</span></span>  
  
```vb

    <Root xmlns="http://www.adventure-works.com">  
        <%= <Child xmlns="http://www.adventure-works.com"/> %>  
    </Root>  
Console.WriteLine(root)  
```  
  
 <span data-ttu-id="50b30-127">该示例产生下面的输出：</span><span class="sxs-lookup"><span data-stu-id="50b30-127">This example produces the following output:</span></span>  
  
```xml  
<Root xmlns="http://www.adventure-works.com">  
  <Child xmlns="http://www.adventure-works.com" />  
</Root>  
```  
  
 <span data-ttu-id="50b30-128">但是，这种方法更不便于使用，更好的方法是使用全局默认命名空间。</span><span class="sxs-lookup"><span data-stu-id="50b30-128">However, this is more cumbersome to use than the global default namespace, which is a better approach.</span></span> <span data-ttu-id="50b30-129">通过使用全局默认命名空间，您可以使用 XML 文本而无需声明命名空间。</span><span class="sxs-lookup"><span data-stu-id="50b30-129">With the global default namespace, you can use XML literals without declaring namespaces.</span></span> <span data-ttu-id="50b30-130">生成的 XML 将处于全局声明的默认命名空间中。</span><span class="sxs-lookup"><span data-stu-id="50b30-130">The resulting XML will be in the globally-declared default namespace.</span></span>  
  
```vb

Module Module1  
    Sub Main()  
        Dim root As XElement = <Root>  
                                   <%= <Child/> %>  
                               </Root>  
        Console.WriteLine(root)  
    End Sub  
End Module  
```  
  
 <span data-ttu-id="50b30-131">该示例产生下面的输出：</span><span class="sxs-lookup"><span data-stu-id="50b30-131">This example produces the following output:</span></span>

  <Child />
