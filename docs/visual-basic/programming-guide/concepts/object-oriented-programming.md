 <span data-ttu-id="8701f-283">“委托”是一种类型，它定义方法签名，可以提供对具有兼容签名的任何方法的引用。</span><span class="sxs-lookup"><span data-stu-id="8701f-283">A *delegate* is a type that defines a method signature, and can provide a reference to any method with a compatible signature.</span></span> <span data-ttu-id="8701f-284">你可以通过委托调用方法。</span><span class="sxs-lookup"><span data-stu-id="8701f-284">You can invoke (or call) the method through the delegate.</span></span> <span data-ttu-id="8701f-285">委托用于将方法作为参数传递给其他方法。</span><span class="sxs-lookup"><span data-stu-id="8701f-285">Delegates are used to pass methods as arguments to other methods.</span></span>
