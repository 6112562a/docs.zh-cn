# <a name="returns-visual-basic"></a>\<返回 > (Visual Basic)
指定属性或函数的返回值。  
  
## <a name="syntax"></a>语法  
  
```xml  
<returns>description</returns>  
```  
  
#### <a name="parameters"></a>参数  
 `description`  
 返回值的说明。  
  
## <a name="remarks"></a>备注  
 使用`<returns>`标记来描述返回值的方法声明的注释中。  
  
 使用 [/doc](../../../visual-basic/reference/command-line-compiler/doc.md) 进行编译可以将文档注释处理到文件中。
