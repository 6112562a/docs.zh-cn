## <a name="remarks"></a><span data-ttu-id="edad4-104">备注</span><span class="sxs-lookup"><span data-stu-id="edad4-104">Remarks</span></span>

<span data-ttu-id="edad4-105">*重载*是一种提供多个定义相同的作用域中的给定属性或过程名称的行为。</span><span class="sxs-lookup"><span data-stu-id="edad4-105">*Overloading* is the practice of supplying more than one definition for a given property or procedure name in the same scope.</span></span> <span data-ttu-id="edad4-106">重新声明属性或具有不同签名过程有时称为*按签名隐藏*。</span><span class="sxs-lookup"><span data-stu-id="edad4-106">Redeclaring a property or procedure with a different signature is sometimes called *hiding by signature*.</span></span>

## <a name="rules"></a><span data-ttu-id="edad4-107">规则</span><span class="sxs-lookup"><span data-stu-id="edad4-107">Rules</span></span>

- <span data-ttu-id="edad4-108">**声明上下文。**</span><span class="sxs-lookup"><span data-stu-id="edad4-108">**Declaration Context.**</span></span> <span data-ttu-id="edad4-109">只能在属性或过程声明语句中使用 `Overloads`。</span><span class="sxs-lookup"><span data-stu-id="edad4-109">You can use `Overloads` only in a property or procedure declaration statement.</span></span>

- <span data-ttu-id="edad4-110">**组合的修饰符。**</span><span class="sxs-lookup"><span data-stu-id="edad4-110">**Combined Modifiers.**</span></span> <span data-ttu-id="edad4-111">不能指定`Overloads`连同[阴影](../../../visual-basic/language-reference/modifiers/shadows.md)同一过程声明中。</span><span class="sxs-lookup"><span data-stu-id="edad4-111">You cannot specify `Overloads` together with [Shadows](../../../visual-basic/language-reference/modifiers/shadows.md) in the same procedure declaration.</span></span>

  - <span data-ttu-id="edad4-115">参数的数量</span><span class="sxs-lookup"><span data-stu-id="edad4-115">the number of parameters</span></span>

  - <span data-ttu-id="edad4-116">参数顺序</span><span class="sxs-lookup"><span data-stu-id="edad4-116">the order of the parameters</span></span>

  - <span data-ttu-id="edad4-117">参数的数据类型</span><span class="sxs-lookup"><span data-stu-id="edad4-117">the data types of the parameters</span></span>

  - <span data-ttu-id="edad4-118">类型参数的数量（针对泛型过程）</span><span class="sxs-lookup"><span data-stu-id="edad4-118">the number of type parameters (for a generic procedure)</span></span>

  - <span data-ttu-id="edad4-119">返回类型（仅针对转换运算符过程）</span><span class="sxs-lookup"><span data-stu-id="edad4-119">the return type (only for a conversion operator procedure)</span></span>

  <span data-ttu-id="edad4-120">所有重载都必须有相同的名称，但每个重载必须在上面的一个或多个方面与所有其他重载不同。</span><span class="sxs-lookup"><span data-stu-id="edad4-120">All overloads must have the same name, but each must differ from all the others in one or more of the preceding respects.</span></span> <span data-ttu-id="edad4-121">这使编译器在代码调用属性或过程时可以区分要使用哪个版本。</span><span class="sxs-lookup"><span data-stu-id="edad4-121">This allows the compiler to distinguish which version to use when code calls the property or procedure.</span></span>

- <span data-ttu-id="edad4-122">**不允许的差异。**</span><span class="sxs-lookup"><span data-stu-id="edad4-122">**Disallowed Differences.**</span></span> <span data-ttu-id="edad4-123">对于重载属性或过程而言，更改下面的一项或多项无效，因为它们不是签名的一部分：</span><span class="sxs-lookup"><span data-stu-id="edad4-123">Changing one or more of the following is not valid for overloading a property or procedure, because they are not part of the signature:</span></span>

  - <span data-ttu-id="edad4-124">它是否返回值（针对过程）</span><span class="sxs-lookup"><span data-stu-id="edad4-124">whether or not it returns a value (for a procedure)</span></span>

  - <span data-ttu-id="edad4-125">返回值的数据类型（除转换运算符之外）</span><span class="sxs-lookup"><span data-stu-id="edad4-125">the data type of the return value (except for a conversion operator)</span></span>

  - <span data-ttu-id="edad4-126">参数或类型参数的名称</span><span class="sxs-lookup"><span data-stu-id="edad4-126">the names of the parameters or type parameters</span></span>

  - <span data-ttu-id="edad4-127">对类型参数的约束（针对泛型过程）</span><span class="sxs-lookup"><span data-stu-id="edad4-127">the constraints on the type parameters (for a generic procedure)</span></span>

  - <span data-ttu-id="edad4-128">参数修饰符关键字（如 `ByRef` 或 `Optional`）</span><span class="sxs-lookup"><span data-stu-id="edad4-128">parameter modifier keywords (such as `ByRef` or `Optional`)</span></span>

  - <span data-ttu-id="edad4-129">属性或过程修饰符关键字（如 `Public` 或 `Shared`）</span><span class="sxs-lookup"><span data-stu-id="edad4-129">property or procedure modifier keywords (such as `Public` or `Shared`)</span></span>

- <span data-ttu-id="edad4-130">**可选的修饰符。**</span><span class="sxs-lookup"><span data-stu-id="edad4-130">**Optional Modifier.**</span></span> <span data-ttu-id="edad4-131">当在同一个类中定义多个重载属性或过程时，不必使用 `Overloads` 修饰符。</span><span class="sxs-lookup"><span data-stu-id="edad4-131">You do not have to use the `Overloads` modifier when you are defining multiple overloaded properties or procedures in the same class.</span></span> <span data-ttu-id="edad4-132">然而，如果在某个声明中使用 `Overloads`，则必须在所有的声明中使用它。</span><span class="sxs-lookup"><span data-stu-id="edad4-132">However, if you use `Overloads` in one of the declarations, you must use it in all of them.</span></span>

- <span data-ttu-id="edad4-133">**隐藏和重载。**</span><span class="sxs-lookup"><span data-stu-id="edad4-133">**Shadowing and Overloading.**</span></span> <span data-ttu-id="edad4-134">`Overloads` 此外可以使用卷影的现有成员，或一组重载的成员，在基类中。</span><span class="sxs-lookup"><span data-stu-id="edad4-134">`Overloads` can also be used to shadow an existing member, or set of overloaded members, in a base class.</span></span> <span data-ttu-id="edad4-135">以这种方式使用 `Overloads` 时，应使用与基类成员相同的名称和参数列表来声明属性或方法，并且不提供 `Shadows` 关键字。</span><span class="sxs-lookup"><span data-stu-id="edad4-135">When you use `Overloads` in this way, you declare the property or method with the same name and the same parameter list as the base class member, and you do not supply the `Shadows` keyword.</span></span>
