# <a name="array-declared-as-for-loop-control-variable-cannot-be-declared-with-an-initial-size"></a><span data-ttu-id="a324c-102">声明为 For Each 循环控制变量的数组在声明时不能指定初始大小值</span><span class="sxs-lookup"><span data-stu-id="a324c-102">Array declared as for loop control variable cannot be declared with an initial size</span></span>

<span data-ttu-id="a324c-103">`For Each` 循环使用数组作为其迭代变量，但会初始化该数组。</span><span class="sxs-lookup"><span data-stu-id="a324c-103">A `For Each` loop uses an array as its iteration variable but initializes that array.</span></span>

<span data-ttu-id="a324c-104">**错误 ID：** BC32039</span><span class="sxs-lookup"><span data-stu-id="a324c-104">**Error ID:** BC32039</span></span>

## <a name="example"></a><span data-ttu-id="a324c-105">示例</span><span class="sxs-lookup"><span data-stu-id="a324c-105">Example</span></span>

<span data-ttu-id="a324c-106">下面的示例生成 bc32039：</span><span class="sxs-lookup"><span data-stu-id="a324c-106">The following example generates bc32039:</span></span>

```vb  
Dim arrayList As New List(Of Integer())
For Each listElement(1) As Integer In arrayList

Next
```  

## <a name="to-correct-this-error"></a><span data-ttu-id="a324c-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="a324c-107">To correct this error</span></span>  

<span data-ttu-id="a324c-108">从迭代变量的声明中删除初始化，如以下示例中所示：</span><span class="sxs-lookup"><span data-stu-id="a324c-108">Remove the initialization from the declaration of the iteration variable as shown in the following example:</span></span>

```vb
Dim arrayList As New List(Of Integer())
For Each listElement() As Integer In arrayList

Next
```

<span data-ttu-id="a324c-109">您也可以使用类型推理：</span><span class="sxs-lookup"><span data-stu-id="a324c-109">or you can use type inference:</span></span>

```vb
Dim arrayList As New List(Of Integer())
For Each listElement In arrayList

Next
```
