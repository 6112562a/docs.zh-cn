# <a name="device-io-error"></a>设备 I/O 错误
应用程序正在使用的设备如打印机或磁盘驱动器时，输入或输出错误出现。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保该设备正常，然后重试该操作。
