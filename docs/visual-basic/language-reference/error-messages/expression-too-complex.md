# <a name="expression-too-complex"></a>表达式过于复杂
浮点表达式包含太多的嵌套子表达式。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将表达式拆分为任意多个单独的表达式中，根据需要以防止再次出现错误。
