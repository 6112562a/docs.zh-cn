# <a name="array-subscript-expression-missing"></a>缺少数组下标表达式
数组初始化省略了一个或多个定义数组边界的下标。 例如，该语句可能包含表达式`myArray (5,5,,10)`，其中省略了第三个下标。  
  
 **错误 ID:** BC30306  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   提供缺少的下标。
