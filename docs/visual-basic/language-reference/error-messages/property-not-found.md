# <a name="property-not-found"></a>找不到属性
此对象不支持指定的属性。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查属性的名称的拼写。  
  
2. 请查看对象的文档，请确保您未尝试访问类似于"text"属性的内容时对象实际上支持"标题"或类似的命名属性。
