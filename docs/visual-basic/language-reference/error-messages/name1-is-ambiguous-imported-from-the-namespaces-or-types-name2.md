# <a name="name1-is-ambiguous-imported-from-the-namespaces-or-types-name2"></a>'\<name1 > 不明确，从命名空间或类型的导入\<name2 >
你提供的名称不明确，因此与另一个名称冲突。 Visual Basic 编译器不具有任何冲突解决规则;你必须自己区分名称。  
  
 **错误 ID:** BC30561  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 删除命名空间导入，消除名称歧义。  
  
2. 完全限定名称。
