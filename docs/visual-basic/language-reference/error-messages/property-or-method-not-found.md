# <a name="property-or-method-not-found"></a>未找到属性或方法
未定义引用的对象方法或对象属性。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 可能有拼写错误的对象的名称。 若要查看哪些属性和方法定义的对象，显示对象浏览器。 选择适当的对象库，若要查看可用属性和方法的列表。
