# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>可以&#39;t 打开&#39;&lt;文件名&gt;&#39;以进行写入
指定的文件不能打开进行写入，可能因为已经打开。  
  
 **错误 ID:** BC2012  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  关闭文件并重新打开它。  
  
2.  检查文件的权限。
