# <a name="string-constants-must-end-with-a-double-quote"></a>字符串常量必须以双引号结束
字符串常量必须开始和结束加上引号。  
  
 **ErrorID:** BC30648  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保文本字符串结尾引号 （"）。 如果粘贴其他文本编辑器中的值，请确保粘贴的字符是有效的引号，而不是类似，如"智能"或"大"的引号字符 （") 或两个单引号 （'）。
