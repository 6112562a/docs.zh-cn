# <a name="procedure-call-or-argument-is-not-valid-visual-basic"></a>过程调用或自变量无效 (Visual Basic)
无法完成调用的某些部分。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查参数以确保未超出允许的值的允许的范围。
