# <a name="identifier-expected"></a>应输入标识符
不是可识别的已声明的元素名称的编程元素出现在上下文要求元素名称的位置。 一个可能的原因是以外的某处指定了一个属性的语句开始处。  
  
 **错误 ID:** BC30203  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证该语句中的任何属性的所有位于开头。  
  
-   验证语句中的所有元素名称都拼写正确。
