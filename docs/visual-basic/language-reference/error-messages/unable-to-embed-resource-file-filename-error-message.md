# <a name="unable-to-embed-resource-file-filename-error-message"></a>无法嵌入资源文件 "\<filename >"： \<错误消息 >
Visual Basic 编译器调用程序集链接器（Al.exe，也称作 Alink）生成包含清单的程序集。 该链接器已报告在将本机 COM+ 资源文件直接嵌入到程序集时出错。  
  
 **错误 ID：** BC30143  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查引用的错误消息并查阅[al.exe](../../../framework/tools/al-exe-assembly-linker.md)。 获取进一步的说明和建议。  
  
2. 如果仍然出现错误，则收集有关该情况的信息并通知 Microsoft 产品支持服务。
