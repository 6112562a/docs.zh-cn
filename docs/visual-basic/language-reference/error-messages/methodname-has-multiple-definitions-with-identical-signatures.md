# <a name="39ltmethodnamegt39-has-multiple-definitions-with-identical-signatures"></a>&#39;&lt;methodname&gt; &#39;具有多个带有相同签名的定义
一个`Function`或`Sub`过程声明使用相同的过程名称和参数列表与以前的声明。 一个可能的原因是尝试重载原始过程。 重载的过程必须具有不同的参数列表。  
  
 **错误 ID:** BC30269  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   过程名称或参数列表中，更改或删除重复的声明。
