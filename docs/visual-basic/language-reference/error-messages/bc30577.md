# <a name="addressof-operand-must-be-the-name-of-a-method-without-parentheses"></a><span data-ttu-id="7a39e-102">“AddressOf”操作数必须是方法名（不带括号）</span><span class="sxs-lookup"><span data-stu-id="7a39e-102">'AddressOf' operand must be the name of a method (without parentheses)</span></span>

<span data-ttu-id="7a39e-103">`AddressOf` 运算符创建引用特定过程的过程委托实例。</span><span class="sxs-lookup"><span data-stu-id="7a39e-103">The `AddressOf` operator creates a procedure delegate instance that references a specific procedure.</span></span> <span data-ttu-id="7a39e-104">语法如下所示：</span><span class="sxs-lookup"><span data-stu-id="7a39e-104">The syntax is as follows:</span></span>

```vb
AddressOf procedurename
```

<span data-ttu-id="7a39e-105">在 `AddressOf` 后的参数中插入括号，其中无需任何内容。</span><span class="sxs-lookup"><span data-stu-id="7a39e-105">You inserted parentheses around the argument following `AddressOf`, where none are needed.</span></span>

<span data-ttu-id="7a39e-106">**错误 ID：** BC30577</span><span class="sxs-lookup"><span data-stu-id="7a39e-106">**Error ID:** BC30577</span></span>

## <a name="example"></a><span data-ttu-id="7a39e-107">示例</span><span class="sxs-lookup"><span data-stu-id="7a39e-107">Example</span></span>

<span data-ttu-id="7a39e-108">下面的示例生成 bc30577：</span><span class="sxs-lookup"><span data-stu-id="7a39e-108">The following example generates bc30577:</span></span>

```vb
Public Sub CountZeroToTen()
    For i = 0 To 10
        Console.WriteLine($"Counted: {i}")
        Threading.Thread.Sleep(500)
    Next
End Sub

Sub Main()
    ' Any of the following two lines generates bc30577.
    'Dim t As New Threading.Thread(AddressOf(CountZeroToTen))
    'Dim t As New Threading.Thread(AddressOf CountZeroToTen())
    t.Start()
End Sub
```

## <a name="to-correct-this-error"></a><span data-ttu-id="7a39e-109">更正此错误</span><span class="sxs-lookup"><span data-stu-id="7a39e-109">To correct this error</span></span>

1. <span data-ttu-id="7a39e-110">删除后面的参数两边的括号，如以下示例中所示 `AddressOf`：</span><span class="sxs-lookup"><span data-stu-id="7a39e-110">Remove the parentheses around the argument following `AddressOf` as shown in the following example:</span></span>

    ```vb

            Console.WriteLine($"Counted: {i}")
            Threading.Thread.Sleep(500)
        Next
    End Sub

    Sub Main()
        Dim t As New Threading.Thread(AddressOf CountZeroToTen)
        t.Start()
    End Sub
    ```

2. <span data-ttu-id="7a39e-111">请确保参数为方法名称。</span><span class="sxs-lookup"><span data-stu-id="7a39e-111">Make sure the argument is a method name.</span></span>
