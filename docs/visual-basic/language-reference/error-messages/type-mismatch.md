# <a name="type-mismatch-visual-basic"></a>类型不匹配 (Visual Basic)
试图以无效的方式将值转换为另一种类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查分配，确保它是有效的。  
  
2. 请确保未将对象传递到需要单个属性或值的过程。  
  
3. 请确保未使用表达式所需的模块或项目名称。
