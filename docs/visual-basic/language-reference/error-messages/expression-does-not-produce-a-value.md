# <a name="expression-does-not-produce-a-value"></a>表达式不产生值
已尝试使用不会生成中生成值的上下文，例如，调用的值的表达式`Sub`的上下文中其中`Function`预期。  
  
 **错误 ID:** BC30491  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将表达式更改为所生成的值。
