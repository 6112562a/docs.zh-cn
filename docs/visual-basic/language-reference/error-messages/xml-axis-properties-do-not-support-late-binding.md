# <a name="xml-axis-properties-do-not-support-late-binding"></a>XML 轴属性不支持后期绑定
非类型化的对象引用了 XML 轴属性。  
  
 **错误 ID:** BC31168  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 请确保该对象是强类型<xref:System.Xml.Linq.XElement>之前引用 XML 轴属性的对象。
