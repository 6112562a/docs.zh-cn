# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>此关键字已与该集合的某个元素相关联
指定键的集合成员已标识另一个成员的集合。 密钥是一个字符串中指定`Add`唯一标识集合中的特定成员的方法。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 此成员使用不同的密钥。
