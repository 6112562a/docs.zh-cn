## <a name="example"></a><span data-ttu-id="07fa0-113">示例</span><span class="sxs-lookup"><span data-stu-id="07fa0-113">Example</span></span>

<span data-ttu-id="07fa0-114">下面的示例生成 bc42025：</span><span class="sxs-lookup"><span data-stu-id="07fa0-114">The following example generates bc42025:</span></span>

```vb
Public Class TestClass
    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program
    Public Sub Main()
        Dim tc As New TestClass()
        tc.SayHello() ' BC42025.
    End Sub
End Module
```

## <a name="to-address-this-warning"></a><span data-ttu-id="07fa0-115">解决此警告</span><span class="sxs-lookup"><span data-stu-id="07fa0-115">To address this warning</span></span>

```vb

    Public Shared Sub SayHello()
        Console.WriteLine("Hello")
    End Sub
End Class

Module Program

        TestClass.SayHello()
    End Sub
End Module
```
