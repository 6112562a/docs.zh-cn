# <a name="some-subkeys-cannot-be-deleted"></a>无法删除某些子项
尝试删除注册表项，但操作失败，因为无法删除某些子项。 通常这是由于缺少权限。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保您具有足够权限删除指定的子项。
