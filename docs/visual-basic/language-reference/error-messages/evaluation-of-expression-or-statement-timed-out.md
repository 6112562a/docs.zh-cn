# <a name="evaluation-of-expression-or-statement-timed-out"></a>表达式或语句的计算已超时
表达式的计算未及时完成。  
  
 **错误 ID:** BC30722  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 验证输入的代码正确。  
  
2. 简化你的表达式，使花费更少的时间来执行。
