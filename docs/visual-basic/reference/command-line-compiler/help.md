# <a name="-help---visual-basic"></a>-help，-? (Visual Basic)
显示编译器选项。  
  
## <a name="syntax"></a>语法  
  
```  
-help  
' -or-  
-?  
```  
  
## <a name="remarks"></a>备注  
 如果将此选项包含在编译时，会创建任何输出文件并不进行编译。  
  
> [!NOTE]
>  `-help`选项不适用于从 Visual Studio 开发环境中，仅当从命令行编译时便可。  
  
## <a name="example"></a>示例  
 下面的代码显示从命令行帮助。  
  
```  
vbc -help  
```
