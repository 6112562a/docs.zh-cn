## <a name="example"></a><span data-ttu-id="c28f1-126">示例</span><span class="sxs-lookup"><span data-stu-id="c28f1-126">Example</span></span>  
 <span data-ttu-id="c28f1-127">下面的示例演示了 Visual Basic 编译器插入到 PE 默认清单。</span><span class="sxs-lookup"><span data-stu-id="c28f1-127">The following example shows the default manifest that the Visual Basic compiler inserts into a PE.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="c28f1-128">编译器将插入到清单 XML 的标准应用程序名称 MyApplication.app。</span><span class="sxs-lookup"><span data-stu-id="c28f1-128">The compiler inserts a standard application name MyApplication.app into the manifest XML.</span></span> <span data-ttu-id="c28f1-129">这是一种使应用程序可以在 Windows Server 2003 Service Pack 3 上运行的解决方法。</span><span class="sxs-lookup"><span data-stu-id="c28f1-129">This is a workaround to enable applications to run on Windows Server 2003 Service Pack 3.</span></span>  
  
```xml

<assembly xmlns="urn:schemas-microsoft-com:asm.v1" manifestVersion="1.0">  
  <assemblyIdentity version="1.0.0.0" name="MyApplication.app"/>  
  <trustInfo xmlns="urn:schemas-microsoft-com:asm.v2">  
    <security>  
      <requestedPrivileges xmlns="urn:schemas-microsoft-com:asm.v3">  
        <requestedExecutionLevel level="asInvoker"/>  
      </requestedPrivileges>  
    </security>  
  </trustInfo>  
</assembly>  
```
