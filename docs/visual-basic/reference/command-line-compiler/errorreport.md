# <a name="-errorreport"></a><span data-ttu-id="21561-102">-errorreport</span><span class="sxs-lookup"><span data-stu-id="21561-102">-errorreport</span></span>

<span data-ttu-id="21561-103">指定 Visual Basic 编译器应报告内部编译器错误的方式。</span><span class="sxs-lookup"><span data-stu-id="21561-103">Specifies how the Visual Basic compiler should report internal compiler errors.</span></span>

## <a name="syntax"></a><span data-ttu-id="21561-104">语法</span><span class="sxs-lookup"><span data-stu-id="21561-104">Syntax</span></span>

```
-errorreport:{ prompt | queue | send | none }
```
