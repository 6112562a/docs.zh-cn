- <span data-ttu-id="fb148-124">指针算术包含固定大小。</span><span class="sxs-lookup"><span data-stu-id="fb148-124">Pointer arithmetic that includes constant sizes.</span></span>

- <span data-ttu-id="fb148-131">如果你在选项上设置 `x86` 标志且之后应用程序在 64 位计算机上运行，则应用程序将在 WOW 子系统中运行而不是在本机上运行。</span><span class="sxs-lookup"><span data-stu-id="fb148-131">If you set the `x86` flag on the option and the application is subsequently run on a 64-bit machine, the application will run in the WOW subsystem instead of running natively.</span></span>  
  
 <span data-ttu-id="fb148-132">在 64 位 Windows 操作系统上：</span><span class="sxs-lookup"><span data-stu-id="fb148-132">On a 64-bit Windows operating system:</span></span>
