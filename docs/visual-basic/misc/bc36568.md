# <a name="elementname-is-not-a-method-parameter"></a>\<elementname > 不是方法参数
过程调用提供了一个方法未定义的额外参数。  
  
 **错误 ID:** BC36568  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从参数列表中删除多余的参数。
