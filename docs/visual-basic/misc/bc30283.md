# <a name="sub-new-cannot-be-declared-overrides"></a>“Sub New”不能声明为“Overrides”
构造函数显示它重写继承构造函数。 不能重写构造函数。  
  
 **错误 ID:** BC30283  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从 `Overrides` 声明中删除 `Sub` 关键字。
