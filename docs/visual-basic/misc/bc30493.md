# <a name="constructor-must-be-declared-as-a-sub-not-as-a-function"></a>构造函数必须声明为 Sub，而不能声明为 Function。
你已尝试声明 `Function New`。 构造函数必须声明为 `Sub New`。  
  
 **错误 ID:** BC30493  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 使用 `Sub` 而非 `Function`。
