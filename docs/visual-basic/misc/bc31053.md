<span data-ttu-id="3453a-103">在无效位置遇到 `Implements` 语句。</span><span class="sxs-lookup"><span data-stu-id="3453a-103">An `Implements` statement was encountered in an invalid location.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="3453a-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="3453a-105">To correct this error</span></span>

    ```  
    End Class  
    ```
