# <a name="cannot-convert-start-value-of-type-lttype1gt-and-step-value-of-type-lttype2gt-to-a-common-type"></a>无法转换类型的起始值&lt;type1&gt;和类型的步长值&lt;type2&gt;为通用类型
指定的 `For...Next` 控制值具有不同的类型，因此无效，其原因是它们不能转换为通用类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  更改一个或多个值的类型，以便所有值都可以转换为通用类型。
