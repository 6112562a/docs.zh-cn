# <a name="cannot-delete-a-registry-hive"></a>不能删除注册表配置单元
尝试删除注册表配置单元。 配置单元是顶级注册表项（例如 `HKEY_CURRENT_USER` 和 `HKEY_LOCAL_MACHINE`），无法删除。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查你尝试删除的注册表项，并确保提供正确的路径。
