# <a name="return-statements-are-not-valid-in-the-immediate-window"></a>“Return”语句在“即时”窗口中无效
`Return` 语句执行分支，但不允许用于调试上下文。  
  
 **错误 ID:** BC30147  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   不要在“即时” `Return`**窗口中发出** 语句。
