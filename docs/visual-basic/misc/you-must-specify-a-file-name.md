# <a name="you-must-specify-a-file-name"></a>必须指定文件名
你试图执行的操作（如移动或复制文件）需要提供文件名。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 提供必要的文件名。  
  
- 请确保该路径不以目录分隔符字符结尾 (\\)。
