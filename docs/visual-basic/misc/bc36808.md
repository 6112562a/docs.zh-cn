# <a name="xml-attributes-cannot-be-selected-from-type-type"></a><span data-ttu-id="34fd1-102">无法从类型“type”选择 XML 特性</span><span class="sxs-lookup"><span data-stu-id="34fd1-102">XML attributes cannot be selected from type 'type'</span></span>
<span data-ttu-id="34fd1-103">已针对不属于类型 <xref:System.Xml.Linq.XElement> or `IEnumerable(Of XElement)`的对象引用了一个 XML 特性。</span><span class="sxs-lookup"><span data-stu-id="34fd1-103">An XML attribute has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement> or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="34fd1-104">有关更多信息，请参见 [XML Attribute Axis Property](../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)。</span><span class="sxs-lookup"><span data-stu-id="34fd1-104">For more information, see [XML Attribute Axis Property](../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md).</span></span>  
  
```vb  
' Generates an error.  
Dim var = "sample text".@attr  
```  
  
 <span data-ttu-id="34fd1-105">**错误 ID:** BC36808</span><span class="sxs-lookup"><span data-stu-id="34fd1-105">**Error ID:** BC36808</span></span>
