# <a name="xml-attributes-cannot-be-selected-from-type-type"></a><span data-ttu-id="52043-102">无法从类型“type”选择 XML 特性</span><span class="sxs-lookup"><span data-stu-id="52043-102">XML attributes cannot be selected from type 'type'</span></span>

```vb

```  
  
 <span data-ttu-id="52043-105">**错误 ID:** BC36808</span><span class="sxs-lookup"><span data-stu-id="52043-105">**Error ID:** BC36808</span></span>
