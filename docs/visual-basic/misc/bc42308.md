# <a name="xml-comment-parameter-must-have-a-name-attribute"></a>XML 注释参数必须具有 "name" 属性
XML 注释参数必须具有 "name" 属性。 将忽略 XML 注释。  
  
 XML 标记缺少所需的 `name` 特性。  
  
 **错误 ID:** BC42308  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 为该标记提供所需的 `name` 特性。
