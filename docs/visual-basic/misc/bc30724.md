# <a name="error-statements-are-not-valid-in-the-immediate-window"></a>“Error”语句在即时窗口中无效
`Error` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30724  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从调试代码中删除 `Error` 语句。
