# <a name="argument-per-is-not-valid"></a>参数“Per”无效。
为 `Per`提供的参数无效，该参数必须是一个 `Double` ，用于指定从 1 到 `NPer`范围内的付款周期。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保参数在有效范围内。
