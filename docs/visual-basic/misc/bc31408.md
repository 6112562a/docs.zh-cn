# <a name="ltspecifier1gt-and-ltspecifier2gt-cannot-be-combined"></a>'&lt;specifier1&gt;和&lt;specifier2&gt;不能组合
同时使用了两个不兼容的说明符（如 `Shadows` 和 `Overrides` ）。  
  
 **错误 ID:** BC31408  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除其中一个说明符。
