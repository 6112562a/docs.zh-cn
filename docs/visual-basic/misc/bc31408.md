# <a name="specifier1-and-specifier2-cannot-be-combined"></a>'\<specifier1 > 和\<specifier2 > 不能组合
同时使用了两个不兼容的说明符（如 `Shadows` 和 `Overrides` ）。  
  
 **错误 ID:** BC31408  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除其中一个说明符。
