# <a name="end-tag-tagname-expected"></a>结束标记\</tagName > 预期
XML 元素文本中的结束标记与对应的开始标记不匹配。  
  
 **错误 ID:** BC31150  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将 XML 结束标记修改为与该 XML 元素文本的 XML 开始标记匹配。
