# <a name="implicit-variable-variablename-is-invalid-because-of-message"></a>隐式变量\<variablename > 是无效的由于\<消息 >'
调试过程中生成了无效的隐式变量。  
  
 **错误 ID:** BC36016  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查引用的消息以寻找原因和可能的纠正方法。  
  
2. 如果仍然出现错误，则收集有关该情况的信息并通知 Microsoft 产品支持服务。
