# <a name="disk-not-ready"></a>磁盘未准备好
对于指定的驱动器，没有磁盘或者驱动器门处于打开状态。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查指定的驱动器，必要时插入磁盘。  
  
2.  关闭驱动器门。
