# <a name="interfacename1-cannot-implement-methodname-because-there-is-no-matching-method-on-interface-interfacename2"></a>"\<1 >" 无法实现 "\<>", 因为接口 "\<i \<>" 上没有匹配的方法 >
当第一个接口继承的接口上没有匹配的方法时，你已尝试在该接口上实现方法。  
  
 **错误 ID:** BC30401  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 请确保你没有键错方法的名称或提供缺少的方法。
