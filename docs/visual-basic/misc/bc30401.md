# <a name="ltinterfacename1gt-cannot-implement-ltmethodnamegt-because-there-is-no-matching-ltmethodgt-on-interface-ltinterfacename2gt"></a>'&lt;interfacename1&gt;不能实现&lt;methodname&gt;因为不存在匹配&lt;方法&gt;接口上'&lt;interfacename2&gt;
当第一个接口继承的接口上没有匹配的方法时，你已尝试在该接口上实现方法。  
  
 **错误 ID:** BC30401  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保你没有键错方法的名称或提供缺少的方法。
