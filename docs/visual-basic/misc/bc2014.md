# <a name="the-value-value-is-invalid-for-option-optionname"></a>值\<值 > 是无效的选项\<选项名称 >
为无效命令行选项提供了值。  
  
 **错误 ID:** BC2014  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查值，以找出其中的无效值。
