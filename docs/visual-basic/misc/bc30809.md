# <a name="wend-statements-are-no-longer-supported-use-end-while-statements-instead"></a>不再支持“Wend”语句；请改用“End While”语句
`End While` 构建可替换 `Wend` 关键字。  
  
 **错误 ID:** BC30809  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `End While` 结束 `While` 块。
