# <a name="xml-processing-instruction-name-name-is-not-valid"></a>XML 处理指令名“name”无效
为 XML 处理指令提供的名称无效。  
  
 **错误 ID:** BC31174  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更正 XML 处理指令名中的任何错误或将其删除。
