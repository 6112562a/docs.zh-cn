# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="11534-102">“Implements”子句中不允许类型参数</span><span class="sxs-lookup"><span data-stu-id="11534-102">Type parameter not allowed in 'Implements' clause</span></span>
<span data-ttu-id="11534-103">泛型类型中的 `Implements` 子句将类型形参指定为要实现的成员。</span><span class="sxs-lookup"><span data-stu-id="11534-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="11534-104">`Implements` 子句必须指定一个接口和一个成员。</span><span class="sxs-lookup"><span data-stu-id="11534-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="11534-105">它可以将类型形参传递给该接口，但不能将其传递给该成员，也不能将其用作该成员的名称。</span><span class="sxs-lookup"><span data-stu-id="11534-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="11534-106">以下语句可能会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="11534-106">The following statements can generate this error.</span></span>  
  
```  
Class c1(Of t)  
    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 <span data-ttu-id="11534-107">**错误 ID:** BC32056</span><span class="sxs-lookup"><span data-stu-id="11534-107">**Error ID:** BC32056</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="11534-108">更正此错误</span><span class="sxs-lookup"><span data-stu-id="11534-108">To correct this error</span></span>  
  
-   <span data-ttu-id="11534-109">在 `Implements` 关键字后指定接口的接口名称和真实成员。</span><span class="sxs-lookup"><span data-stu-id="11534-109">Specify the interface name and a genuine member of the interface following the `Implements` keyword.</span></span> <span data-ttu-id="11534-110">可以根据需要将类型形参传递给接口。</span><span class="sxs-lookup"><span data-stu-id="11534-110">You can pass the type parameter to the interface if appropriate.</span></span>  
  
    ```  
    Public Sub doSomething() Implements i1(Of t).doSomething  
    ```
