# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="f5f6c-102">“Implements”子句中不允许类型形参</span><span class="sxs-lookup"><span data-stu-id="f5f6c-102">Type parameter not allowed in 'Implements' clause</span></span>
<span data-ttu-id="f5f6c-103">泛型类型中的 `Implements` 子句将类型形参指定为要实现的成员。</span><span class="sxs-lookup"><span data-stu-id="f5f6c-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="f5f6c-104">`Implements` 子句必须指定一个接口和一个成员。</span><span class="sxs-lookup"><span data-stu-id="f5f6c-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="f5f6c-105">它可以将类型形参传递给该接口，但不能将其传递给该成员，也不能将其用作该成员的名称。</span><span class="sxs-lookup"><span data-stu-id="f5f6c-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="f5f6c-106">以下语句可能会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="f5f6c-106">The following statements can generate this error.</span></span>  
  
```vb  
Class c1(Of t)  
    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 <span data-ttu-id="f5f6c-107">**错误 ID：** BC32056</span><span class="sxs-lookup"><span data-stu-id="f5f6c-107">**Error ID:** BC32056</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f5f6c-108">更正此错误</span><span class="sxs-lookup"><span data-stu-id="f5f6c-108">To correct this error</span></span>

    ```vb

    ```
