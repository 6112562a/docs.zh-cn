# <a name="byref-parameter-ltparameternamegt-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="2c33a-102">ByRef 参数&lt;parametername&gt;不能在 lambda 表达式中使用</span><span class="sxs-lookup"><span data-stu-id="2c33a-102">'ByRef' parameter '&lt;parametername&gt;' cannot be used in a lambda expression</span></span>
<span data-ttu-id="2c33a-103">在 `Sub` 或函数中声明的 lambda 表达式不可使用该 `ByRef` 或函数的任何 `Sub` 参数。</span><span class="sxs-lookup"><span data-stu-id="2c33a-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="2c33a-104">例如，下面的代码会导致此错误，因为在 lambda 表达式中使用了 `ByRef` 参数 `n` 。</span><span class="sxs-lookup"><span data-stu-id="2c33a-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="2c33a-105">**错误 ID:** BC36639</span><span class="sxs-lookup"><span data-stu-id="2c33a-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2c33a-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="2c33a-106">To correct this error</span></span>  
  
-   <span data-ttu-id="2c33a-107">将 `ByRef` 参数分配到局部变量，并在 lambda 表达式中使用该局部变量，如以下代码所示：</span><span class="sxs-lookup"><span data-stu-id="2c33a-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
