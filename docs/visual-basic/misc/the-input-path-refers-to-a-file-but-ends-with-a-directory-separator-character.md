# <a name="the-input-path-refers-to-a-file-but-ends-with-a-directory-separator-character"></a>输入的路径引用了一个文件，但是以目录分隔符字符结尾
尝试的操作要求提供文件名，但提供的名称以目录分隔符 (`\`) 结尾。 这可能是由于不正确解析文件路径造成的。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查提供的文件路径以确保它是正确的。  
  
- 删除冗余的字符。
