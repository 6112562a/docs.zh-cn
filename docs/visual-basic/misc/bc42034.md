# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>已使用另一个 GUID 和校验和值声明了文件名
已尝试声明一个文件，但此文件已存在并具有不同的 GUID 和校验值。  
  
 **错误 ID:** BC42034  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查文件名和路径，以确保两者均正确无误。  
  
-   删除不再需要的冗余文件。
