# <a name="ltnamegt-necessary-for-compiling-this-construct-cannot-be-found"></a>'&lt;名称&gt;，无法找到编译此结构所需的
无法找到编译必需的元素。  
  
 **错误 ID:** BC30930  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  提供必要的帮助程序元素。  
  
2.  确保代码是有效的托管代码。
