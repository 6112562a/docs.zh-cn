# <a name="ltproperty1gt-and-ltproperty2gt-cannot-overload-each-other-because-only-one-is-declared-default"></a>'&lt;property1&gt;和&lt;property2&gt;无法重载对方，因为只有一个声明为 Default
如果某个属性指定 `Default`，则在该名称上重载的所有属性也必须指定 `Default`。  
  
 **错误 ID:** BC30361  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保所有重载的属性均声明为 `Default`。
