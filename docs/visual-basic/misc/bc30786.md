# <a name="declare-statements-in-a-module-cannot-be-declared-ltspecifiergt"></a>声明语句在模块中的不能声明为&lt;说明符&gt;
`Declare` 声明具有在 `Module` 声明中无效的说明符。 模块永远不能实例化，不支持继承且不能实现接口。  
  
 **错误 ID:** BC30786  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
