# <a name="type-of-variablename-cannot-be-inferred-from-an-expression-containing-variablename"></a><span data-ttu-id="96649-102">类型\<variablename > 不能从一个表达式，其中包含推断出\<变量名 >'</span><span class="sxs-lookup"><span data-stu-id="96649-102">Type of '\<variablename>' cannot be inferred from an expression containing '\<variablename>'</span></span>
<span data-ttu-id="96649-103">如果在声明中使用变量建立其初始值，则编译器无法推断该变量的数据类型。</span><span class="sxs-lookup"><span data-stu-id="96649-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="96649-104">例如，在 `Option Infer` 设置为 `On`的情况下，下面的示例不编译：</span><span class="sxs-lookup"><span data-stu-id="96649-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
- <span data-ttu-id="96649-105">声明</span><span class="sxs-lookup"><span data-stu-id="96649-105">Declarations</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
- <span data-ttu-id="96649-106">`For` 循环</span><span class="sxs-lookup"><span data-stu-id="96649-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
- <span data-ttu-id="96649-107">`For Each` 循环</span><span class="sxs-lookup"><span data-stu-id="96649-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="96649-108">**错误 ID:** BC30980</span><span class="sxs-lookup"><span data-stu-id="96649-108">**Error ID:** BC30980</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="96649-109">更正此错误</span><span class="sxs-lookup"><span data-stu-id="96649-109">To correct this error</span></span>  
  
- <span data-ttu-id="96649-110">如果两个变量用于指代不同的值，则更改正在声明的变量的名称。</span><span class="sxs-lookup"><span data-stu-id="96649-110">If the two variables were intended to refer to different values, change the name of the variable that you are declaring.</span></span>
