# <a name="array-exceeds-the-limit-of-32-dimensions"></a>数组超过了 32 维数限制。
数组的维数不能超过 32。  
  
 **错误 ID:** BC30052  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 减少数组声明中的维数。
