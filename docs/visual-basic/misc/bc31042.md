# <a name="sub-new-cannot-implement-interface-members"></a>“Sub New”无法实现接口成员
`Sub New` 是构造函数，它不能实现成员。  
  
 **错误 ID:** BC31042  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从 `Implements` 过程删除 `Sub New` 语句。
