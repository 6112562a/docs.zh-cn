# <a name="next-expected"></a>应为“Next”
出现 `On Error Resume` 构造而没有结尾的 `Next`。  
  
 **错误 ID:** BC30003  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `Next` 关键字添加到 `On Error` 语句的末尾。
