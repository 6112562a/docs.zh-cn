## <a name="to-correct-this-error"></a><span data-ttu-id="b3039-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="b3039-107">To correct this error</span></span>  
  
1. <span data-ttu-id="b3039-108">使用 <xref:System.DBNull> 而非 `Null`。</span><span class="sxs-lookup"><span data-stu-id="b3039-108">Use <xref:System.DBNull> instead of `Null`.</span></span> <span data-ttu-id="b3039-109">下面的示例演示这一操作。</span><span class="sxs-lookup"><span data-stu-id="b3039-109">The following example demonstrates this.</span></span>  
  
    ```vb  
    Sub TestDBNull()  
        Dim t As DataTable  
        ' Assume the DataGrid is bound to a DataTable.  
        t = CType(DataGrid1.DataSource, DataTable)  
        Dim r As DataRow  
        r = t.Rows(datagrid1.CurrentCell.RowNumber)  
        r.BeginEdit  
        r(1) = System.DBNull.Value ' Assign DBNull to the record.  
        r.EndEdit  
        r.AcceptChanges  
        If r.IsNull(1) Then  
            MsgBox("")  
        End If  
    End Sub  
    ```

    ```vb  
        If (cls Is Nothing) Then  
            MsgBox("cls is Nothing")  
        End If  
        cls = Nothing ' Assign Nothing to the class variable cls.  
    End Sub  
    ```
