# <a name="reference-required-to-module-ltmodulenamegt-containing-the-definition-for-event-lteventnamegt"></a>需要对模块引用&lt;modulename&gt;包含事件的定义&lt;eventname&gt;
需要对模块“<`modulename`>”（包含事件“<`eventname`>”的定义）的引用。 请向项目中添加一个。  
  
 事件在项目不直接引用的模块中定义。 Visual Basic 编译器需要引用以避免多义性的多个模块中定义该事件。  
  
 **错误 ID:** BC30006  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将未引用模块的名称包括在项目引用中。
