# <a name="inherits-statement-must-precede-all-declarations-in-a-class"></a>“Inherits”语句必须位于类中的所有声明之前
`Inherits` 语句必须位于类定义的开头。  
  
 **错误 ID:** BC30683  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `Inherits` 语句移动到类的顶部。
