# <a name="key-cannot-be-deleted-because-it-has-subkeys"></a>无法删除项，因为它具有子项
无法删除该项，因为它具有子项。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 在尝试此操作之前，请确定并删除指定项的子项。
