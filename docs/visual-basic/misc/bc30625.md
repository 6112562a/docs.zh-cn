# <a name="module-statement-must-end-with-a-matching-end-module"></a>“Module”语句必须以匹配的“End Module”结束
Module 块必须以 `End Module` 语句结束。  
  
 **错误 ID:** BC30625  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将 `End Module` 语句添加到 `Module` 块末尾。
