# <a name="managed-classes-derived-from-a-com-class-cannot-be-called-late-bound"></a>不能以后期绑定方式调用从 COM 类派生的托管类。

尝试对从 COM 类派生的托管类进行后期绑定调用，但不支持这种调用。

## <a name="to-correct-the-problem"></a>更正此问题

进行早期绑定调用。
