# <a name="operator-is-not-valid-for-type-lttypenamegt"></a>运算符不是有效的类型&lt;typename&gt;
运算符不适合于类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保该运算符适合于正运算的类型。
