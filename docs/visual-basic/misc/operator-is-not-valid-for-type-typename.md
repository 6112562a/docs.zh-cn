# <a name="operator-is-not-valid-for-type-typename"></a>运算符不是有效类型为\<类型名称 >
运算符不适合于类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 确保该运算符适合于正运算的类型。
