<span data-ttu-id="ad584-103">已在扩展方法的调用中指定类型实参，该扩展方法没有泛型形参或没有尚未指定类型的泛型形参。</span><span class="sxs-lookup"><span data-stu-id="ad584-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="ad584-104">例如，下面的代码会导致此错误。</span><span class="sxs-lookup"><span data-stu-id="ad584-104">For example, the following code causes this error.</span></span>
