# <a name="property-ltpropertynamegt-cannot-be-set-to-nothing"></a>属性&lt;propertyname&gt;不能设置为 Nothing
该属性要求除 `Nothing`以外的值。 如果计算值时有误，则可能会出现此结果。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   提供一个有效值。
