# <a name="file-io-of-a-structure-with-field-fieldname-of-type-typename-is-not-valid"></a>文件 I/O 具有字段的结构\<fieldname > 类型的\<类型名称 > 无效
试图向文件写入的结构包含不支持的类型。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查结构来确定不支持的类型出现的位置，然后将其更改为有效类型。
