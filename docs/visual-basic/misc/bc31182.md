# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>特性值无效；应为“value1”或“value2”
为 XML 文档文本中的特性指定了无效的值。 XML 文档文本的有效特性值基于 XML 1.0 规范。  
  
 **错误 ID:** BC31182  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将特性值更改为符合 XML 1.0 规范的有效值。
