# <a name="ltpropertyname1gt-cannot-override-ltpropertyname2gt-because-they-differ-by-readonly-or-writeonly"></a>'&lt;propertyname1&gt;不能重写&lt;propertyname2&gt;因为它们的差别在于 ReadOnly 还是 WriteOnly
已尝试用仅与第一个属性在 `ReadOnly` 或 `WriteOnly`上不同的第二个属性重写属性。  
  
 **错误 ID:** BC30362  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保属性不仅由 `ReadOnly` 和 `WriteOnly`区分。
