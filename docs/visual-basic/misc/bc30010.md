# <a name="reference-required-to-module-ltmodulenamegt-containing-the-implemented-interface-ltinterfacenamegt"></a>需要对模块引用&lt;modulename&gt;包含实现的接口&lt;interfacename&gt;
需要对模块引用\<模块名称 > 包含实现的接口\<interfacename >。 请向项目中添加一个。  
  
 在你的项目中未直接引用的模块中定义接口。 Visual Basic 编译器需要一个引用以避免多义性的接口定义多个模块中。  
  
 **错误 ID:** BC30010  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将未引用模块的名称包括在项目引用中。
