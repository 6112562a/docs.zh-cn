# <a name="targetfilepath-specifies-an-existing-folder"></a>TargetFilePath 指定现有文件夹
参数 `TargetFilePath` 指定现有的目录，但必须指定新目录的名称。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改 `TargetFilePath` 以指定新的而不是现有的目录。
