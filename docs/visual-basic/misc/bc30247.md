# <a name="modifier-is-not-valid-on-a-local-variable-declaration"></a>\<修饰符 > 在局部变量声明上无效
过程内的 `Dim` 语句包含无效的关键字，如 `Overloads`。  
  
 **错误 ID:** BC30247  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从 `Dim` 语句中删除无效的关键字。
