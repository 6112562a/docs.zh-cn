# <a name="expected-closing--for-embedded-expression"></a>嵌入表达式需要有结束的“%>”
XML 文本中的嵌入表达式缺少必需的结束字符 `%>` 。  
  
 **错误 ID:** BC31159  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   向嵌入表达式末尾添加必需的结束字符 `%>` 。
