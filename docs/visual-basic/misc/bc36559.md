<span data-ttu-id="bcf8e-103">直到建立匿名类型属性的类型，它才可用于建立其他属性的类型。</span><span class="sxs-lookup"><span data-stu-id="bcf8e-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="bcf8e-104">例如，在下面的声明中， `.IDName = .LastName` 无效，因为尚未初始化 `.LastName` 。</span><span class="sxs-lookup"><span data-stu-id="bcf8e-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>  
  
```  
' Not valid.   
' Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}   
```
