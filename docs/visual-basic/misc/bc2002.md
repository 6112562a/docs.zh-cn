# <a name="source-file-filename-specified-multiple-times"></a>源文件\<文件名 > 指定多次
多次指定某文件。  
  
 **错误 ID:** BC2002  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除多余的文件说明符。
