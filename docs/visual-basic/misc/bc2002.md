# <a name="source-file-ltfilenamegt-specified-multiple-times"></a>源文件&lt;文件名&gt;指定多次
多次指定某文件。  
  
 **错误 ID:** BC2002  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的文件说明符。
