# <a name="methods-cannot-be-declared-static"></a>方法不能声明为“Static”
`Static` 修饰符不能应用于整个过程。  
  
 **错误 ID:** BC30810  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 使用 `Static` 修饰符将过程中的各个项声明为静态项。
