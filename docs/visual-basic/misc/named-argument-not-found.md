# <a name="named-argument-not-found-visual-basic"></a>未找到命名参数 (Visual Basic)
除非命名参数出现在过程定义中，否则不能在过程调用中使用命名参数。 你指定了一个命名参数，但未将过程未定义为接受该名称的参数。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查参数名称的拼写。
