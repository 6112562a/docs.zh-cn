# <a name="type1-typename-and-type2-typename-conflict-in-namespace-namespacename"></a>\<类型 1 >\<类型名称 > 和\<y p e 2 >\<typename >' 中相冲突\<命名空间 >\<命名空间名称 >
已声明的两个元素（如类和模块）在同一命名空间中具有相同的名称。  
  
 **错误 ID:** BC30179  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将其中一个元素的声明移到其他命名空间。  
  
- 更改其中一个元素的名称。
