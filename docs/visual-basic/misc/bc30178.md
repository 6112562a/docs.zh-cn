# <a name="specifier-is-duplicated"></a>说明符重复
某一关键字多次出现同一声明中。  
  
 **错误 ID:** BC30178  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从声明中删除重复出现的关键字。
