# <a name="conflicts-with-ltname1gt-which-is-implicitly-created-for-ltname2gt-in-lttypegt-ltname3gt"></a>与冲突&lt;name1&gt;，这隐式创建&lt;name2&gt;中&lt;类型&gt;'&lt;name3&gt;
类型成员的名称与另一个隐式创建的成员相冲突。  
  
 **错误 ID:** BC31058  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重命名显式声明的成员，以消除命名冲突。
