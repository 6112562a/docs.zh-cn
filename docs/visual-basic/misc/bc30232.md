# <a name="implemented-type-must-be-an-interface"></a>实现的类型必须是接口
`Implements` 语句不指定接口。 类只能实现接口。  
  
 **错误 ID:** BC30232  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保接口名称拼写正确。  
  
2. 如果该语句指定一个类，请考虑使用 `Inherits` 语句。
