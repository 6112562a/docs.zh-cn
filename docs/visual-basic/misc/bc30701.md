# <a name="nothing-cannot-be-evaluated"></a>无法计算“Nothing”的值
在调试窗口中使用了 `Nothing` 关键字。  
  
 **错误 ID:** BC30701  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 请勿在调试期间使用 `Nothing` 。
