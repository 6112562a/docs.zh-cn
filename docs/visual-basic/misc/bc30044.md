# <a name="ltkeywordgt-is-not-valid-within-a-structure"></a>'&lt;关键字&gt;在结构中无效
结构是值类型，而不是引用类型。 结构不是从类创建的实例，因此 `Me`、 `MyClass`和 `MyBase` 关键字在结构中没有意义。  
  
 **错误 ID:** BC30044  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将结构更改为类，或从过程中删除关键字。
