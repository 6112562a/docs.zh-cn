# <a name="typename-is-a-class-type-and-cannot-be-used-as-an-expression"></a>\<类型名称 > 是类类型，不能用作表达式
类型名称出现在要求表达式的位置。 表达式必须包含变量、常量、文本、属性和 `Function` 过程调用的组合。  
  
 **错误 ID:** BC30109  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除类型名称，并使用有效的元素构造表达式。
