# <a name="project-level-conditional-compilation-constant-is-not-valid-error"></a>项目级条件编译常量无效：\<错误 >
为项目指定了无效的条件编译选项。  
  
 **错误 ID:** BC31031  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 验证与此项目结合使用的条件编译常量。
