# <a name="no-accessible-overloaded-ltmethodnamegt-can-be-called-with-these-arguments-without-a-widening-conversion-ltlistgt"></a>没有可访问重载&lt;methodname&gt;可以使用这些自变量进行扩大转换名为：&lt;列表&gt;
调用了重载方法，但是方法不可与未进行扩大转换的所提供的参数列表相匹配。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   指定 `Option Strict Off`。  
  
-   更改参数以匹配重载方法的其中一个签名。
