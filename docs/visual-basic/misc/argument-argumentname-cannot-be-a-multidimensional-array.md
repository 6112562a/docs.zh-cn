# <a name="argument-ltargumentnamegt-cannot-be-a-multidimensional-array"></a>参数&lt;argumentname&gt;不能为多维数组
数组参数的秩大于 1，但它只能是一维数组。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将参数秩更改为 1。
