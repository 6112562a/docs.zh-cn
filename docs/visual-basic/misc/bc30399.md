# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>"MyBase" 不能与方法 "\<方法名称 >" 一起使用, 因为它已声明为 "MustOverride"
试图在声明为 `MyBase` 的方法中使用 `MustOverride`。  
  
 **错误 ID:** BC30399  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除 `MustOverride` 声明。
