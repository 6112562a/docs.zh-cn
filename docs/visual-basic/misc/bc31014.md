# <a name="unable-to-find-entry-point-ltnamegt-in-dll-ltfilenamegt-lterrorgt"></a>找不到入口点 '&lt;名称&gt;在 DLL&lt;filename&gt;:&lt;错误&gt;
未能在指定的 DLL 中找到过程名称。  
  
 **错误 ID:** BC31014  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证过程名称是否拼写正确，以及大小写是否与 DLL 中函数的大小写匹配。
