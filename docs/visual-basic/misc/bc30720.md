# <a name="on-error-statements-are-not-valid-in-the-immediate-window"></a>“On Error”语句在即时窗口中无效
`On Error` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30720  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从调试代码中删除 `On Error` 语句。
