# <a name="badly-formed-xml-file-commentfile-cannot-be-included"></a>XML 的格式不正确。 文件\<commentfile > 不能包含
一个 XML 注释中的 `<include>` 标记引用不存在的文件或包含格式不正确的 XML 的文件。  
  
 **错误 ID:** BC42321  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证文件是否存在。  
  
2.  验证该文件中的 XML 是否有效。
