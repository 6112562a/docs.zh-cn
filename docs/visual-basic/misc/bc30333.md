# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-lttype3gt-is-not-a-reference-type"></a>类型的值&lt;type1&gt;不能转换为&lt;type2&gt;由于'&lt;type3&gt;不是引用类型
语句试图将一个数组类型转换为另一个数组类型，但目标数组元素的数据类型不是引用类型。  
  
 **错误 ID:** BC30333  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查两个数组的元素数据类型以确定错误源。
