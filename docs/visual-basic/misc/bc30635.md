# <a name="readonly-and-writeonly-cannot-be-combined"></a>"ReadOnly" 不能与 "WriteOnly" 组合
`ReadOnly` 和 `WriteOnly` 修饰符互相排斥。  
  
 **错误 ID:** BC30635  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   指定 `ReadOnly` 或 `WriteOnly`，但不能同时指定这两者。
