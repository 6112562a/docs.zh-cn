# <a name="endif-statements-are-no-longer-supported-use-end-if-instead"></a>不再支持“EndIf”语句；请改用“End If”
不再支持 `EndIf` 语句。  
  
 **错误 ID:** BC30826  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 使用 `End If` 终止 `If` 语句。
