## <a name="example"></a><span data-ttu-id="91a37-110">示例</span><span class="sxs-lookup"><span data-stu-id="91a37-110">Example</span></span>  
 <span data-ttu-id="91a37-111">以下示例中的第一个参数指示 `Print` 方法扩展 `String` 数据类型。</span><span class="sxs-lookup"><span data-stu-id="91a37-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>

    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="91a37-112">当如下调用扩展方法时，该方法中的参数 `str` 绑定到 `greeting`，它是调用 `String` 的 `Print`的实例。</span><span class="sxs-lookup"><span data-stu-id="91a37-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="91a37-113">编译器将使用 `greeting` 作为扩展方法 `Print`的参数。</span><span class="sxs-lookup"><span data-stu-id="91a37-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

```
