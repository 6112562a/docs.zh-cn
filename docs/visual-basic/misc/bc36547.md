# <a name="anonymous-type-member-or-property-ltpropertynamegt-is-already-declared"></a><span data-ttu-id="0b9c3-102">匿名类型成员或属性&lt;propertyname&gt;已声明</span><span class="sxs-lookup"><span data-stu-id="0b9c3-102">Anonymous type member or property '&lt;propertyname&gt;' is already declared</span></span>
<span data-ttu-id="0b9c3-103">属性名在匿名类型的声明中只能建立一次。</span><span class="sxs-lookup"><span data-stu-id="0b9c3-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="0b9c3-104">例如，下面的声明无效：</span><span class="sxs-lookup"><span data-stu-id="0b9c3-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="0b9c3-105">**错误 ID:** BC36547</span><span class="sxs-lookup"><span data-stu-id="0b9c3-105">**Error ID:** BC36547</span></span>
