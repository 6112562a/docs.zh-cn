# <a name="public-member-ltmembernamegt-on-type-lttypenamegt-not-found"></a>公共成员&lt;membername&gt;类型上&lt;typename&gt;找不到
找不到你尝试访问的成员。 访问后期绑定对象的非公共成员时，可能出现此错误。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  如果该成员存在，请确保它被声明为 `Public`。
