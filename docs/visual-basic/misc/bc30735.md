# <a name="type-in-a-module-cannot-be-declared-specifier"></a>不能声明为模块中的类型\<说明符 >
在模块中声明了具有无效修饰符的类型。  
  
 **错误 ID:** BC30735  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 不要对模块中声明的类型使用 `Protected` 或 `Protected Friend` 修饰符。
