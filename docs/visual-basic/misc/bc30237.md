# <a name="parameter-already-declared-with-name-ltargumentnamegt"></a>已用名称声明的参数&lt;argumentname&gt;
过程声明定义了具有相同名称的两个参数。 过程声明中的所有参数名称必须都是唯一的。  
  
 **错误 ID:** BC30237  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改其中一个参数的名称。
