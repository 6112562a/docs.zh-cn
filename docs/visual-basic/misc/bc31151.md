# <a name="element-is-missing-an-end-tag"></a>元素缺少结束标记
XML 元素文本缺少 XML 结束标记或空标记标识符“/”（例如 `<tagName />`）。  
  
 **错误 ID:** BC31151  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 添加 XML 标记的结束标记，或向 XML 标记添加空标记标识符“/”。
