# <a name="ltmembernamegt-is-already-declared-by-ltprocedurenamegt-which-was-generated-for-this-ltcontainernamegt"></a>'&lt;membername&gt;已通过声明'&lt;过程名称&gt;，其生成的此&lt;containername&gt;
所声明的某个成员与由公共语言运行时声明的某个成员同名。  
  
 **错误 ID:** BC30733  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改成员名称以消除冲突。
