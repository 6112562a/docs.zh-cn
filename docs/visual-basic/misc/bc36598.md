# <a name="instance-of-restricted-type-lttypenamegt-cannot-be-used-in-a-query-expression"></a>受限制类型的实例&lt;typename&gt;不能在查询表达式中使用
不能在 lambda 表达式或查询表达式中使用受限类型的实例，例如 <xref:System.ArgIterator>、 <xref:System.RuntimeArgumentHandle>和 <xref:System.TypedReference>。  
  
 **错误 ID:** BC36598
