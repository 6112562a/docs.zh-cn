# <a name="set-parameter-cannot-be-declared-ltspecifiergt"></a>不能声明为 set 参数&lt;说明符&gt;
在 `Set` 属性过程使用参数指定了无效的修饰符。  
  
 **错误 ID:** BC31065  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除无效的修饰符，如 `ByRef`、 `ParamArray`和 `Optional`。
