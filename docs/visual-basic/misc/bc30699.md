# <a name="membername-is-not-declared-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>"\<成员名称 >" 未声明或包含它的模块未在调试会话中加载
找不到指定的成员。  
  
 **错误 ID：** BC30699  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 验证是否定义了成员并且拼写正确。  
  
2. 访问在该模块中声明的成员之一。 在某些情况下，调试环境找不到成员，因为尚未加载在其中声明成员的模块。
