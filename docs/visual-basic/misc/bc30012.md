# <a name="if-block-must-end-with-a-matching-end-if"></a>“#If” 块必须以匹配的 “#End If” 结束
`#If` 是条件编译指令。 `#If` 指令未终止 `#End If` 块。  
  
 **错误 ID:** BC30012  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 添加 `#End If` 指令至条件编译块的末尾。
