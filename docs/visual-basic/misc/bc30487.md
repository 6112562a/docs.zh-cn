# <a name="operator-ltoperatornamegt-is-not-defined-for-type-lttypenamegt"></a>运算符&lt;运算符名称&gt;没有为类型定义'&lt;typename&gt;
尝试对某个类型（如`+`）使用没有为其定义的一元运算符（ `-`、 `Not`或 `String`）。  
  
 **错误 ID:** BC30487  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将一元运算符替换为适用于该表达式的相应运算符。
