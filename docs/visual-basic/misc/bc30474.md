# <a name="method1-cannot-override-method2-because-it-is-a-declare-statement"></a>'\<method1 > 不能重写\<method2 > 是 Declare 语句
你试图重写用 `Declare` 语句声明的基类名称上的委托。  
  
 **错误 ID:** BC30474  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 更改重写的成员，使其不再是 `Declare` 语句。  
  
2. 请勿尝试重写此方法。
