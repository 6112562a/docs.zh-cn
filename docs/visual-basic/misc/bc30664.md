# <a name="conversions-from-lttypename1gt-to-lttypename2gt-must-be-explicit"></a>从转换&lt;typename1&gt;到&lt;typename2&gt;必须为显式
无法隐式实现此转换。  
  
 **错误 ID:** BC30664  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `CType`或某个特定转换函数，如 `CInt` 或 `CDec`。
