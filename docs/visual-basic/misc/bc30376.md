# <a name="new-cannot-be-used-on-class-ltclassnamegt-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>New 不能在类上使用&lt;classname&gt;因为它包含尚未重写的 MustOverride 成员
已在包含未被重写的 `New` 成员的类上尝试使用 `MustOverride` 。  
  
 **错误 ID:** BC30376  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除 `New` 语句。
