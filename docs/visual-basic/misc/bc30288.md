# <a name="local-variable-variablename-is-already-declared-in-the-current-block"></a>本地变量\<变量名 > 已在当前块中声明
在同一个块（可以是过程或控制块，如 `If...Then`）中声明了具有相同名称的两个变量。  
  
 **错误 ID:** BC30288  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将其中一个声明移动到其他块，或更改其中某个变量的名称。
