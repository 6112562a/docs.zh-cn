# <a name="expected-closing-gt-for-xml-cdata-section"></a>需要有结束]]&gt;XML CDATA 部分
XML CDATA 文本缺少必需的结束字符“]]>”。  
  
 **错误 ID:** BC31162  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   向 XML CDATA 文本末尾添加必需的结束字符“]]>”。
