# <a name="type-lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>类型&lt;typename&gt;因为它未声明为 Public 不能用作特性
特性必须是公开的。  
  
 **错误 ID:** BC31516  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  向特性类中添加 `Public` 修饰符。
