# <a name="value-ltvaluename1gt-cannot-be-converted-to-ltvaluename2gt"></a>值&lt;valuename1&gt;不能转换为&lt;值名称 2&gt&gt;
在两个不兼容的类型之间尝试执行了类型转换。  
  
 **错误 ID:** BC30742  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证正在被强制转换的类型。
