# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>值\<valuename1 > 无法转换为\<值名称 2&gt >
在两个不兼容的类型之间尝试执行了类型转换。  
  
 **错误 ID:** BC30742  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 验证正在被强制转换的类型。
