# <a name="ltfunctionnamegt-is-not-declared-visual-basic-error"></a>'&lt;functionname&gt;未声明 （Visual Basic 错误）
\<functionname > 未声明。 文件 I/O 功能在 Microsoft.VisualBasic 命名空间中可用。  
  
 **错误 ID:** BC30815  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用在 `My.Computer.FileSystem`中定义的文件 I/O 功能、 `System.IO` 命名空间、Visual Basic 运行时函数或 `Microsoft.VisualBasic.FileSystem` 对象。
