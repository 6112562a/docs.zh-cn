# <a name="events-cannot-have-a-return-type"></a>事件不能有返回类型
事件可以接受参数，但不能直接返回值。  
  
 **错误 ID:** BC30032  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从 `Event` 语句中删除返回类型。
