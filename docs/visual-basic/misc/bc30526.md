# <a name="property-ltpropertynamegt-is-readonly"></a>属性&lt;propertyname&gt;是 ReadOnly
尝试为声明为 `ReadOnly`的属性赋值。  
  
 **错误 ID:** BC30526  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从属性声明中删除 `ReadOnly` 说明符。
