# <a name="statement-cannot-appear-within-a-property-body"></a>语句不能出现在属性体内
语句不能出现在属性体内。 假定已到达属性末尾。  
  
 属性中出现意外的语句。  
  
 **错误 ID:** BC30634  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   验证属性所用的语法是否正确，以及是否以 `End Property` 构造终止。
