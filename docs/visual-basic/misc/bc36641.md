# <a name="lambda-parameter-parameter-hides-a-variable-in-an-enclosing-block-a-previously-defined-range-variable-or-an-implicitly-declared-variable-in-a-query-expression"></a>Lambda 参数\<参数 > 隐藏封闭块、 以前定义的范围变量或在查询表达式中隐式声明的变量中的变量。
Lambda 表达式中的变量与之前在同个作用域内定义的变量同名。 这可以是位于嵌套的 lambda 表达式代码的封闭块中的变量，可以是 LINQ 查询中之前定义的范围变量或者是 LINQ 查询的隐式声明的变量。  
  
 **错误 ID:** BC36641  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 确保在 lambda 表达式中的所有变量都具有唯一名称，该名称不能与同一作用域中已有变量名称重复。
