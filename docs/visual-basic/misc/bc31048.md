# <a name="properties-with-no-required-parameters-cannot-be-declared-default"></a>不带必需参数的属性不能声明为“Default”
指定属性不带必需参数，但带有 `Default` 修饰符。  
  
 **错误 ID:** BC31048  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 声明默认属性的必需参数。  
  
- 从属性定义中删除 `Default` 修饰符。
