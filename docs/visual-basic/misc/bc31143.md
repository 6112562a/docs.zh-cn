 <span data-ttu-id="4146a-105">以下代码阐释了这些失败的转换。</span><span class="sxs-lookup"><span data-stu-id="4146a-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="4146a-106">委托是 `FunDel`。</span><span class="sxs-lookup"><span data-stu-id="4146a-106">The delegate is `FunDel`.</span></span>
