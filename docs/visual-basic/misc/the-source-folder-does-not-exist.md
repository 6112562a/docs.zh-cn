# <a name="the-source-folder-does-not-exist"></a>项目文件夹不存在
尝试的操作引用了不存在的源文件夹。 这可能是由于提供的文件路径不正确。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查正在使用的路径是否正确，尤其是使用相对路径时。
