# <a name="argument-ltargument1gt-must-be-less-than-or-equal-to-the-length-of-argument-ltargument2gt"></a>参数&lt;argument1&gt;'必须小于或等于自变量的长度'&lt;argument2&gt;
参数的长度超过了允许的限制，其由第二个参数的长度确定。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  确保该参数的值的长度在有效范围内。  
  
2.  如果参数由某个计算提供，则检查该计算所处理的值。
