# <a name="unrecognized-option-optionname-ignored"></a>无法识别的选项\<选项名称 >; 已忽略
指定了一个命令行中无效的选项。  
  
 **错误 ID:** BC2007  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查指定选项以找出无效的那个选项。
