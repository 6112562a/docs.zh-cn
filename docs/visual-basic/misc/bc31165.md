# <a name="expected-beginning--for-an-xml-tag"></a>需要有开始\<XML 标记
XML 文本的 XML 标记缺少必需的开始字符“<”。  
  
 **错误 ID:** BC31165  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   添加“<”字符到 XML 文本的 XML 标记的起始位置。
