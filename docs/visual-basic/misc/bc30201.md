# <a name="expression-expected"></a>应输入表达式
`+` 或 `=` 等运算符的右操作数需要表达式，但没有出现可识别的表达式。  
  
 **错误 ID:** BC30201  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 为运算符的右操作数提供有效的表达式。
