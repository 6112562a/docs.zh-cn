# <a name="events-in-interfaces-cannot-be-declared-implements"></a>不能声明为接口中的事件\<实现 >
在接口中声明的事件不能实现其他接口的事件。  
  
 **错误 ID:** BC30688  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 删除 `Implements` 语句。  
  
2. 实现类或结构中的事件。
