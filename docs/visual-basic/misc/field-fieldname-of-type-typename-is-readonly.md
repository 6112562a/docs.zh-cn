# <a name="field-fieldname-of-type-typename-is-readonly"></a>字段\<fieldname > 类型的\<类型名称 > 是 ReadOnly
试图向到声明为 `ReadOnly`的字段进行写入。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改此字段，使其不再声明为 `ReadOnly`。  
  
-   - 或 -  
  
-   不要尝试向此字段进行写入。
