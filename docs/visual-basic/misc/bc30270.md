# <a name="specifier-is-not-valid-on-an-interface-method-declaration"></a>\<说明符 > 在接口方法声明上无效
接口内的 `Function` 或 `Sub` 语句包含无效的关键字，如 `Implements`。 接口只能定义成员，而不能实现它们。  
  
 **错误 ID:** BC30270  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  从声明语句中删除无效的关键字。  
  
2.  将接口成员的实现移动到实现该接口的类。
