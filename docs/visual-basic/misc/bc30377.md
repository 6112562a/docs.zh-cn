# <a name="method-ltmethodnamegt-is-already-declared-in-interface-ltinterfacenamegt"></a>方法&lt;methodname&gt;中已声明了接口&lt;interfacename&gt;
你通过复制第二次声明了某个接口方法。  
  
 **错误 ID:** BC30377  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除重复的声明。
