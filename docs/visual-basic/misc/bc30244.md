# <a name="modifier-is-not-valid-on-a-declare"></a>\<修饰符 > 在 declare 中无效
`Declare` 语句包含无效的关键字，如 `ReadOnly`。  
  
 **错误 ID:** BC30244  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从 `Declare` 语句中删除无效的关键字。
