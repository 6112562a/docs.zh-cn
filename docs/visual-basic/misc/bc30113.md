# <a name="ltnamespacegt-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>'&lt;命名空间&gt;不是有效的名称，不能用作根命名空间名称
合格命名空间名称不是有效的元素名称。 名称只能包含字母字符、数字和下划线，且必须以字母字符或下划线 (`_`) 开头。  
  
 **错误 ID:** BC30113  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   如果命名空间是一组嵌套的命名空间的一部分，请检查任何外部名称中是否包含了无效字符。
