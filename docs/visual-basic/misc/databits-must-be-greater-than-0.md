# <a name="databits-must-be-greater-than-0"></a>DataBits 必须大于 0
为 `DataBits` 方法指定的 `My.Computer.Ports.OpenSerialPort` 参数必须大于零。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `DataBits` 参数的值更改为正数值。
