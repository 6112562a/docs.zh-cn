# <a name="assemblyname-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>\<程序集名称 > 是一个程序集，不能作为模块引用
尝试使用程序集作为模块。  
  
 **错误 ID:** BC31077  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改对模块的引用。
