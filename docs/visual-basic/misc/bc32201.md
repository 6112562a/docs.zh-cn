# <a name="property-parameters-cannot-have-the-name-value"></a>属性参数的名称不能为“Value”
`Property` 语句中的参数名为 `Value`。 此名称保留在 `Set` 过程中被分配给该属性的新值的属性定义内。  
  
 **错误 ID:** BC32201  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 更改 `Property` 语句中的参数名称。
