# <a name="arrays-cannot-be-declared-with-new"></a><span data-ttu-id="f2c21-102">不能用“New”声明数组</span><span class="sxs-lookup"><span data-stu-id="f2c21-102">Arrays cannot be declared with 'New'</span></span>
<span data-ttu-id="f2c21-103">`New` 关键字只能在数组声明的初始化部分中出现。</span><span class="sxs-lookup"><span data-stu-id="f2c21-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="f2c21-104">这意味着 `New` 必须在等号 (`=`) 右侧，以便它可以创建要赋给数组变量的新数组类型。</span><span class="sxs-lookup"><span data-stu-id="f2c21-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="f2c21-105">类初始化的快捷方式对数组不可用。</span><span class="sxs-lookup"><span data-stu-id="f2c21-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="f2c21-106">以下两个代码行都是有效的，而且是等效，因为它们初始化类中的对象。</span><span class="sxs-lookup"><span data-stu-id="f2c21-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="f2c21-107">但是，数组初始化与类初始化不能使用相同的快捷方式。</span><span class="sxs-lookup"><span data-stu-id="f2c21-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="f2c21-108">请注意，数组的 `New` 子句必须同时包含圆括号 `()`和大括号 `{}`。</span><span class="sxs-lookup"><span data-stu-id="f2c21-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="f2c21-109">圆括号指定新类型为数组，大括号提供初始化值。</span><span class="sxs-lookup"><span data-stu-id="f2c21-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="f2c21-110">即使编译器为空，也就是说即使你不初始化任何数组值，编译器也需要大括号。</span><span class="sxs-lookup"><span data-stu-id="f2c21-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="f2c21-111">**错误 ID:** BC30053</span><span class="sxs-lookup"><span data-stu-id="f2c21-111">**Error ID:** BC30053</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f2c21-112">更正此错误</span><span class="sxs-lookup"><span data-stu-id="f2c21-112">To correct this error</span></span>  
  
-   <span data-ttu-id="f2c21-113">将 `Dim myDates() As New Date` 等语句替换为 `Dim myDates() As Date = New Date() {}`等语句。</span><span class="sxs-lookup"><span data-stu-id="f2c21-113">Replace a statement such as `Dim myDates() As New Date` with a statement such as `Dim myDates() As Date = New Date() {}`.</span></span>
