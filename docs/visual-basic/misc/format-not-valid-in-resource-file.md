# <a name="format-not-valid-in-resource-file"></a>资源文件中的格式无效
你的项目中的资源文件未遵循标准的 Windows 资源文件格式或资源文件和 Visual Basic 项目之间不匹配。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 重新创建资源源和资源文件，并遵循 Windows 软件开发工具包中所述的语法。  
  
2. 从项目中删除无效的资源文件。
