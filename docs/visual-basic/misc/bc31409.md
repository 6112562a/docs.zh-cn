# <a name="ltmethodgt-ltmethodname1gt-must-be-declared-overloads-because-another-ltmethodname2gt-is-declared-overloads"></a>&lt;方法&gt;'&lt;methodname1&gt;必须声明为 Overloads 因为另一个&lt;methodname2&gt;声明为 Overloads
另一个具有相同名称的属性或方法使用 `Overloads` 修饰符声明。  
  
 **错误 ID:** BC31409  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  将 `Overloads` 修饰符添加到属性或方法。  
  
2.  从所有具有相同名称的属性或方法中删除 `Overloads` 修饰符。
