# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>\<说明符 > 在结构声明中无效
你使用了在 `Dim` 声明上无效的说明符，例如 `Override`或 `Structure` 。  
  
 **错误 ID:** BC30395  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除说明符。
