# <a name="xml-comment-tag-include-must-have-a-ltattributegt-attribute"></a>XML 注释标记 include 必须具有&lt;特性&gt;属性
XML 注释标记 include 必须具有\<属性 > 特性。 将忽略 XML 注释。  
  
 传递给 `file` include `path` 标记的 `<`或`>` 特性缺少或不完整。  
  
 **错误 ID:** BC42310  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   添加或完成所需的特性\<包括 > 标记。
