# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>默认程序集引用\<引用 > 具有已被自动添加; 已忽略
已尝试添加默认程序集引用，但已添加该程序集，因此已忽略尝试。  
  
 **错误 ID:** BC2024  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除多余代码。
