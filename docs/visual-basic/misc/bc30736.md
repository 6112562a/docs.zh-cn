# <a name="expression-cannot-be-evaluated-at-this-time"></a>此时无法计算表达式
在当前上下文中无法计算指定的表达式。  
  
 **错误 ID:** BC30736  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 更改表达式的语法。
