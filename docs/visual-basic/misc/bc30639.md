# <a name="properties-cannot-be-declared-ltmodifiernamegt"></a>不能声明为属性&lt;modifiername&gt;
`Property` 声明中使用的语法存在错误。 此错误通常指示指定关键字（如 `WithEvents`）不能在 `Property` 声明中使用。  
  
 **错误 ID:** BC30639  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查 `Property` 声明的语法。
