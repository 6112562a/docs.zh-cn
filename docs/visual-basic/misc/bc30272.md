# <a name="argumentname-is-not-a-parameter-of-methoddeclaration"></a>'\<argumentname > 不是参数的\<methoddeclaration >
过程调用提供一个不是由过程定义的附加参数。  
  
 **错误 ID:** BC30272  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从参数列表中删除多余的参数。
