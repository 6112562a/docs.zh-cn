# <a name="class-ltclassnamegt-does-not-implement-the-systemcollectionsicollection-interface"></a>类的&lt;classname&gt;不实现 System.Collections.ICollection 接口
类没有实现 `System.Collections.ICollection` 接口，该接口定义了所有集合的大小、枚举器和同步方法。 可能需要重新安装 Visual Studio。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  重新安装 Visual Studio。
