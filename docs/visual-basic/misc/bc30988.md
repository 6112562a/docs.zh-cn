# <a name="type-or-with-expected"></a><span data-ttu-id="f7d7b-102">应为类型或“With”</span><span class="sxs-lookup"><span data-stu-id="f7d7b-102">Type or 'With' expected</span></span>
<span data-ttu-id="f7d7b-103">在声明类的实例时， `New` 关键字必须后跟类型名称或 `With`。</span><span class="sxs-lookup"><span data-stu-id="f7d7b-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="f7d7b-104">例如，以下语句各将 `client` 声明为 `Customer` 类的实例。</span><span class="sxs-lookup"><span data-stu-id="f7d7b-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="f7d7b-105">类型名称 `Customer` 在 `New`之后。</span><span class="sxs-lookup"><span data-stu-id="f7d7b-105">The type name `Customer` follows `New`.</span></span>  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="f7d7b-106">自 [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)]起，你可以将对象声明为匿名类型的实例，此情况下未指定数据类型。</span><span class="sxs-lookup"><span data-stu-id="f7d7b-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="f7d7b-107">在匿名类型声明中，关键字 `With` 在 `New`之后。</span><span class="sxs-lookup"><span data-stu-id="f7d7b-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
```  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```
