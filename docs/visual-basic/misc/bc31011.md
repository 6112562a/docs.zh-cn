# <a name="unable-to-load-referenced-library-ltfilenamegt-lterrorgt"></a>无法加载引用的库 '&lt;文件名&gt;:&lt;错误&gt;
无法加载此项目引用的库。  
  
 **错误 ID:** BC31011  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  验证文件是否处于指定位置，并且未被另一个应用程序锁定。
