# <a name="a-log-has-already-been-created-with-this-name-on-this-machine"></a>此计算机上已创建了具有此名称的日志
尝试使用已在使用中的名称创建日志。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除冲突的日志。  
  
-   使用不同的日志名。
