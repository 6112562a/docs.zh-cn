# <a name="cannot-implement-interface-interfacename1-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-interfacename2-for-some-type-arguments"></a><span data-ttu-id="34386-102">无法实现接口\<interfacename1 >' 由于其实现可能与另一个实现接口的实现冲突\<interfacename2 > 对于某些类型参数</span><span class="sxs-lookup"><span data-stu-id="34386-102">Cannot implement interface '\<interfacename1>' because its implementation could conflict with the implementation of another implemented interface '\<interfacename2>' for some type arguments</span></span>
<span data-ttu-id="34386-103">类声明包含指定两个或多个接口的 `Implements` 语句，但其中至少一个接口为泛型对象，且两个实现都可能对类型实参的某些值冲突。</span><span class="sxs-lookup"><span data-stu-id="34386-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="34386-104">以下语句可能会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="34386-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1  
    Sub testSub(ByVal arg As String)  
End Interface  
Public Interface iFace2(Of t)  
    Sub testSub(ByVal arg As t)  
End Interface  
Public Class testClass  
    Implements iFace1, iFace2(Of String)  
End Class  
```  
  
 <span data-ttu-id="34386-105">因为 `iFace2` 是使用 `String`构造的，因此 `testClass` 必须使用相同签名实现 `testSub` 的两个版本。</span><span class="sxs-lookup"><span data-stu-id="34386-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="34386-106">如果这样做，则在访问哪个版本方面存在歧义。</span><span class="sxs-lookup"><span data-stu-id="34386-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="34386-107">**错误 ID:** BC32072</span><span class="sxs-lookup"><span data-stu-id="34386-107">**Error ID:** BC32072</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="34386-108">更正此错误</span><span class="sxs-lookup"><span data-stu-id="34386-108">To correct this error</span></span>  
  
-   <span data-ttu-id="34386-109">更改提供给泛型接口的类型实参，以消除冲突。</span><span class="sxs-lookup"><span data-stu-id="34386-109">Change the type argument supplied to the generic interface so that there is no conflict.</span></span>  
  
     <span data-ttu-id="34386-110">或</span><span class="sxs-lookup"><span data-stu-id="34386-110">-or-</span></span>
