# <a name="resource-name-resourcename-cannot-be-used-more-than-once"></a>资源名称\<资源名称 > 不能多次使用
动态链接库或可执行文件中指定资源的名称被多次使用。  
  
 **错误 ID:** BC31502  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   唯一的命名资源。
