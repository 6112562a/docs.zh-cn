# <a name="registry-key-ltkeynamegt-could-not-be-created"></a>注册表项&lt;keyname&gt;无法创建
无法创建指定的注册表项，可能是因为没有足够的安全权限。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保你有足够的安全权限，可以访问该注册表项。
