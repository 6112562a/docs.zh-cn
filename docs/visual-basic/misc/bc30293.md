# <a name="lterrorgt-ltstructurename1gt-contains-ltstructurename2gt"></a>&lt;错误&gt;:&lt;structurename1&gt;包含&lt;structurename2&gt;
当一个结构将自身指定为自己的成员之一时，会出现循环结构定义。  
  
 **错误 ID:** BC30293  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改结构成员的名称。
