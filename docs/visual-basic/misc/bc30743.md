# <a name="type-typename-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>未定义类型 "\<typename >" 或在调试会话中未加载包含它的模块
引用的类型不可用，或未定义。  
  
 **错误 ID：** BC30743  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 验证类型名称拼写是否正确。  
  
2. 在当前类或模块中定义类型。
