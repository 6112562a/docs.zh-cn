# <a name="method-ltmethodname1gt-must-be-declared-private-in-order-to-implement-partial-method-ltmethodname2gt"></a><span data-ttu-id="a2fbd-102">方法&lt;methodname1&gt;必须声明为 Private 才能实现分部方法&lt;methodname2&gt;</span><span class="sxs-lookup"><span data-stu-id="a2fbd-102">Method '&lt;methodname1&gt;' must be declared 'Private' in order to implement partial method '&lt;methodname2&gt;'</span></span>
<span data-ttu-id="a2fbd-103">分部方法的实现必须声明为 `Private`。</span><span class="sxs-lookup"><span data-stu-id="a2fbd-103">The implementation of a partial method must be declared `Private`.</span></span> <span data-ttu-id="a2fbd-104">例如，下面的代码会导致此错误。</span><span class="sxs-lookup"><span data-stu-id="a2fbd-104">For example, the following code causes this error.</span></span>  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,   
    ' causes this error.   
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 <span data-ttu-id="a2fbd-105">**错误 ID:** BC31441</span><span class="sxs-lookup"><span data-stu-id="a2fbd-105">**Error ID:** BC31441</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a2fbd-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="a2fbd-106">To correct this error</span></span>

```vb  
    MsgBox("Value was changed to " & Me.Quantity)  
End Sub  
```
