# <a name="structure-ltstructurenamegt-cannot-be-indexed-because-it-has-no-default-property"></a>结构&lt;structurename&gt;不能将编制索引，因为它没有默认属性
试图对没有默认属性的结构使用默认属性语法。  
  
 **错误 ID:** BC30690  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用标准属性语法访问该结构的属性。
