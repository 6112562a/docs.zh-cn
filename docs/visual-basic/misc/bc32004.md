# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>在项目 "\<projectname1 >" 中声明了 "\<name >"，该项目未被项目 "\<projectname2 >" 引用
在当前项目未引用的项目中定义了一个编程元素。  
  
 **错误 ID：** BC32004  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 在当前项目中，添加对包含所需元素的项目的引用。
