# <a name="ltnamegt-is-declared-in-project-ltprojectname1gt-which-is-not-referenced-by-project-ltprojectname2gt"></a>'&lt;名称&gt;在项目中声明'&lt;projectname1&gt;'，其未引用的项目'&lt;项目名称 2>&gt;
在当前项目未引用的项目中定义了一个编程元素。  
  
 **错误 ID:** BC32004  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在当前项目中，添加对包含所需元素的项目的引用。
