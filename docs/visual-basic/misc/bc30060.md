# <a name="conversion-from-lttype1gt-to-lttype2gt-cannot-occur-in-a-constant-expression"></a>从转换&lt;type1&gt;到&lt;type2&gt;不能在常量表达式中
`Const` 语句的初始化表达式计算出无法转换为所声明的常量类型的数据类型。  
  
 **错误 ID:** BC30060  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  使用可转换为对常量声明的类型的数据类型的表达式初始化常量。
