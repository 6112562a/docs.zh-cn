# <a name="type-of-argument-ltargumentnamegt-is-lttypenamegt-which-is-not-numeric"></a>自变量的类型&lt;argumentname&gt;是&lt;typename&gt;，它不是数值
在需要数值参数的情况下出现了非数值参数。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保参数是数值。
