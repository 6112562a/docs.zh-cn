# <a name="expected-closing--for-xml-entity"></a>XML 实体需要有结束的“;”
XML 实体引用（例如， `>`）缺少右分号 (;)。  
  
 **错误 ID:** BC31178  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在 XML 实体引用末尾添加分号。
