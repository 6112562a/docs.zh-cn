# <a name="xml-dtds-are-not-supported"></a>不支持 XML DTD
文档类型定义 (DTD) 已被指定为 XML 文本。  
  
 **错误 ID:** BC31175  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除 DTD。 使用 XML 架构定义 (XSD) 来指定 XML 文档的架构。
