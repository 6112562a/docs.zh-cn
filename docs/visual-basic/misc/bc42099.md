# <a name="unused-local-constant-ltconstantgt"></a>未使用的局部常量:&lt;常量&gt;
声明了局部常量，但未在代码中的任何位置使用它。  
  
 **错误 ID:** BC42099  
  
### <a name="to-remove-this-warning"></a>删除此警告  
  
1.  此错误是一条警告，可以忽略。 若要删除此警告，请删除代码中未使用的常量声明。
