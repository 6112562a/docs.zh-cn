```  
    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface

    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="9a381-105">如果构造或实现 `derivedInterface` 时同时向 `t1` 和 `t2`提供相同类型，则它必须继承类型参数相同的两个版本的 `interfaceX` 。</span><span class="sxs-lookup"><span data-stu-id="9a381-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="9a381-106">如果这样做，则在访问哪个版本方面存在歧义。</span><span class="sxs-lookup"><span data-stu-id="9a381-106">Doing so would produce an ambiguity about which version to access.</span></span>
