# <a name="expression-name-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>表达式\<名称 > 不是一个过程，但作为过程调用的目标出现
你已使过程调用成为了一个非过程的表达式。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 请确保过程的名称拼写正确。  
  
2. 为该过程提供的 <`name`> 调用。
