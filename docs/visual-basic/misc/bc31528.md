# <a name="construct-makes-an-indirect-reference-to-assembly-ltassemblynamegt-which-contains-lttypenamegt"></a>构造函数间接引用程序集 '&lt;assemblyname&gt;，其中包含&lt;typename&gt;
构造函数间接引用程序集 '\<程序集名称 >，其中包含\<类型名称 >。 添加对的文件引用\<文件名 > 到你的项目。  
  
 表达式使用类、结构、接口、枚举或委托等类型，但程序集没有对定义该类型的程序集的项目引用。  
  
 **错误 ID:** BC31528  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在项目属性中，添加对包含某程序集的文件的引用，该程序集定义正在使用的类型。
