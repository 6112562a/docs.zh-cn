# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>没有为参数指定的参数&lt;parametername&gt;的扩展方法&lt;methodname&gt;中定义'&lt;typename&gt;
尚未为扩展方法中所需的形参提供实参。  
  
 **错误 ID:** BC36586  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   为指定形参提供实参。
