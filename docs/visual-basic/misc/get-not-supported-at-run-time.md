# <a name="get-not-supported-at-run-time"></a>运行时不支持 Get 语句
你试图在运行时读取仅在设计时才可访问的属性。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查该属性并确定可以在什么情况下对其进行设置。  
  
2. 删除对该属性的引用。
