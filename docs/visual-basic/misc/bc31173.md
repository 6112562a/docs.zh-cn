# <a name="missing-required-white-space"></a>缺少必需的空白区域
指定了 XML 文本，但缺少必需的分隔符空间。 必需的分隔符空间的一个示例是 XML 元素文本中后续特性之间的空间。  
  
 **错误 ID:** BC31173  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 添加必需的空间分隔符。
