# <a name="event-lteventnamegt-cannot-be-handled-because-it-is-not-accessible-from-ltnamegt"></a>事件&lt;eventname&gt;不能处理，因为不能从访问&lt;名称&gt;
你试图处理不可访问的事件。 例如，如果 `Handles` 变量共享，则处理事件的方法也必须共享。  
  
 **错误 ID:** BC30585  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  请确保该事件可访问。
