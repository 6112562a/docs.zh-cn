# <a name="field-or-property-ltmembernamegt-is-not-found"></a>字段或属性&lt;membername&gt;找不到
为特性引用的字段或特性未定义。  
  
 **错误 ID:** BC30661  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查字段或属性的拼写。  
  
2.  请验证你的项目是否可以访问正确的特性类，并将其添加到项目引用中（如有必要）。
