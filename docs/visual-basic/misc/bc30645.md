# <a name="attribute-attributename-cannot-be-applied-to-a-method-with-optional-parameters"></a>特性 "\<attributename >" 不能应用于具有可选参数的方法
该特性仅可用于使用所需参数或不带参数的方法。  
  
 **错误 ID:** BC30645  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 定义不带可选参数的方法。  
  
2. 使用可应用于具有可选参数的方法的特性。  
  
3. 定义可以用在此上下文中的自定义特性。
