# <a name="character-character-0xlthexadecimalgt-is-not-allowed-in-xml-literal"></a>字符 character (0x&lt;十六进制&gt;) 中不允许使用 XML 文本
在 XML 文本中使用了不受支持的字符。  
  
 **错误 ID:** BC31176  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除不受支持的字符。
