# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>Imports 别名&lt;名称&gt;冲突&lt;name2&gt;根命名空间中声明
`Imports` 语句中使用的别名名称与根命名空间中声明的另一个名称冲突。  
  
 **错误 ID:** BC31403  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   选择 `Imports` 语句使用的唯一别名名称。
