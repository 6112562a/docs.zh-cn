# <a name="properties-in-a-module-cannot-be-declared-specifier"></a>不能声明为模块中的属性 '\<说明符 >
你尝试对模块中的属性使用说明符，例如 `MustOverride`、 `NotOverridable`或 `Overridable`。  
  
 **错误 ID:** BC30503  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除说明符。
