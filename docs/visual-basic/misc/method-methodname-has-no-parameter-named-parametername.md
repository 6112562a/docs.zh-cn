# <a name="method-ltmethodnamegt-has-no-parameter-named-ltparameternamegt"></a>方法&lt;methodname&gt;有没有参数命名为&lt;parametername&gt;
提供的命名参数与目标方法的所有参数均不匹配。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  确保正确提供了命名参数。
