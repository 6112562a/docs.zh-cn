# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-declared-in-ltnamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;type1&gt; '&lt;typename&gt;和&lt;y p e 2&gt; &lt;typename&gt;，声明中的&lt;名称&gt;，在发生冲突&lt;命名空间&gt; '&lt;命名空间名称&gt;
已声明的两个元素（如类和模块）在同一命名空间中具有相同的名称。  
  
 **错误 ID:** BC30175  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将其中一个元素的声明移到其他命名空间。  
  
-   更改其中一个元素的名称。
