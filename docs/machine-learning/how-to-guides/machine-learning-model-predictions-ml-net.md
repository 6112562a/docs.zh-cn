## <a name="create-data-models"></a><span data-ttu-id="06b5e-105">创建数据模型</span><span class="sxs-lookup"><span data-stu-id="06b5e-105">Create data models</span></span>

### <a name="input-data"></a><span data-ttu-id="06b5e-106">输入数据</span><span class="sxs-lookup"><span data-stu-id="06b5e-106">Input data</span></span>

```csharp
public class HousingData
{
    [LoadColumn(0)]
    public float Size { get; set; }

    [LoadColumn(1, 3)]
    [VectorType(3)]

    [LoadColumn(4)]
```

### <a name="output-data"></a><span data-ttu-id="06b5e-107">输出数据</span><span class="sxs-lookup"><span data-stu-id="06b5e-107">Output data</span></span>

<span data-ttu-id="06b5e-108">与 `Features` 和 `Label` 输入列名一样，ML.NET 为模型生成的预测值列提供默认名称。</span><span class="sxs-lookup"><span data-stu-id="06b5e-108">Like the `Features` and `Label` input column names, ML.NET has default names for the predicted value columns produced by a model.</span></span> <span data-ttu-id="06b5e-109">名称可能因任务而异。</span><span class="sxs-lookup"><span data-stu-id="06b5e-109">Depending on the task the name may differ.</span></span>

<span data-ttu-id="06b5e-110">由于此示例中使用的算法是线性回归算法，输出列的默认名称为 `Score`，它由 `PredictedPrice` 属性上的 [`ColumnName`](xref:Microsoft.ML.Data.ColumnNameAttribute) 特性定义。</span><span class="sxs-lookup"><span data-stu-id="06b5e-110">Because the algorithm used in this sample is a linear regression algorithm, the default name of the output column is `Score` which is defined by the [`ColumnName`](xref:Microsoft.ML.Data.ColumnNameAttribute) attribute on the `PredictedPrice` property.</span></span>

```csharp
class HousingPrediction : HousingData
{
    [ColumnName("Score")]
    public float PredictedPrice { get; set; }
}
```

<span data-ttu-id="06b5e-111">`HousingPrediction` 数据模型继承自 `HousingData`，这让它可以轻松地将原始输入数据与模型生成的输出一起可视化。</span><span class="sxs-lookup"><span data-stu-id="06b5e-111">The `HousingPrediction` data model inherits from `HousingData` to make it easy to visualize the original input data along with the output generated by the model.</span></span>  

## <a name="set-up-a-prediction-pipeline"></a><span data-ttu-id="06b5e-112">设置预测管道</span><span class="sxs-lookup"><span data-stu-id="06b5e-112">Set up a prediction pipeline</span></span>

<span data-ttu-id="06b5e-113">无论是进行单一预测还是批量预测，都需要将预测管道加载到应用程序中。</span><span class="sxs-lookup"><span data-stu-id="06b5e-113">Whether making a single or batch prediction, the prediction pipeline needs to be loaded into the application.</span></span> <span data-ttu-id="06b5e-114">此管道包含数据预处理转换以及经过训练的模型。</span><span class="sxs-lookup"><span data-stu-id="06b5e-114">This pipeline contains both the data pre-processing transformations as well as the trained model.</span></span> <span data-ttu-id="06b5e-115">下面的代码片段从名为 `model.zip` 的文件中加载预测管道。</span><span class="sxs-lookup"><span data-stu-id="06b5e-115">The code snippet below loads the prediction pipeline from a file named `model.zip`.</span></span>

```csharp
//Create MLContext

DataViewSchema predictionPipelineSchema;
ITransformer predictionPipeline = mlContext.Model.Load("model.zip", out predictionPipelineSchema);
```

## <a name="single-prediction"></a><span data-ttu-id="06b5e-116">单一预测</span><span class="sxs-lookup"><span data-stu-id="06b5e-116">Single prediction</span></span>

<span data-ttu-id="06b5e-117">若要进行单一预测，请使用加载的预测管道创建 [`PredictionEngine`](xref:Microsoft.ML.PredictionEngine%602)。</span><span class="sxs-lookup"><span data-stu-id="06b5e-117">To make a single prediction, create a [`PredictionEngine`](xref:Microsoft.ML.PredictionEngine%602) using the loaded prediction pipeline.</span></span>

```csharp
// Create PredictionEngines
PredictionEngine<HousingData, HousingPrediction> predictionEngine = mlContext.Model.CreatePredictionEngine<HousingData, HousingPrediction>(predictionPipeline);
```

<span data-ttu-id="06b5e-118">然后，使用 [`Predict`](xref:Microsoft.ML.PredictionEngineBase%602.Predict*) 方法并将输入数据作为参数传入。</span><span class="sxs-lookup"><span data-stu-id="06b5e-118">Then, use the [`Predict`](xref:Microsoft.ML.PredictionEngineBase%602.Predict*) method and pass in your input data as a parameter.</span></span> <span data-ttu-id="06b5e-119">请注意，使用 [`Predict`](xref:Microsoft.ML.PredictionEngineBase%602.Predict*) 方法不要求输入为 [`IDataView`](xref:Microsoft.ML.IDataView)。</span><span class="sxs-lookup"><span data-stu-id="06b5e-119">Notice that using the [`Predict`](xref:Microsoft.ML.PredictionEngineBase%602.Predict*) method does not require the input to be an [`IDataView`](xref:Microsoft.ML.IDataView)).</span></span> <span data-ttu-id="06b5e-120">这是因为它可以方便地内在化输入数据类型操作，以便能够传入输入数据类型的对象。</span><span class="sxs-lookup"><span data-stu-id="06b5e-120">This is because it conveniently internalizes the input data type manipulation so you can pass in an object of the input data type.</span></span> <span data-ttu-id="06b5e-121">此外，由于 `CurrentPrice` 是尝试使用新数据进行预测的目标或标签，假设此时没有用于它的值。</span><span class="sxs-lookup"><span data-stu-id="06b5e-121">Additionally, since `CurrentPrice` is the target or label you're trying to predict using new data, it's assumed there is no value for it at the moment.</span></span>

```csharp
// Input Data
HousingData inputData = new HousingData
{
    Size = 900f,
    HistoricalPrices = new float[] { 155000f, 190000f, 220000f }
};

// Get Prediction
HousingPrediction prediction = predictionEngine.Predict(inputData);
```

<span data-ttu-id="06b5e-122">如果访问 `prediction` 对象的 `Score` 属性，则应获得类似于 `150079` 的值。</span><span class="sxs-lookup"><span data-stu-id="06b5e-122">If you access the `Score` property of the `prediction` object, you should get a value similar to `150079`.</span></span>
