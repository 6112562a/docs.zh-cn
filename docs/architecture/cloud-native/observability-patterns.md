## <a name="when-to-use-alerts"></a><span data-ttu-id="03937-171">何时使用警报</span><span class="sxs-lookup"><span data-stu-id="03937-171">When to use alerts</span></span>

## <a name="alerts"></a><span data-ttu-id="03937-177">警报</span><span class="sxs-lookup"><span data-stu-id="03937-177">Alerts</span></span>
