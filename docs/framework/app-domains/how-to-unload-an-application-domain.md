 <span data-ttu-id="9325f-106">加载到应用程序域中的所有程序集都会被移除，无法再使用。</span><span class="sxs-lookup"><span data-stu-id="9325f-106">Assemblies loaded into the application domain are removed and are no longer available.</span></span> <span data-ttu-id="9325f-107">如果应用程序域中的程序集不特定于域，则程序集的数据会保留在内存中，直到整个进程关闭。</span><span class="sxs-lookup"><span data-stu-id="9325f-107">If an assembly in the application domain is domain-neutral, data for the assembly remains in memory until the entire process is shut down.</span></span> <span data-ttu-id="9325f-108">除了关闭整个进程，没有机制可以卸载非特定于域的程序集。</span><span class="sxs-lookup"><span data-stu-id="9325f-108">There is no mechanism to unload a domain-neutral assembly other than shutting down the entire process.</span></span> <span data-ttu-id="9325f-109">在某些情况下，卸载应用程序域的请求会不起作用，并导致 <xref:System.CannotUnloadAppDomainException>。</span><span class="sxs-lookup"><span data-stu-id="9325f-109">There are situations where the request to unload an application domain does not work and results in a <xref:System.CannotUnloadAppDomainException>.</span></span>
