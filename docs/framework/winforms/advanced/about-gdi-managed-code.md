# <a name="about-gdi-managed-code"></a>关于 GDI+ 托管代码
GDI + 是 Windows 操作系统提供二维矢量图形、 图像处理和版式的部分。 GDI + 通过添加新功能和优化了现有功能，则改进了 GDI （包含在早期版本的 Windows 的图形设备接口）。  
  
 GDI + 托管的类接口 （一组包装） 是.NET Framework，用于构建、 部署和运行 XML Web 服务和其他应用程序的环境的一部分。  
  
 本部分提供有关 GDI + API 程序员使用的信息的托管代码。
