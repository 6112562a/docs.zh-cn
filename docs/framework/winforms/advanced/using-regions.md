# <a name="using-regions"></a>使用区域
GDI +<xref:System.Drawing.Region>类允许你定义的自定义形状。 形状可以组成线条、 多边形和曲线。  
  
 有关区域的两个常见用途是命中测试和剪裁。 命中测试确定是否在特定的区域的屏幕中单击了鼠标。 剪辑是将绘图限制在特定的区域。
