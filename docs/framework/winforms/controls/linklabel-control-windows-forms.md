# <a name="linklabel-control-windows-forms"></a>LinkLabel 控件（Windows 窗体）
利用 Windows 窗体 `LinkLabel` 控件，你可以将 Web 样式链接添加到 Windows 窗体应用程序。 您可以使用 `LinkLabel` 控件来查找可对其使用 <xref:System.Windows.Forms.Label> 控件的所有内容;您还可以将部分文本设置为指向对象或网页的链接。

 列出如何指定链接标签中 Web 样式链接的颜色和下划线。

 提供有关在单击链接时打开窗体或网页的操作说明。

 演示如何在用户单击 Windows 窗体 `LinkLabel` 控件时在默认浏览器中显示网页。
