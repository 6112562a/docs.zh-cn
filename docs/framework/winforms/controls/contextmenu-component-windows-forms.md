# <a name="contextmenu-component-windows-forms"></a>ContextMenu 组件（Windows 窗体）
> [!NOTE]
> 尽管`MenuStrip`和<xref:System.Windows.Forms.ContextMenu> <xref:System.Windows.Forms.MainMenu> <xref:System.Windows.Forms.MainMenu> <xref:System.Windows.Forms.ContextMenu>将功能替换为以前版本的和控件, 并将其保留以实现后向兼容性和将来使用 (如果你选择)。 `ContextMenuStrip`  
  
 Windows 窗体`ContextMenu`组件用于向用户提供与所选对象关联的常用命令的快捷菜单。 快捷菜单中的项通常是显示在应用程序中其他位置的主菜单中的项的子集。 快捷菜单通常通过右键单击鼠标提供。 在 Windows 窗体它们与其他控件相关联。
