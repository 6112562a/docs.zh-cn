# <a name="how-to-create-access-keys-with-windows-forms-label-controls"></a><span data-ttu-id="919eb-102">如何：使用 Windows 窗体 Label 控件创建访问键</span><span class="sxs-lookup"><span data-stu-id="919eb-102">How to: Create Access Keys with Windows Forms Label Controls</span></span>
<span data-ttu-id="919eb-103">Windows 窗体<xref:System.Windows.Forms.Label>控件可用于定义其他控件的访问键。</span><span class="sxs-lookup"><span data-stu-id="919eb-103">Windows Forms <xref:System.Windows.Forms.Label> controls can be used to define access keys for other controls.</span></span> <span data-ttu-id="919eb-104">在标签控件中定义访问键时，用户可以同时按 ALT 键和指定的字符，以便将焦点移到 Tab 键顺序中排在其后面的控件。</span><span class="sxs-lookup"><span data-stu-id="919eb-104">When you define an access key in a label control, the user can press the ALT key plus the character you designate to move the focus to the control that follows it in the tab order.</span></span> <span data-ttu-id="919eb-105">由于标签不能接收焦点，因此焦点会自动移到 Tab 键顺序中的下一个控件。</span><span class="sxs-lookup"><span data-stu-id="919eb-105">Because labels cannot receive focus, focus automatically moves to the next control in the tab order.</span></span> <span data-ttu-id="919eb-106">使用此方法可将访问键分配给文本框、组合框、列表框和数据网格。</span><span class="sxs-lookup"><span data-stu-id="919eb-106">Use this technique to assign access keys to text boxes, combo boxes, list boxes, and data grids.</span></span>  
  
### <a name="to-assign-an-access-key-to-a-control-with-a-label"></a><span data-ttu-id="919eb-107">为具有标签的控件分配访问键</span><span class="sxs-lookup"><span data-stu-id="919eb-107">To assign an access key to a control with a label</span></span>  
  
1. <span data-ttu-id="919eb-108">首先绘制标签, 然后绘制另一个控件。</span><span class="sxs-lookup"><span data-stu-id="919eb-108">Draw the label first, and then draw the other control.</span></span>  
  
     <span data-ttu-id="919eb-109">或</span><span class="sxs-lookup"><span data-stu-id="919eb-109">-or-</span></span>  
  
     <span data-ttu-id="919eb-110">按任意顺序绘制控件, 并将标签<xref:System.Windows.Forms.Control.TabIndex%2A>的属性设置为一个小于另一个控件的。</span><span class="sxs-lookup"><span data-stu-id="919eb-110">Draw the controls in any order and set the <xref:System.Windows.Forms.Control.TabIndex%2A> property of the label to one less than the other control.</span></span>  
  
2. <span data-ttu-id="919eb-111">将标签的<xref:System.Windows.Forms.Label.UseMnemonic%2A>属性设置为`true`。</span><span class="sxs-lookup"><span data-stu-id="919eb-111">Set the label's <xref:System.Windows.Forms.Label.UseMnemonic%2A> property to `true`.</span></span>  
  
3. <span data-ttu-id="919eb-112">在标签的 <xref:System.Windows.Forms.Label.Text%2A> 属性中使用与号 (&) 来分配标签的访问键。</span><span class="sxs-lookup"><span data-stu-id="919eb-112">Use an ampersand (&) in the label's <xref:System.Windows.Forms.Label.Text%2A> property to assign the access key for the label.</span></span> <span data-ttu-id="919eb-113">有关详细信息，请参阅[为 Windows 窗体控件创建访问键](how-to-create-access-keys-for-windows-forms-controls.md)。</span><span class="sxs-lookup"><span data-stu-id="919eb-113">For more information, see [Creating Access Keys for Windows Forms Controls](how-to-create-access-keys-for-windows-forms-controls.md).</span></span>  
  
    > [!NOTE]
    > <span data-ttu-id="919eb-114">可能需要显示标签控件中的与号，而不使用它们来创建访问键。</span><span class="sxs-lookup"><span data-stu-id="919eb-114">You may want to display ampersands in a label control, rather than use them to create access keys.</span></span> <span data-ttu-id="919eb-115">如果将标签控件绑定到记录集中的一个字段，而该记录集中的数据包含与号，则可能会出现这种情况。</span><span class="sxs-lookup"><span data-stu-id="919eb-115">This may occur if you bind a label control to a field in a recordset where the data includes ampersands.</span></span> <span data-ttu-id="919eb-116">若要在标签控件中显示与号，请将 <xref:System.Windows.Forms.Label.UseMnemonic%2A> 属性设置为 `false`。</span><span class="sxs-lookup"><span data-stu-id="919eb-116">To display ampersands in a label control, set the <xref:System.Windows.Forms.Label.UseMnemonic%2A> property to `false`.</span></span> <span data-ttu-id="919eb-117">若要显示与号，同时还要设置访问键，请将 <xref:System.Windows.Forms.Label.UseMnemonic%2A> 属性设置为 `true`，并用一个与号 (&) 来表示访问键，用两个与号来表示与号。</span><span class="sxs-lookup"><span data-stu-id="919eb-117">If you wish to display ampersands and also have an access key, set the <xref:System.Windows.Forms.Label.UseMnemonic%2A> property to `true` and indicate the access key with one ampersand (&) and the ampersand to display with two ampersands.</span></span>
