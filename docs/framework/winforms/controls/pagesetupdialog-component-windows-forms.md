# <a name="pagesetupdialog-component-windows-forms"></a>PageSetupDialog 组件（Windows 窗体）
Windows 窗体<xref:System.Windows.Forms.PageSetupDialog>组件是一个预先配置的对话框，用于设置页面详细信息，以便在基于 Windows 的应用程序中打印。 在基于 Windows 的应用程序中使用它作为用户设置页面首选项以代替配置自己的对话框的简单解决方案。 您可以允许用户设置边框和边距调整、标题和页脚以及纵向与横向。 利用标准的 Windows 对话框，你可以创建其基本功能可立即为用户所熟悉的应用程序。
