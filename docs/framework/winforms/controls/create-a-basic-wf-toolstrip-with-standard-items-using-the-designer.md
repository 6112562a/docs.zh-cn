# <a name="how-to-create-a-basic-windows-forms-toolstrip-with-standard-items-using-the-designer"></a>如何：使用设计器创建包含标准项的基本 Windows 窗体 ToolStrip
下面的过程演示如何创建<xref:System.Windows.Forms.ToolStrip>和添加七个<xref:System.Windows.Forms.ToolStripButton>表示典型任务的控件。

## <a name="to-add-standard-items-in-the-designer"></a>在设计器中添加标准项

1. <xref:System.Windows.Forms.ToolStrip>创建控件。

2. 在的<xref:System.Windows.Forms.ToolStrip>右上角, 单击 "智能任务" 箭头以显示 " **ToolStrip 任务**" 窗格。

3. 在 " **ToolStrip 任务**" 窗格中, 选择 "**插入标准项**"。
