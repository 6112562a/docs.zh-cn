# <a name="label-control-windows-forms"></a>Label 控件（Windows 窗体）
> [!IMPORTANT]
> <xref:System.Windows.Forms.ToolStripLabel> 控件将替换功能并将其添加到 <xref:System.Windows.Forms.Label> 控件。 可以将 <xref:System.Windows.Forms.ToolStripLabel> 与其他新控件（如 <xref:System.Windows.Forms.ToolStripDropDown>）结合使用。 但是，如果您选择，则会保留 <xref:System.Windows.Forms.Label> 控件以实现向后兼容性和将来使用。  
  
 Windows 窗体 <xref:System.Windows.Forms.Label> 控件用于显示用户无法编辑的文本或图像。 它们用于标识窗体上的对象-例如，提供特定控件在被单击时所要执行的操作的说明，或者显示为响应应用程序中的运行时事件或进程而显示的信息。 由于 <xref:System.Windows.Forms.Label> 控件不能接收焦点，因此也可以使用它来创建其他控件的访问键。
