# <a name="notifyicon-component-windows-forms"></a>NotifyIcon 구성 요소(Windows Forms)
Windows 窗体 `NotifyIcon` 组件会在任务栏的状态通知区域中显示图标，用于在后台运行且不具有用户界面的进程。 예를 들어 작업 표시줄의 상태 알림 영역에서 아이콘을 클릭하여 액세스할 수 있는 바이러스 방지 프로그램이 있습니다.
