# <a name="fontdialog-component-windows-forms"></a>FontDialog 구성 요소(Windows Forms)
Windows Forms <xref:System.Windows.Forms.FontDialog> 구성 요소는 미리 구성된 대화 상자입니다. 它与 Windows 操作系统公开的 "**字体**" 对话框相同。 组件继承自 <xref:System.Windows.Forms.CommonDialog> 类。  
  
## <a name="in-this-section"></a>섹션 내용  
 [FontDialog 구성 요소 개요](fontdialog-component-overview-windows-forms.md)  
 介绍 <xref:System.Windows.Forms.FontDialog> 组件的一般概念，您可以使用该组件显示预配置的对话框。 用户可以使用该对话框操作字体及其设置。  
  
 [방법: FontDialog 구성 요소를 사용하여 글꼴 목록 표시](how-to-show-a-font-list-with-the-fontdialog-component.md)  
 说明如何在运行时通过 <xref:System.Windows.Forms.FontDialog> 组件的实例选择字体。
