# <a name="picturebox-control-overview-windows-forms"></a>PictureBox 컨트롤 개요(Windows Forms)
"Windows 窗体" <xref:System.Windows.Forms.PictureBox> 控件用于以位图、GIF、JPEG、图元文件或图标格式显示图形。  
  
## <a name="key-properties-and-methods"></a>键属性和方法  
 显示的图片由 <xref:System.Windows.Forms.PictureBox.Image%2A> 属性确定，该属性可在运行时或设计时设置。 您也可以通过设置 <xref:System.Windows.Forms.PictureBox.ImageLocation%2A> 属性来指定图像，然后使用 <xref:System.Windows.Forms.PictureBox.Load%2A> 方法以异步方式加载图像，或使用 <xref:System.Windows.Forms.PictureBox.LoadAsync%2A> 方法以异步方式加载图像。 <xref:System.Windows.Forms.PictureBox.SizeMode%2A> 属性控制图像和控件如何彼此吻合。 有关详细信息，请参阅[如何：在运行时修改图片的大小或位置](how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md)。
