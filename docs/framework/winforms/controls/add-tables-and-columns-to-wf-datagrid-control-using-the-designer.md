### <a name="to-add-a-table-to-the-datagrid-control-in-the-designer"></a><span data-ttu-id="09a00-113">在设计器中向 DataGrid 控件添加表</span><span class="sxs-lookup"><span data-stu-id="09a00-113">To add a table to the DataGrid control in the designer</span></span>

1. <span data-ttu-id="09a00-114">若要在表中显示数据，必须首先将 <xref:System.Windows.Forms.DataGrid> 控件绑定到数据集。</span><span class="sxs-lookup"><span data-stu-id="09a00-114">In order to display data in the table, you must first bind the <xref:System.Windows.Forms.DataGrid> control to a dataset.</span></span> <span data-ttu-id="09a00-115">有关详细信息，请参阅[如何：使用设计器将 Windows 窗体 DataGrid 控件绑定到数据源](bind-wf-datagrid-control-to-a-data-source-using-the-designer.md)。</span><span class="sxs-lookup"><span data-stu-id="09a00-115">For more information, see [How to: Bind the Windows Forms DataGrid Control to a Data Source Using the Designer](bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span></span>

2. <span data-ttu-id="09a00-116">选择属性窗口中 <xref:System.Windows.Forms.DataGrid> 控件的 <xref:System.Windows.Forms.DataGrid.TableStyles%2A> 属性，然后单击省略号按钮（![中的省略号按钮（...），以显示 " **DataGridTableStyle 集合编辑器**" 的属性旁的 "属性窗口"。</span><span class="sxs-lookup"><span data-stu-id="09a00-116">Select the <xref:System.Windows.Forms.DataGrid> control's <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property in the Properties window, and then click the ellipsis button (![The Ellipsis button (...) in the Properties window of Visual Studio.](./media/visual-studio-ellipsis-button.png)) next to the property to display the **DataGridTableStyle Collection Editor**.</span></span>

3. <span data-ttu-id="09a00-117">在集合编辑器中，单击 "**添加**" 以插入表样式。</span><span class="sxs-lookup"><span data-stu-id="09a00-117">In the collection editor, click **Add** to insert a table style.</span></span>

4. <span data-ttu-id="09a00-118">单击 **"确定"** 关闭 "集合编辑器"，然后单击 "<xref:System.Windows.Forms.DataGrid.TableStyles%2A>" 属性旁边的省略号按钮以重新打开它。</span><span class="sxs-lookup"><span data-stu-id="09a00-118">Click **OK** to close the collection editor, and then reopen it by clicking the ellipsis button next to the <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property.</span></span>

     <span data-ttu-id="09a00-119">重新打开 "集合编辑器" 时，绑定到该控件的任何数据表都将出现在表样式的 "<xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A>" 属性的下拉列表中。</span><span class="sxs-lookup"><span data-stu-id="09a00-119">When you reopen the collection editor, any data tables bound to the control will appear in the drop-down list for the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> property of the table style.</span></span>

5. <span data-ttu-id="09a00-120">在集合编辑器的 "**成员**" 框中，单击表样式。</span><span class="sxs-lookup"><span data-stu-id="09a00-120">In the **Members** box of the collection editor, click the table style.</span></span>

6. <span data-ttu-id="09a00-121">在 "集合编辑器" 的 "**属性**" 框中，选择要显示的表的 <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> 值。</span><span class="sxs-lookup"><span data-stu-id="09a00-121">In the **Properties** box of the collection editor, select the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> value for the table you want to display.</span></span>

2. <span data-ttu-id="09a00-125">在集合编辑器中，单击 "**添加**" 以插入列样式，或者单击 "**添加**" 旁边的向下箭头以指定列类型。</span><span class="sxs-lookup"><span data-stu-id="09a00-125">In the collection editor, click **Add** to insert a column style or click the down arrow next to **Add** to specify a column type.</span></span>
