# <a name="combobox-control-windows-forms"></a>ComboBox 컨트롤(Windows Forms)
"Windows 窗体" `ComboBox` 控件用于在下拉组合框中显示数据。 默认情况下，`ComboBox` 控件出现在两个部分中：顶部是一个文本框，用户可以在其中键入列表项。 第二部分是显示项列表的列表框，用户可以从中选择一个项。
