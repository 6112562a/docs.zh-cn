# <a name="button-control-windows-forms"></a>Button 控件（Windows 窗体）
Windows 窗体 `Button` 控件允许用户通过单击来执行某项操作。 `Button` 控件可以显示文本和图像。 单击该按钮时，看上去它像是被按下并释放。

 说明如何将 `Button` 控件指定为接受按钮（也称为默认按钮）。
