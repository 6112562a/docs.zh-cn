# <a name="mainmenu-component-windows-forms"></a>MainMenu 组件（Windows 窗体）
> [!NOTE]
> 尽管 `MenuStrip` 和 `ContextMenuStrip` 将功能替换并添加到以前版本的 `MainMenu` 和 <xref:System.Windows.Forms.ContextMenu> 控件中，但 `MainMenu` 和 <xref:System.Windows.Forms.ContextMenu> 会保留，以实现向后兼容性和将来使用（如果你选择）。  
  
 Windows 窗体 `MainMenu` 组件在运行时显示菜单。
