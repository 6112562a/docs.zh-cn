### <a name="to-display-the-current-date-at-the-bottom-of-the-control"></a><span data-ttu-id="ac592-117">若要在控件的底部显示当前日期</span><span class="sxs-lookup"><span data-stu-id="ac592-117">To display the current date at the bottom of the control</span></span>
