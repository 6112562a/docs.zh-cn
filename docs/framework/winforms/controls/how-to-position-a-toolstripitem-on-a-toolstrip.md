# <a name="how-to-position-a-toolstripitem-on-a-toolstrip"></a>如何：将 ToolStripItem 置于 ToolStrip
也可以移动或添加<xref:System.Windows.Forms.ToolStripItem>左侧或右侧<xref:System.Windows.Forms.ToolStrip>。  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-left-side-of-a-toolstrip"></a>若要移动或将 ToolStripItem 添加到左侧和右侧的 ToolStrip  
  
1.  设置<xref:System.Windows.Forms.ToolStripItemAlignment>的属性<xref:System.Windows.Forms.ToolStripItem>到<xref:System.Windows.Forms.ToolStripItemAlignment.Left>。  
  
### <a name="to-move-or-add-a-toolstripitem-to-the-right-side-of-a-toolstrip"></a>若要移动或 ToolStrip 的右侧添加 ToolStripItem  
  
1.  设置<xref:System.Windows.Forms.ToolStripItemAlignment>的属性<xref:System.Windows.Forms.ToolStripItem>到<xref:System.Windows.Forms.ToolStripItemAlignment.Right>。
