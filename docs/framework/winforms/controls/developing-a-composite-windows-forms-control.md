# <a name="develop-a-composite-windows-forms-control"></a>开发复合 Windows 窗体控件

다른 Windows Forms 컨트롤을 결합하여 복합 Windows Forms 컨트롤을 개발할 수 있습니다. 派生自 <xref:System.Web.UI.UserControl> 的复合控件称为用户控件。 기본 클래스 <xref:System.Windows.Forms.UserControl>은 자식 컨트롤에 대한 키보드 라우팅을 제공하므로 자식 컨트롤이 포커스를 받을 수 있습니다. 有关用户控件的示例，请参阅[如何：在 Windows 窗体控件中应用特性](how-to-apply-attributes-in-windows-forms-controls.md)中的 <xref:System.Windows.Forms.UserControl> 示例。
