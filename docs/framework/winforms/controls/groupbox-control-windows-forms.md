# <a name="groupbox-control-windows-forms"></a>GroupBox 컨트롤(Windows Forms)
Windows 窗体 <xref:System.Windows.Forms.GroupBox> 控件用于为其他控件提供可识别的分组。 通常，使用分组框按函数细分窗体。 例如，你可能有一个订单窗体，它指定邮寄选项，例如要使用哪一整夜运输公司。 对分组框中的所有选项进行分组可为用户提供逻辑视觉提示。 <xref:System.Windows.Forms.GroupBox> 控件类似于 <xref:System.Windows.Forms.Panel> 控件;但是，只有 <xref:System.Windows.Forms.GroupBox> 控件显示标题，并且只有 <xref:System.Windows.Forms.Panel> 控件才能有滚动条。
