# <a name="sqlstreamchars-constructor"></a>SqlStreamChars 构造函数

初始化 `SqlStreamChars` 类的新实例。 包含此构造函数的程序集具有与 SQLAccess.dll 友元关系。 它被用于 SQL server 上。 对于其他数据库，使用提供该数据库的宿主机制。

```csharp
protected SqlStreamChars ();
```

## <a name="remarks"></a>备注

> [!WARNING]
> `SqlStreamChars`构造函数受到保护，不适合在代码中直接使用。
>
> Microsoft 不支持在生产应用程序在任何情况下使用此字段。
