# <a name="connectiongroup-class"></a>ConnectionGroup 类

类`ConnectionGroup`对<xref:System.Net.ServicePoint>上下文中的连接列表进行分组，用于维护网络资源的上下文（例如，代理和单独的客户端）。

## <a name="syntax"></a>语法
  
```csharp  
internal class ConnectionGroup
```

> [!WARNING]
> 该`ConnectionGroup`类是内部的，不应直接在代码中使用。
>
> 在任何情况下，Microsoft 都不支持在生产应用程序中使用此类。

## <a name="requirements"></a>要求

**命名空间：**<xref:System.Net>

**装配：** 系统（系统中）

**.NET 框架版本：** 自 2.0 起可用。
