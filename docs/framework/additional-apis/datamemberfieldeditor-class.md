# <a name="datamemberfieldeditor-class"></a>DataMemberFieldEditor 类

通过列出要从中选择值源的 `DataSource` 对象的所有属性，为编辑数据绑定对象的属性提供用户界面。  
  
## <a name="syntax"></a>语法
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> `DataMemberFieldEditor` 类是内部的，不应在代码中直接使用。
> 
> 在任何情况下，Microsoft 不支持在生产应用程序中使用此类。
