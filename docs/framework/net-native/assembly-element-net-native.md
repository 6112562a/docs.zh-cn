## <a name="remarks"></a><span data-ttu-id="cac9c-178">备注</span><span class="sxs-lookup"><span data-stu-id="cac9c-178">Remarks</span></span>  
 <span data-ttu-id="cac9c-179">`<Assembly>` 元素为一个程序集中的所有类型定义运行时策略。</span><span class="sxs-lookup"><span data-stu-id="cac9c-179">The `<Assembly>` element defines runtime policy for all the types in an assembly.</span></span> <span data-ttu-id="cac9c-180">它不同于 [\<Library>](library-element-net-native.md) 元素，后者指定一个库但依赖其子元素来定义运行时反射策略。</span><span class="sxs-lookup"><span data-stu-id="cac9c-180">It differs from the [\<Library>](library-element-net-native.md) element, which specifies a library but depends on its child elements to define runtime reflection policy.</span></span> <span data-ttu-id="cac9c-181">`<Assembly>` 元素将应用到一个程序集中的所有类型，除非这些类型遭到一个子元素的替代。</span><span class="sxs-lookup"><span data-stu-id="cac9c-181">The `<Assembly>` element applies to all the types in an assembly unless they are overridden by a child element.</span></span>  
  
 <span data-ttu-id="cac9c-182">以下示例展示了该如何通过分配给 `Name` 特性一个“\*Application\*”值，从而将运行时策略应用到应用包内的程序集中的所有类型。</span><span class="sxs-lookup"><span data-stu-id="cac9c-182">The following example shows how you can apply runtime policy to all the types in assemblies within your app package by assigning the `Name` attribute a value of "\*Application\*".</span></span> <span data-ttu-id="cac9c-183">`<Assembly>` 元素必须是 [\<Application>](application-element-net-native.md) 元素的子元素。</span><span class="sxs-lookup"><span data-stu-id="cac9c-183">The `<Assembly>` element must be a child of the [\<Application>](application-element-net-native.md) element.</span></span>  
  
```xml  
<Directives xmlns="http://schemas.microsoft.com/netfx/2013/01/metadata">   
  <Application>   
     <Assembly Name="*Application*" Dynamic="Required All" />   
  </Application>   
</Directives>  
```  
  
 <span data-ttu-id="cac9c-184">`Activate`、`Browse`、`Dynamic` 和 `Serialize` 特性都是可选项。</span><span class="sxs-lookup"><span data-stu-id="cac9c-184">The `Activate`, `Browse`, `Dynamic`, and `Serialize` attributes are all optional.</span></span> <span data-ttu-id="cac9c-185">然而，`<Assembly>` 元素必须至少包含这些属性中的一个。</span><span class="sxs-lookup"><span data-stu-id="cac9c-185">However, the `<Assembly>` element must contain at least one of these attributes.</span></span>
