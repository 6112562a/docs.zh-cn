# <a name="extending-windows-workflow-foundation"></a>扩展 Windows Workflow Foundation
以下部分介绍如何在 Visual Studio 2010 以外的重新承载环境中将 Windows Workflow Foundation （WF）与自定义活动和设计器进行扩展。

## <a name="in-this-section"></a>本节内容
 [自定义工作流设计体验](customizing-the-workflow-design-experience.md)指示设计自定义活动和重新承载 Windows 工作流设计器的方案如何在 .NET Framework 4 中大大简化。 由于新的活动设计器编程模型是基于 Windows Presentation Foundation （WPF）构建的，因此开发和部署现在更容易且更灵活。
