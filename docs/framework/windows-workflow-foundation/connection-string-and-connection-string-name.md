# <a name="connection-string-and-connection-string-name"></a>连接字符串和连接字符串名称
**连接字符串**属性指定 SQL 工作流实例存储应使用连接到持久性数据库的连接字符串。 此参数为可选参数。 **连接字符串名称**属性指定 SQL 工作流实例存储应使用连接到持久性数据库的已命名的连接字符串的名称。 此参数为可选参数。 如果不希望 SQL 工作流实例存储区以使用默认命名连接字符串应指定连接字符串名称属性或连接字符串属性的值**DefaultSqlWorkflowInstanceStoreConnectionString**.
