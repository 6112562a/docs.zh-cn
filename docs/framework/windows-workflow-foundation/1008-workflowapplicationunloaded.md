# <a name="1008---workflowapplicationunloaded"></a>1008 - WorkflowApplicationUnloaded
## <a name="properties"></a>属性  
  
|||  
|-|-|  
|Id|1008|  
|关键字|WFRuntime|  
|级别|信息|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述  
 指示工作流应用程序已卸载。  
  
## <a name="message"></a>消息  
 WorkflowInstance Id“%1”已卸载。
