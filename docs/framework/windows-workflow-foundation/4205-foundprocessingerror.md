# <a name="4205---foundprocessingerror"></a>4205 - FoundProcessingError

|||  
|-|-|  
|Id|4205|  
|关键字|WFInstanceStore|  
|级别|Error|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|
