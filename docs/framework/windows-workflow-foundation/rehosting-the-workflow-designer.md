# <a name="rehosting-the-workflow-designer"></a>重新承载工作流设计器
可在 Visual Studio 2012 之外的环境中重新承载 Windows 工作流设计器，以创建、修改和监视工作流。

 <xref:System.Activities.Presentation.WorkflowDesigner> 类型是画布、属性网格和其他元素的包装，它公开一个基本的编程模型，用于处理大多数设计器重新承载方案。 在 Windows Presentation Foundation （WPF）应用程序中承载 <xref:System.Activities.Presentation.WorkflowDesigner> 是工作流设计器的常见重新承载方案。
