 [在 Windows 中跟踪](tracking-events-into-event-tracing-in-windows.md)事件跟踪事件演示如何在工作流服务上启用 [!INCLUDE[wf1](../../../../includes/wf1-md.md)] 跟踪，并在 Windows 事件跟踪（ETW）中发出跟踪事件。
