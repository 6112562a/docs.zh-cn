# <a name="441--stopsignpostevent1"></a>441- StopSignpostEvent1

|||  
|-|-|  
|Id|441|  
|关键字|疑难解答|  
|级别|信息|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/分析|  
  
## <a name="description"></a>描述  
 在活动跟踪中，指示消息是否已完成跨越发送或接收中的活动边界。  
  
## <a name="message"></a>消息  
 活动边界。
