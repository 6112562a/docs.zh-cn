# <a name="bitmap-effects-overview"></a><span data-ttu-id="17a3a-102">位图效果概述</span><span class="sxs-lookup"><span data-stu-id="17a3a-102">Bitmap Effects Overview</span></span>
<span data-ttu-id="17a3a-103">使用位图效果，设计人员和开发人员可以将视觉效果应用于呈现的 Windows Presentation Foundation （WPF）内容。</span><span class="sxs-lookup"><span data-stu-id="17a3a-103">Bitmap effects enable designers and developers to apply visual effects to rendered Windows Presentation Foundation (WPF) content.</span></span> <span data-ttu-id="17a3a-104">例如，位图效果允许您轻松地将 <xref:System.Windows.Media.Effects.DropShadowBitmapEffect> 效果或模糊效果应用于图像或按钮。</span><span class="sxs-lookup"><span data-stu-id="17a3a-104">For example, bitmap effects allow you to easily apply a <xref:System.Windows.Media.Effects.DropShadowBitmapEffect> effect or a blur effect to an image or a button.</span></span>

> <span data-ttu-id="17a3a-105">在 .NET Framework 4 或更高版本中，<xref:System.Windows.Media.Effects.BitmapEffect> 类已过时。</span><span class="sxs-lookup"><span data-stu-id="17a3a-105">In the .NET Framework 4 or later, the <xref:System.Windows.Media.Effects.BitmapEffect> class is obsolete.</span></span> <span data-ttu-id="17a3a-106">如果尝试使用 <xref:System.Windows.Media.Effects.BitmapEffect> 类，将会收到过时的异常。</span><span class="sxs-lookup"><span data-stu-id="17a3a-106">If you try to use the <xref:System.Windows.Media.Effects.BitmapEffect> class, you will get an obsolete exception.</span></span> <span data-ttu-id="17a3a-107"><xref:System.Windows.Media.Effects.BitmapEffect> 类的未过时替代项是 <xref:System.Windows.Media.Effects.Effect> 类。</span><span class="sxs-lookup"><span data-stu-id="17a3a-107">The non-obsolete alternative to the <xref:System.Windows.Media.Effects.BitmapEffect> class is the <xref:System.Windows.Media.Effects.Effect> class.</span></span> <span data-ttu-id="17a3a-108">在大多数情况下，<xref:System.Windows.Media.Effects.Effect> 类的速度要快得多。</span><span class="sxs-lookup"><span data-stu-id="17a3a-108">In most situations, the <xref:System.Windows.Media.Effects.Effect> class is significantly faster.</span></span>

## <a name="wpf-bitmap-effects"></a><span data-ttu-id="17a3a-109">WPF 位图效果</span><span class="sxs-lookup"><span data-stu-id="17a3a-109">WPF Bitmap Effects</span></span>  
 <span data-ttu-id="17a3a-110">位图效果（<xref:System.Windows.Media.Effects.BitmapEffect> 对象）是简单的像素处理操作。</span><span class="sxs-lookup"><span data-stu-id="17a3a-110">Bitmap effects (<xref:System.Windows.Media.Effects.BitmapEffect> object) are simple pixel processing operations.</span></span> <span data-ttu-id="17a3a-111">位图效果会将 <xref:System.Windows.Media.Imaging.BitmapSource> 作为输入，并在应用效果后生成新 <xref:System.Windows.Media.Imaging.BitmapSource>，如模糊或投影。</span><span class="sxs-lookup"><span data-stu-id="17a3a-111">A bitmap effect takes a <xref:System.Windows.Media.Imaging.BitmapSource> as an input and produces a new <xref:System.Windows.Media.Imaging.BitmapSource> after applying the effect, such as a blur or drop shadow.</span></span> <span data-ttu-id="17a3a-112">每个位图效果都公开了可控制筛选属性的属性，如 <xref:System.Windows.Media.Effects.BlurBitmapEffect>的 <xref:System.Windows.Media.Effects.BlurBitmapEffect.Radius%2A>。</span><span class="sxs-lookup"><span data-stu-id="17a3a-112">Each bitmap effect exposes properties that can control the filtering properties, such as <xref:System.Windows.Media.Effects.BlurBitmapEffect.Radius%2A> of <xref:System.Windows.Media.Effects.BlurBitmapEffect>.</span></span>  
  
 <span data-ttu-id="17a3a-113">作为一种特殊情况，在 [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]中，可以将效果设置为实时 <xref:System.Windows.Media.Visual> 对象（如 <xref:System.Windows.Controls.Button> 或 <xref:System.Windows.Controls.TextBox>）的属性。</span><span class="sxs-lookup"><span data-stu-id="17a3a-113">As a special case, in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)], effects can be set as properties on live <xref:System.Windows.Media.Visual> objects, such as a <xref:System.Windows.Controls.Button> or <xref:System.Windows.Controls.TextBox>.</span></span> <span data-ttu-id="17a3a-114">像素处理在运行时应用并呈现。</span><span class="sxs-lookup"><span data-stu-id="17a3a-114">The pixel processing is applied and rendered at run-time.</span></span> <span data-ttu-id="17a3a-115">在这种情况下，在呈现时，<xref:System.Windows.Media.Visual> 会自动转换为其 <xref:System.Windows.Media.Imaging.BitmapSource> 等效项，并作为输入送进 <xref:System.Windows.Media.Effects.BitmapEffect>。</span><span class="sxs-lookup"><span data-stu-id="17a3a-115">In this case, at the time of rendering, a <xref:System.Windows.Media.Visual> is automatically converted to its <xref:System.Windows.Media.Imaging.BitmapSource> equivalent and is fed as input to the <xref:System.Windows.Media.Effects.BitmapEffect>.</span></span> <span data-ttu-id="17a3a-116">输出替换 <xref:System.Windows.Media.Visual> 对象的默认呈现行为。</span><span class="sxs-lookup"><span data-stu-id="17a3a-116">The output replaces the <xref:System.Windows.Media.Visual> object's default rendering behavior.</span></span> <span data-ttu-id="17a3a-117">这就是 <xref:System.Windows.Media.Effects.BitmapEffect> 对象强制仅在软件中呈现视觉对象的原因，即，在应用效果时不会对视觉对象进行硬件加速。</span><span class="sxs-lookup"><span data-stu-id="17a3a-117">This is why <xref:System.Windows.Media.Effects.BitmapEffect> objects force visuals to render in software only i.e. no hardware acceleration on visuals when effects are applied.</span></span>  
  
- <span data-ttu-id="17a3a-118"><xref:System.Windows.Media.Effects.BlurBitmapEffect> 模拟显得不到焦点的对象。</span><span class="sxs-lookup"><span data-stu-id="17a3a-118"><xref:System.Windows.Media.Effects.BlurBitmapEffect> simulates an object that appears out-of-focus.</span></span>  
  
- <span data-ttu-id="17a3a-119"><xref:System.Windows.Media.Effects.OuterGlowBitmapEffect> 在对象的周围创建颜色光环。</span><span class="sxs-lookup"><span data-stu-id="17a3a-119"><xref:System.Windows.Media.Effects.OuterGlowBitmapEffect> creates a halo of color around the perimeter of an object.</span></span>  
  
- <span data-ttu-id="17a3a-120"><xref:System.Windows.Media.Effects.DropShadowBitmapEffect> 在对象后面创建阴影。</span><span class="sxs-lookup"><span data-stu-id="17a3a-120"><xref:System.Windows.Media.Effects.DropShadowBitmapEffect> creates a shadow behind an object.</span></span>  
  
- <span data-ttu-id="17a3a-121"><xref:System.Windows.Media.Effects.BevelBitmapEffect> 创建一个斜角，它根据指定的曲线提升图像表面。</span><span class="sxs-lookup"><span data-stu-id="17a3a-121"><xref:System.Windows.Media.Effects.BevelBitmapEffect> creates a bevel which raises the surface of an image according to a specified curve.</span></span>  
  
- <span data-ttu-id="17a3a-122"><xref:System.Windows.Media.Effects.EmbossBitmapEffect> 创建一个 <xref:System.Windows.Media.Visual> 的凹凸映射，以获得来自人工光源的深度和纹理。</span><span class="sxs-lookup"><span data-stu-id="17a3a-122"><xref:System.Windows.Media.Effects.EmbossBitmapEffect> creates a bump mapping of a <xref:System.Windows.Media.Visual> to give the impression of depth and texture from an artificial light source.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="17a3a-123">WPF 位图效果在软件模式下呈现。</span><span class="sxs-lookup"><span data-stu-id="17a3a-123">WPF bitmap effects are rendered in software mode.</span></span> <span data-ttu-id="17a3a-124">应用效果的任何对象也都将以软件形式呈现。</span><span class="sxs-lookup"><span data-stu-id="17a3a-124">Any object that applies an effect will also be rendered in software.</span></span> <span data-ttu-id="17a3a-125">在大型视觉对象上使用位图效果或对位图效果的属性进行动画处理时，性能的下降幅度最大。</span><span class="sxs-lookup"><span data-stu-id="17a3a-125">Performance is degraded the most when using Bitmap effects on large visuals or animating properties of a Bitmap effect.</span></span> <span data-ttu-id="17a3a-126">这并不表示完全不应该以这种方式使用位图效果，而是应谨慎使用，并进行全面测试以确保用户得到预期的体验。</span><span class="sxs-lookup"><span data-stu-id="17a3a-126">This is not to say that you should not use Bitmap effects in this way at all, but you should use caution and test thoroughly to ensure that your users are getting the experience you expect.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="17a3a-127">WPF 位图效果不支持部分信任的执行。</span><span class="sxs-lookup"><span data-stu-id="17a3a-127">WPF bitmap effects do not support partial trust execution.</span></span> <span data-ttu-id="17a3a-128">应用程序必须具有完全信任权限才能使用位图效果。</span><span class="sxs-lookup"><span data-stu-id="17a3a-128">An application must have full trust permissions to use bitmap effects.</span></span>
