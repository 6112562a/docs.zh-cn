# <a name="binding-markup-extension"></a><span data-ttu-id="fcf9a-102">绑定标记扩展</span><span class="sxs-lookup"><span data-stu-id="fcf9a-102">Binding Markup Extension</span></span>
<span data-ttu-id="fcf9a-103">将属性值延迟为数据绑定值，创建中间表达式对象并在运行时解释应用于该元素及其绑定的数据上下文。</span><span class="sxs-lookup"><span data-stu-id="fcf9a-103">Defers a property value to be a data-bound value, creating an intermediate expression object and interpreting the data context that applies to the element and its binding at run time.</span></span>  
  
## <a name="binding-expression-usage"></a><span data-ttu-id="fcf9a-104">绑定表达式使用情况</span><span class="sxs-lookup"><span data-stu-id="fcf9a-104">Binding Expression Usage</span></span>  
  
```xaml  
<object property="{Binding}" .../>  
-or-  
<object property="{Binding  bindProp1=value1[, bindPropN=valueN]*}" ...  
/>  
-or-  
<object property="{Binding path}" .../>  
-or  
<object property="{Binding path[, bindPropN=valueN]*}" .../>  
```
