# <a name="label"></a>Label
<xref:System.Windows.Controls.Label> 控件通常在 [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)]中提供信息。  过去，<xref:System.Windows.Controls.Label> 仅包含文本，但由于 [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] 附带的 <xref:System.Windows.Controls.Label> 是 <xref:System.Windows.Controls.ContentControl>，因此它可以包含文本或 <xref:System.Windows.UIElement>。  
  
 <xref:System.Windows.Controls.Label> 提供对访问密钥的功能和视觉支持。 它经常用于启用对控件（如 <xref:System.Windows.Controls.TextBox>）的快速键盘访问。 若要将 <xref:System.Windows.Controls.Label> 分配到 <xref:System.Windows.Controls.Control>，请将 <xref:System.Windows.Controls.Label.Target%2A?displayProperty=nameWithType> 属性设置为当用户按下访问键时应获得焦点的控件。  
  
 下图显示了以 <xref:System.Windows.Controls.ComboBox>为目标 <xref:System.Windows.Controls.Label> "主题"。  当用户按下时，<xref:System.Windows.Controls.ComboBox> 将接收焦点。  有关详细信息，请参阅[如何：设置标签的目标属性](https://docs.microsoft.com/previous-versions/dotnet/netframework-3.5/ms752101(v=vs.90))。
