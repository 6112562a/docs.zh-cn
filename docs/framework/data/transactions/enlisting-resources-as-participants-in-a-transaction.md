<span data-ttu-id="dc9fe-112">资源管理器的持久性（反之为可变性）是指资源管理器是否支持故障恢复。</span><span class="sxs-lookup"><span data-stu-id="dc9fe-112">The durability (or conversely the volatility) of a resource manager refers
