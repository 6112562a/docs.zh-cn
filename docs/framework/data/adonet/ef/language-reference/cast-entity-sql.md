## <a name="remarks"></a><span data-ttu-id="a0c79-115">备注</span><span class="sxs-lookup"><span data-stu-id="a0c79-115">Remarks</span></span>  
 <span data-ttu-id="a0c79-116">强制转换表达式的语义与 [!INCLUDE[tsql](../../../../../../includes/tsql-md.md)] CONVERT 表达式类似。</span><span class="sxs-lookup"><span data-stu-id="a0c79-116">The cast expression has similar semantics to the [!INCLUDE[tsql](../../../../../../includes/tsql-md.md)] CONVERT expression.</span></span> <span data-ttu-id="a0c79-117">强制转换表达式用于将一种类型的值转换为另一种类型的值。</span><span class="sxs-lookup"><span data-stu-id="a0c79-117">The cast expression is used to convert a value of one type into a value of another type.</span></span>  
  
```  
CAST( e as T )  
```  
  
 <span data-ttu-id="a0c79-118">如果 e 具有某种类型 S，且 S 可转换为 T，则上面的表达式是有效的强制转换表达式。</span><span class="sxs-lookup"><span data-stu-id="a0c79-118">If e is of some type S, and S is convertible to T, then the above expression is a valid cast expression.</span></span> <span data-ttu-id="a0c79-119">T 必须为基元（标量）类型。</span><span class="sxs-lookup"><span data-stu-id="a0c79-119">T must be a primitive (scalar) type.</span></span>  
  
 <span data-ttu-id="a0c79-120">精度和标量方面的值可以选择在强制转换为 `Edm.Decimal`时提供。</span><span class="sxs-lookup"><span data-stu-id="a0c79-120">Values for the precision and scale facets may optionally be provided when casting to `Edm.Decimal`.</span></span> <span data-ttu-id="a0c79-121">如果未显式提供，则精度和标量的默认值分别为 18 和 0。</span><span class="sxs-lookup"><span data-stu-id="a0c79-121">If not explicitly provided, the default values for precision and scale are 18 and 0, respectively.</span></span> <span data-ttu-id="a0c79-122">具体而言， `Decimal`支持以下重载：</span><span class="sxs-lookup"><span data-stu-id="a0c79-122">Specifically, the following overloads are supported for `Decimal`:</span></span>  
  
- `CAST( d as Edm.Decimal );`  
  
- `CAST( d as Edm.Decimal(precision) );`  
  
- `CAST( d as Edm.Decimal(precision, scale) );`  
  
 <span data-ttu-id="a0c79-123">使用强制转换表达式视为显式转换。</span><span class="sxs-lookup"><span data-stu-id="a0c79-123">The use of a cast expression is considered an explicit conversion.</span></span> <span data-ttu-id="a0c79-124">显式转换可能截断数据或丧失精度。</span><span class="sxs-lookup"><span data-stu-id="a0c79-124">Explicit conversions might truncate data or lose precision.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="a0c79-125">仅对基元类型和枚举成员类型支持 CAST。</span><span class="sxs-lookup"><span data-stu-id="a0c79-125">CAST is only supported over primitive types and enumeration member types.</span></span>
