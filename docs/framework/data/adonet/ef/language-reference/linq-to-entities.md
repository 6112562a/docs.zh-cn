## <a name="query-conversion"></a><span data-ttu-id="69f08-129">查询转换</span><span class="sxs-lookup"><span data-stu-id="69f08-129">Query Conversion</span></span>  
 <span data-ttu-id="69f08-130">若要针对实体框架执行 LINQ to Entities 查询，必须将 LINQ 查询转换为可针对实体框架执行的命令目录树表示形式。</span><span class="sxs-lookup"><span data-stu-id="69f08-130">To execute a LINQ to Entities query against the Entity Framework, the LINQ query must be converted to a command tree representation that can be executed against the Entity Framework.</span></span>  
  
 <span data-ttu-id="69f08-131">LINQ to Entities 查询组成 LINQ 标准查询运算符 (如<xref:System.Linq.Queryable.Select%2A>， <xref:System.Linq.Queryable.Where%2A>，和<xref:System.Linq.Queryable.GroupBy%2A>) 和 (x > 10、 10、contact.lastname 等等） 的表达式。</span><span class="sxs-lookup"><span data-stu-id="69f08-131">LINQ to Entities queries are comprised of LINQ standard query operators (such as <xref:System.Linq.Queryable.Select%2A>, <xref:System.Linq.Queryable.Where%2A>, and <xref:System.Linq.Queryable.GroupBy%2A>) and expressions (x > 10, Contact.LastName, and so on).</span></span> <span data-ttu-id="69f08-132">LINQ 运算符并非由类定义，而是由类中的方法定义。</span><span class="sxs-lookup"><span data-stu-id="69f08-132">LINQ operators are not defined by a class, but rather are methods on a class.</span></span> <span data-ttu-id="69f08-133">在 LINQ 中，表达式可包含 <xref:System.Linq.Expressions> 命名空间内的类型所允许的任何内容，并且，通过扩展，可包含可在 lambda 函数中表示的任何内容。</span><span class="sxs-lookup"><span data-stu-id="69f08-133">In LINQ, expressions can contain anything allowed by types within the <xref:System.Linq.Expressions> namespace and, by extension, anything that can be represented in a lambda function.</span></span> <span data-ttu-id="69f08-134">这是实体框架允许的表达式的超集，这些表达式在定义上限于数据库所允许的操作，并且受到 <xref:System.Data.Objects.ObjectQuery%601> 支持。</span><span class="sxs-lookup"><span data-stu-id="69f08-134">This is a superset of the expressions that are allowed by the Entity Framework, which are by definition restricted to operations allowed on the database, and supported by <xref:System.Data.Objects.ObjectQuery%601>.</span></span>  
  
 <span data-ttu-id="69f08-135">在实体框架中，运算符和表达式同时由单一类型层次结构表示，这些运算符和表达式随后会放入命令目录树中。</span><span class="sxs-lookup"><span data-stu-id="69f08-135">In the Entity Framework, both operators and expressions are represented by a single type hierarchy, which are then placed in a command tree.</span></span> <span data-ttu-id="69f08-136">实体框架使用命令目录树来执行此查询。</span><span class="sxs-lookup"><span data-stu-id="69f08-136">The command tree is used by the Entity Framework to execute the query.</span></span> <span data-ttu-id="69f08-137">如果 LINQ 查询无法表示为命令目录树，则当转换查询时，将引发异常。</span><span class="sxs-lookup"><span data-stu-id="69f08-137">If the LINQ query cannot be expressed as a command tree, an exception will be thrown when the query is being converted.</span></span> <span data-ttu-id="69f08-138">LINQ to Entities 查询的转换涉及两个子转换：标准查询运算符的转换和表达式转换。</span><span class="sxs-lookup"><span data-stu-id="69f08-138">The conversion of LINQ to Entities queries involves two sub-conversions: the conversion of the standard query operators, and the conversion of the expressions.</span></span>  
  
 <span data-ttu-id="69f08-139">一些 LINQ 标准查询运算符在 LINQ to Entities 中没有有效的转换。</span><span class="sxs-lookup"><span data-stu-id="69f08-139">There are a number of LINQ standard query operators that do not have a valid translation in LINQ to Entities.</span></span> <span data-ttu-id="69f08-140">尝试使用这些运算符将在查询转换期间导致异常。</span><span class="sxs-lookup"><span data-stu-id="69f08-140">Attempts to use these operators will result in an exception at query translation time.</span></span> <span data-ttu-id="69f08-141">有关支持的 LINQ to Entities 运算符的列表，请参阅[支持和不受支持的 LINQ 方法 (LINQ to Entities)](../../../../../../docs/framework/data/adonet/ef/language-reference/supported-and-unsupported-linq-methods-linq-to-entities.md)。</span><span class="sxs-lookup"><span data-stu-id="69f08-141">For a list of supported LINQ to Entities operators, see [Supported and Unsupported LINQ Methods (LINQ to Entities)](../../../../../../docs/framework/data/adonet/ef/language-reference/supported-and-unsupported-linq-methods-linq-to-entities.md).</span></span>  
  
 <span data-ttu-id="69f08-142">有关使用 LINQ to Entities 中的标准查询运算符的详细信息，请参阅[LINQ to Entities 查询中的标准查询运算符](../../../../../../docs/framework/data/adonet/ef/language-reference/standard-query-operators-in-linq-to-entities-queries.md)。</span><span class="sxs-lookup"><span data-stu-id="69f08-142">For more information about using the standard query operators in LINQ to Entities, see [Standard Query Operators in LINQ to Entities Queries](../../../../../../docs/framework/data/adonet/ef/language-reference/standard-query-operators-in-linq-to-entities-queries.md).</span></span>
