# <a name="queries-in-linq-to-entities"></a><span data-ttu-id="f2928-102">LINQ to Entities 中的查询</span><span class="sxs-lookup"><span data-stu-id="f2928-102">Queries in LINQ to Entities</span></span>
<span data-ttu-id="f2928-103">查询是一种从数据源检索数据的表达式。</span><span class="sxs-lookup"><span data-stu-id="f2928-103">A query is an expression that retrieves data from a data source.</span></span> <span data-ttu-id="f2928-104">查询通常用专用查询语言表示，如用于关系数据库的 SQL 和用于 XML 的 XQuery。</span><span class="sxs-lookup"><span data-stu-id="f2928-104">Queries are usually expressed in a specialized query language, such as SQL for relational databases and XQuery for XML.</span></span> <span data-ttu-id="f2928-105">因此，开发人员对于他们查询的每种类型的数据源或数据格式，都不得不学习一种新的查询语言。</span><span class="sxs-lookup"><span data-stu-id="f2928-105">Therefore, developers have had to learn a new query language for each type of data source or data format that they query.</span></span> <span data-ttu-id="f2928-106">语言集成查询 (LINQ) 为跨各种数据源和格式处理数据提供了一种更简单的、一致的模型。</span><span class="sxs-lookup"><span data-stu-id="f2928-106">Language-Integrated Query (LINQ) offers a simpler, consistent model for working with data across various kinds of data sources and formats.</span></span> <span data-ttu-id="f2928-107">在 LINQ 查询中，您始终可以使用编程对象。</span><span class="sxs-lookup"><span data-stu-id="f2928-107">In a LINQ query, you always work with programming objects.</span></span>  
  
 <span data-ttu-id="f2928-108">LINQ 查询操作包含三个操作：获得一个或多个数据源、创建查询并执行查询。</span><span class="sxs-lookup"><span data-stu-id="f2928-108">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>  
  
 <span data-ttu-id="f2928-109">可以通过 LINQ 查询实现 <xref:System.Collections.Generic.IEnumerable%601> 泛型接口或 <xref:System.Linq.IQueryable%601> 泛型接口的数据源。</span><span class="sxs-lookup"><span data-stu-id="f2928-109">Data sources that implement the <xref:System.Collections.Generic.IEnumerable%601> generic interface or the <xref:System.Linq.IQueryable%601> generic interface can be queried through LINQ.</span></span> <span data-ttu-id="f2928-110">实现泛型<xref:System.Linq.IQueryable%601>接口的<xref:System.Data.Objects.ObjectQuery%601>泛型类的实例充当 LINQ 到实体查询的数据源。</span><span class="sxs-lookup"><span data-stu-id="f2928-110">Instances of the generic <xref:System.Data.Objects.ObjectQuery%601> class, which implements the generic <xref:System.Linq.IQueryable%601> interface, serve as the data source for LINQ to Entities queries.</span></span> <span data-ttu-id="f2928-111"><xref:System.Data.Objects.ObjectQuery%601> 泛型类表示一个查询，该查询返回零个或多个类型化对象的集合。</span><span class="sxs-lookup"><span data-stu-id="f2928-111">The <xref:System.Data.Objects.ObjectQuery%601> generic class represents a query that returns a collection of zero or more typed objects.</span></span> <span data-ttu-id="f2928-112">还可以让编译器使用 C# 关键字`var`（Visual Basic 中的 Dim）推断实体的类型。</span><span class="sxs-lookup"><span data-stu-id="f2928-112">You can also let the compiler infer the type of an entity by using the C# keyword `var` (Dim in Visual Basic).</span></span>  
  
 <span data-ttu-id="f2928-113">在查询中，您可以确切指定要从数据源检索哪些信息。</span><span class="sxs-lookup"><span data-stu-id="f2928-113">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="f2928-114">查询也可以指定返回信息之前信息的排序、分组和表现方式。</span><span class="sxs-lookup"><span data-stu-id="f2928-114">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="f2928-115">在 LINQ 中，查询存储在变量中。</span><span class="sxs-lookup"><span data-stu-id="f2928-115">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="f2928-116">如果查询返回一系列值，则查询变量本身必须为可查询的类型。</span><span class="sxs-lookup"><span data-stu-id="f2928-116">If the query returns a sequence of values, the query variable itself must be a queryable type.</span></span> <span data-ttu-id="f2928-117">此查询变量不执行任何操作，也不返回任何数据；它只存储查询信息。</span><span class="sxs-lookup"><span data-stu-id="f2928-117">This query variable takes no action and returns no data; it only stores the query information.</span></span> <span data-ttu-id="f2928-118">创建查询后必须执行该查询以检索任何数据。</span><span class="sxs-lookup"><span data-stu-id="f2928-118">After you create a query you must execute that query to retrieve any data.</span></span>  
  
## <a name="query-syntax"></a><span data-ttu-id="f2928-119">查询语法</span><span class="sxs-lookup"><span data-stu-id="f2928-119">Query Syntax</span></span>  
 <span data-ttu-id="f2928-120">可以通过两种语法编写 LINQ to Entities 查询：查询表达式语法和基于方法的查询语法。</span><span class="sxs-lookup"><span data-stu-id="f2928-120">LINQ to Entities queries can be composed in two different syntaxes: query expression syntax and method-based query syntax.</span></span> <span data-ttu-id="f2928-121">查询表达式语法是 C# 3.0 和 Visual Basic 9.0 中的新增功能，它由一组用类似于 Transact-SQL 或 XQuery 的声明性语法所编写的子句组成。</span><span class="sxs-lookup"><span data-stu-id="f2928-121">Query expression syntax is new in C# 3.0 and Visual Basic 9.0, and it consists of a set of clauses written in a declarative syntax similar to Transact-SQL or XQuery.</span></span> <span data-ttu-id="f2928-122">但是，.NET Framework 通用语言运行时 （CLR） 无法读取查询表达式语法本身。</span><span class="sxs-lookup"><span data-stu-id="f2928-122">However, the .NET Framework common language runtime (CLR) cannot read the query expression syntax itself.</span></span> <span data-ttu-id="f2928-123">因此，在编译时，查询表达式将转换为 CLR 能理解的形式，即方法调用。</span><span class="sxs-lookup"><span data-stu-id="f2928-123">Therefore, at compile time, query expressions are translated to something that the CLR does understand: method calls.</span></span> <span data-ttu-id="f2928-124">这些方法称为*标准查询运算符*。</span><span class="sxs-lookup"><span data-stu-id="f2928-124">These methods are known as the *standard query operators*.</span></span> <span data-ttu-id="f2928-125">作为开发人员，您可以选择使用方法语法而不使用查询语法直接调用这些方法。</span><span class="sxs-lookup"><span data-stu-id="f2928-125">As a developer, you have the option of calling them directly by using method syntax, instead of using query syntax.</span></span> <span data-ttu-id="f2928-126">有关详细信息，请参阅 [LINQ 中的查询语法和方法语法](../../../../../csharp/programming-guide/concepts/linq/query-syntax-and-method-syntax-in-linq.md)。</span><span class="sxs-lookup"><span data-stu-id="f2928-126">For more information, see [Query Syntax and Method Syntax in LINQ](../../../../../csharp/programming-guide/concepts/linq/query-syntax-and-method-syntax-in-linq.md).</span></span>
