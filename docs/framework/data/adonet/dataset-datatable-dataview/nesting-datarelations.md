# <a name="nesting-datarelations"></a><span data-ttu-id="dbf92-102">嵌套 DataRelation</span><span class="sxs-lookup"><span data-stu-id="dbf92-102">Nesting DataRelations</span></span>
<span data-ttu-id="dbf92-103">在数据的关系表示形式中，各个表都包含使用一个列或一组列来相互关联的行。</span><span class="sxs-lookup"><span data-stu-id="dbf92-103">In a relational representation of data, individual tables contain rows that are related to one another using a column or set of columns.</span></span> <span data-ttu-id="dbf92-104">在 ADO.NET <xref:System.Data.DataSet> 中，表之间的关系使用 <xref:System.Data.DataRelation> 来实现。</span><span class="sxs-lookup"><span data-stu-id="dbf92-104">In the ADO.NET <xref:System.Data.DataSet>, the relationship between tables is implemented using a <xref:System.Data.DataRelation>.</span></span> <span data-ttu-id="dbf92-105">当您创建**DataRelation**，仅通过关系管理列的父-子关系。</span><span class="sxs-lookup"><span data-stu-id="dbf92-105">When you create a **DataRelation**, the parent-child relationships of the columns are managed only through the relation.</span></span> <span data-ttu-id="dbf92-106">表和列是独立的实体。</span><span class="sxs-lookup"><span data-stu-id="dbf92-106">The tables and columns are separate entities.</span></span> <span data-ttu-id="dbf92-107">在 XML 提供的数据的分层表示形式中，父子关系通过包含嵌套子元素的父元素来表示。</span><span class="sxs-lookup"><span data-stu-id="dbf92-107">In the hierarchical representation of data that XML provides, the parent-child relationships are represented by parent elements that contain nested child elements.</span></span>  
  
 <span data-ttu-id="dbf92-108">为了便于子对象的嵌套时**数据集**与同步<xref:System.Xml.XmlDataDocument>或使用 XML 数据形式写入**WriteXml**，则**DataRelation**公开**嵌套**属性。</span><span class="sxs-lookup"><span data-stu-id="dbf92-108">To facilitate the nesting of child objects when a **DataSet** is synchronized with an <xref:System.Xml.XmlDataDocument> or written as XML data using **WriteXml**, the **DataRelation** exposes a **Nested** property.</span></span> <span data-ttu-id="dbf92-109">设置**嵌套**的属性**DataRelation**到**true**导致子行嵌套在父列时以 XML 数据形式编写的关系或与同步**XmlDataDocument**。</span><span class="sxs-lookup"><span data-stu-id="dbf92-109">Setting the **Nested** property of a **DataRelation** to **true** causes the child rows of the relation to be nested within the parent column when written as XML data or synchronized with an **XmlDataDocument**.</span></span> <span data-ttu-id="dbf92-110">**嵌套**的属性**DataRelation**是**false**，默认情况下。</span><span class="sxs-lookup"><span data-stu-id="dbf92-110">The **Nested** property of the **DataRelation** is **false**, by default.</span></span>  
  
 <span data-ttu-id="dbf92-111">例如，请考虑以下**数据集**。</span><span class="sxs-lookup"><span data-stu-id="dbf92-111">For example, consider the following **DataSet**.</span></span>  
  
```vb  
' Assumes connection is a valid SqlConnection.  
Dim customerAdapter As SqlDataAdapter = New SqlDataAdapter( _  
  "SELECT CustomerID, CompanyName FROM Customers", connection)  
Dim orderAdapter As SqlDataAdapter = New SqlDataAdapter( _  
  "SELECT OrderID, CustomerID, OrderDate FROM Orders", connection)  
  
connection.Open()  
  
Dim dataSet As DataSet = New DataSet("CustomerOrders")  
customerAdapter.Fill(dataSet, "Customers")  
orderAdapter.Fill(dataSet, "Orders")  
  
connection.Close()  
  
Dim customerOrders As DataRelation = dataSet.Relations.Add( _  
  "CustOrders", dataSet.Tables("Customers").Columns("CustomerID"), _  
  dataSet.Tables("Orders").Columns("CustomerID"))  
```  
  
```csharp  
// Assumes connection is a valid SqlConnection.  
SqlDataAdapter customerAdapter = new SqlDataAdapter(  
  "SELECT CustomerID, CompanyName FROM Customers", connection);  
SqlDataAdapter orderAdapter = new SqlDataAdapter(  
  "SELECT OrderID, CustomerID, OrderDate FROM Orders", connection);  
  
connection.Open();  
  
DataSet dataSet = new DataSet("CustomerOrders");  
customerAdapter.Fill(dataSet, "Customers");  
orderAdapter.Fill(dataSet, "Orders");  
  
connection.Close();  
  
DataRelation customerOrders = dataSet.Relations.Add(  
  "CustOrders", dataSet.Tables["Customers"].Columns["CustomerID"],  
  dataSet.Tables["Orders"].Columns["CustomerID"]);  
```  
  
 <span data-ttu-id="dbf92-112">因为**嵌套**的属性**DataRelation**对象未设置为**true**此**数据集**，不嵌套的子对象在父元素时这**数据集**表示为 XML 数据。</span><span class="sxs-lookup"><span data-stu-id="dbf92-112">Because the **Nested** property of the **DataRelation** object is not set to **true** for this **DataSet**, the child objects are not nested within the parent elements when this **DataSet** is represented as XML data.</span></span> <span data-ttu-id="dbf92-113">转换的 XML 表示形式**数据集**，其中包含相关**数据集**与非嵌套数据关系可能导致性能降低。</span><span class="sxs-lookup"><span data-stu-id="dbf92-113">Transforming the XML representation of a **DataSet** that contains related **DataSet**s with non-nested data relations can cause slow performance.</span></span> <span data-ttu-id="dbf92-114">建议您嵌套数据关系。</span><span class="sxs-lookup"><span data-stu-id="dbf92-114">We recommend that you nest the data relations.</span></span> <span data-ttu-id="dbf92-115">若要执行此操作，设置**嵌套**属性设置为**true**。</span><span class="sxs-lookup"><span data-stu-id="dbf92-115">To do this, set the **Nested** property to **true**.</span></span> <span data-ttu-id="dbf92-116">然后在使用上下分层 XPath 查询表达式的 XSLT 样式表中编写代码以定位和转换数据。</span><span class="sxs-lookup"><span data-stu-id="dbf92-116">Then write code in the XSLT style sheet that uses top-down hierarchical XPath query expressions to locate and transform the data.</span></span>
