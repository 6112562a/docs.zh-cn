 <span data-ttu-id="ffef1-111">下面的代码显示了从该架构强类型化**数据集**为该示例生成。</span><span class="sxs-lookup"><span data-stu-id="ffef1-111">The following code shows the schema from which the strongly typed **DataSet** is generated for this sample.</span></span>

 <span data-ttu-id="ffef1-112">请注意，从该唯一信息**OrderDetails**并**产品**原始 XML 文档的元素包含的架构中**数据集**。</span><span class="sxs-lookup"><span data-stu-id="ffef1-112">Notice that only information from the **OrderDetails** and **Products** elements of the original XML document are included in the schema for the **DataSet**.</span></span> <span data-ttu-id="ffef1-113">同步**数据集**与**XmlDataDocument**元素中不包含可确保**数据集**与 XML 文档将保持不变。</span><span class="sxs-lookup"><span data-stu-id="ffef1-113">Synchronizing the **DataSet** with an **XmlDataDocument** ensures that the elements not included in the **DataSet** will persist with the XML document.</span></span>  
  
 <span data-ttu-id="ffef1-114">使用强类型化**数据集**生成的 XML 架构 (具有的命名空间**Northwind.FillOrder**)，可以同步公开原始 XML 文档的一部分**数据集**与**XmlDataDocument**源 XML 文档中加载。</span><span class="sxs-lookup"><span data-stu-id="ffef1-114">With the strongly typed **DataSet** generated from the XML Schema (with a namespace of **Northwind.FillOrder**), a portion of the original XML document can be exposed by synchronizing the **DataSet** with the **XmlDataDocument** loaded from the source XML document.</span></span> <span data-ttu-id="ffef1-115">请注意，**数据集**从生成的架构包含结构，但没有数据。</span><span class="sxs-lookup"><span data-stu-id="ffef1-115">Notice that the **DataSet** generated from the schema contains structure but no data.</span></span> <span data-ttu-id="ffef1-116">将数据加载到 XML 时填充**XmlDataDocument**。</span><span class="sxs-lookup"><span data-stu-id="ffef1-116">The data is filled in when you load the XML into the **XmlDataDocument**.</span></span> <span data-ttu-id="ffef1-117">如果你尝试加载**XmlDataDocument**的已同步**数据集**已包含数据，将引发异常。</span><span class="sxs-lookup"><span data-stu-id="ffef1-117">If you attempt to load an **XmlDataDocument** that has been synchronized with a **DataSet** that already contains data, an exception will be thrown.</span></span>  
  
 <span data-ttu-id="ffef1-118">之后**数据集**(与**XmlDataDocument**) 已更新**XmlDataDocument**可以写出经过修改的 XML 文档被忽略的元素**数据集**仍保持不变，如下所示。</span><span class="sxs-lookup"><span data-stu-id="ffef1-118">After the **DataSet** (and the **XmlDataDocument**) has been updated, the **XmlDataDocument** can then write out the modified XML document with the elements ignored by the **DataSet** still intact, as shown below.</span></span> <span data-ttu-id="ffef1-119">在订单方案中，当填充订单项后，经过修改的 XML 文档将传递到订单过程的下一步（可能会传递到公司货运部）。</span><span class="sxs-lookup"><span data-stu-id="ffef1-119">In the purchase order scenario, after the order items have been filled, the modified XML document can then be passed on to the next step in the order process, perhaps to the company's shipping department.</span></span>  
  
```vb  
Imports System  
Imports System.Data  
Imports System.Xml  
Imports Northwind.FillOrder  
  
Public class Sample  
  Public Shared Sub Main()  
  
    Dim orderDS As OrderDetail = New OrderDetail  
  
    Dim xmlDocument As XmlDataDocument = New XmlDataDocument(orderDS)   
  
    xmlDocument.Load("Order.xml")  
  
    Dim orderItem As OrderDetail.LineItem  
    Dim product As OrderDetail.Product  
  
    For Each orderItem In orderDS.LineItems  
      product = orderItem.Product  
  
      ' Remove quantity from the current stock.  
      product.UnitsInStock = CType(product.UnitsInStock - orderItem.Quantity, Short)  
  
      ' If the remaining stock is less than the reorder level, order more.  
      If ((product.UnitsInStock + product.UnitsOnOrder) < product.ReorderLevel) Then  
        product.UnitsOnOrder = CType(product.UnitsOnOrder + product.ReorderLevel, Short)  
      End If  
    Next  
  
    xmlDocument.Save("Order_out.xml")  
  End Sub  
End Class  
```  
  
```csharp  
using System;  
using System.Data;  
using System.Xml;  
using Northwind.FillOrder;  
  
public class Sample  
{  
  public static void Main()  
  {  
    OrderDetail orderDS = new OrderDetail();   
  
    XmlDataDocument xmlDocument = new XmlDataDocument(orderDS);   
  
    xmlDocument.Load("Order.xml");  
  
    foreach (OrderDetail.LineItem orderItem in orderDS.LineItems)  
    {  
      OrderDetail.Product product = orderItem.Product;  
  
      // Remove quantity from the current stock.  
      product.UnitsInStock = (short)(product.UnitsInStock - orderItem.Quantity);  
  
      // If the remaining stock is less than the reorder level, order more.  
      if ((product.UnitsInStock + product.UnitsOnOrder) < product.ReorderLevel)  
        product.UnitsOnOrder = (short)(product.UnitsOnOrder + product.ReorderLevel);  
    }  
  
    xmlDocument.Save("Order_out.xml");  
  }  
}  
```
