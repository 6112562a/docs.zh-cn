# <a name="data-encryption-in-sql-server"></a><span data-ttu-id="0d7cf-102">SQL Server 中的数据加密</span><span class="sxs-lookup"><span data-stu-id="0d7cf-102">Data Encryption in SQL Server</span></span>
<span data-ttu-id="0d7cf-103">SQL Server 提供了使用证书、非对称密钥或对称密钥对数据进行加密和解密的功能。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-103">SQL Server provides functions to encrypt and decrypt data using a certificate, asymmetric key, or symmetric key.</span></span> <span data-ttu-id="0d7cf-104">它在一个内部证书存储中管理所有这些证书或密钥。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-104">It manages all of these in an internal certificate store.</span></span> <span data-ttu-id="0d7cf-105">该存储使用加密层次结构，可在层次结构中的上一层级别保护证书和密钥。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-105">The store uses an encryption hierarchy that secures certificates and keys at one level with the layer above it in the hierarchy.</span></span> <span data-ttu-id="0d7cf-106">SQL Server 的此功能区域称为“机密存储”。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-106">This feature area of SQL Server is called Secret Storage.</span></span>  
  
 <span data-ttu-id="0d7cf-107">加密功能支持的最快加密模式是对称密钥加密。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-107">The fastest mode of encryption supported by the encryption functions is symmetric key encryption.</span></span> <span data-ttu-id="0d7cf-108">此模式适用于处理大量数据。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-108">This mode is suitable for handling large volumes of data.</span></span> <span data-ttu-id="0d7cf-109">可以通过证书、密码或其他对称密钥加密对称密钥。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-109">The symmetric keys can be encrypted by certificates, passwords or other symmetric keys.</span></span>  
  
## <a name="keys-and-algorithms"></a><span data-ttu-id="0d7cf-110">密钥和算法</span><span class="sxs-lookup"><span data-stu-id="0d7cf-110">Keys and Algorithms</span></span>  
 <span data-ttu-id="0d7cf-111">SQL Server 支持多种对称密钥加密算法，包括 DES、Triple DES、RC2、RC4、128 位 RC4、DESX、128 位 AES、192 位 AES 和 256 位 AES。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-111">SQL Server supports several symmetric key encryption algorithms, including DES, Triple DES, RC2, RC4, 128-bit RC4, DESX, 128-bit AES, 192-bit AES, and 256-bit AES.</span></span> <span data-ttu-id="0d7cf-112">这些算法使用 Windows 加密 API 实现。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-112">The algorithms are implemented using the Windows Crypto API.</span></span>  
  
 <span data-ttu-id="0d7cf-113">在数据库连接范围内，SQL Server 可以保留多个开放式对称密钥。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-113">Within the scope of a database connection, SQL Server can maintain multiple open symmetric keys.</span></span> <span data-ttu-id="0d7cf-114">开放式密钥从存储中进行检索，并可用于解密数据。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-114">An open key is retrieved from the store and is available for decrypting data.</span></span> <span data-ttu-id="0d7cf-115">解密一段数据时，不需要指定要使用的对称密钥。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-115">When a piece of data is decrypted, there is no need to specify the symmetric key to use.</span></span> <span data-ttu-id="0d7cf-116">每个加密的值均包含用于加密该值的密钥的密钥标识符（密钥 GUID）。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-116">Each encrypted value contains the key identifier (key GUID) of the key used to encrypt it.</span></span> <span data-ttu-id="0d7cf-117">如果已解密正确的密钥并且该密钥已开放，则引擎会将加密的字节流匹配到开放式对称密钥。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-117">The engine matches the encrypted byte stream to an open symmetric key, if the correct key has been decrypted and is open.</span></span> <span data-ttu-id="0d7cf-118">然后使用此密钥执行解密并返回数据。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-118">This key is then used to perform decryption and return the data.</span></span> <span data-ttu-id="0d7cf-119">如果正确的密钥未开放，则返回 NULL。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-119">If the correct key is not open, NULL is returned.</span></span>  
  
 <span data-ttu-id="0d7cf-120">有关演示如何使用在数据库中加密数据的示例，请参阅[加密数据列](/sql/relational-databases/security/encryption/encrypt-a-column-of-data)。</span><span class="sxs-lookup"><span data-stu-id="0d7cf-120">For an example that shows how to work with encrypted data in a database, see [Encrypt a Column of Data](/sql/relational-databases/security/encryption/encrypt-a-column-of-data).</span></span>
