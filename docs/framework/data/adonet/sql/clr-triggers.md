# <a name="clr-triggers"></a>CLR 触发器
触发器是在语言事件执行时自动运行的一种特殊类型的存储过程。 因为 Microsoft SQL Server 与 .NET Framework 公共语言运行库 (CLR) 相集成，所以您可以使用任何 .NET Framework 语言创建 CLR 触发器。  
  
 有关详细信息，请参阅[SQL Server 文档](/sql)正在使用的 SQL Server 的版本。
