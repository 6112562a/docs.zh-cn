## <a name="xtypearguments"></a><span data-ttu-id="1b64e-106">x:TypeArguments</span><span class="sxs-lookup"><span data-stu-id="1b64e-106">x:TypeArguments</span></span>

 <span data-ttu-id="1b64e-110">因为`x:TypeArguments`采用一个字符串，并具有类型转换器支持，它通常在 XAML 特性用法中声明。</span><span class="sxs-lookup"><span data-stu-id="1b64e-110">Because `x:TypeArguments` takes a string, and has type converter backing, it is typically declared in XAML usage as an attribute.</span></span>
