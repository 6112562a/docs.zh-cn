# <a name="icorprofilerinfogetfunctionfromip-method"></a>ICorProfilerInfo::GetFunctionFromIP 方法
映射到的托管的代码指令指针`FunctionID`。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetFunctionFromIP(  
    [in]  LPCBYTE    ip,  
    [out] FunctionID *pFunctionId);  
```  
  
## <a name="parameters"></a>参数  
 `ip`  
 [in]托管代码中的指令指针。  
  
 `pFunctionId`  
 [out]返回的函数 id。
