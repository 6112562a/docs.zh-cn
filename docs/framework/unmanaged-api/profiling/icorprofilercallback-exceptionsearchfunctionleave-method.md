# <a name="icorprofilercallbackexceptionsearchfunctionleave-method"></a>ICorProfilerCallback::ExceptionSearchFunctionLeave 方法

## <a name="syntax"></a>语法  
  
```cpp  
HRESULT ExceptionSearchFunctionLeave();  
```
