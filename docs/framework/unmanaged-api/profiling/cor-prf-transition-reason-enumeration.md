# <a name="corprftransitionreason-enumeration"></a>COR_PRF_TRANSITION_REASON 枚举
指示从托管代码向非托管代码转换或从非托管代码向托管代码转换的原因。  
  
## <a name="syntax"></a>语法  
  
```cpp  
typedef enum {  
    COR_PRF_TRANSITION_CALL,  
    COR_PRF_TRANSITION_RETURN  
} COR_PRF_TRANSITION_REASON;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`COR_PRF_TRANSITION_CALL`|转换是由于执行某一函数的调用。|  
|`COR_PRF_TRANSITION_RETURN`|转换是由于从函数返回。|  
  
## <a name="remarks"></a>备注  
 在探查器时，将发生转换，接收[icorprofilercallback:: Managedtounmanagedtransition](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback-managedtounmanagedtransition-method.md)或[icorprofilercallback:: Unmanagedtomanagedtransition](../../../../docs/framework/unmanaged-api/profiling/icorprofilercallback-unmanagedtomanagedtransition-method.md)回调，其中任一提供的值为`COR_PRF_TRANSITION_REASON`枚举来指示转换的原因。
