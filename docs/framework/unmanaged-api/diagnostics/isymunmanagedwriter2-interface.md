# <a name="isymunmanagedwriter2-interface"></a>ISymUnmanagedWriter2 接口
表示符号编写器，并提供定义文档、序列点、词法范围和变量的方法。 此接口扩展[ISymUnmanagedWriter](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedwriter-interface.md)接口。  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[DefineConstant2 方法](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedwriter2-defineconstant2-method.md)|定义常数值的名称。|  
|[DefineGlobalVariable2 方法](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedwriter2-defineglobalvariable2-method.md)|定义一个全局变量。|  
|[DefineLocalVariable2 方法](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanagedwriter2-definelocalvariable2-method.md)|在当前词法范围内定义单个变量。|  
  
## <a name="requirements"></a>要求  
 **标头：** CorSym，CorSym
