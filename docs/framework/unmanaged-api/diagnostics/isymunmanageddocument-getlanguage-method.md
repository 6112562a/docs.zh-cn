# <a name="isymunmanageddocumentgetlanguage-method"></a>ISymUnmanagedDocument::GetLanguage 方法
获取此文档的语言标识符  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetLanguage(  
    [out, retval]  GUID*  pRetVal);  
```  
  
## <a name="parameters"></a>参数  
 `pRetVal`  
 [out]指向一个变量来接收语言标识符的指针。  
  
## <a name="return-value"></a>返回值  
 如果该方法成功，则为 S_OK。
