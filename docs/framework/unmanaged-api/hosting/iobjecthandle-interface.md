# <a name="iobjecthandle-interface"></a>IObjectHandle 接口
提供了一种方法，用于从间接寻址解包按值封送对象。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Unwrap 方法](../../../../docs/framework/unmanaged-api/hosting/iobjecthandle-unwrap-method.md)|从间接寻址将按值封送对象解包。|
