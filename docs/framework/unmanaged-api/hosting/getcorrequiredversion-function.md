# <a name="getcorrequiredversion-function"></a>GetCORRequiredVersion 函数
获取所需的公共语言运行时 (CLR) 版本数量。  
  
 .NET Framework 4 中已弃用此函数。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetCORRequiredVersion (  
    [out] LPWSTR   pbuffer,  
    [in]  DWORD    cchBuffer,  
    [out] DWORD   *dwLength  
);  
```  
  
## <a name="parameters"></a>参数  
 `pbuffer`  
 [out]包含指定的版本号的字符串的缓冲区。  
  
 `cchBuffer`  
 [in]以字节为单位的缓冲区的大小。  
  
 `dwLength`  
 [out]在缓冲区中返回的字节数。
