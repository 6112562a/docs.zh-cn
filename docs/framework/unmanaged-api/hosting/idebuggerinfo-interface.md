# <a name="idebuggerinfo-interface"></a>IDebuggerInfo 接口
提供用于获取有关调试服务状态的信息的方法。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[IsDebuggerAttached 方法](../../../../docs/framework/unmanaged-api/hosting/idebuggerinfo-isdebuggerattached-method.md)|获取一个值，该值指示托管调试器是否已附加到此进程。|
