# <a name="eclrassemblyidentityflags-enumeration"></a>ECLRAssemblyIdentityFlags 枚举

```cpp  
typedef enum _CLRAssemblyIdentityFlags {  
    CLR_ASSEMBLY_IDENTITY_FLAGS_DEFAULT = 0  
} ECLRAssemblyIdentityFlags;  
```
