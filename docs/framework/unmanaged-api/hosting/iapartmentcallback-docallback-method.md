# <a name="iapartmentcallbackdocallback-method"></a>IApartmentCallback::DoCallback 方法
执行单元内的指定的函数。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT _stdcall DoCallback(  
    [in] SIZE_T pFunc,  
    [in] SIZE_T pData  
);  
```  
  
## <a name="parameters"></a>参数  
 `pFunc`  
 [in]指向要在单元内执行的函数的指针。  
  
 `pData`  
 [in]指向函数的参数的指针。
