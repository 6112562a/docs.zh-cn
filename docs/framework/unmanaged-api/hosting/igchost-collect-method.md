# <a name="igchostcollect-method"></a>IGCHost::Collect 方法
强制给定生成，而不考虑当前的垃圾回收状态发生的集合。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Collect (  
    [in] LONG Generation  
);  
```  
  
## <a name="parameters"></a>参数  
 `Generation`  
 [in]要对其执行垃圾回收生成。 值为-1 指示所有代将都进行垃圾回收。
