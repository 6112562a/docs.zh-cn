# <a name="imanagedobjectgetserializedbuffer-method"></a>IManagedObject::GetSerializedBuffer 方法
获取此托管对象的字符串表示形式。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetSerializedBuffer (  
    [out] BSTR *pBSTR  
);  
```  
  
## <a name="parameters"></a>参数  
 `pBSTR`  
 [out]指向为序列化的对象的字符串的指针。  
  
## <a name="remarks"></a>备注  
 `GetSerializedBuffer`方法序列化的对象，因此它可以封送到客户端。
