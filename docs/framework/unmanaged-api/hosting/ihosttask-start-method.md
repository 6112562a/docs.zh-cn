# <a name="ihosttaskstart-method"></a>IHostTask::Start 方法
请求宿主将当前[IHostTask](../../../../docs/framework/unmanaged-api/hosting/ihosttask-interface.md)实例表示的任务从挂起状态移动到实时状态，在此状态下可以执行代码。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Start ();  
```  
  
## <a name="return-value"></a>返回值  
  
|HRESULT|描述|  
|-------------|-----------------|  
|S_OK|已成功启动。|  
|E_FAIL|发生未知的灾难性故障。 当方法返回 E_FAIL 时，公共语言运行时（CLR）在该过程中将不再可用。 对宿主方法的后续调用会返回 HOST_E_CLRNOTAVAILABLE。|  
  
## <a name="remarks"></a>备注  
 `Start` 总是返回的 HRESULT 值为 S_OK，但发生灾难性故障的情况除外。
