# <a name="iobjecthandleunwrap-method"></a>IObjectHandle::Unwrap 方法
从间接寻址将按值封送对象解包。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Unwrap (  
    [out, retval] VARIANT *ppv  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppv`  
 弄指向要解包的对象的指针。
