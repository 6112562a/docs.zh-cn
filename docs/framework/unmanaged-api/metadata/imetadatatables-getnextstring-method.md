# <a name="imetadatatablesgetnextstring-method"></a>IMetaDataTables::GetNextString 方法
获取当前表列中的下一个字符串的索引。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetNextString (   
    [in]  ULONG   ixString,  
    [out] ULONG   *pNext  
);  
```  
  
## <a name="parameters"></a>参数  
 `ixString`  
 中字符串表列的索引值。  
  
 `pNext`  
 弄指向列中的下一个字符串的索引的指针。
