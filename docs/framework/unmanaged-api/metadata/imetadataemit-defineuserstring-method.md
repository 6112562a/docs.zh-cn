# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString 方法
获取指定文本字符串的元数据令牌。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT DefineUserString (
    [in]  LPCWSTR     szString,
    [in]  ULONG       cchString,
    [out] mdString    *pstk
);  
```  
  
## <a name="parameters"></a>parameters  
 `szString`  
 [在]要存储的用户字符串。  
  
 `cchString`  
 [在]中的`szString`宽字符的计数。  
  
 `pstk`  
 [出]分配的字符串令牌。
