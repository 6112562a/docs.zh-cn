# <a name="iceegengetstring-method"></a>ICeeGen::GetString 方法
获取存储在指定的相对虚拟地址的字符串。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetString (  
    [in]  ULONG      RVA,   
    [out] LPWSTR     *lpString  
);  
```  
  
#### <a name="parameters"></a>参数  
 `RVA`  
 [in]要返回的字符串相对虚拟地址。  
  
 `lpString`  
 [out]返回的字符串。
