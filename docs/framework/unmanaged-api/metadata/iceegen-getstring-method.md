# <a name="iceegengetstring-method"></a>ICeeGen::GetString 方法
获取存储在指定相对虚拟地址的字符串。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetString (  
    [in]  ULONG      RVA,
    [out] LPWSTR     *lpString  
);  
```  
  
## <a name="parameters"></a>parameters  
 `RVA`  
 [在]要返回的字符串的相对虚拟地址。  
  
 `lpString`  
 [出]返回的字符串。
