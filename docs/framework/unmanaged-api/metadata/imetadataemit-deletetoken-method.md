# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken 方法
从当前的元数据范围中删除指定的标记。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT DeleteToken (   
    [in]  mdToken     tkObj   
);  
```  
  
## <a name="parameters"></a>参数  
 `tkObj`  
 [in]要删除的标记。
