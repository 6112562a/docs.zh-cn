# <a name="cornativelinkflags-enumeration"></a>CorNativeLinkFlags 枚举
提供链接本机代码时链接器使用的标志值。  
  
## <a name="syntax"></a>语法  
  
```cpp  
typedef enum  
{  
    nlfNone         = 0x00,  
    nlfLastError    = 0x01,  
    nlfNoMangle     = 0x02,  
    nlfMaxValue     = 0x03  
} CorNativeLinkFlags;  
```  
  
## <a name="members"></a>Members  
  
|成员|说明|  
|------------|-----------------|  
|`nlfNone`|指示无标志。|  
|`nlfLastError`|指示 `setLastError` 关键字。|  
|`nlfNoMangle`|指示 `nomangle` 关键字。|  
|`nlfMaxValue`|未使用。|
