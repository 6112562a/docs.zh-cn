# <a name="imetadataerroronerror-method"></a>IMetaDataError::OnError 方法
提供元数据合并期间发生的错误通知。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT OnError (  
    [in] HRESULT   hrError,
    [in] mdToken   token  
);  
```  
  
## <a name="parameters"></a>parameters  
 `hrError`  
 [在]HRESULT 错误值返回到调用方法。  
  
 `token`  
 [在]发生错误时正在合并的代码对象的元数据令牌。
