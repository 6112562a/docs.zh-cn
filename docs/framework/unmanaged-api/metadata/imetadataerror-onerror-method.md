# <a name="imetadataerroronerror-method"></a>IMetaDataError::OnError 方法
提供元数据合并过程中发生的错误的通知。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT OnError (  
    [in] HRESULT   hrError,   
    [in] mdToken   token  
);  
```  
  
#### <a name="parameters"></a>参数  
 `hrError`  
 [in]HRESULT 错误值返回给调用的方法。  
  
 `token`  
 [in]发生错误时合并代码对象的元数据标记。
