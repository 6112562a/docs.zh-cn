# <a name="imetadatafiltermarktoken-method"></a>IMetaDataFilter::MarkToken 方法
设置一个值，该值指示已处理指定的元数据标记。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT MarkToken (  
    [in] mdToken   tk  
);  
```  
  
## <a name="parameters"></a>参数  
 `tk`  
 中标记为已处理的标记。
