# <a name="imetadataimportfindtyperef-method"></a>IMetaDataImport::FindTypeRef 方法
获取一个指针，该指针指向指定范围内且具有指定名称的 <xref:System.Type> 引用的 TypeRef 标记。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT FindTypeRef (  
   [in] mdToken        tkResolutionScope,  
   [in]  LPCWSTR       szName,  
   [out] mdTypeRef     *ptr  
);  
```  
  
## <a name="parameters"></a>参数  
 `tkResolutionScope`  
 中一个 ModuleRef、AssemblyRef 或 TypeRef 标记，分别指定定义了类型引用的模块、程序集或类型。  
  
 `szName`  
 中要搜索的类型引用的名称。  
  
 `ptr`  
 弄指向匹配的 TypeRef 标记的指针。
