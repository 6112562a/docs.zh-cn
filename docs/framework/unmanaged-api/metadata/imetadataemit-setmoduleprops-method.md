# <a name="imetadataemitsetmoduleprops-method"></a>IMetaDataEmit::SetModuleProps 方法

## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SetModuleProps (
    [in]  LPCWSTR     szName  
);  
```  
  
## <a name="parameters"></a>parameters  
 `szName`  
 [在]Unicode 中的模块名称。 这只是文件名，而不是完整的路径名称。
