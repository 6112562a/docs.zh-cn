# <a name="corsavesize-enumeration"></a>CorSaveSize 枚举
包含一些值，用于指示查询保存操作的大小时所需的精度级别。  
  
## <a name="syntax"></a>语法  
  
```  
typedef enum CorSaveSize {  
    cssAccurate                = 0x0000,   
    cssQuick                   = 0x0001,   
    cssDiscardTransientCAs     = 0x0002  
} CorSaveSize;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`cssAccurate`|指定返回值应为完全。|  
|`cssQuick`|指定应估计的返回值。|  
|`cssDiscardTransientCAs`|指定应删除丢弃的类型。|
