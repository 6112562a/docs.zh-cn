# <a name="metadata-structures"></a>元数据结构
本节描述元数据 API 使用的非托管结构。  
  
## <a name="in-this-section"></a>本节内容  
 ASSEMBLYMETADATA  
 存储程序集元数据，供以后使用。  
  
 [COR_FIELD_OFFSET 结构](../../../../docs/framework/unmanaged-api/metadata/cor-field-offset-structure.md)  
 存储一个类中的指定字段的偏移量。  
  
 [COR_NATIVE_LINK 结构](../../../../docs/framework/unmanaged-api/metadata/cor-native-link-structure.md)  
 包含用于链接本机代码的信息。  
  
 [CVStruct 结构](../../../../docs/framework/unmanaged-api/metadata/cvstruct-structure.md)  
 包含在安装模块或复合图像时所使用的信息。  
  
 [OSINFO 结构](../../../../docs/framework/unmanaged-api/metadata/osinfo-structure.md)  
 包含有关操作系统的信息。
