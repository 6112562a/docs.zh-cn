# <a name="imetadataassemblyimportfindexportedtypebyname-method"></a>IMetaDataAssemblyImport::FindExportedTypeByName 方法
获取指向导出类型的指针，给定其名称和封闭类型。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT FindExportedTypeByName (  
    [in]  LPCWSTR           szName,
    [in]  mdToken           mdtExportedType,
    [out] mdExportedType    *ptkExportedType  
);  
```  
  
## <a name="parameters"></a>parameters  
 `szName`  
 [在]导出类型的名称。  
  
 `mdtExportedType`  
 [在]导出类型的封闭类的元数据令牌。 此值是`mdExportedTypeNil`请求的导出类型不是嵌套类型。  
  
 `ptkExportedType`  
 [出]指向表示导出类型的`mdExportedType`令牌的指针。  
  
## <a name="remarks"></a>备注  
 该方法`FindExportedTypeByName`使用通用语言运行时采用的标准规则来解决引用。
