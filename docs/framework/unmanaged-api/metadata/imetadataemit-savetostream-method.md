# <a name="imetadataemitsavetostream-method"></a>IMetaDataEmit::SaveToStream 方法
将当前作用域中的所有元数据保存到指定的`IStream`。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SaveToStream (
    [in]  IStream     *pIStream,  
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>parameters  
 `pIStream`  
 [在]要保存到的可写流。  
  
 `dwSaveFlags`  
 [in] 保留。 必须为零。
