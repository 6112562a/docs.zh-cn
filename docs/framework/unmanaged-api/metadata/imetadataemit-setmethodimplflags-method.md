# <a name="imetadataemitsetmethodimplflags-method"></a>IMetaDataEmit::SetMethodImplFlags 方法
设置或更新指定令牌引用的继承方法实现的元数据签名。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SetMethodImplFlags (
    [in]  mdMethodDef   md,
    [in]  DWORD         dwImplFlags
);  
```  
  
## <a name="parameters"></a>parameters  
 `md`  
 [在]要更改的方法的令牌。  
  
 `dwImplFlags`  
 [在][CorMethodImpl](../../../../docs/framework/unmanaged-api/metadata/cormethodimpl-enumeration.md)枚举的值的组合，用于指定方法实现要素。
