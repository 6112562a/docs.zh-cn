# <a name="cvstruct-structure"></a>CVStruct 结构
包含在安装模块或复合图像时所使用的信息。  
  
## <a name="syntax"></a>语法  
  
```cpp  
typedef struct {  
    short Major;  
    short Minor;  
    short Sub;  
    short Build;  
} CVStruct;  
```  
  
## <a name="members"></a>Members  
  
|成员|描述|  
|------------|-----------------|  
|主要|Major version build number.|  
|次要|Minor version build number.|  
|Sub|Sub-build number.|  
|生成|Build number.|
