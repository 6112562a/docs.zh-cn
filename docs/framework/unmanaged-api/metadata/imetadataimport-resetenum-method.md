# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum 方法
将指定的枚举器重置到指定位置。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,   
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>参数  
 `hEnum`  
 中要重置的枚举数。  
  
 `ulPos`  
 中要放置枚举器的新位置。
