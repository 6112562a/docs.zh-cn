# <a name="imetadataimportresetenum-method"></a>IMetaDataImport::ResetEnum 方法
将指定的枚举器重置到指定位置。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,
   [in] ULONG       ulPos  
);  
```  
  
## <a name="parameters"></a>parameters  
 `hEnum`  
 [在]要重置的枚举器。  
  
 `ulPos`  
 [在]放置枚举器的新位置。
