# <a name="imetadataemit2savedelta-method"></a>IMetaDataEmit2::SaveDelta 方法
将当前编辑和继续会话的更改保存到指定的文件。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>parameters  
 `szFile`  
 [在]要保存的文件名将发生更改。  
  
 `dwSaveFlags`  
 [in] 保留。 必须为零。
