# <a name="imetadatatablesgetuserstringheapsize-method"></a>IMetaDataTables::GetUserStringHeapSize 方法
获取以字节为单位的用户字符串堆的大小。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetUserStringHeapSize (  
    [out] ULONG   *pcbBlobs  
);  
```  
  
## <a name="parameters"></a>参数  
 `pcbBlobs`  
 [out]指向以字节为单位的用户字符串堆的大小的指针。
