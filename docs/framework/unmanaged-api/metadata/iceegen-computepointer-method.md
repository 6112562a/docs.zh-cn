# <a name="iceegencomputepointer-method"></a>ICeeGen::ComputePointer 方法
确定指定代码节的缓冲区。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT ComputePointer (  
    [in]  HCEESECTION  section,  
    [in]  ULONG        RVA,   
    [out] UCHAR        **lpBuffer  
);  
```  
  
## <a name="parameters"></a>参数  
 `section`  
 中要为其返回缓冲区的代码部分。  
  
 `RVA`  
 中要为其获取指针的方法的相对虚拟地址。  
  
 `lpBuffer`  
 弄指向返回的缓冲区的指针。
