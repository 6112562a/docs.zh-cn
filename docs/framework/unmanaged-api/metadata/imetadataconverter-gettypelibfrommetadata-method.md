# <a name="imetadataconvertergettypelibfrommetadata-method"></a>IMetaDataConverter::GetTypeLibFromMetaData 方法
获取指向表示具有指定`ITypeLib`库和模块名称的类型库的实例的指针。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetTypeLibFromMetaData (  
    [in]  BSTR     strModule,
    [in]  BSTR     strTlbName,
    [out] ITypeLib **ppITL  
);  
```  
  
## <a name="parameters"></a>parameters  
 `strModule`  
 [在]类型库模块的名称。  
  
 `strTlbName`  
 [在]类型库的名称。  
  
 `ppITL`  
 [出]指向接收表示类型库的`ITypeLib`实例地址的位置的指针。
