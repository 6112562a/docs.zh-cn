# <a name="imetadatatablesgetstring-method"></a>IMetaDataTables::GetString 方法
从当前引用作用域中的表列获取指定索引处的字符串。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetString (
    [in]  ULONG       ixString,  
    [out] const char  **ppString  
);  
```  
  
## <a name="parameters"></a>parameters  
 `ixString`  
 [在]开始搜索下一个值的索引。  
  
 `ppString`  
 [出]指向返回的字符串值的指针。
