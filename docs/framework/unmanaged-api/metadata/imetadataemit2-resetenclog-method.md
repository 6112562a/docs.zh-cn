# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog 方法
重置编辑并继续日志并启动新的会话。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT ResetENCLog ();  
```  
  
## <a name="parameters"></a>参数  
 `ResetENCLog` 不采用任何参数。
