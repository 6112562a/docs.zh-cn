# <a name="imetadataemitsetmethodprops-method"></a>IMetaDataEmit::SetMethodProps 方法
设置或更新由之前调用[IMetaDataEmit：:DefineMethod）](../../../../docs/framework/unmanaged-api/metadata/imetadataemit-definemethod-method.md)定义的方法的要素（存储在指定的相对虚拟地址）  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SetMethodProps (
    [in]  mdMethodDef md,
    [in]  DWORD       dwMethodFlags,  
    [in]  ULONG       ulCodeRVA,
    [in]  DWORD       dwImplFlags
);  
```  
  
## <a name="parameters"></a>parameters  
 `md`  
 [在]要更改的方法的令牌。  
  
 `dwMethodFlags`  
 [在]成员属性。  
  
 `ulCodeRVA`  
 [在]代码的地址。  
  
 `dwImplFlags`  
 [在]方法的实现标志。
