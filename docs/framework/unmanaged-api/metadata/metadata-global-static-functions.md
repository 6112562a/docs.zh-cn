# <a name="metadata-global-static-functions"></a>元数据全局静态函数
从.NET Framework 4 的文档开始`_Cor*`， `CoInitialize*`， `CoUninitialize*`，和`CoEEShutDownCOM`函数，与承载相关，位于[已弃用的 CLR 承载函数](../../../../docs/framework/unmanaged-api/hosting/deprecated-clr-hosting-functions.md)。 没有与元数据相关的全局静态函数。
