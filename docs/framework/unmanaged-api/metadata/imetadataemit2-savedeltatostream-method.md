# <a name="imetadataemit2savedeltatostream-method"></a>IMetaDataEmit2::SaveDeltaToStream 方法
将更改从当前编辑和继续会话保存到指定的流。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SaveDeltaToStream (  
    [in] IStream     *pIStream,
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>parameters  
 `pIStream`  
 [在]指向要保存到的可写流的接口指针。  
  
 `dwSaveFlags`  
 [in] 保留。 此值必须为零。
