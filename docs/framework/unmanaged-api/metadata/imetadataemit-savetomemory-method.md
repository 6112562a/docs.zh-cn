# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory 方法
将保存到指定的内存区域的当前作用域中的所有元数据。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SaveToMemory (   
    [out]  void        *pbData,   
    [in]   ULONG       cbData   
);  
```  
  
## <a name="parameters"></a>参数  
 `pbData`  
 [out]要开始编写元数据地址。  
  
 `cbData`  
 [in]以字节为单位分配的内存大小。
