# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory 方法
将当前作用域中的所有元数据保存到指定的内存区域。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SaveToMemory (
    [out]  void        *pbData,
    [in]   ULONG       cbData
);  
```  
  
## <a name="parameters"></a>parameters  
 `pbData`  
 [出]开始写入元数据的地址。  
  
 `cbData`  
 [在]已分配内存的大小（以字节为单位）。
