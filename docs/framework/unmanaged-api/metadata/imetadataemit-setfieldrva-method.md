# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA 方法
为指定令牌引用的字段的相对虚拟地址设置全局变量值。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SetFieldRVA (
    [in]  mdFieldDef  fd,
    [in]  ULONG       ulRVA
);  
```  
  
## <a name="parameters"></a>parameters  
 `fd`  
 [在]目标字段的令牌。  
  
 `ulRVA`  
 [在]代码或数据区域的地址。
