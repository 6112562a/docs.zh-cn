# <a name="imetadataimportisvalidtoken-method"></a>IMetaDataImport::IsValidToken 方法
获取指示指定的标记是否包含对代码对象的有效引用的值。  
  
## <a name="syntax"></a>语法  
  
```cpp  
BOOL IsValidToken (  
   [in] mdToken     tk  
);  
```  
  
## <a name="parameters"></a>参数  
 `tk`  
 中要检查其引用有效性的标记。  
  
## <a name="return-value"></a>返回值  
 如果 `tk` 是当前范围内的有效元数据标记，则 `true`。 否则为 `false`。
