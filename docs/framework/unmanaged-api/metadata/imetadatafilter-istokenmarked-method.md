# <a name="imetadatafilteristokenmarked-method"></a>IMetaDataFilter::IsTokenMarked 方法
获取指示指定元数据令牌是否已标记为已处理的值。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT IsTokenMarked (  
    [in]  mdToken  tk,
    [out] BOOL     *pIsMarked  
);  
```  
  
## <a name="parameters"></a>parameters  
 `tk`  
 [在]要检查处理标记的令牌。  
  
 `pIsMarked`  
 [出]`true`已处理的值`tk`;如果已处理，则为否则`false`。
