# <a name="iassemblycacheitemcommit-method"></a>IAssemblyCacheItem::Commit 方法
将缓存的程序集引用提交到内存。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Commit (  
    [in] DWORD dwFlags,
    [out, optional] ULONG *pulDisposition  
);  
```  
  
## <a name="parameters"></a>parameters  
 `dwFlags`  
 [在]在 Fusion.idl 中定义的标志。  
  
 `pulDisposition`  
 [退出，可选]指示操作结果的值。
