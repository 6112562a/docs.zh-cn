# <a name="fusion-enumerations"></a>合成枚举
本部分描述合成 API 使用的非托管的枚举。  
  
## <a name="in-this-section"></a>本节内容  
 [ASM_CACHE_FLAGS 枚举](../../../../docs/framework/unmanaged-api/fusion/asm-cache-flags-enumeration.md)  
 指示表示程序集的源[IAssemblyCacheItem](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-interface.md)全局程序集缓存中。  
  
 [ASM_CMP_FLAGS 枚举](../../../../docs/framework/unmanaged-api/fusion/asm-cmp-flags-enumeration.md)  
 指示版本、 生成、 区域性、 签名和等等的两个程序集的比较[iassemblyname:: Isequal](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-isequal-method.md)方法。  
  
 [ASM_DISPLAY_FLAGS 枚举](../../../../docs/framework/unmanaged-api/fusion/asm-display-flags-enumeration.md)  
 指示版本、 版本、 区域性、 签名和等等，将通过检索其显示名称的程序集[iassemblyname:: Getdisplayname](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-getdisplayname-method.md)方法。  
  
 [ASM_NAME 枚举](../../../../docs/framework/unmanaged-api/fusion/asm-name-enumeration.md)  
 指示版本、 生成、 区域性、 签名和等等的程序集将检索到或通过设置其属性[IAssemblyName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md)方法。  
  
 [AssemblyComparisonResult 枚举](../../../../docs/framework/unmanaged-api/fusion/assemblycomparisonresult-enumeration.md)  
 指示两个程序集标识的等效性，由[CompareAssemblyIdentity](../../../../docs/framework/unmanaged-api/fusion/compareassemblyidentity-function.md)函数。  
  
 [CREATE_ASM_NAME_OBJ_FLAGS 枚举](../../../../docs/framework/unmanaged-api/fusion/create-asm-name-obj-flags-enumeration.md)  
 指定的特性`IAssemblyName`对象时通过构造[CreateAssemblyNameObject](../../../../docs/framework/unmanaged-api/fusion/createassemblynameobject-function.md)函数。
