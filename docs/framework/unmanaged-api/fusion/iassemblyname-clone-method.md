# <a name="iassemblynameclone-method"></a>IAssemblyName::Clone 方法
创建此[IAssemblyName](iassemblyname-interface.md)对象的浅表副本。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Clone (  
    [out] IAssemblyName **pName  
);  
```  
  
## <a name="parameters"></a>参数  
 `pName`  
 弄此 `IAssemblyName` 对象的返回副本。
