# <a name="itypelibresolver-interface"></a>ITypeLibResolver 接口
提供[ResolveTypeLib 方法](resolvetypelib-method.md)，该方法解析类型库的文件路径。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[ResolveTypeLib 方法](resolvetypelib-method.md)|通过返回类型库的完全限定路径来解析该类型库的简单名称。|
