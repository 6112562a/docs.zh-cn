# <a name="closeassembly-method"></a>CloseAssembly 方法
确定程序集操作。 在开始新的程序集或未绑定的模块之前调用此方法。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT CloseAssembly(  
    mdAssembly AssemblyID  
) PURE;  
```  
  
#### <a name="parameters"></a>参数  
 `AssemblyID`  
 程序集的 ID。  
  
## <a name="return-value"></a>返回值  
 如果该方法成功，返回，则为 S_OK。  
  
## <a name="requirements"></a>要求  
 需要 alink.h。
