# <a name="icordebugfunctiongettoken-method"></a>ICorDebugFunction::GetToken 方法
获取此函数的元数据标记。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetToken (  
    [out] mdMethodDef *pMethodDef  
);  
```  
  
## <a name="parameters"></a>参数  
 `pMethodDef`  
 弄指向 `mdMethodDef` 引用此函数的元数据的标记的指针。
