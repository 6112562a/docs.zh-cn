# <a name="icordebugevalabort-method"></a>ICorDebugEval::Abort 方法
中止当前正在执行此 ICorDebugEval 对象的计算。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Abort ();  
```  
  
## <a name="remarks"></a>备注  
 如果评估嵌套的并且不是最新的项，`Abort`方法可能会失败。
