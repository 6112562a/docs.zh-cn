# <a name="icordebugclassgettoken-method"></a>ICorDebugClass::GetToken 方法
获取`TypeDef`元数据标记所引用的此类定义。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetToken (  
    [out] mdTypeDef          *pTypeDef  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pTypeDef`  
 [out]一个指向`mdTypeDef`引用此类定义的令牌。
