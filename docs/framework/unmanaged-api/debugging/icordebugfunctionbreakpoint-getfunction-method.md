# <a name="icordebugfunctionbreakpointgetfunction-method"></a>ICorDebugFunctionBreakpoint::GetFunction 方法
获取一个指向 ICorDebugFunction 的接口指针，该指针引用在其中设置断点的函数。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetFunction (  
    [out] ICorDebugFunction  **ppFunction  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppFunction`  
 弄一个指针，指向设置断点的函数的地址。
