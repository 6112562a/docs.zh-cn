# <a name="icordebugthreadenumnext-method"></a>ICorDebugThreadEnum::Next 方法
从当前位置开始，获取枚举中指定的 ICorDebugThread 实例的数目。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Next (  
    [in] ULONG celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugThread *threads[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>参数  
 `celt`  
 中要检索的 `ICorDebugThread` 实例的数目。  
  
 `threads`  
 弄指针的数组，其中每个都指向表示线程的 `ICorDebugThread` 对象。  
  
 `pceltFetched`  
 弄一个指针，指向实际返回的 `ICorDebugThread` 实例的数目。 如果 `celt` 为1，则此值可以为 null。
