# <a name="icordebugilcode-interface"></a>ICorDebugILCode 接口
[仅在 .NET Framework 4.5.2 及更高版本中受支持]  
  
 表示中间语言 (IL) 代码段。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetEHClauses 方法](icordebugilcode-getehclauses-method.md)|返回指向为此 IL 定义的异常处理 (EH) 子句列表的指针。|
