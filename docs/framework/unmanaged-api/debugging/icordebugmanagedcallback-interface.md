## <a name="remarks"></a><span data-ttu-id="2c430-160">备注</span><span class="sxs-lookup"><span data-stu-id="2c430-160">Remarks</span></span>  
 <span data-ttu-id="2c430-161">所有回调都是序列化的, 在同一线程中调用, 并在进程处于已同步状态的情况下调用。</span><span class="sxs-lookup"><span data-stu-id="2c430-161">All callbacks are serialized, called in the same thread, and called with the process in the synchronized state.</span></span>  
  
 <span data-ttu-id="2c430-162">每个回调实现都必须调用[ICorDebugController:: Continue](../../../../docs/framework/unmanaged-api/debugging/icordebugcontroller-continue-method.md)以继续执行。</span><span class="sxs-lookup"><span data-stu-id="2c430-162">Each callback implementation must call [ICorDebugController::Continue](../../../../docs/framework/unmanaged-api/debugging/icordebugcontroller-continue-method.md) to resume execution.</span></span> <span data-ttu-id="2c430-163">如果`ICorDebugController::Continue`在回调返回之前未调用, 则该进程将保持停止状态, 并且在调用之前`ICorDebugController::Continue`不会进行更多的事件回调。</span><span class="sxs-lookup"><span data-stu-id="2c430-163">If `ICorDebugController::Continue` is not called before the callback returns, the process will remain stopped and no more event callbacks will occur until `ICorDebugController::Continue` is called.</span></span>  
  
 <span data-ttu-id="2c430-164">如果调试程序正在 .NET Framework 版本2.0 应用程序进行调试, 则必须实现[ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) 。</span><span class="sxs-lookup"><span data-stu-id="2c430-164">A debugger must implement [ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) if it is debugging .NET Framework version 2.0 applications.</span></span> <span data-ttu-id="2c430-165">`ICorDebugManagedCallback` 或`ICorDebugManagedCallback2`的实例作为回调对象传递给[ICorDebug:: SetManagedHandler](../../../../docs/framework/unmanaged-api/debugging/icordebug-setmanagedhandler-method.md)。</span><span class="sxs-lookup"><span data-stu-id="2c430-165">An instance of `ICorDebugManagedCallback` or `ICorDebugManagedCallback2` is passed as the callback object to [ICorDebug::SetManagedHandler](../../../../docs/framework/unmanaged-api/debugging/icordebug-setmanagedhandler-method.md).</span></span>
