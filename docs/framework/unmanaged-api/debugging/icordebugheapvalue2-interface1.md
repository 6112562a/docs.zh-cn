ICorDebugHeapValue 的公共语言运行时 (CLR) 处理提供支持的扩展。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[CreateHandle 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugheapvalue2-createhandle-method.md)|为此创建指定类型的句柄`ICorDebugHeapValue2`对象。|
