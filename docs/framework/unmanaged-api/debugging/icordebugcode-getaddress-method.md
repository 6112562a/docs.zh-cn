# <a name="icordebugcodegetaddress-method"></a>ICorDebugCode::GetAddress 方法
获取此 "ICorDebugCode" 接口表示的代码段的相对虚拟地址（RVA）。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetAddress (  
    [out] CORDB_ADDRESS *pStart  
);  
```  
  
## <a name="parameters"></a>参数  
 `pStart`  
 弄指向代码段的 RVA 的指针。
