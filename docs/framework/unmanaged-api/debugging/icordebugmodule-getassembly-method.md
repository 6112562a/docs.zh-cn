# <a name="icordebugmodulegetassembly-method"></a>ICorDebugModule::GetAssembly 方法
获取包含此模块的程序集。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetAssembly(  
    [out] ICorDebugAssembly **ppAssembly  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppAssembly`  
 弄指向 ICorDebugAssembly 对象的指针，该对象表示包含此模块的程序集。
