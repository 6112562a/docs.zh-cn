# <a name="cordebuggenerationtypes-enumeration"></a>CorDebugGenerationTypes 枚举
指定托管堆上内存区域的生成。  
  
## <a name="syntax"></a>语法  
  
```cpp  
typedef enum CorDebugGenerationTypes {  
    CorDebug_Gen0 = 0,  
    CorDebug_Gen1 = 1,  
    CorDebug_Gen2 = 2,  
    CorDebug_LOH  = 3,  
} CorDebugRegionTypes;  
```  
  
## <a name="members"></a>成员  
  
|成员名称|说明|  
|-----------------|-----------------|  
|`CorDebug_Gen0`|第 0 代。|  
|`CorDebug_Gen1`|第 1 代。|  
|`CorDebug_Gen2`|第 2 代。|  
|`CorDebug_LOH`|大型对象堆。|
