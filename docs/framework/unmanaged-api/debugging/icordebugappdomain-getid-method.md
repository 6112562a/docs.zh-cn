# <a name="icordebugappdomaingetid-method"></a>ICorDebugAppDomain::GetId 方法
获取应用程序域的唯一标识符。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetID (  
    [out] ULONG32   *pId  
);  
```  
  
## <a name="parameters"></a>参数  
 `pId`  
 弄应用程序域的唯一标识符。  
  
## <a name="remarks"></a>备注  
 应用程序域的标识符在包含进程中是唯一的。
