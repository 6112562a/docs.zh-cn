# <a name="icordebugchaingetreason-method"></a>ICorDebugChain::GetReason 方法
获取生成此调用链的原因。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetReason (  
    [out] CorDebugChainReason *pReason  
);  
```  
  
## <a name="parameters"></a>参数  
 `pReason`  
 [out]指向表示此调用链生成原因 CorDebugChainReason 枚举值 （按位组合） 的指针。
