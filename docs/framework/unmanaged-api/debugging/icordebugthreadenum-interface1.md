# <a name="icordebugthreadenum-interface"></a>ICorDebugThreadEnum 接口
实现 ICorDebugEnum 方法并枚举 ICorDebugThread 数组。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugthreadenum-next-method.md)|从当前位置开始，从枚举中获取指定数目的 `ICorDebugThread` 实例。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
