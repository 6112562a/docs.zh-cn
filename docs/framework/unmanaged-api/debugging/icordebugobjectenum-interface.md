# <a name="icordebugobjectenum-interface"></a>ICorDebugObjectEnum 接口

实现 ICorDebugEnum 方法，并按它们的相对虚拟地址（Rva）枚举对象的数组。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](icordebugobjectenum-next-method.md)|从当前位置开始，获取枚举中指定数量的对象的 Rva。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
