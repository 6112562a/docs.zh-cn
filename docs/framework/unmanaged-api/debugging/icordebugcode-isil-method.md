# <a name="icordebugcodeisil-method"></a>ICorDebugCode::IsIL 方法
获取一个值，该值指示是否此"ICorDebugCode"表示 Microsoft 中间语言 (MSIL) 编译的代码。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT IsIL (  
    [out] BOOL       *pbIL  
);  
```  
  
## <a name="parameters"></a>参数  
 `pbIL`  
 [out]`true`如果此`ICorDebugCode`表示的代码在 MSIL 中编译; 否则为`false`。
