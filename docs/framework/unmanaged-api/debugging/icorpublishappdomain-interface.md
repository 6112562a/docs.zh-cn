# <a name="icorpublishappdomain-interface"></a>ICorPublishAppDomain 接口
表示并提供关于应用程序域的信息。

|方法|描述|  
|------------|-----------------|  
|[GetID 方法](../../../../docs/framework/unmanaged-api/debugging/icorpublishappdomain-getid-method.md)|获取此 `ICorPublishAppDomain`的唯一标识符。|  
|[GetName 方法](../../../../docs/framework/unmanaged-api/debugging/icorpublishappdomain-getname-method.md)|获取此 `ICorPublishAppDomain`引用的应用程序域的名称。|
