# <a name="icordebuginitialize-method"></a>ICorDebug::Initialize 方法
初始化 `ICorDebug` 对象。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Initialize ();  
```  
  
## <a name="remarks"></a>备注  
 调试器必须在创建时调用 `Initialize`，才能初始化调试服务。 在调用 `ICorDebug` 上的任何其他方法之前，必须先调用此方法。
