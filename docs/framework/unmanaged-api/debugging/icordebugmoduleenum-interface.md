# <a name="icordebugmoduleenum-interface"></a>ICorDebugModuleEnum 接口

实现 ICorDebugEnum 方法，并枚举 icor 调试模块数组。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugmoduleenum-next-method.md)|获取指定的数目的`ICorDebugModule`从当前位置开始枚举的实例。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
>  此接口不支持跨计算机或跨进程远程调用。
