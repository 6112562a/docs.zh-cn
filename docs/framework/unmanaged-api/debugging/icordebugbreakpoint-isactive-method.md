# <a name="icordebugbreakpointisactive-method"></a>ICorDebugBreakpoint::IsActive 方法
获取一个值，该值指示此 `ICorDebugBreakpoint` 是否处于活动状态。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT IsActive (  
    [out] BOOL *pbActive  
);  
```  
  
## <a name="parameters"></a>参数  
 `pbActive`  
 [out] 如果此断点是活动的，则 `true`;否则，`false`。
