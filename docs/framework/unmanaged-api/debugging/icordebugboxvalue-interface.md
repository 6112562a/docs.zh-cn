# <a name="icordebugboxvalue-interface"></a>ICorDebugBoxValue 接口

表示装箱值类对象的 "ICorDebugHeapValue" 的子类。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetObject 方法](icordebugboxvalue-getobject-method.md)|获取指向装箱的 "ICorDebugObjectValue" 实例的接口指针。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
