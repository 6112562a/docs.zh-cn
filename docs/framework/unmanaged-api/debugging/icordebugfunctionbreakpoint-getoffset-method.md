# <a name="icordebugfunctionbreakpointgetoffset-method"></a>ICorDebugFunctionBreakpoint::GetOffset 方法
获取断点的函数中的偏移量。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetOffset (  
    [out] ULONG32  *pnOffset  
);  
```  
  
## <a name="parameters"></a>参数  
 `pnOffset`  
 [out]指向该断点的偏移量的指针。
