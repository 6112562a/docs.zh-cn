# <a name="icordebugassemblyenum-interface"></a>ICorDebugAssemblyEnum 接口

实现 ICorDebugEnum 方法并枚举 ICorDebugAssembly 数组。  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[Next 方法](icordebugassemblyenum-next-method.md)|从当前位置开始，获取`ICorDebugAssembly`枚举中指定数量的实例。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
