# <a name="icordebughandlevaluedispose-method"></a>ICorDebugHandleValue::Dispose 方法
释放此 ICorDebugHandleValue 对象引用的句柄，无需显式释放接口指针。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Dispose ();  
```
