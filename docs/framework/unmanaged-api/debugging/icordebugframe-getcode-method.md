# <a name="icordebugframegetcode-method"></a>ICorDebugFrame::GetCode 方法
获取一个指向与此堆栈帧关联的代码。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetCode (  
    [out] ICorDebugCode      **ppCode  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppCode`  
 [out]指向一个 ICorDebugCode 对象，表示此帧与关联的代码的地址的指针。
