# <a name="icordebugstepper2-interface"></a>ICorDebugStepper2 接口
为 "仅我的代码" (JMC) 调试提供支持。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[SetJMC 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugstepper2-setjmc-method.md)|设置一个值, 该值指定此 ICorDebugStepper 是否仅通过应用程序开发人员编写的代码执行步骤。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
