# <a name="icordebugmemorybuffergetstartaddress-method"></a>ICorDebugMemoryBuffer：： GetStartAddress 方法
获取内存缓冲区的起始地址。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetStartAddress(  
   [out] LPCVOID *address  
);  
```  
  
## <a name="parameters"></a>参数  
 `address`  
 [out] 指向内存缓冲区起始地址的指针。  
  
## <a name="remarks"></a>备注  
  
> [!WARNING]
> 此方法仅适用于 .NET Native。
