# <a name="icordebugcode4-interface"></a>ICorDebugCode4 接口
提供一个方法，该方法使调试器可以枚举函数中的局部变量和参数。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[EnumerateVariableHomes 方法](icordebugcode4-enumeratevariablehomes-method.md)|获取函数中局部变量和参数的枚举器。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
