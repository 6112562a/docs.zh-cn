# <a name="icordebuggetprocess-method"></a>ICorDebug::GetProcess 方法
获取指向指定进程的 "ICorDebugProcess" 实例的指针。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetProcess (  
    [in] DWORD               dwProcessId,  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
## <a name="parameters"></a>参数  
 `dwProcessId`  
 中进程的 ID。  
  
 `ppProcess`  
 弄指向指定进程的`ICorDebugProcess`实例地址的指针。
