# <a name="icordebugcontrollerisrunning-method"></a>ICorDebugController::IsRunning 方法
获取一个值，该值指示进程中的线程当前是否可以自由运行。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT IsRunning (  
    [out] BOOL *pbRunning  
);  
```  
  
## <a name="parameters"></a>参数  
 `pbRunning`  
 弄指向一个值的指针，如果进程中的线程自由运行，则该值为 `true`;否则，`false`。
