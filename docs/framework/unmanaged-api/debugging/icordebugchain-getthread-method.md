# <a name="icordebugchaingetthread-method"></a>ICorDebugChain::GetThread 方法
获取此调用链所属的物理线程。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetThread (  
    [out] ICorDebugThread    **ppThread  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppThread`  
 弄指向 ICorDebugThread 对象的指针，该对象表示此调用链所属的物理线程。
