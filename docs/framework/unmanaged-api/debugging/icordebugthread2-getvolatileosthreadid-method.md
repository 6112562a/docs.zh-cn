# <a name="icordebugthread2getvolatileosthreadid-method"></a>ICorDebugThread2::GetVolatileOSThreadID 方法
获取此 ICorDebugThread2 操作系统线程标识符。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetVolatileOSThreadID (  
    [out] DWORD    *pdwTid  
);  
```  
  
## <a name="parameters"></a>参数  
 `pdwTid`  
 [out]此线程由操作系统线程标识符。
