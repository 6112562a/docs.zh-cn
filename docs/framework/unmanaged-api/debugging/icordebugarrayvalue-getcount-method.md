# <a name="icordebugarrayvaluegetcount-method"></a>ICorDebugArrayValue::GetCount 方法
获取数组中的元素总数。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetCount (  
    [out] ULONG32 *pnCount  
);  
```  
  
## <a name="parameters"></a>参数  
 `pnCount`  
 弄指向数组中元素总数的指针。
