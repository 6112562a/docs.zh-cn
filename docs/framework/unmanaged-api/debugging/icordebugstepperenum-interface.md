# <a name="icordebugstepperenum-interface"></a>ICorDebugStepperEnum 接口
实现 ICorDebugEnum 方法，并枚举 ICorDebugStepper 数组。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugstepperenum-next-method.md)|从当前位置开始，从枚举中获取指定数目的 `ICorDebugStepper` 实例。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
