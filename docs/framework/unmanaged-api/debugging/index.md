# <a name="debugging-unmanaged-api-reference"></a>调试（非托管 API 参考）
调试 API 允许调试器对在公共语言运行时 (CLR) 环境中运行的代码进行调试。 要调试的代码可以是 CLR 支持的任何类型的代码。
