# <a name="icordebugappdomainattach-method"></a>ICorDebugAppDomain::Attach 方法
将调试器附加到应用程序域。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Attach ();  
```  
  
## <a name="remarks"></a>备注  
 调试器必须附加到应用程序域才能接收事件，并以启用调试的应用程序域。
