# <a name="icordebugappdomainattach-method"></a>ICorDebugAppDomain::Attach 方法
将调试器附加到应用程序域。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Attach ();  
```  
  
## <a name="remarks"></a>备注  
 调试器必须附加到应用程序域，才能接收事件和启用应用程序域的调试。
