# <a name="iclrmetadatalocator-interface"></a>ICLRMetadataLocator 接口
由数据访问服务层用于在目标进程中查找程序集的元数据。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[GetMetadata 方法](../../../../docs/framework/unmanaged-api/debugging/iclrmetadatalocator-getmetadata-method.md)|从目标进程中检索图像的元数据。|  
  
## <a name="remarks"></a>备注  
 API 客户端（即调试器）必须针对特定的目标进程实现此接口。 例如，实时过程的实现与内存转储的实现不同。
