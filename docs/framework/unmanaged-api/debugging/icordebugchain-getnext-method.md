# <a name="icordebugchaingetnext-method"></a>ICorDebugChain::GetNext 方法
获取线程的下一帧链。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetNext (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppChain`  
 [out]指向一个 ICorDebugChain 对象，表示下一个线程的帧链的地址的指针。 如果此证书链的最后一个链`ppChain`为 null。
