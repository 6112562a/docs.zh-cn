# <a name="icordebugchaingetnext-method"></a>ICorDebugChain::GetNext 方法
获取线程的下一个帧链。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetNext (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppChain`  
 弄指向 ICorDebugChain 对象的地址的指针，该对象表示线程的下一个帧链。 如果此链是最后一个链，则 `ppChain` 为 null。
