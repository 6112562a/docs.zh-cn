# <a name="icordebugdatatarget2getsymbolproviderforimage-method"></a>ICorDebugDataTarget2::GetSymbolProviderForImage 方法
返回该模块基址中的模块符号提供程序。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetSymbolProviderForImage(  
    [in] CORDB_ADDRESS imageBaseAddress,
    [out] ICorDebugSymbolProvider **ppSymProvider  
);  
```  
  
## <a name="parameters"></a>parameters  
 `imageBaseAddress`  
 [在]表示模块基本地址的[CORDB_ADDRESS](../../../../docs/framework/unmanaged-api/common-data-types-unmanaged-api-reference.md)值。  
  
 `ppSymProvider`  
 [出]指向[ICorDebugSymbol 提供程序](icordebugsymbolprovider-interface.md)对象地址的指针。  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
> 此方法仅适用于 .NET Native。
