# <a name="icordebugprocessgetthread-method"></a>ICorDebugProcess::GetThread 方法
获取此进程的线程，该线程具有指定的操作系统（OS）线程 ID。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetThread(  
    [in] DWORD dwThreadId,  
    [out] ICorDebugThread **ppThread);  
```  
  
## <a name="parameters"></a>参数  
 `dwThreadId`  
 中要检索的线程的 OS 线程 ID。  
  
 `ppThread`  
 弄指向表示线程的 ICorDebugThread 对象的地址的指针。
