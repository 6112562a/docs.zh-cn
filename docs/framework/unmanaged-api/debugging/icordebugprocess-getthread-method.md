# <a name="icordebugprocessgetthread-method"></a>ICorDebugProcess::GetThread 方法
获取具有指定的操作系统 (OS) 线程 id。 此进程的线程  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetThread(  
    [in] DWORD dwThreadId,  
    [out] ICorDebugThread **ppThread);  
```  
  
## <a name="parameters"></a>参数  
 `dwThreadId`  
 [in]操作系统线程的线程 ID 来检索。  
  
 `ppThread`  
 [out]指向一个 ICorDebugThread 对象，表示在线程的地址的指针。
