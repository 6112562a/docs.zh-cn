# <a name="configuration-element"></a>\<配置 > 元素

公共语言运行时和 .NET Framework 应用程序所使用的每个配置文件中的根元素。

**\<configuration>**

## <a name="syntax"></a>语法

```xml
<configuration>
  <!-- Configuration settings -->
</configuration>
```

## <a name="attributes"></a>特性

无

## <a name="parent-element"></a>父元素

无
