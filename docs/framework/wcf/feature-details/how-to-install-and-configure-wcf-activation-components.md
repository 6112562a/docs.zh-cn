### <a name="to-configure-the-was-to-support-tcp-activation"></a><span data-ttu-id="ca325-115">配置 WAS 以支持 TCP 激活</span><span class="sxs-lookup"><span data-stu-id="ca325-115">To configure the WAS to support TCP activation</span></span>  
  
1. <span data-ttu-id="ca325-116">若要支持 net.tcp 激活，必须首先将默认的网站绑定到一个 net.tcp 端口。</span><span class="sxs-lookup"><span data-stu-id="ca325-116">To support net.tcp activation, the default Web site must first be bound to a net.tcp port.</span></span> <span data-ttu-id="ca325-117">可以通过使用随 [!INCLUDE[iisver](../../../../includes/iisver-md.md)] 管理工具集安装的 Appcmd.exe 来执行此操作。</span><span class="sxs-lookup"><span data-stu-id="ca325-117">You can do this by using Appcmd.exe, which is installed with the [!INCLUDE[iisver](../../../../includes/iisver-md.md)] management toolset.</span></span> <span data-ttu-id="ca325-118">在管理员级别命令提示符窗口中，运行以下命令。</span><span class="sxs-lookup"><span data-stu-id="ca325-118">In an administrator-level Command Prompt window, run the following command.</span></span>  
  
    ```  
    %windir%\system32\inetsrv\appcmd.exe set site "Default Web Site" -+bindings.[protocol='net.tcp',bindingInformation='808:*']  
    ```  
  
    > [!NOTE]
    >  <span data-ttu-id="ca325-119">此命令是单行文本。</span><span class="sxs-lookup"><span data-stu-id="ca325-119">This command is a single line of text.</span></span> <span data-ttu-id="ca325-120">此命令将 net.tcp 网站绑定添加到以任何主机名侦听 TCP 端口 808 的默认网站。</span><span class="sxs-lookup"><span data-stu-id="ca325-120">This command adds a net.tcp site binding to the default Web site listening on TCP port 808 with any host name.</span></span>  
  
2. <span data-ttu-id="ca325-121">尽管网站内的所有应用程序共享一个公共 net.tcp 绑定，但是每个应用程序可以单独启用 net.tcp 支持。</span><span class="sxs-lookup"><span data-stu-id="ca325-121">Although all applications within a site share a common net.tcp binding, each application can enable net.tcp support individually.</span></span> <span data-ttu-id="ca325-122">若要启用应用程序的 net.tcp，请从管理员级别命令提示符运行以下命令。</span><span class="sxs-lookup"><span data-stu-id="ca325-122">To enable net.tcp for the application, run the following command from an administrator-level command prompt.</span></span>  
  
    ```  
    %windir%\system32\inetsrv\appcmd.exe set app   
    "Default Web Site/<WCF Application>" /enabledProtocols:http,net.tcp  
    ```  
  
    > [!NOTE]
    >  <span data-ttu-id="ca325-123">此命令是单行文本。</span><span class="sxs-lookup"><span data-stu-id="ca325-123">This command is a single line of text.</span></span> <span data-ttu-id="ca325-124">此命令启用 /\<*WCF 应用程序*> 应用程序使用同时访问`http://localhost/<WCF Application>`和`net.tcp://localhost/<WCF Application>`。</span><span class="sxs-lookup"><span data-stu-id="ca325-124">This command enables the /\<*WCF Application*> application to be accessed using both `http://localhost/<WCF Application>` and `net.tcp://localhost/<WCF Application>`.</span></span>
  
     <span data-ttu-id="ca325-125">移除为此示例添加的 net.tcp 网站绑定。</span><span class="sxs-lookup"><span data-stu-id="ca325-125">Remove the net.tcp site binding you added for this sample.</span></span>  
  
     <span data-ttu-id="ca325-126">为方便起见，在位于示例目录中名为 RemoveNetTcpSiteBinding.cmd 的批处理文件中实现以下两个步骤。</span><span class="sxs-lookup"><span data-stu-id="ca325-126">As a convenience, the following two steps are implemented in a batch file called RemoveNetTcpSiteBinding.cmd located in the sample directory.</span></span>  
  
    1. <span data-ttu-id="ca325-127">通过在管理员级别命令提示符窗口中运行以下命令，从启用的协议列表移除 net.tcp。</span><span class="sxs-lookup"><span data-stu-id="ca325-127">Remove net.tcp from the list of enabled protocols by running the following command in an administrator-level Command Prompt window.</span></span>  
  
        ```  
        %windir%\system32\inetsrv\appcmd.exe set app   
        "Default Web Site/servicemodelsamples<WCF Application>" " /enabledProtocols:http  
        ```  
  
        > [!NOTE]
        >  <span data-ttu-id="ca325-128">此命令是单行文本。</span><span class="sxs-lookup"><span data-stu-id="ca325-128">This command is a single line of text.</span></span>  
  
    2. <span data-ttu-id="ca325-129">通过在提升的命令提示符窗口中运行以下命令移除 net.tcp 网站绑定：</span><span class="sxs-lookup"><span data-stu-id="ca325-129">Remove the net.tcp site binding by running the following command in an elevated Command Prompt window:</span></span>  
  
        ```  
        %windir%\system32\inetsrv\appcmd.exe set site "Default Web Site"   
        --bindings.[protocol='net.tcp',bindingInformation='808:*']  
        ```  
  
        > [!NOTE]
        >  <span data-ttu-id="ca325-130">此命令是单行文本。</span><span class="sxs-lookup"><span data-stu-id="ca325-130">This command is a single line of text.</span></span>  
  
### <a name="to-remove-nettcp-from-the-list-of-enabled-protocols"></a><span data-ttu-id="ca325-131">从启用的协议列表中移除 net.tcp</span><span class="sxs-lookup"><span data-stu-id="ca325-131">To remove net.tcp from the list of enabled protocols</span></span>
