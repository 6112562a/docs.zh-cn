# <a name="metadata"></a>元数据
Windows Communication Foundation (WCF) 提供了用于导出、 发布、 检索和导入服务元数据的基础结构。 WCF 服务使用元数据来描述如何与服务的终结点交互，以便诸如 Svcutil.exe 之类的工具，可以自动生成用于访问服务的客户端代码。
