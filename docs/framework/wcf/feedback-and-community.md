# <a name="feedback-and-community"></a>反馈和社区
我们非常感谢您的评论和问题有关 Windows Communication Foundation (WCF) 或[!INCLUDE[infocard](../../../includes/infocard-md.md)]文档。 在 SDK 中的每个页面底部都有一句“将有关此主题的意见发送给 Microsoft”。 点击“注释”链接可发送有关特定主题的观点或建议。  
  
 如果你正在寻找的详细信息，或者想要共享使用的其他人感兴趣 WCF，请尝试[MSDN Windows Communication Foundation 论坛](https://go.microsoft.com/fwlink/?LinkId=186463)。
