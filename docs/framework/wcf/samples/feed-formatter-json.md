## <a name="architecture-of-the-sample"></a><span data-ttu-id="6a010-104">示例的体系结构</span><span class="sxs-lookup"><span data-stu-id="6a010-104">Architecture of the Sample</span></span>

## <a name="writing-json-feeds"></a><span data-ttu-id="6a010-109">编写 JSON 源</span><span class="sxs-lookup"><span data-stu-id="6a010-109">Writing JSON feeds</span></span>  
 <span data-ttu-id="6a010-110">通过与 `JsonFeedFormatter` 一起使用 <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer>（在本示例中实现），可实现编写 JSON 源，如下面的示例代码所示。</span><span class="sxs-lookup"><span data-stu-id="6a010-110">Writing a JSON feed can be accomplished by using the `JsonFeedFormatter` (implemented in this sample) with the <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer> as shown in the following sample code.</span></span>  
  
```  
//Basic feed with sample data  
SyndicationFeed feed = new SyndicationFeed("Custom JSON feed", "A Syndication extensibility sample", null);  
feed.LastUpdatedTime = DateTime.Now;  
feed.Items = from s in new string[] { "hello", "world" }  
select new SyndicationItem()  
{  
    Summary = SyndicationContent.CreatePlaintextContent(s)  
};  
  
//Write the feed out to a MemoryStream in JSON format  
DataContractJsonSerializer writeSerializer = new DataContractJsonSerializer(typeof(JsonFeedFormatter));  
writeSerializer.WriteObject(stream, new JsonFeedFormatter(feed));  
```
