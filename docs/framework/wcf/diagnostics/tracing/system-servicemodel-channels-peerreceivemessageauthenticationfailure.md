# <a name="systemservicemodelchannelspeerreceivemessageauthenticationfailure"></a>System.ServiceModel.Channels.PeerReceiveMessageAuthenticationFailure
未能对消息源进行身份验证  
  
## <a name="description"></a>描述  
 尝试验证传入消息的身份并在绑定上启用消息身份验证时，会出现此跟踪。
