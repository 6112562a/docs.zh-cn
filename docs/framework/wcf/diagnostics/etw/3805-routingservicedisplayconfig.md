## <a name="properties"></a>属性  
  
|||  
|-|-|  
|Id|3805|  
|关键字|RoutingServices|  
|级别|详细|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述  
 当使用 RouteOnHeadersOnly 配置路由服务时，发出此事件。  
  
## <a name="message"></a>消息  
 使用 RouteOnHeadersOnly %1、SoapProcessingEnabled %2 和 EnsureOrderedDispatch %3 配置路由服务。
