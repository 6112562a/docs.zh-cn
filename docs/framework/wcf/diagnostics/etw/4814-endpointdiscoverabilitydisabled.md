## <a name="properties"></a>属性  
  
|||  
|-|-|  
|Id|4814|  
|关键字|发现|  
|级别|信息|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/调试|  
  
## <a name="description"></a>描述  
 当禁用了终结点的可发现性时，发出此事件。  
  
## <a name="message"></a>消息  
 已禁用 EndpointAddress 为“%1”和 ListenUri 为“%2”的终结点的可发现性。
