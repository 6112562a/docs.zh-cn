## <a name="properties"></a>属性  
  
|||  
|-|-|  
|Id|1443|  
|关键字|配额|  
|级别|Error|  
|通道|Microsoft-Windows-应用程序服务器-应用程序/分析|

 当对于 MSMQ 消息超过最大重试周期时，将发出此事件。

 对于 id 为“%1”的 MSMQ 消息，已超过最大重试周期数
