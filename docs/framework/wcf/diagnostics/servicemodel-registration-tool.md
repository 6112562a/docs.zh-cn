# <a name="servicemodel-registration-tool"></a>ServiceModel 注册工具
ServiceModel 注册工具是一个命令行工具，可用于管理 ServiceModel 在单个计算机上的注册。 此工具的详细信息，请参阅[ServiceModel 注册工具 (ServiceModelReg.exe)](../../../../docs/framework/wcf/servicemodelreg-exe.md)。
