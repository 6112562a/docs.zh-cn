# <a name="endpoint-calls-per-second"></a>终结点：Calls Per Second（每秒调用次数）
计数器名称：每秒调用。  
  
## <a name="description"></a>描述  
 一秒内对此终结点的调用次数。  
  
 此计数器为性能计数器类型[PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649)，使用以下公式计算其值。  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
