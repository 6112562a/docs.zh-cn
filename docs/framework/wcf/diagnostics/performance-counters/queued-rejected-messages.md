# <a name="queued-rejected-messages"></a>Queued Rejected Messages（已排队的被拒绝消息数）
计数器名称：已拒绝的排队的消息。  
  
## <a name="description"></a>描述  
 此服务中被排队传输拒绝的消息的数目。  
  
 请参阅[病毒消息处理](https://go.microsoft.com/fwlink/?LinkID=96233)有关何时消息被拒绝的详细信息。
