# <a name="queue-dropped-messages-per-second"></a>每秒丢弃的排队消息数
计数器名称：每秒丢弃的排队的消息。  
  
## <a name="description"></a>描述  
 此服务处的排队传输机制每秒丢弃的消息数。  
  
 此计数器为性能计数器类型[PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649)，使用以下公式计算其值。  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
