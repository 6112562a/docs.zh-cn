# <a name="queued-poison-messages"></a>Queued Poison Messages（已排队的病毒消息数）
计数器名称：排队病毒消息数。  
  
## <a name="description"></a>描述  
 此服务中由排队传输标记为中毒的消息的数目。  
  
 请参阅[病毒消息处理](https://go.microsoft.com/fwlink/?LinkID=96233)有关何时将消息标记为中毒的更多信息。
