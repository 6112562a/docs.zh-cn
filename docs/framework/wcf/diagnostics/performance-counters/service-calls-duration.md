# <a name="service-calls-duration"></a>服务：调用持续时间
计数器名称：调用持续时间  
  
## <a name="description"></a>描述  
 调用此服务的平均持续时间。 根据此公式计算的平均持续时间：(N1-N0)/(D1-D0).
