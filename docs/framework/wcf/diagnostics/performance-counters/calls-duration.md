# <a name="calls-duration"></a>调用持续时间
计数器名称：调用持续时间  
  
## <a name="description"></a>描述  
 调用该操作的平均持续时间。 根据此公式计算的平均持续时间：(N1-N0)/(D1-D0).  
  
> [!WARNING]
>  在异步 WCF 服务中使用时将调用持续时间计数器将始终返回-1。
