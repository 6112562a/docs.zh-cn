# <a name="messagequeueduplicatedsocketleak"></a>MessageQueueDuplicatedSocketLeak
ID:165  
  
 严重性：Error  
  
 类别：SMSvcHost  
  
## <a name="description"></a>描述  
 此事件指示调度重复套接字时发生错误。 此句柄当前已在进程中泄漏。 此事件将列出源、异常、进程名称和进程 ID。
