# <a name="missingnecessaryenhancedkeyusage"></a>MissingNecessaryEnhancedKeyUsage
ID:157  
  
 对应Error  
  
 类别TransactionBridge  
  
## <a name="description"></a>描述  
 此事件指示具有特定主题名称和指纹的标识证书不在其 EnhancedKeyUsages 之间提供特定的数据。 该事件将列出进程名称和进程 ID。
