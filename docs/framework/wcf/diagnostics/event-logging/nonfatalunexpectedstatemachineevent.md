# <a name="nonfatalunexpectedstatemachineevent"></a>NonFatalUnexpectedStateMachineEvent
ID:148  
  
 对应Error  
  
 类别TransactionBridge  
  
## <a name="description"></a>描述  
 此事件指示状态机尝试处理意外事件。 此事件不被视为是致命的。 此事件将列出事务 ID、状态机、当前状态、历史记录、事件名称、事件详细信息、进程名称和进程 ID。
