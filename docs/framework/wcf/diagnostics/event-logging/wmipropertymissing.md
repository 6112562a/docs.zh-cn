# <a name="wmipropertymissing"></a>WmiPropertyMissing
ID:129  
  
 严重性：Error  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示出现类型不匹配。 指定的 CIM 类没有特定类型的指定属性。 该事件将列出进程名称和进程 ID。
