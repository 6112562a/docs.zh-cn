# <a name="sslnoprivatekey"></a>SslNoPrivateKey
ID:154  
  
 对应Error  
  
 类别TransactionBridge  
  
## <a name="description"></a>描述  
 此事件指示带有特定主题名称和指纹的标识证书不具有私钥。 该事件将列出进程名称和进程 ID。
