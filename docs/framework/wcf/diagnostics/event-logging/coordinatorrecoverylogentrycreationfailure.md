# <a name="coordinatorrecoverylogentrycreationfailure"></a>CoordinatorRecoveryLogEntryCreationFailure
ID:140  
  
 对应Error  
  
 类别TransactionBridge  
  
## <a name="description"></a>描述  
 无法为协调器列入生成恢复日志项。 事务将中止。 事件将列出事务 ID、原因、异常、进程名称和进程 ID。
