# <a name="wmigetobjectfailed"></a>WmiGetObjectFailed
ID:120  
  
 严重性：Error  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示未处理 WMI GetObject 请求。 此事件将列出 WMI 对象、异常、进程名称和进程 ID。
