# <a name="serviceauthorizationsuccess"></a>ServiceAuthorizationSuccess
ID:176  
  
 对应信息性  
  
 类别SecurityAudit  
  
## <a name="description"></a>描述  
 此事件指示服务授权成功。 此事件列出了服务、操作、客户端标识、授权上下文、活动 ID 和 ServiceAuthorizationManager。
