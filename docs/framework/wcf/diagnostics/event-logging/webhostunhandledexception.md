# <a name="webhostunhandledexception"></a>WebHostUnhandledException
ID:109  
  
 对应Error  
  
 类别ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示发生了 WebHost 未处理的异常。 此事件列出了发送方信息、异常、进程名称和进程 ID。
