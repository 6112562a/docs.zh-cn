# <a name="invariantassertionfailed"></a>InvariantAssertionFailed
ID:106  
  
 严重性：Error  
  
 类别：跟踪  
  
## <a name="description"></a>描述  
 此事件指示固定断言为 false。 此事件将列出消息、进程名称和进程 ID。
