# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed
ID:123  
  
 对应Error  
  
 类别ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示尚未处理 WMI CreateInstance 请求。 该事件列出了类名、异常、进程名称和进程 ID。
