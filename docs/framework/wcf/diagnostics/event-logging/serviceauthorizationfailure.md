# <a name="serviceauthorizationfailure"></a>ServiceAuthorizationFailure
ID:168  
  
 对应Error  
  
 类别SecurityAudit  
  
## <a name="description"></a>描述  
 此事件指示服务授权失败。 此事件列出了服务、操作、客户端标识、AuthorizationContext、活动 ID 和 ServiceAuthorizationManager。
