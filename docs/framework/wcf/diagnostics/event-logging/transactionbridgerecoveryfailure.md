# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
ID:146  
  
 严重性：Error  
  
 类别：TransactionBridge  
  
## <a name="description"></a>描述  
 此事件指示恢复期间 MSDTC TransactionBridge 失败。 这是错误状态，因此 MSDTC 服务将终止。 该事件列出了异常、进程名称和进程 ID。 .
