# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
ID:122  
  
 严重性：Error  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示未处理 WMI DeleteInstance 请求。 该事件列出了异常、进程名称和进程 ID。
