# <a name="wcf-system-requirements"></a>WCF 系统需求

Windows Communication Foundation （WCF）是用于创建分布式应用程序的通信基础结构。 下面列出了允许 WCF 运行的要求。

## <a name="system-requirements"></a>系统要求

默认情况下，WCF 安装在 [!INCLUDE[wv](../../../includes/wv-md.md)] 上。

还可以在 [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)]、Windows Server 2003 R2 或 [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] 上安装 WCF。

> [!NOTE]
> WCF 的消息队列（MSMQ）功能仅在 [!INCLUDE[wv](../../../includes/wv-md.md)]、Windows Server 2003 R2、[!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] 和 Windows XP Professional 上受支持。
