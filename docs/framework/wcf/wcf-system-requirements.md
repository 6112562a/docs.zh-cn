# <a name="wcf-system-requirements"></a>WCF 系统需求

Windows Communication Foundation （WCF）是用于创建分布式应用程序的通信基础结构。 下面列出了允许 WCF 运行的要求。

## <a name="system-requirements"></a>系统要求

默认情况下，在上[!INCLUDE[wv](../../../includes/wv-md.md)]安装 WCF。

WCF 还可以安装在、 [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)] [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]或[!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]上。

> [!NOTE]
> WCF 的消息队列（MSMQ）功能[!INCLUDE[wv](../../../includes/wv-md.md)]仅在[!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]、 [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]、和[!INCLUDE[wxppro](../../../includes/wxppro-md.md)]上受支持。
