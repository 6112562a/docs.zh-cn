## <a name="adaptive-proxies"></a><span data-ttu-id="c0f0c-107">自适应代理</span><span class="sxs-lookup"><span data-stu-id="c0f0c-107">Adaptive Proxies</span></span>

 <span data-ttu-id="c0f0c-111">自适应代理是通过配置脚本配置的（请参阅[自动代理检测](../../../docs/framework/network-programming/automatic-proxy-detection.md)）。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-111">Adaptive proxies are configured by a configuration script (see [Automatic Proxy Detection](../../../docs/framework/network-programming/automatic-proxy-detection.md)).</span></span> <span data-ttu-id="c0f0c-112">脚本针对每个协议生成一组应用程序协议和一个代理。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-112">The script generates a set of application protocols and a proxy for each protocol.</span></span>  
  
 <span data-ttu-id="c0f0c-113">网络环境的变化可能要求系统使用一组新代理。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-113">Changes in the network environment may require that the system use a new set of proxies.</span></span> <span data-ttu-id="c0f0c-114">如果网络连接不可用或新的网络连接已初始化，系统必须在新环境中发现相应的配置脚本源并运行新脚本。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-114">If a network connection goes down or a new network connection is initialized, the system must discover the appropriate source of the configuration script in the new environment and run the new script.</span></span>  
  
 <span data-ttu-id="c0f0c-115">可以使用配置文件中的 [`<proxy>`](../configure-apps/file-schema/network/proxy-element-network-settings.md) 元素的 `usesystemdefault` 属性。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-115">You can use the `usesystemdefault` attribute of the [`<proxy>`](../configure-apps/file-schema/network/proxy-element-network-settings.md) element in your configuration file.</span></span> <span data-ttu-id="c0f0c-116">`usesystemdefault` 属性控制是否应从用户的 Internet Explorer 代理设置读取静态代理设置（代理地址、跳过列表和跳过本地）。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-116">The `usesystemdefault` attribute controls whether the static proxy settings (proxy address, bypass list, and bypass on local) should be read from the Internet Explorer proxy settings for the user.</span></span> <span data-ttu-id="c0f0c-117">如果此值设置为 `true`，那么将使用来自 Internet Explorer 的静态代理设置。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-117">If this value is set to `true`, the static proxy settings from Internet Explorer will be used.</span></span> <span data-ttu-id="c0f0c-118">如果此值设置为 `false` 或未设置，那么静态代理设置可在配置中指定并将替代 Internet Explorer 代理设置。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-118">If this value is `false` or not set, the static proxy settings can be specified in the configuration and will override the Internet Explorer proxy settings.</span></span> <span data-ttu-id="c0f0c-119">此外，此值必须设置为 `false` 或不设置，才能启用自适应代理。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-119">This value must also be set to `false` or not set for adaptive proxies to be enabled.</span></span>  
  
 <span data-ttu-id="c0f0c-120">以下例子显示了典型的自适应代理配置。</span><span class="sxs-lookup"><span data-stu-id="c0f0c-120">The following example shows a typical adaptive proxy configuration.</span></span>  
  
```xml  
<system.net>  
    <defaultProxy>  
      <proxy usesystemdefault="false" />
    </defaultProxy>  
</system.net>  
```
