## <a name="the-new-packagereference-element"></a><span data-ttu-id="2121d-110">新 \<PackageReference> 元素</span><span class="sxs-lookup"><span data-stu-id="2121d-110">The new \<PackageReference> element</span></span>
<span data-ttu-id="2121d-111">`<PackageReference>` 具有下列基本结构：</span><span class="sxs-lookup"><span data-stu-id="2121d-111">The `<PackageReference>` has the following basic structure:</span></span>
