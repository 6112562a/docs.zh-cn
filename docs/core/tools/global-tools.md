### <a name="what-could-go-wrong"></a><span data-ttu-id="ac707-152">可能出现的问题</span><span class="sxs-lookup"><span data-stu-id="ac707-152">What could go wrong</span></span>

<span data-ttu-id="ac707-153">全局工具是[依赖框架的应用程序](../deploying/index.md#framework-dependent-deployments-fdd)，也就是说它们依赖于计算机上安装的 .NET Core 运行时。</span><span class="sxs-lookup"><span data-stu-id="ac707-153">Global Tools are [framework-dependent applications](../deploying/index.md#framework-dependent-deployments-fdd), which means they rely on a .NET Core runtime installed on your machine.</span></span> <span data-ttu-id="ac707-154">如果没有找到所需的运行时，则遵循常规的 .NET Core 运行时前滚规则，例如：</span><span class="sxs-lookup"><span data-stu-id="ac707-154">If the expected runtime is not found, they follow normal .NET Core runtime roll-forward rules such as:</span></span>

<span data-ttu-id="ac707-161">如果应用程序无法找到合适的运行时，它就无法运行并报告错误。</span><span class="sxs-lookup"><span data-stu-id="ac707-161">If an application cannot find an appropriate runtime, it fails to run and reports an error.</span></span>
