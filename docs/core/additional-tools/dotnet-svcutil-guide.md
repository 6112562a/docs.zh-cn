<span data-ttu-id="f3dcf-105">类似于 .NET Framework 项目的[服务模型元数据 - svcutil](../../framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md) 工具，dotnet svcutil 是用于生成 Web 服务引用的命令行工具，与 .NET Core 和 .NET Standard 项目兼容。</span><span class="sxs-lookup"><span data-stu-id="f3dcf-105">Similar to the [**Service Model Metadata - svcutil**](../../framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md) tool for .NET Framework projects, the **dotnet-svcutil** is a command-line tool for generating a web service reference compatible with .NET Core and .NET Standard projects.</span></span>

## <a name="getting-started"></a><span data-ttu-id="f3dcf-113">入门</span><span class="sxs-lookup"><span data-stu-id="f3dcf-113">Getting started</span></span>

<span data-ttu-id="f3dcf-114">下面的示例将指导你完成将 Web 服务引用添加到 .NET Core 控制台项目并调用该服务所需的步骤。</span><span class="sxs-lookup"><span data-stu-id="f3dcf-114">The following example walks you through the steps required to add a web service reference to a .NET Core console project and invoke the service.</span></span> <span data-ttu-id="f3dcf-115">你将创建名为“HelloSvcutil”的 .NET Core 控制台应用程序，并将引用添加到实现以下协定的 Web 服务：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-115">You will create a .NET Core console application named _HelloSvcutil_ and will add a reference to a web service that implements the following contract:</span></span>

```csharp
[ServiceContract]
public interface ISayHello
{
    [OperationContract]
    string Hello(string name);
}
```

<span data-ttu-id="f3dcf-116">在此示例中，将假定 Web 服务托管在以下地址中：`http://contoso.com/SayHello.svc`</span><span class="sxs-lookup"><span data-stu-id="f3dcf-116">For this example, the web service will be assumed to be hosted at the following address: `http://contoso.com/SayHello.svc`</span></span>

<span data-ttu-id="f3dcf-117">从 Windows、macOS 或 Linux 命令窗口执行以下步骤：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-117">From a Windows, macOS, or Linux command window perform the following steps:</span></span>

1. <span data-ttu-id="f3dcf-118">为项目创建一个名为“HelloSvcutil”的目录，并将其设置为当前目录，如以下示例所示：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-118">Create a directory named _HelloSvcutil_ for your project and make it your current directory, as in the following example:</span></span>

```console

cd HelloSvcutil
```

2. <span data-ttu-id="f3dcf-119">在该目录中使用 [`dotnet new`](../tools/dotnet-new.md) 命令创建新的 C# 控制台项目项目，如下所示：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-119">Create a new C# console project in that directory using the [`dotnet new`](../tools/dotnet-new.md) command as follows:</span></span>


3. <span data-ttu-id="f3dcf-120">在编辑器中打开 `HelloSvcutil.csproj` 项目文件，编辑 `Project` 元素，并使用下面的代码添加 [`dotnet-svcutil` NuGet 包](https://nuget.org/packages/dotnet-svcutil)作为 CLI 工具引用：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-120">Open the `HelloSvcutil.csproj` project file in your editor, edit the `Project` element, and add the [`dotnet-svcutil` NuGet package](https://nuget.org/packages/dotnet-svcutil) as a CLI tool reference, using the following code:</span></span>

```xml

  <DotNetCliToolReference Include="dotnet-svcutil" Version="1.0.*" />
</ItemGroup>
```

4. <span data-ttu-id="f3dcf-121">使用 [`dotnet restore`](../tools/dotnet-restore.md) 命令还原 dotnet-svcutil 包，如下所示：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-121">Restore the _dotnet-svcutil_ package using the [`dotnet restore`](../tools/dotnet-restore.md) command as follows:</span></span>

```console

```

5. <span data-ttu-id="f3dcf-122">运行 _dotnet_ 与 _svcutil_ 命令生成 Web 服务引用文件，如下所示：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-122">Run _dotnet_ with the _svcutil_ command to generate the web service reference file as follows:</span></span>

```console

```
<span data-ttu-id="f3dcf-123">生成的文件保存为 _HelloSvcutil/ServiceReference1/Reference.cs_。</span><span class="sxs-lookup"><span data-stu-id="f3dcf-123">The generated file is saved as _HelloSvcutil/ServiceReference1/Reference.cs_.</span></span> <span data-ttu-id="f3dcf-124">_dotnet_svcutil_ 工具还向项目添加代理代码所需的适当 WCF 包作为包引用。</span><span class="sxs-lookup"><span data-stu-id="f3dcf-124">The _dotnet_svcutil_ tool also adds to the project the appropriate WCF packages required by the proxy code as package references.</span></span>

6. <span data-ttu-id="f3dcf-125">使用 [`dotnet restore`](../tools/dotnet-restore.md) 命令还原 WCF 包，如下所示：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-125">Restore the WCF packages using the [`dotnet restore`](../tools/dotnet-restore.md) command as follows:</span></span>

```console
dotnet restore
```

7. <span data-ttu-id="f3dcf-126">在编辑器中打开 `Program.cs` 文件，编辑 `Main()` 方法，然后使用下列代码替换自动生成的代码来调用 Web 服务：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-126">Open the `Program.cs` file in your editor, edit the `Main()` method, and replace the auto-generated code with the following code to invoke the web service:</span></span>

```csharp
    var client = new SayHelloClient();
    Console.WriteLine(client.HelloAsync("dotnet-svcutil").Result);
```

8. <span data-ttu-id="f3dcf-127">使用 [`dotnet run`](../tools/dotnet-run.md) 命令运行应用程序，如下所示：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-127">Run the application using the [`dotnet run`](../tools/dotnet-run.md) command as follows:</span></span>

```console
dotnet run
```
<span data-ttu-id="f3dcf-128">您应看到以下输出：“Hello dotnet-svcutil!”</span><span class="sxs-lookup"><span data-stu-id="f3dcf-128">You should see the following output: "Hello dotnet-svcutil!"</span></span>

<span data-ttu-id="f3dcf-129">有关 `dotnet-svcutil` 工具参数的详细说明，请调用传递帮助参数的工具，如下所示：</span><span class="sxs-lookup"><span data-stu-id="f3dcf-129">For a detailed description of the `dotnet-svcutil` tool parameters, invoke the tool passing the help parameter as follows:</span></span>
