<span data-ttu-id="f0ae5-184">需要添加到项目中的第一个 `<ItemGroup>` 节点将包括“App.xaml”文件，该文件代表你的应用使用的启动配置和资源  。</span><span class="sxs-lookup"><span data-stu-id="f0ae5-184">The first `<ItemGroup>` node you need to add to your project includes the **App.xaml** file that represents the startup config and resources your app uses.</span></span> <span data-ttu-id="f0ae5-185">“App.xaml”文件还会附带“App.xaml.cs”文件，但它将自动包含在不同的 `<ItemGroup>` 中   。</span><span class="sxs-lookup"><span data-stu-id="f0ae5-185">The **App.xaml** file also has an accompanying **App.xaml.cs** file, but it will be automatically included in a different `<ItemGroup>`.</span></span>

```xml
    <ApplicationDefinition Include="..\MyWPFApp\App.xaml">
      <Generator>MSBuild:Compile</Generator>
    </ApplicationDefinition>
  </ItemGroup>
```

<span data-ttu-id="f0ae5-186">接下来，将以下 `<ItemGroup>` 节点添加到项目中。</span><span class="sxs-lookup"><span data-stu-id="f0ae5-186">Next, add the following `<ItemGroup>` node to your project.</span></span> <span data-ttu-id="f0ae5-187">每个语句都包含一个文件 glob 模式，其中包含子目录。</span><span class="sxs-lookup"><span data-stu-id="f0ae5-187">Each statement includes a file glob pattern that includes child directories.</span></span> <span data-ttu-id="f0ae5-188">它包括项目的源代码、任何设置文件和任何资源。</span><span class="sxs-lookup"><span data-stu-id="f0ae5-188">It includes the source code for your project, any settings files, and any resources.</span></span> <span data-ttu-id="f0ae5-189">将会显式排除“obj”目录  。</span><span class="sxs-lookup"><span data-stu-id="f0ae5-189">The **obj** directory is explicitly excluded.</span></span>

```xml
    <Compile Include="..\MyWPFApp\**\*.cs" Exclude="..\MyWPFApp\obj\**" />
```

<span data-ttu-id="f0ae5-190">接下来，包括另一个 `<ItemGroup>` 节点，该节点包含项目中每个“xaml”文件（“App.xaml”文件除外）的 `<Page>` 条目   。</span><span class="sxs-lookup"><span data-stu-id="f0ae5-190">Next, include another `<ItemGroup>` node that contains a `<Page>` entry for every **xaml** file in your project except the **App.xaml** file.</span></span> <span data-ttu-id="f0ae5-191">它们包含“xaml”格式的所有窗口、页面和资源  。</span><span class="sxs-lookup"><span data-stu-id="f0ae5-191">These contain all of the windows, pages, and resources that are in **xaml** format.</span></span> <span data-ttu-id="f0ae5-192">不能在此处使用 glob 模式，必须为每个文件添加一个条目并指出使用的 `<Generator>`。</span><span class="sxs-lookup"><span data-stu-id="f0ae5-192">You cannot use a glob pattern here and must add an entry for every file and indicate the `<Generator>` used.</span></span>

```xml
  <ItemGroup>
    <Page Include="..\MyWPFApp\MainWindow.xaml">
      <Generator>MSBuild:Compile</Generator>
    </Page>
  </ItemGroup>
```
