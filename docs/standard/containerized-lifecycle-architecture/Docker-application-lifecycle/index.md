# <a name="introduction-tothe-docker-application-life-cycle"></a>Docker 应用程序生命周期简介
 
容器化应用程序的生命周期是一个从开发人员开始的旅程。 开发人员选择实现容器和 Docker ，因为它消除了部署和 IT 操作中的摩擦，最终帮助每个用户提高敏捷性、端到端工作效率和速度。
