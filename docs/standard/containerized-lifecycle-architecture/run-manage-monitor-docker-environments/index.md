# <a name="run-manage-and-monitor-docker-production-environments"></a>运行、管理和监视 Docker 生产环境

设想：企业应用程序需要使用高可用性和高可伸缩性; 才能运行IT 操作需要能够管理和监视环境及应用程序本身。

容器化 Docker 应用程序生命周期的最后一个关键点在于，如何能够在可缩放的高可用性 (HA) 生产环境中运行、管理和监视应用程序。

如何在生产环境中运行容器化应用程序（基础体系结构和平台技术）也与此电子书第 1 章中介绍的选定基础结构和开发平台密切相关，并完全建立在这些基础结构和平台的基础之上。 本章检验 Microsoft 和其他供应商提供的、可用于有效运行高度可伸缩的 HA 分布式应用程序的特定产品和技术，并介绍如何从 IT 视角管理和监视它们。
