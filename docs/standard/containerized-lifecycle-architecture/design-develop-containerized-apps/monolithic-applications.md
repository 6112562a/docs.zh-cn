## <a name="monolithic-application-deployed-as-a-container"></a><span data-ttu-id="fef9b-145">部署为容器的整体式应用程序</span><span class="sxs-lookup"><span data-stu-id="fef9b-145">Monolithic application deployed as a container</span></span>

<span data-ttu-id="fef9b-148">将更新部署为 Docker 映像会快得多，并且网络效率更高。</span><span class="sxs-lookup"><span data-stu-id="fef9b-148">Deploying updates as Docker images is far faster and network efficient.</span></span> <span data-ttu-id="fef9b-149">以秒为单位，加快了推出速度通常启动 docker 容器。</span><span class="sxs-lookup"><span data-stu-id="fef9b-149">Docker containers typically start in seconds, speeding rollouts.</span></span> <span data-ttu-id="fef9b-150">拆除 Docker 容器非常简单，只调用`docker stop`命令，通常少于一秒内完成。</span><span class="sxs-lookup"><span data-stu-id="fef9b-150">Tearing down a Docker container is as easy as invoking the `docker stop` command, typically completing in less than a second.</span></span>
