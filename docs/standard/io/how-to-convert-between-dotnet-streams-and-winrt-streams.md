<span data-ttu-id="c40ac-132">[System.IO.WindowsRuntimeStreamExtensions.AsInputStream](https://msdn.microsoft.com/library/system.io.windowsruntimestreamextensions.asinputstream.aspx) 将适用于 Windows 应用商店应用的 .NET 子集中的托管流转换为 Windows 运行时中的输入流。</span><span class="sxs-lookup"><span data-stu-id="c40ac-132">[System.IO.WindowsRuntimeStreamExtensions.AsInputStream](https://msdn.microsoft.com/library/system.io.windowsruntimestreamextensions.asinputstream.aspx) Converts a managed stream in the .NET for Windows Store apps subset to an input stream in the Windows Runtime.</span></span>
