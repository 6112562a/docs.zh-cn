# <a name="how-to-convert-numeric-user-input-in-web-controls-to-numbers"></a><span data-ttu-id="b55fe-102">如何：将用户在 Web 控件中输入的数值转换成数字</span><span class="sxs-lookup"><span data-stu-id="b55fe-102">How to: Convert Numeric User Input in Web Controls to Numbers</span></span>
<span data-ttu-id="b55fe-103">由于世界各地的人都可以查看网页，因此用户能够在 <xref:System.Web.UI.WebControls.TextBox> 控件中以几乎无限种格式输入数字数据。</span><span class="sxs-lookup"><span data-stu-id="b55fe-103">Because a Web page can be displayed anywhere in the world, users can input numeric data into a <xref:System.Web.UI.WebControls.TextBox> control in an almost unlimited number of formats.</span></span> <span data-ttu-id="b55fe-104">所以，请务必确定网页用户的区域设置和区域性。</span><span class="sxs-lookup"><span data-stu-id="b55fe-104">As a result, it is very important to determine the locale and culture of the Web page's user.</span></span> <span data-ttu-id="b55fe-105">分析用户输入后，可以应用用户的区域设置和区域性定义的格式设置约定。</span><span class="sxs-lookup"><span data-stu-id="b55fe-105">When you parse user input, you can then apply the formatting conventions defined by the user's locale and culture.</span></span>  
  
### <a name="to-convert-numeric-input-from-a-web-textbox-control-to-a-number"></a><span data-ttu-id="b55fe-106">将 Web 文本框控件中的数字输入转换为数字的具体步骤</span><span class="sxs-lookup"><span data-stu-id="b55fe-106">To convert numeric input from a Web TextBox control to a number</span></span>  
  
1. <span data-ttu-id="b55fe-107">确定是否已填充 <xref:System.Web.HttpRequest.UserLanguages%2A?displayProperty=nameWithType> 属性返回的字符串数组。</span><span class="sxs-lookup"><span data-stu-id="b55fe-107">Determine whether the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A?displayProperty=nameWithType> property is populated.</span></span> <span data-ttu-id="b55fe-108">如果没有，请继续执行第 6 步。</span><span class="sxs-lookup"><span data-stu-id="b55fe-108">If it is not, continue to step 6.</span></span>  
  
2. <span data-ttu-id="b55fe-109">如果已填充 <xref:System.Web.HttpRequest.UserLanguages%2A> 属性返回的字符串数组，请检索其中第一个元素。</span><span class="sxs-lookup"><span data-stu-id="b55fe-109">If the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A> property is populated, retrieve its first element.</span></span> <span data-ttu-id="b55fe-110">第一个元素指明用户的默认或首选语言和区域。</span><span class="sxs-lookup"><span data-stu-id="b55fe-110">The first element indicates the user's default or preferred language and region.</span></span>  
  
3. <span data-ttu-id="b55fe-111">通过调用 <xref:System.Globalization.CultureInfo.%23ctor%28System.String%2CSystem.Boolean%29?displayProperty=nameWithType> 构造函数，实例化表示用户首选区域性的 <xref:System.Globalization.CultureInfo> 对象。</span><span class="sxs-lookup"><span data-stu-id="b55fe-111">Instantiate a <xref:System.Globalization.CultureInfo> object that represents the user's preferred culture by calling the <xref:System.Globalization.CultureInfo.%23ctor%28System.String%2CSystem.Boolean%29?displayProperty=nameWithType> constructor.</span></span>  
  
4. <span data-ttu-id="b55fe-112">调用要将用户输入转换为的数字类型的 `TryParse` 或 `Parse` 方法。</span><span class="sxs-lookup"><span data-stu-id="b55fe-112">Call either the `TryParse` or the `Parse` method of the numeric type that you want to convert the user's input to.</span></span> <span data-ttu-id="b55fe-113">重载 `TryParse` 或 `Parse` 方法（带 `provider` 参数），并将它传递到下面两个对象之一：</span><span class="sxs-lookup"><span data-stu-id="b55fe-113">Use an overload of the `TryParse` or the `Parse` method with a `provider` parameter, and pass it either of the following:</span></span>  
  
    - <span data-ttu-id="b55fe-114">第 3 步中创建的 <xref:System.Globalization.CultureInfo> 对象。</span><span class="sxs-lookup"><span data-stu-id="b55fe-114">The <xref:System.Globalization.CultureInfo> object created in step 3.</span></span>  
  
    - <span data-ttu-id="b55fe-115">第 3 步中创建的 <xref:System.Globalization.CultureInfo> 对象的 <xref:System.Globalization.CultureInfo.NumberFormat%2A> 属性返回的 <xref:System.Globalization.NumberFormatInfo> 对象。</span><span class="sxs-lookup"><span data-stu-id="b55fe-115">The <xref:System.Globalization.NumberFormatInfo> object that is returned by the <xref:System.Globalization.CultureInfo.NumberFormat%2A> property of the <xref:System.Globalization.CultureInfo> object created in step 3.</span></span>  
  
5. <span data-ttu-id="b55fe-116">如果转换失败，请对 <xref:System.Web.HttpRequest.UserLanguages%2A> 属性返回的字符串数组中的每个剩余元素重复执行第 2 步到第 4 步。</span><span class="sxs-lookup"><span data-stu-id="b55fe-116">If the conversion fails, repeat steps 2 through 4 for each remaining element in the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A> property.</span></span>  
  
6. <span data-ttu-id="b55fe-117">如果转换仍失败或 <xref:System.Web.HttpRequest.UserLanguages%2A> 属性返回的字符串数组为空，请使用 <xref:System.Globalization.CultureInfo.InvariantCulture%2A?displayProperty=nameWithType> 属性返回的固定区域性分析字符串。</span><span class="sxs-lookup"><span data-stu-id="b55fe-117">If the conversion still fails or if the string array returned by the <xref:System.Web.HttpRequest.UserLanguages%2A> property is empty, parse the string by using the invariant culture, which is returned by the <xref:System.Globalization.CultureInfo.InvariantCulture%2A?displayProperty=nameWithType> property.</span></span>
