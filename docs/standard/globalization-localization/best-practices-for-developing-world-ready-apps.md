## <a name="globalization-best-practices"></a><span data-ttu-id="bf3d9-104">全球化最佳做法</span><span class="sxs-lookup"><span data-stu-id="bf3d9-104">Globalization Best Practices</span></span>  
  
1.  <span data-ttu-id="bf3d9-105">在内部使应用程序代码成为 Unicode。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-105">Make your application Unicode internally.</span></span>

    -   <span data-ttu-id="bf3d9-108">对于字符串比较，使用 <xref:System.Globalization.CompareInfo> 类。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-108">For string comparisons, use the <xref:System.Globalization.CompareInfo> class.</span></span>  
  
    -   <span data-ttu-id="bf3d9-109">对于日期和时间格式化，使用 <xref:System.Globalization.DateTimeFormatInfo> 类。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-109">For date and time formatting, use the <xref:System.Globalization.DateTimeFormatInfo> class.</span></span>  
  
    -   <span data-ttu-id="bf3d9-110">对于数字格式化，使用 <xref:System.Globalization.NumberFormatInfo> 类。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-110">For numeric formatting, use the <xref:System.Globalization.NumberFormatInfo> class.</span></span>  
  
    -   <span data-ttu-id="bf3d9-111">对于公历和非公历，使用 <xref:System.Globalization.Calendar> 类或特定的日历实现之一。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-111">For Gregorian and non-Gregorian calendars, use the <xref:System.Globalization.Calendar> class or one of the specific calendar implementations.</span></span>  
  
3.  <span data-ttu-id="bf3d9-112">在适当的情况下，使用 <xref:System.Globalization.CultureInfo?displayProperty=nameWithType> 类提供的区域性属性设置。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-112">Use the culture property settings provided by the <xref:System.Globalization.CultureInfo?displayProperty=nameWithType> class in the appropriate situations.</span></span> <span data-ttu-id="bf3d9-113">使用 <xref:System.Globalization.CultureInfo.CurrentCulture%2A?displayProperty=nameWithType> 属性来执行格式化任务，如日期和时间或数字的格式化。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-113">Use the <xref:System.Globalization.CultureInfo.CurrentCulture%2A?displayProperty=nameWithType> property for formatting tasks, such as date and time or numeric formatting.</span></span> <span data-ttu-id="bf3d9-114">使用 <xref:System.Globalization.CultureInfo.CurrentUICulture%2A?displayProperty=nameWithType> 属性来检索资源。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-114">Use the <xref:System.Globalization.CultureInfo.CurrentUICulture%2A?displayProperty=nameWithType> property to retrieve resources.</span></span> <span data-ttu-id="bf3d9-115">请注意，`CurrentCulture` 和 `CurrentUICulture` 属性可以通过线程进行设置。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-115">Note that the `CurrentCulture` and `CurrentUICulture` properties can be set per thread.</span></span>  
  
4.  <span data-ttu-id="bf3d9-116">通过使用 <xref:System.Text> 命名空间中的编码类，使应用程序能够与各种编码相互进行数据读写。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-116">Enable your application to read and write data to and from a variety of encodings by using the encoding classes in the <xref:System.Text> namespace.</span></span> <span data-ttu-id="bf3d9-117">不要采用 ASCII 数据。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-117">Do not assume ASCII data.</span></span> <span data-ttu-id="bf3d9-118">假定在用户可以输入文本的任何位置都将提供国际字符。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-118">Assume that international characters will be supplied anywhere a user can enter text.</span></span> <span data-ttu-id="bf3d9-119">例如，应用程序应接受服务器名、目录、文件名、用户名和 URL 中的国际字符。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-119">For example, the application should accept international characters in server names, directories, file names, user names, and URLs.</span></span>  
  
5.  <span data-ttu-id="bf3d9-120">使用 <xref:System.Text.UTF8Encoding> 类时，出于安全原因，应使用此类提供的错误检测功能。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-120">When using the <xref:System.Text.UTF8Encoding> class, for security reasons, use the error detection feature offered by this class.</span></span> <span data-ttu-id="bf3d9-121">为了打开错误检测功能，应用程序会使用具有一个 `throwOnInvalidBytes` 参数的构造函数来创建该类的一个实例，并将该参数的值设置为 `true`。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-121">To turn on the error detection feature, the application creates an instance of the class using the constructor that takes a `throwOnInvalidBytes` parameter and sets the value of this parameter to `true`.</span></span>  
  
6.  <span data-ttu-id="bf3d9-122">尽可能将字符串按整个字符串处理，而不是按一系列个别字符处理。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-122">Whenever possible, handle strings as entire strings instead of as a series of individual characters.</span></span> <span data-ttu-id="bf3d9-123">这在排序或搜索子字符串时尤为重要。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-123">This is especially important when sorting or searching for substrings.</span></span> <span data-ttu-id="bf3d9-124">这可以防止与分析组合字符有关的问题。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-124">This will prevent problems associated with parsing combined characters.</span></span>  
  
7.  <span data-ttu-id="bf3d9-125">使用 <xref:System.Drawing> 命名空间提供的类来显示文本。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-125">Display text using the classes provided by the <xref:System.Drawing> namespace.</span></span>  
  
8.  <span data-ttu-id="bf3d9-126">为保持操作系统间的一致性，不要允许用户设置重写 <xref:System.Globalization.CultureInfo>。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-126">For consistency across operating systems, do not allow user settings to override <xref:System.Globalization.CultureInfo>.</span></span> <span data-ttu-id="bf3d9-127">使用接受 `CultureInfo` 参数的 `useUserOverride` 构造函数并将此参数设置为 `false`。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-127">Use the `CultureInfo` constructor that accepts a `useUserOverride` parameter and set it to `false`.</span></span>  
  
9. <span data-ttu-id="bf3d9-128">在国际操作系统版本上使用国际数据来测试应用程序功能。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-128">Test your application functionality on international operating system versions, using international data.</span></span>  
  
10. <span data-ttu-id="bf3d9-129">如果安全决策基于字符串比较或大小写更改操作的结果，请让应用程序执行不区分区域性的操作。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-129">If a security decision is based on the result of a string comparison or case change operation, have the application perform a culture-insensitive operation.</span></span> <span data-ttu-id="bf3d9-130">这种做法可确保结果不会受 `CultureInfo.CurrentCulture` 值的影响。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-130">This practice ensures that the result is not affected by the value of `CultureInfo.CurrentCulture`.</span></span> <span data-ttu-id="bf3d9-131">有关展示了区域性敏感型字符串比较如何产生不一致结果的示例，请参阅[字符串使用最佳做法](../../../docs/standard/base-types/best-practices-strings.md)的“使用当前区域性的字符串比较”部分。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-131">See the "String Comparisons that Use the Current Culture" section of [Best Practices for Using Strings](../../../docs/standard/base-types/best-practices-strings.md) for an example that demonstrates how culture-sensitive string comparisons can produce inconsistent results.</span></span>  
  
## <a name="localization-best-practices"></a><span data-ttu-id="bf3d9-132">本地化最佳做法</span><span class="sxs-lookup"><span data-stu-id="bf3d9-132">Localization Best Practices</span></span>  
  
1.  <span data-ttu-id="bf3d9-133">将所有可本地化的资源移动到单独的纯资源 DLL 中。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-133">Move all localizable resources to separate resource-only DLLs.</span></span> <span data-ttu-id="bf3d9-134">可本地化的资源包括用户界面元素，如字符串、错误消息、对话框、菜单以及嵌入的对象资源。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-134">Localizable resources include user interface elements, such as strings, error messages, dialog boxes, menus, and embedded object resources.</span></span>  
  
2.  <span data-ttu-id="bf3d9-135">不要对字符串或用户界面资源进行硬编码。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-135">Do not hardcode strings or user interface resources.</span></span>  
  
3.  <span data-ttu-id="bf3d9-136">不要将不可本地化的资源放在纯资源 DLL 中。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-136">Do not put nonlocalizable resources into the resource-only DLLs.</span></span> <span data-ttu-id="bf3d9-137">否则会使翻译人员产生困惑。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-137">This causes confusion for translators.</span></span>  
  
4.  <span data-ttu-id="bf3d9-138">不要使用在运行时从串联词组生成的复合字符串。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-138">Do not use composite strings that are built at run time from concatenated phrases.</span></span> <span data-ttu-id="bf3d9-139">复合字符串难以本地化，因为它们往往采用英语语法顺序，而此顺序并不适用于所有语言。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-139">Composite strings are difficult to localize because they often assume an English grammatical order that does not apply to all languages.</span></span>  
  
5.  <span data-ttu-id="bf3d9-140">避免不明确的结构，如“Empty Folder”，因为根据字符串组成部分的语法规则，这些字符串可能产生不同的翻译。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-140">Avoid ambiguous constructs such as "Empty Folder" where the strings can be translated differently depending on the grammatical roles of the string components.</span></span> <span data-ttu-id="bf3d9-141">例如，“empty”既可以是一个动词，也可以是一个形容词，因此在诸如意大利语或法语等语言中就可能导致不同的翻译。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-141">For example, "empty" can be either a verb or an adjective, which can lead to different translations in languages such as Italian or French.</span></span>  
  
6.  <span data-ttu-id="bf3d9-142">避免在应用程序中使用包含文本的图像和图标。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-142">Avoid using images and icons that contain text in your application.</span></span> <span data-ttu-id="bf3d9-143">本地化这些图像和图标的成本是很大的。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-143">They are expensive to localize.</span></span>  
  
7.  <span data-ttu-id="bf3d9-144">允许在用户界面中为字符串长度的扩展保留足够的空间。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-144">Allow plenty of room for the length of strings to expand in the user interface.</span></span> <span data-ttu-id="bf3d9-145">在某些语言中，词组所需的空间可能比在其他语言中多 50-75%。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-145">In some languages, phrases can require 50-75 percent more space than they need in other languages.</span></span>  
  
8.  <span data-ttu-id="bf3d9-146">使用 <xref:System.Resources.ResourceManager?displayProperty=nameWithType> 类来根据区域性检索资源。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-146">Use the <xref:System.Resources.ResourceManager?displayProperty=nameWithType> class to retrieve resources based on culture.</span></span>  
  
9. <span data-ttu-id="bf3d9-147">使用 Visual Studio 创建 Windows 窗体对话框，以便可以使用 [Windows 窗体资源编辑器 (Winres.exe)](../../../docs/framework/tools/winres-exe-windows-forms-resource-editor.md) 对它们进行本地化。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-147">Use Visual Studio to create Windows Forms dialog boxes, so that they can be localized using the [Windows Forms Resource Editor (Winres.exe)](../../../docs/framework/tools/winres-exe-windows-forms-resource-editor.md).</span></span> <span data-ttu-id="bf3d9-148">不要对 Windows 窗体对话框进行手动编码。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-148">Do not code Windows Forms dialog boxes by hand.</span></span>  
  
10. <span data-ttu-id="bf3d9-149">安排进行专业本地化工作（翻译）。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-149">Arrange for professional localization (translation).</span></span>  
  
11. <span data-ttu-id="bf3d9-150">有关创建并本地化资源的完整说明，请参阅[应用中的资源](../../../docs/framework/resources/index.md)。</span><span class="sxs-lookup"><span data-stu-id="bf3d9-150">For a complete description of creating and localizing resources, see [Resources in Applications](../../../docs/framework/resources/index.md).</span></span>
