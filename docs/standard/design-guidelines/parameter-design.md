 <span data-ttu-id="89020-176">您应验证数组不处理前为 null。</span><span class="sxs-lookup"><span data-stu-id="89020-176">You should validate that the array is not null before processing.</span></span>  
  
 <span data-ttu-id="89020-177">**X DO NOT** 使用`varargs`方法，也称为旁边的省略号。</span><span class="sxs-lookup"><span data-stu-id="89020-177">**X DO NOT** use the `varargs` methods, otherwise known as the ellipsis.</span></span>  
  
 <span data-ttu-id="89020-178">某些 CLR 语言，例如，c + +，支持替代约定，以便传递变量参数列表调用`varargs`方法。</span><span class="sxs-lookup"><span data-stu-id="89020-178">Some CLR languages, such as C++, support an alternative convention for passing variable parameter lists called `varargs` methods.</span></span> <span data-ttu-id="89020-179">约定不应在框架中，因为它不符合 CLS。</span><span class="sxs-lookup"><span data-stu-id="89020-179">The convention should not be used in frameworks, because it is not CLS compliant.</span></span>  
  
### <a name="pointer-parameters"></a><span data-ttu-id="89020-180">指针参数</span><span class="sxs-lookup"><span data-stu-id="89020-180">Pointer Parameters</span></span>  
 <span data-ttu-id="89020-181">一般情况下，指针不应出现在一个设计良好的托管的代码框架的公共图面区域。</span><span class="sxs-lookup"><span data-stu-id="89020-181">In general, pointers should not appear in the public surface area of a well-designed managed code framework.</span></span> <span data-ttu-id="89020-182">大多数情况下，应封装的指针。</span><span class="sxs-lookup"><span data-stu-id="89020-182">Most of the time, pointers should be encapsulated.</span></span> <span data-ttu-id="89020-183">但是，在某些情况下指针所需的互操作性方面的考虑，并且在这种情况下使用指针适合。</span><span class="sxs-lookup"><span data-stu-id="89020-183">However, in some cases pointers are required for interoperability reasons, and using pointers in such cases is appropriate.</span></span>  
  
 <span data-ttu-id="89020-184">**✓ DO** 为采用指针自变量，任何成员提供一种替代方法，因为不符合 cls 的指针。</span><span class="sxs-lookup"><span data-stu-id="89020-184">**✓ DO** provide an alternative for any member that takes a pointer argument, because pointers are not CLS-compliant.</span></span>  
  
 <span data-ttu-id="89020-185">**X AVOID** 执行检查的指针自变量的高开销的参数。</span><span class="sxs-lookup"><span data-stu-id="89020-185">**X AVOID** doing expensive argument checking of pointer arguments.</span></span>  
  
 <span data-ttu-id="89020-186">**✓ DO** 设计具有使用指针的成员时遵循指针相关的常见约定。</span><span class="sxs-lookup"><span data-stu-id="89020-186">**✓ DO** follow common pointer-related conventions when designing members with pointers.</span></span>  
  
 <span data-ttu-id="89020-187">例如，没有必要将传递的起始索引，因为简单指针算法可用于实现相同的结果。</span><span class="sxs-lookup"><span data-stu-id="89020-187">For example, there is no need to pass the start index, because simple pointer arithmetic can be used to accomplish the same result.</span></span>
