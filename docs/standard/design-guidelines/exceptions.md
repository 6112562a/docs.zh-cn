# <a name="design-guidelines-for-exceptions"></a>异常设计准则
与基于返回值的错误报告相比，异常处理具有许多优点。 良好的框架设计有助于应用程序开发人员理解异常的优势。 本节讨论异常的优势，并提供如何有效使用它们的准则。
