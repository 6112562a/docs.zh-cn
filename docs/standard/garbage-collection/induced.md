# <a name="induced-collections"></a><span data-ttu-id="9f53f-102">被动回收</span><span class="sxs-lookup"><span data-stu-id="9f53f-102">Induced Collections</span></span>
<span data-ttu-id="9f53f-103">在大多数情况下，垃圾回收器可以确定执行回收的最佳时间，应让其独立运行。</span><span class="sxs-lookup"><span data-stu-id="9f53f-103">In most cases, the garbage collector can determine the best time to perform a collection, and you should let it run independently.</span></span> <span data-ttu-id="9f53f-104">在某些不常见的情况下，强制回收可以提高应用程序的性能。</span><span class="sxs-lookup"><span data-stu-id="9f53f-104">There are rare situations when a forced collection might improve your application's performance.</span></span> <span data-ttu-id="9f53f-105">在这种情况下，可以使用 <xref:System.GC.Collect%2A?displayProperty=nameWithType> 方法强制执行垃圾回收，从而诱导垃圾回收。</span><span class="sxs-lookup"><span data-stu-id="9f53f-105">In these cases, you can induce garbage collection by using the <xref:System.GC.Collect%2A?displayProperty=nameWithType> method to force a garbage collection.</span></span>  
  
 <span data-ttu-id="9f53f-106">如果应用代码中特定点使用的内存量大量减少，请使用 <xref:System.GC.Collect%2A?displayProperty=nameWithType> 方法。</span><span class="sxs-lookup"><span data-stu-id="9f53f-106">Use the <xref:System.GC.Collect%2A?displayProperty=nameWithType> method when there is a significant reduction in the amount of memory being used at a specific point in your application's code.</span></span> <span data-ttu-id="9f53f-107">例如，如果应用使用包含多个控件的复杂对话框，那么在对话框关闭时调用 <xref:System.GC.Collect%2A> 可以立即回收对话框占用的内存，从而提升性能。</span><span class="sxs-lookup"><span data-stu-id="9f53f-107">For example, if your application uses a complex dialog box that has several controls, calling <xref:System.GC.Collect%2A> when the dialog box is closed could improve performance by immediately reclaiming the memory used by the dialog box.</span></span> <span data-ttu-id="9f53f-108">请确保应用程序不会过于频繁地引发垃圾回收，否则当垃圾回收器无效率地尝试回收对象时，可能会使性能降低。</span><span class="sxs-lookup"><span data-stu-id="9f53f-108">Be sure that your application is not inducing garbage collection too frequently, because that can decrease performance if the garbage collector is trying to reclaim objects at non-optimal times.</span></span> <span data-ttu-id="9f53f-109">可以向 <xref:System.GC.Collect%2A> 方法提供 <xref:System.GCCollectionMode.Optimized?displayProperty=nameWithType> 枚举值，以便仅在回收能够提高效率时才进行回收，如下一部分所述。</span><span class="sxs-lookup"><span data-stu-id="9f53f-109">You can supply a <xref:System.GCCollectionMode.Optimized?displayProperty=nameWithType> enumeration value to the <xref:System.GC.Collect%2A> method to collect only when collection would be productive, as discussed in the next section.</span></span>
